#### மைய கருத்துக்களை மேம்படுத்தல் (01-CoreConcepts/)
- **புதிய கிளையண்ட் அடிப்படை - ரூட்ஸ்**: ரூட்ஸ் கிளையண்ட் அடிப்படையின் விரிவான ஆவணத்துடன் சேர்க்கப்பட்டது, இதனால் சர்வர்கள் கோப்பு அமைப்பு எல்லைகளைப் புரிந்து கொள்ளவும் அணுகல் உரிமைகளை நிர்ணயம் செய்யவும் முடியும்
- **கருவி குறிப்புரை**: கருவி நடத்தை குறிப்புரைகள் (`readOnlyHint`, `destructiveHint`) குறித்த ஆவணங்கள் சேர்க்கப்பட்டன, இது கருவி செயல்பாட்டின் சிறந்த முடிவுகளை வழங்குகிறது
- **த samplingகருவி அழைப்பு**: மாதிரித்தலைக் குறிக்கும் Sampling ஆவணத்தில் `tools` மற்றும் `toolChoice` ஆகிய அளவுருக்கள் சேர்க்கப்பட்டன, இது மாதிரித்தல் கோரிக்கைகளின் போது கருவி அழைப்பை இயக்கு
- **URL வகை வெளிப்படுத்தல்**: சர்வர் துவக்கம் செய்யப்பட்ட வெளி வலை தொடர்புகளுக்கான URL அடிப்படைக் கிளை வழிதலைக் குறிக்கும் ஆவணம் சேர்க்கப்பட்டது
- **பணி (திருப்பி சேகரிப்பது) (சோதனை)**: நிலையான செயல்பாட்டு முனைவுகள் மற்றும் பின்னர் முடிவுகள் பெறுதலைக் குறிக்கும் புதிய பணி அம்சம் ஆவணத்தில் சேர்க்கப்பட்டது
- **படிகள் ஆதரவு**: கருவி, வளங்கள், வள வார்ப்புருக்கள் மற்றும் உத்தரவுகள் இப்போது கூடுதல் தகவல்களாக படிக்களை உள்ளடக்கக்கூடியதாகக் குறிப்பிடப்பட்டது

#### விவரக்குறிப்பு இணக்கத்தினை உறுதி செய்தல்
- **ப்ரோட்டோக்கால் பதிப்பு**: அனைத்து ஆவணங்களும் தற்போதைய MCP விவரக்குறிப்பு 2025-11-25-ன் குறிப்பு இருப்பதைக் கண்டறிந்து உறுதி செய்தது

- **அடிப்படைக் கருத்துக்கள் ஆவணங்கள்**: சர்வர் அடிப்படைகள் (வளங்கள், உத்தரவுகள், கருவிகள்) மற்றும் கிளையண்ட் அடிப்படைகள் (திரைவேட்பு, வெளிப்படுத்து, பதிவு, ரூட்ஸ்) சரிபார்க்கப்பட்டன  
- **போக்குவரத்து முறைகள்**: STDIO மற்றும் ஸ்ட்ரீமிங் HTTP போக்குவரத்து ஆவணங்கள் சரியானவை என உறுதி செய்தது

#### மேம்பட்ட சர்வர் பயன்படுத்தல் (03-GettingStarted/10-advanced)
- **புது அத்தியாயம் சேர்க்கப்பட்டது**: வழமையான மற்றும் குறைந்த நிலை சர்வர் வளைக்களங்களை உள்ளடக்கிய மேம்பட்ட MCP சர்வர் பயன்படுத்தல் குறித்த விரிவான வழிகாட்டி அறிமுகம்.
  - **வழமையான vs குறைந்த நிலை சர்வர்**: இரு அணுகுமுறைகளுக்கும் Python மற்றும் TypeScript இல் விரிவான ஒப்பீடு மற்றும் குறியீடு எடுத்துக்காட்டுகள்.
  - **ஹேண்ட்லர் அடிப்படையிலான வடிவமைப்பு**: விரிவான மற்றும் நெடுங்கால சர்வர் செயலாக்கங்களுக்கான ஹேண்ட்லர்-அடிப்படை கருவி/வளம்/கேள்வி மேலாண்மை விளக்கம்.
  - **நடைமுறை மாதிரிகள்**: மேம்பட்ட அம்சங்கள் மற்றும் கட்டமைப்புக்கு குறைந்த நிலை சர்வர் மாதிரிகள் பயன்பாட்டு நிகழ்வுகள்.

#### எளிய அங்கீகாரம் (03-GettingStarted/11-simple-auth)
- **புது அத்தியாயம் சேர்க்கப்பட்டது**: MCP சர்வர்களில் எளிய அங்கீகாரத்தை செயல்படுத்த படிப்படியாக வழிகாட்டல்.
  - **அங்கீகாரக் கருத்துக்கள்**: அங்கீகாரம் மற்றும் அங்கீகார மறுமொழிவின் தெளிவான விளக்கமும் அட்டை மேலாண்மையும்.
  - **அடிப்படை அங்கீகாரம் நடைமுறை**: Python (Starlette) மற்றும் TypeScript (Express) இல் மிடில்வேர் அடிப்படையிலான அங்கீகாரம் மாதிரிகள் மற்றும் குறியீடு எடுத்துக்காட்டுகள்.
  - **மேம்பட்ட பாதுகாப்பிற்கான முன்னேற்றம்**: எளிய அங்கீகாரத்துடன் துவங்கி OAuth 2.1 மற்றும் RBAC க்கு முன்னேற்ற வழிகாட்டிகள், மேம்பட்ட பாதுகாப்பு தொகுப்புக்களுக்கு குறிகள்.

இந்த இணைப்புகள் திடமான, பாதுகாப்பான மற்றும் நிலையான MCP சர்வர் உருவாக்கக்கான நடைமுறைகளை வழங்குகின்றன, அடிப்படை கருத்துக்களுடன் மேம்பட்ட உற்பத்தி மாதிரிகளின் இடையே பாலமாக உள்ளன.

  - சர்வர் மூலக்கூறுகள்: வளங்கள் (தரவு மூலங்கள்), வினாக்கள் (வார்ப்புறைகள்), கருவிகள் (செயல்படுத்தக்கூடிய செயலிகள்) விரிவான விளக்கங்கள் மற்றும் எடுத்துக்காட்டுகளுடன்
  - கிளையன்ட் மூலக்கூறுகள்: மாதிரிகள் எடுக்கும் (LLM முடிச்சுகள்), கேள்விவிடை பெறல் (பயனர் உள்ளீடு), பதிவேட்டல் (பிழைத்தடை/கண்காணிப்பு)
  - தற்போதைய கண்டுபிடிப்பு (`*/list`), பெற்றல் (`*/get`), மற்றும் செயல்படுத்தல் (`*/call`) முறைமைகள் கூடுதல் தகவல்களுடன் புதுப்பிக்கப்பட்டது
- **பிரோட்டோக்கால் கட்டமைப்பு**: இரண்டு அடுக்குக் கட்டமைப்பு மாதிரியை அறிமுகம் செய்தல்
  - தரவு அடுக்கு: JSON-RPC 2.0 அடிப்படையுடன் வாழ்கால மேலாண்மை மற்றும் மூலக்கூறுகளுடனான
  - போக்குவரத்து அடுக்கு: STDIO (உள்ளூர்) மற்றும் Streamable HTTP SSE (தொலை) போக்குவரத்து முறைகள்
- **பாதுகாப்பு அமைப்பு**: தெளிவான பயனர் சம்மதம், தரவு தனியுரிமை பாதுகாப்பு, கருவி செயல்பாட்டு பாதுகாப்பு மற்றும் போக்குவரத்து அடுப்பு பாதுகாப்பு ஆகியவற்றைக் கொண்ட விரிவான பாதுகாப்பு கோட்பாடுகள்
- **தொடர்பு குண்டிகள்**: ஆரம்பிப்பது, கண்டுபிடித்தல், செயல்படுத்தல் மற்றும் அறிவிப்பு ஓட்டங்களை காண்பிக்கும் புதிய பிரோட்டோக்கால் செய்தி வடிவங்கள்
- **குறியீடு எடுத்துக்காட்டுகள்**: நவீன MCP SDK மாதிரிகளுக்கேற்ற .NET, Java, Python, JavaScript போன்ற பலமொழிக் எடுத்துக்காட்டுகளுடன் புதுப்பிக்கப்பட்டது

#### பாதுகாப்பு (02-Security/) - விரிவான பாதுகாப்பு மறுசீரமைப்பு  
- **தரநிலை ஒத்திசைவு**: MCP Specification 2025-06-18 பாதுகாப்பு தேவைகளுடன் முழுமையாக ஒத்திசைவு செய்யப்பட்டல்
- **அங்கீகாரம் முன்னேற்றம்**: தனிப்பயனாக்கப்பட்ட OAuth சர்வர்களிலிருந்து வெளிப்புற அடையாள வழங்குநர் ஒப்படைப்பு (Microsoft Entra ID) வரை தாக்கியுள்ள மாற்றங்கள் ஆவணப்படுத்தப்பட்டுள்ளன
- **AI-சிறப்பு முப்பண்புக் குற்றவியல் ஆய்வு**: நவீன AI தாக்கு வழிகளுக்கான மேம்பட்ட உள்ளடக்கம்
  - உண்மையான உலகின் எடுத்துக்காட்டுகளுடன் தொகுக்கப்பட்ட குத்துதல் தாக்கு நிகழ்வுகள்
  - கருவி விஷம் உண்டாக்கலை மற்றும் "ரக் புல்" தாக்கு விதிகள்
  - சூழல் கவனச்சாளத்தன்மை விஷம் மற்றும் மாதிரி குழப்பம் தாக்கல்கள்
- **Microsoft AI பாதுகாப்பு தீர்வுகள்**: Microsoft பாதுகாப்பு சூழலை விரிவாக வழங்கல்
  - AI Prompt Shields மேம்பட்ட கண்டறிதல், ஸ்பாட்லைட்டிங் மற்றும் பகுப்பாய்வு முறைகளுடன்
  - Azure Content Safety இணைப்பு மாதிரிகள்
  - GitHub முன்னேறிய பாதுகாப்பு வழங்கல் சங்கிலி பாதுகாப்பிற்கு
- **முன்னேறிய முப்பண்பு தடுப்பு**: 
  - MCP-சிறப்பு அமர்வு கடத்தல் மற்றும் குறியாக்க அமர்வு ID தேவைகளுடன்
  - MCP பிராக்ஸி சூழலில் குழப்பமான உதவியாளர் பிரச்சினைகள் மற்றும் தெளிவான சம்மதக் கோரிக்கைகள்
  - கட்டாயமான சரிபார்ப்புடன் டோக்கன் பாஸ்த்ரூ பாதிப்புகள்
- **வழங்கல் சங்கிலி பாதுகாப்பு**: அடிப்படை மாதிரிகள், நெமுறைக் சேவைகள், சூழல் வழங்குநர்கள் மற்றும் மூன்றாம் பக்கம் APIகளுடன் விரிவடையப்பட்டது
- **அடிப்படை பாதுகாப்பு**: பூஜ்ய நம்பிக்கை கட்டமைப்பு மற்றும் Microsoft பாதுகாப்பு சூழல் போக்குவரத்திற்கு மேலதிக ஒருங்கிணைப்பு
- **வளம் அமைப்பு**: வகைப்படுத்தப்பட்ட வள இணைப்புகள் (அதிகாரப்பூர்வ ஆவணங்கள், தரநிலைகள், ஆய்வு, Microsoft தீர்வுகள், நடைமுறை வழிகாட்டிகள்)

### ஆவண தரம் மேம்பாடுகள்
- **மைல்களைக் கூடி கற்றல் நோக்கங்கள்**: குறிப்பிட்ட மற்றும் செயல்படுத்தக்கூடிய முடிவுகளுடன் மேம்படுத்தப்பட்டது
