# MCP இல் முன்னேற்றமான தலைப்புகள்

[![Advanced MCP: Secure, Scalable, and Multi-modal AI Agents](../../../translated_images/ta/06.42259eaf91fccfc6.webp)](https://youtu.be/4yjmGvJzYdY)

_(இந்த பாடத்தின் வீடியோவை பார்க்க மேலுள்ள படத்தை கிளிக் செய்யவும்)_

இந்த அத்தியாயம் Model Context Protocol (MCP) செயலாக்கத்தில் முன்னேற்றமான தலைப்புகளை, அதாவது பல்வேறு முறை ஒருங்கிணைப்பு, பரிமாணம், பாதுகாப்பு சிறந்த நடைமுறைகள் மற்றும் நிறுவன ஒருங்கிணைப்பு ஆகியவற்றை விவரிக்கிறது. இத்தலைப்புகள் நவீன AI அமைப்புகளின் தேவைகளை பூர்த்தி செய்யக்கூடிய, வலுவான மற்றும் தயாரிப்பு ரெடி MCP செயலிகள் உருவாக்குவதற்கான முக்கிய அம்சங்கள்.

## பார்வை

இந்த பாடத்தில் Model Context Protocol செயலாக்கத்தில் முன்னேற்றத்திற்கான கொள்கைகள் ஆராயப்படுகின்றன, குறிப்பாக பல்வேறு முறை ஒருங்கிணைப்பு, பரிமாணம், பாதுகாப்பு சிறந்த நடைமுறைகள் மற்றும் நிறுவன ஒருங்கிணைப்பு ஆகியவற்றுக்கு கவனம் செலுத்தப்பட்டுள்ளது. இத்தலைப்புகள் நிறுவன சூழல்களில் கோட்பாட்டிற்கு ஏற்ப MCP செயலிகளை உருவாக்குவதற்கு அவசியம்.

## கற்றல் குறிக்கோள்கள்

இந்த பாடத்தின் முடிவில், நீங்கள்:

- MCP கட்டமைப்புகளில் பல்வேறு முறை செயல்பாடுகளை செயல்படுத்த முடியும்
- உயர் தேவையான சூழல்களுக்கு பரிமாற்றக்கூடிய MCP கட்டமைப்புகளை வடிவமைக்க முடியும்
- MCP புதிய பாதுகாப்பு கொள்கைகளுடன் பாதுகாப்பு சிறந்த நடைமுறைகளை பின்பற்ற முடியும்
- MCP ஐ நிறுவன AI அமைப்புகளுடன் ஒருங்கிணைக்க முடியும்
- தயாரிப்பு சூழல்களில் செயல்திறன் மற்றும் நம்பகத்தன்மையை மேம்படுத்த முடியும்

> **MCP விவரக்குறிப்பு 2025-11-25 இல் புதியது**: விவரக்குறிப்பில் தற்போது பிள்ளைகள் செயல்பாடுகளுக்கான (Task) சோதனை ஆதரவு, கருவி குறிச்சொற்கள் (Tool Annotations) பாதுகாப்புக்கான செயல்திறன்களுக்கு, URL முறை கோரிக்கைகள் (URL Mode Elicitation), மற்றும் மேம்படுத்தப்பட்ட ரூட்ஸ் (Roots) - வாந்து வேலைப்படுத்தல் மேலாண்மை ஆகியவை அடக்கப்பட்டுள்ளன. முழு விவரங்களுக்கு [MCP Specification changelog](https://spec.modelcontextprotocol.io/) பார்க்கவும்.

## முக்கிய எடுத்துக்காட்டுகள்

- பல முறை MCP செயல்படுத்தல்கள், உரை செயலாக்கத்திற்கு அப்பாற்பட்டு AI திறன்களை விரிவாக்குகின்றன
- பரிமாற்றம் நிறுவன பயன்பாட்டிற்கான முக்கிய அம்சம், அது அகலம் மற்றும் உயரம் முறை பரிமாற்றங்களின் மூலம் எடுத்துரைக்கப்படுகிறது
- விரிவான பாதுகாப்பு நடவடிக்கைகள் தரவை பாதுகாக்கும் மற்றும் சரியான அணுகல் கட்டுப்பாட்டை உறுதி செய்கின்றன
- Azure OpenAI மற்றும் Microsoft AI Foundry போன்ற தளங்களுடன் நிறுவன ஒருங்கிணைப்பு MCP திறன்களை மேம்படுத்துகின்றது
- முன்னேற்ற MCP செயல்படுத்தல்கள் மேம்பட்ட கட்டமைப்புகளும் கவனமான வள மேலாண்மையும் பெறுகின்றன

## பயிற்சி

ஒரு குறிப்பிட்ட பயன்பாட்டிற்கான நிறுவனம் தரமான MCP செயல்படுத்தலை வடிவமைக்கவும்:

1. உங்கள் பயன்பாட்டிற்கான பல முறை தேவைகளை கண்டறியவும்
2. நுண்ணறிவு தரவை பாதுகாக்க தேவையான பாதுகாப்பு கட்டுப்பாடுகளை பரிசீலிக்கவும்
3. மாறுபடும் பளுவைக் கையாளக்கூடிய பரிமாற்றக்கூடிய கட்டமைப்பை வடிவமைக்கவும்
4. நிறுவன AI அமைப்புகளுடன் ஒருங்கிணைப்பு புள்ளிகளை திட்டமிடவும்
5. செயல்திறன் குறைபாடுகள் மற்றும் அவற்றை குறைக்கும் மூலோபாயங்களை ஆவணப்படுத்தவும்
