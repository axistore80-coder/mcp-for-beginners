# 🚀 ஆரம்பநிலை பயனர்களுக்கான மாடல் கான்டெக்ஸ்ட் புரோட்டோகால் (MCP) பாடத்திட்டம்

## **C#, Java, JavaScript, Rust, Python, மற்றும் TypeScript ஆகிய மொழிகளில் கையேட்டுடன் MCP கற்றுக்கொள்ளுங்கள்**

## 🧠 மாடல் கான்டெக்ஸ்ட் புரோட்டோகால் பாடத்திட்டத்தின் கண்ணோட்டம்
மாடல் கான்டெக்ஸ்ட் புரோட்டோகாலுக்கான உங்கள் பயணத்திற்கு வரவேற்கிறோம்! AI பயன்பாடுகள் எப்படி வெவ்வேறு கருவிகள் மற்றும் சேவைகளுடன் தொடர்பு கொள்கின்றன என்று நீங்கள் எப்போதாவது கேள்விப்பட்டிருந்தால், வளர்த்துநோக்கப்படுகிற அந்த அழகான தீர்வை நீங்கள் கண்டறியப் போகிறீர்கள்.

MCP என்பது AI பயன்பாடுகளுக்கு உலகளாவிய மொழிபெயர்ப்பாளர் போல உள்ளது - USB போர்ட் எந்த சாதனத்தையும் கணினியுடன் இணைக்கும் விதமாக, MCP AI மாதிரிகள் எந்தவொரு கருவி அல்லது சேவையையும் ஒரே விதமான முறையில் தொடர்பு கொள்ள உதவுகிறது. நீங்கள் முதலில் ஒரு செயற்கை நுண்ணறிவு செயற்கை சிலுவை கட்டுமானம் செய்தாலும், அல்லது அதிக சிக்கலான AI செயல்முறைகளை உருவாக்கினாலும், MCP ஐ புரிந்துகொள்ளுதல் உங்கள் செயலிகளை மேலும் திறனான மற்றும் தகுந்தவை ஆக்க உதவும்.

இந்த பாடத்திட்டம் உங்கள் கற்றல் பயணத்திற்கு பொறுமையாகவும் கவனமாகவும் வடிவமைக்கப்பட்டுள்ளது. நீங்கள் ஏற்கனவே புரிந்துகொண்ட எளிய கருத்துகளிலிருந்து தொடங்கி, உங்கள் பிடித்த நிரலாக்க மொழியில் கையேட்டு பயிற்சிகள் மூலம் உங்கள் திறனை படிப்படியாக கட்டமைப்போம். ஒவ்வொரு படியும் தெளிவான விளக்கங்கள், நடைமுறை எடுத்துக்காட்டுகள் மற்றும் மிகவும் உத்வேகத்துடன் நிறைந்திருக்கும்.

இந்த பயணத்தை முடிக்கும் போது, உங்கள் சொந்த MCP சர்வர்களை உருவாக்கும் திறம் உண்டாகும், பிரபலமான AI தளங்களுடன் ஒருங்கிணைப்பது அடங்கிய பல திறன்களை கற்றுக்கொள்வீர்கள், மேலும் இந்த தொழில்நுட்பம் AI வளர்ச்சியின் எதிர்காலத்தை எப்படி மாற்றிக் கொண்டிருக்கிறது என்பதை புரிந்துகொள்ளுவீர்கள். வாருங்கள் இந்த ஆச்சரியமான பயணத்தை ஒன்றாகத் தொடங்குவோம்!

## கற்றல் நோக்கங்கள்

இந்த பாடத்திட்டத்தின் முடிவில், நீங்கள் புதிய திறன்களின் மீது நம்பிக்கை மற்றும் உற்சாகம் கொண்டிருப்பீர்கள். இதோ நீங்கள் அடையும் விஷயங்கள்:

• **MCP அடிப்படைகளை புரிந்துகொள்ளுதல்**: மாடல் கான்டெக்ஸ்ட் புரோட்டோக்கால் என்பது என்ன மற்றும் அது AI பயன்பாடுகள் எப்படி ஒன்றாக வேலை செய்வதை மாற்றிக் கொண்டிருக்கிறது என்பதைக் கற்றுக் கொள்வீர்கள், உருப்படியாக்கலும் எடுத்துக்காட்டுகளால்.

• **முதலாவது MCP சர்வரை கட்டமைப்பது**: உங்கள் விருப்பமான நிரலாக்க மொழியில் ஒரு செயற்பாடுள்ள MCP சர்வரை உருவாக்குவீர்கள், எளிய எடுத்துக்காட்டுகளுடன் தொடங்கி படிப்படியாக உங்கள் திறனை வளர்த்துக் கொள்வீர்கள்.

• **AI மாதிரிகளை உண்மையான கருவிகளுடன் இணைப்பு**: AI மாதிரிகள் மற்றும் உண்மையான சேவைகள் இடையேயான இடைவெளியை நிரப்புவதைக் கற்றுக் கொள்வீர்கள், உங்கள் பயன்பாடுகளுக்கு புதிய சக்திகளைக் கொடுப்பீர்கள்.

• **பாதுகாப்பு சிறந்த நடைமுறைகளை அமல்படுத்தல்**: உங்கள் MCP நடைமுறைகளை பாதுகாப்பாக வைத்தல் எப்படி என்பதை புரிந்து, உங்கள் பயன்பாடுகளையும் பயனாளர்களையும் பாதுகாக்கும் முறைகளைக் கற்றுக்கொள்வீர்கள்.

• **ஆரம்பித்து வெளியிடுதல்**: உங்கள் MCP திட்டங்களை உருவாக்குவதிலிருந்து உருவாக்கத் தரவிறக்கத் தளத்திற்கு எப்படிப் பராமரிப்பது என்பதையும் உண்மையான உலகில் செயல்படும் பயிற்சிகள் பற்றி அறிந்துகொள்ளுவீர்கள்.

## அடிப்படையான பின்னணி

MCP தனிச்சிறப்புகளை ஆராய்வதற்கு முன், சில அடிப்படைக் கருத்துக்கள் உங்களால் பரிச்சயம் படுத்திக்கொள்ளலாமா என்று பார்த்துக்கொள்வோம். இதில் நீங்கள் நிபுணர் இல்லாவிட்டாலும் கவலை வேண்டாம் - தேவையான அனைத்தையும் நாம் விளக்குவோம்!

### புரோட்டோகால்களைப் புரிதல் (அடிப்படை)

புரோட்டோக்கால் என்பது உரையாடலுக்கான விதிமுறைகள் மாதிரி என நினைக்கலாம். நீங்கள் ஒரு நண்பரை அழைக்கும்போது, நீங்கள் இருவரும் "வணக்கம்" என்று சொல்வீர்கள், உரையாடல் நன்கு நடக்கும் வகையில் தடவாக பேசுவீர்கள் மற்றும் முடிவில் "பிரியாவிடை" சொல்வீர்கள். கணினி நிரல்களுக்கு இதுபோன்ற விதிமுறைகள் தேவையாகும்.

MCP ஒரு புரோட்டோக்கால் - AI மாதிரிகள் மற்றும் பயன்பாடுகள் கருவிகளுடன் மற்றும் சேவைகளுடன் பயனுள்ள "உரையாடல்களை" நடத்தியதற்கு சம்மதிக்கப்பட்ட விதிமுறைகள் தொகுப்பு. மனித உரையாடலை எளிதாக்கும் விதிமுறைகளைப் போல், MCP AI பயன்பாடுகள் உரையாடலை நம்பகத்தன்மையுடன் மற்றும் சக்திவாய்ந்ததாக ஆக்குகிறது.

### கிளையன்ட்-சர்வர் உறவுகள் (நிரல்கள் எப்படி சேர்ந்து வேலை செய்கின்றன)

நீங்கள் தினசரி கிளையன்ட்-சர்வர் உறவுகளைப் பயன்படுத்துகிறீர்கள்! ஒரு வலை உலாவியை (கிளையன்ட்) பயன்படுத்தி வலைத்தளத்தை பார்ப்பதற்கு, நீங்கள் ஒரு வெப்சர்வருக்கு இணைகிறீர்கள், அது பக்க உள்ளடக்கத்தை தருகிறது. உலாவி எப்படி தகவலை கேட்க வேண்டும் என்பதையும், சர்வர் எப்படி பதிலளிக்க வேண்டும் என்பதையும் தெரியும்.

MCP இல் அதே மாதிரி உறவு உள்ளது: AI மாதிரிகள் தகவல் அல்லது செயல் கோரிக்கைகளை கோரும் கிளையன்ட்களாக செயல்படுகிறார்கள், MCP சர்வர்கள் அந்த வசதிகளை வழங்குகிறார்கள். இது AIக்கு ஒரு உதவியாளராக இருக்கும் சர்வருடன் ஒப்பிடலாம், குறிப்பிட்ட பணி செய்யும்படி கேட்க அனுமதிக்கும்.

### தரநிர்ணயம் முக்கியத்துவம் (வசதிகளை ஒருங்கிணைக்கல்)

ஒவ்வொரு கார் உற்பத்தியாளரும் வெவ்வேறு வடிவமான எரிவாயு பம்புகளை பயன்படுத்தினால் - ஒவ்வொரு காருக்கும் ஒரு வெவ்வேறு அடாப்டர் தேவைப்படும்! தரநிர்ணயம் பொதுவான அணுகுமுறைகளை ஒப்புக்கொள்வது, இதனால் வேலைகள் ஒருங்கிணைந்து நடக்கும்.

MCP AI பயன்பாடுகளுக்கு இதைத்தான் செய்கிறது. ஒவ்வொரு AI மாதிரியும் ஒவ்வொரு கருவியுடன் தனிப்பயனர் குறியீடு எழுத வேண்டிய அதிர்ச்சியின்றி, MCP அனைத்து AI மாதிரிகளுக்கும் ஒரு அறியுமுறை வழியைக் கொடுக்கிறது. இதனால், உருவாக்குநர்கள் ஒருமுறை கருவியை வடிவமைத்து பலவித AI அமைப்புகளுடன் வேலை செய்ய செய்யலாம்.

## 🧭 உங்கள் கற்றல் பாதை கண்ணோட்டம்

• **மோடியூல் 2 - MCP இல் பாதுகாப்பு**: பாதுகாப்பு மிகவும் கடுமையாக தோன்றலாம், ஆனால் MCP எப்படி பாதுகாப்பு அம்சங்களுடன் வடிவமைக்கப்பட்டது என்பதையும், உங்கள் பயன்பாடுகளை ஆரம்பத்திலேயே பாதுகாக்க சிறந்த நடைமுறைகளை எளிமையாக கற்பிப்போம்.

### 💻 மாதிரி குறியீட்டு திட்டங்கள்

MCP கற்றுக்கொள்ளும் மிக சுவாரஸ்யமான பகுதிகளில் ஒன்று உங்கள் குறியீட்டு திறன்கள் படிப்படியாக வளரும் தன்மை. எங்கள் குறியீட்டு உதாரணங்களை எளிமையாக துவங்கி, உங்கள் புரிதல் வளர்வதற்கெதிரா மேம்படுமாற்குப் வடிவமைத்துள்ளோம். நாம் கருத்துக்களை அறிமுகப்படுத்துவது எப்படி என்பது இதோ - புரிந்துகொள்ள எளிய குறியீடு ஆனால் உண்மையான MCP கோட்பாடுகளை காட்டும் வகையில், இந்த குறியீடு என்ன செய்கிறது என்பதையே மட்டும் அல்ல, இது ஏன் இப்படியாக கட்டமைக்கப்பட்டுள்ளது மற்றும் பெரிய MCP பயன்பாடுகளில் எப்படி பொருந்துகிறது என்பதையும் புரிந்து கொள்வீர்கள்.

## 📚 படிப்பு கையேடு மற்றும் வளங்கள்

இந்த ரெப்போசிடரி பல வசதிகளை கொண்டுள்ளது, இது உங்களுக்கு திறம்பட பயில மற்றும் வழிநடத்த உதவுகின்றது:

### படிப்பு கையேடு

உங்கள் பயில்வதை வழிநடத்த உதவும் ஒரு விரிவான [படிப்பு கையேடு](./study_guide.md) கிடைக்கிறது. இந்த காட்சி பாடத்திட்ட வரைபடம் அனைத்து தலைப்புகளும் எவ்வாறு இணைக்கப்பட்டுள்ளன என்பதை காட்டி, மாதிரிப் பணிகளை எப்படி திறம்பட பயன்படுத்துவது என்பதில் வழிகாட்டும். இது பெரிய படத்தை பார்க்க விரும்புமவா்கள் மற்றும் காட்சி மூலம் கற்றுக் கொள்வோருக்கு மிகவும் உதவியாக இருக்கும்.

கையேட்டில் உள்ளது:  
- அனைத்து தலைப்புகளையும் காட்டும் காட்சி பாடத்திட்ட வரைபடம்  
- ஒவ்வொரு ரெப்போசிடரி பகுதியில் விரிவான பகுப்பாய்வு  
- மாதிரி செயல்பாடுகளை பயன்படுத்தும் வழிகாட்டல்  
- வேறு திறனுக்கு ஏற்ப பரிந்துரைக்கப்பட்ட கற்றல் பாதைகள்  
- உங்கள் கற்றல் பயணத்தை பூர்த்தி செய்ய கூடுதல் வளங்கள்  

### மாற்ற பட்டியல்

பாடத்திட்டப் பொருட்களில் அனைத்து முக்கியமான புதுப்பிப்புகளைத் தொடரும் விரிவான [மாற்றப் பட்டியல்](./changelog.md) எப்போதும் பராமரிக்கப்படுகிறது, அதனால் நீங்கள் சமீபத்திய மேம்படுத்தல்கள் மற்றும் சேர்ப்புகளை அறிந்துகொள்ளலாம்.  
- புதிய உள்ளடக்கம் சேர்த்தல்  
- கட்டமைப்பு மாற்றங்கள்

## 🎓 உங்கள் MCP பயணம் துவங்கியது

வாழ்த்துக்கள்! உங்கள் நிரலாக்க திறன்களை விரிவுபடுத்தும் மற்றும் AI முன்னேற்றத்தின் முன்னணியில் நீங்கள் இணையும் ஒரு சுவாரஸ்யமான பயணத்தில் முதலாவது படியை நீங்கள் எடுத்துள்ளீர்கள்.

### நீங்கள் ஏற்கனவே செய்துள்ளவை

இந்த அறிமுகத்தை வாசித்து உங்கள் MCP அறிவின் அடித்தளத்தை நீங்கள் ஆரம்பித்திருப்பீர்கள். MCP என்பது என்ன, அது ஏன் முக்கியம் மற்றும் இந்த பாடத்திட்டம் உங்கள் கற்றல் பயணத்துக்கு எவ்வாறு உதவும் என்பதை நீங்கள் புரிந்துள்ளீர்கள். இது ஒரு முக்கிய சாதனையாகும் மற்றும் இந்த முக்கிய தொழில்நுட்பத்தில் உங்கள் நிபுணத்துவத்தின் தொடக்கம் ஆகும்.

### எதிர்கால சம்பவம்

மாடியூல்களை கடந்து செல்லும்போது, எல்லா நிபுணர்களும் ஒருகாலത്ത് துவக்கத்திலேயே இருந்தார்கள் என்பதை நினைவில் கொள்க. இப்போது சிக்கலாக தோன்றக்கூடிய கோளைகள் பயிற்சியால் வழக்கம் ஆகும். ஒவ்வொரு சிறிய படியும் வலிமையான திறன்களை உருவாக்கும், இது உங்கள் மேம்பாட்டு வாழ்க்கையிலே உங்களுக்கு உதவும்.

### உங்கள் ஆதரவு வலைப்பு

நீங்கள் MCP பக்கம் ஆர்வமுள்ள கற்றோர் மற்றும் நிபுணர்களின் சமுதாயத்தில் இணைக்கப்படுகிறீர்கள், அவர்கள் மற்றவர்களுக்கு வெற்றி பெற உதவ ஆர்வமாக இருக்கிறார்கள். நீங்கள் குறியீடு சவாலில் சிக்கினாலும், ஒரு முன்னேற்றத்தை பகிர்ந்துகொள்ள உற்சாகப்படுகிறீர்களானாலும், இந்த சமுதாயம் உங்கள் பயணத்தில் துணை நிற்கிறது.

உங்கள் AI செயலிகளை கட்டும்போது சிக்கல் வந்தால் அல்லது ஏதேனும் கேள்வியிருந்தால், MCP பற்றி மற்ற கற்றோர் மற்றும் அனுபவசாலிகள் நடத்திய விவாதங்களில் கலந்துகொள்ளுங்கள். இது கேள்விகள் வரவேற்கப்பட்டு அறிவு சுதந்திரமாக பகிரப்படும் ஆதரவு சமூகமாகும்.

### துவங்க தயார்?

உங்கள் MCP சாகசம் இப்போது துவங்குகிறது! மாடியூல் 0 மூலம் தொடங்கி உங்கள் முதல் முழுமையான MCP அனுபவத்தை அனுபவிக்கவும், அல்லது நீங்கள் உருவாக்கப்போகும் மாதிரி திட்டங்களை ஆராய்ந்து பாருங்கள். நினைவில் வைக்கவும் – அனைத்து நிபுணர்களும் தற்போது நீங்கள் இருக்கிற இடத்தில் துவங்கினார்கள், பொறுமையும் பயிற்சியால் நீங்கள் எவ்வளவு மிகச்சிறந்ததை அடைய முடியும் என்பதில் நீங்கள் வியத்தகம் அடைவீர்கள்.

நீங்கள் ஒரு புல் ரிக்வெஸ்ட் சமர்ப்பிக்கும் போது, CLA பாட்டு தானாகவே CLA வழங்க வேண்டுமா என்பதையும், PR-ஐ முறையான முறையில் அடையாளப்படுத்துவதையும் செய்யும் (பிரதி நிலைச் சோதனை, கருத்து வளர்ப்பு போன்றவை). பாட்டின் வழிகாட்டுதலைச் செய்யவும். இந்த செயல்முறை உங்கள் அனைத்து ரெப்போக்களுக்கும் ஒருமுறை மட்டுமே தேவைப்படும்.
