# 🚀 初學者的模型上下文協議 (MCP) 課程

## **使用 C#、Java、JavaScript、Rust、Python 和 TypeScript 的實作範例學習 MCP**

## 🧠 模型上下文協議課程概述
歡迎展開你的模型上下文協議之旅！如果你曾經好奇 AI 應用程式如何與不同工具與服務溝通，你現在將發現這套改變開發者打造智能系統方式的優雅解決方案。

你可以把 MCP 想像成 AI 應用程式的萬用翻譯器——就像 USB 連接埠讓你能將任何裝置連接到電腦，MCP 讓 AI 模型能以標準化方式連接到任何工具或服務。無論你是構建第一個聊天機器人，或是在處理複雜的 AI 工作流程，了解 MCP 將賦予你打造更強大且彈性應用的能力。

此課程設計充分考慮你的學習歷程，我們將從你已熟悉的簡單概念開始，並透過你最喜歡的程式語言進行實作練習，逐步建立專業知識。每個步驟包含清晰解說、實際範例，還有滿滿的鼓勵。

完成這段旅程後，你將具備自信，能夠打造自己的 MCP 伺服器，將其整合至熱門 AI 平台，並且理解這項技術如何重塑 AI 開發的未來。讓我們一起開始這段激動人心的冒險吧！

### 官方文件與規範

本課程遵循 **MCP 規範 2025-11-25**（最新穩定釋出版）。MCP 規範採用基於日期的版本管理（YYYY-MM-DD 格式），確保協定版本的清晰追蹤。

隨著你的理解加深，這些資源將越發珍貴，但不必感到壓力要馬上全部閱讀。先從你最感興趣的部分開始！
- 📘 [MCP 文件](https://modelcontextprotocol.io/) — 這是你的首選學習資源，提供步驟式教學與使用指南。文件針對初學者撰寫，附有清楚易懂的範例，讓你能依照自身節奏跟著操作。
- 📜 [MCP 規範](https://modelcontextprotocol.io/specification/2025-11-25) — 把它當作全面參考手冊。隨著你進行課程，你會經常回來查閱細節及探索進階功能。
- 📜 [MCP 規範版本資訊](https://modelcontextprotocol.io/specification/versioning) — 提供協定版本歷史及 MCP 如何使用基於日期版本管理（YYYY-MM-DD 格式）的相關資訊。
- 🧑‍💻 [MCP GitHub 倉庫](https://github.com/modelcontextprotocol) — 在這裡你會找到多種程式語言的 SDK、工具與範例程式碼，是實用範例與隨插即用元件的寶庫。
- 🌐 [MCP 社群](https://github.com/orgs/modelcontextprotocol/discussions) — 加入學習者與資深開發者的討論，這是一個歡迎提問並樂於分享知識的支持性社群。
  
## 學習目標

完成此課程後，你將對新技能感到自信與興奮。你將達成以下成果：

• **理解 MCP 基礎概念**：你將掌握模型上下文協議是什麼、為何它正在改變 AI 應用協同工作的運作方式，並透過易懂的比喻與範例輕鬆理解。

• **建置你的第一個 MCP 伺服器**：你會使用偏好的程式語言建立運作中的 MCP 伺服器，從簡單範例開始，逐步提升技能。

• **連接 AI 模型與真實工具**：你會學會如何將 AI 模型與實際服務橋接，為應用程式帶來強大新功能。

• <strong>實踐安全最佳做法</strong>：你將明白如何確保 MCP 實作的安全，保護應用程式與用戶。

• <strong>自信部署應用</strong>：你會知道如何將 MCP 專案從開發階段帶入產品環境，掌握實務可行的部署策略。

• **加入 MCP 社群**：你將成為不斷壯大的開發者社群一份子，攜手共塑 AI 應用開發的未來。 

## 基礎知識準備

在深入 MCP 細節前，我們先確保你對一些基礎概念感到輕鬆自在。別擔心，即使你不是這些領域的專家，我們會在過程中解釋所需知識！

### 理解協定（基石）

把協定想成一場對話的規則。你打電話給朋友時，你們都知道要說「哈囉」來開始，輪流說話，最後說「再見」結束。電腦程式也需要類似規則來有效溝通。

MCP 是一套協定——一組約定俗成的規則，協助 AI 模型與應用程式順利「對話」各種工具與服務。就像有人類對話規則讓溝通更順暢，MCP 讓 AI 應用溝通更可靠且功能強大。

### 客戶端-伺服器關係（程式如何協作）

你每天都在使用客戶端-伺服器關係！當你用瀏覽器（客戶端）訪問網站時，瀏覽器連到網頁伺服器，伺服器送回頁面內容。瀏覽器知道如何請求資訊，伺服器知道如何回覆。

在 MCP 中，我們有類似的關係：AI 模型是請求資訊或動作的客戶端，MCP 伺服器提供這些功能。就像有一位幫手（伺服器）能接 AI 的請求幫忙執行任務。

### 為何標準化至關重要（讓一切協同運作）

想像每個汽車製造商都用不同形狀的加油槍——你得為每輛車準備不同轉接頭！標準化就是大家同意用相同方式，確保元件能無縫配合。

MCP 為 AI 應用提供這種標準化。不是每個 AI 模型都得寫專屬程式碼來配合每個工具，MCP 創造通用溝通方式。如此一來，開發者只需一次打造工具，就能搭配各種不同 AI 系統。

## 🧭 你的學習路徑概述

你的 MCP 旅程經過精心設計，讓你一步步建立信心與技能。每個階段介紹新概念，同時強化已學內容。

### 🌱 基礎階段：理解基礎知識（模組 0-2）

冒險從這裡開始！我們會用熟悉的比喻與簡單範例帶你認識 MCP 概念。你會明白 MCP 是什麼、為何存在，以及它如何融入 AI 開發的宏大格局。

• **模組 0 - MCP 介紹**：我們將開始探索 MCP 是什麼、為什麼對現代 AI 應用如此重要。你會看到 MCP 實際運作的案例，了解它如何解決開發者常見問題。

• **模組 1 - 核心概念說明**：這裡你將學到 MCP 的基本建構要素。大量比喻與視覺範例幫助你輕鬆理解這些概念。

• **模組 2 - MCP 的安全性**：安全性聽來可能讓人害怕，但我們會展示 MCP 如何內建安全機制，並教你如何實踐一開始就保護應用的最佳做法。

### 🔨 建置階段：打造你的首個實作（模組 3）
現在真正的樂趣開始了！你將親手實踐建立真正的 MCP 伺服器和客戶端。別擔心——我們會從簡單開始，並引導你完成每一個步驟。

本模塊包含多個實作指南，讓你能用你偏好的程式語言進行練習。你將建立你的第一個伺服器，打造一個能連接該伺服器的客戶端，甚至整合像 VS Code 這樣的熱門開發工具。

每個指南都包含完整的程式碼範例、故障排除提示，以及我們為何做出特定設計選擇的說明。到這個階段結束時，你將擁有可實際運作並引以為傲的 MCP 實作！

### 🚀 成長階段：進階概念與實務應用（模組 4-5）

掌握基礎後，你準備好探索更複雜的 MCP 功能。我們將涵蓋實務的實作策略、除錯技巧，以及多模態 AI 整合等進階主題。

你還會學習如何擴展 MCP 實作以用於生產環境，並與 Azure 等雲端平台整合。這些模組將助你打造能應對實務需求的 MCP 解決方案。

### 🌟 精通階段：社群與專精（模組 6-11）

最後階段專注於加入 MCP 社群並在你感興趣的領域專精。你將學習如何貢獻開源 MCP 專案、實作進階的驗證模式，並打造整合資料庫的完整解決方案。

mod 11 特別值得一提——這是一條完整包含 13 個實作實驗的學習路徑，教你建立具備 PostgreSQL 整合的可生產 MCP 伺服器。這就像是一個結業專案，結合了你所學的所有內容！
