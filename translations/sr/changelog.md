**Модул 05 - Адверзаријални агенти (`mcp-adversarial-agents`)**  
- **Безбедносна исправка — инјекција команди**: Замена shell интерполације `execSync` са `execFile` + `promisify` у TypeScript алату `run_python`, елиминишући површину за инјекцију команди (код контролисан LLM-ом се сада прослеђује као литерални argv елемент без укључивања shell-а)  
- **Ожичење петље MCP алата**: Ажурирање Python оркестратора дебате да користи `AsyncAnthropic` клијента (умисто блокирајућег синхроног `Anthropic`), прослеђује уживо `ClientSession` директно сваком агентском потезу, сваки пут захтева дефиниције алата преко `session.list_tools()`, и шаље `tool_use` блокове преко `session.call_tool()` у петљи док модел не емитује коначан текстуални одговор
