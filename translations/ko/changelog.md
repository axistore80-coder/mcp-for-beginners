#### 코드 및 보안 수정

**모듈 05 - 적대적 에이전트 (`mcp-adversarial-agents`)**
- **보안 수정 — 명령어 인젝션 방지**: 타입스크립트의 `run_python` 도구에서 `execSync` 셸 인터폴레이션을 `execFile` + `promisify`로 교체하여 명령어 인젝션 공격면 제거 (LLM 제어 코드는 이제 셸을 통하지 않고 리터럴 argv 요소로 전달됨)
- **MCP 도구 루프 연결 수정**: 파이썬 토론 조율자를 블로킹 동기 `Anthropic` 대신 `AsyncAnthropic` 클라이언트로, 각 에이전트 턴에 라이브 `ClientSession`을 직접 전달하도록 변경. 각 턴마다 `session.list_tools()`로 도구 정의를 가져오고, 모델이 최종 텍스트 응답을 낼 때까지 `session.call_tool()`로 `tool_use` 블록을 루프 내에서 호출

### MCP 사양 2025-11-25 준수 검토

#### 핵심 개념 강화 (01-CoreConcepts/)
- **새 클라이언트 프리미티브 - Roots**: 서버가 파일시스템 경계와 접근 권한을 이해할 수 있도록 하는 Roots 클라이언트 프리미티브에 대한 포괄적 문서 추가
- **도구 주석**: 도구 실행 결정을 향상시키기 위한 `readOnlyHint`, `destructiveHint` 도구 행동 주석 문서 추가
- **샘플링 내 도구 호출**: 샘플링 요청 중 모델 주도 도구 호출을 위한 `tools` 및 `toolChoice` 매개변수 관련 샘플링 문서 업데이트
- **URL 모드 유도**: 서버가 시작하는 외부 웹 상호작용을 위한 URL 기반 유도 문서 추가
- **작업(Task, 실험적)**: 내구 실행 래퍼 및 지연 결과 검색을 위한 실험적 작업 기능 문서화 섹션 추가
- **아이콘 지원**: 도구, 리소스, 리소스 템플릿 및 프롬프트가 추가 메타데이터로 아이콘을 포함할 수 있음을 명시
