**ਮਾਡਿਊਲ 05 - ਵਿਰੋਧੀ ਏਜੰਟ (`mcp-adversarial-agents`)**
- **ਸੁਰੱਖਿਆ ਸੁਧਾਰ — ਕਮਾਂਡ ਇੰਜੈਕਸ਼ਨ**: TypeScript ਦੇ `run_python` ਸਾਧਨ ਵਿੱਚ `execSync` ਸ਼ੈੱਲ ਇੰਟਰਪੋਲੇਸ਼ਨ ਨੂੰ `execFile` + `promisify` ਨਾਲ ਬਦਲਿਆ ਜਾ ਚੁੱਕਾ ਹੈ, ਜਿਸ ਨਾਲ ਕਮਾਂਡ ਇੰਜੈਕਸ਼ਨ ਖ਼ਤਰਾ ਖ਼ਤਮ ਹੋਇਆ (LLM-ਨियੰਤਰਿਤ ਕੋਡ ਹੁਣ ਲਿਟਰਲ argv ਤੱਤ ਵਜੋਂ ਗੁਜ਼ਰਦਾ ਹੈ ਬਿਨਾਂ ਕਿਸੇ ਸ਼ੈੱਲ ਦੀ ਮਦਦ)
- **MCP ਸਾਧਨ ਲੂਪ ਵਾਇਰਿੰਗ**: Python ਵਿਚਕਾਰ ਦੀਆਂ ਗੱਲਬਾਤ ਲਈ debate orchestrator ਨੂੰ `AsyncAnthropic` ਕਲਾਇੰਟ (ਰੁਕਾਬਟ ਵਾਲੇ sync `Anthropic` ਦੀ ਥਾਂ) ਵਰਤਣ ਲਈ ਅਪਡੇਟ ਕੀਤਾ, ਹਰ ਏਜੰਟ ਟਰਨ ਨੂੰ ਸਿੱਧਾ ਲਾਈਵ `ClientSession` ਛੱਡਣ ਲਈ, ਹਰੇਕ ਟਰਨ `session.list_tools()` ਨਾਲ ਸਾਧਨਾਂ ਦੀ ਸੂਚੀ ਪ੍ਰਾਪਤ ਕਰਨਾ, ਅਤੇ ਮਾਡਲ ਵੱਲੋਂ ਆਖਰੀ ਲਿਖਤੀ ਜਵਾਬ ਦੇਣ ਤਕ `tool_use` ਬਲੌਕ ਨੂੰ `session.call_tool()` ਲੂਪ ਵਿੱਚ ਡਿਸਪੈਚ ਕਰਨਾ।

#### ਮੁੱਖ ਸੰਕਲਪ ਸਧਾਰਨ (01-CoreConcepts/)
- **ਨਵਾਂ ਕਲਾਇੰਟ ਪ੍ਰਿਮਿਟਿਵ - Roots**: Roots ਕਲਾਇੰਟ ਪ੍ਰਿਮਿਟਿਵ ਬਾਰੇ ਵਿਸਥਾਰਪੂਰਕ ਦਸਤਾਵੇਜ਼ ਸ਼ਾਮਿਲ ਕਰਕੇ ਸਰਵਰਾਂ ਨੂੰ ਫਾਈਲਸਿਸਟਮ ਸੀਮਾਵਾਂ ਅਤੇ ਪਹੁੰਚ ਅਧਿਕਾਰ ਸਮਝਣ ਯੋਗ ਬਣਾਇਆ
- **ਟੂਲ ਐਨੋਟੇਸ਼ਨਜ਼**: ਸਧਾਰਨ ਫੈਸਲਿਆਂ ਲਈ ਟੂਲ ਦੇ ਵਰਤਾਰ ਵਿਵਰਣ (`readOnlyHint`, `destructiveHint`) ਬਾਰੇ ਦਸਤਾਵੇਜ਼ ਸ਼ਾਮਿਲ
- **ਸੈਂਪਲਿੰਗ ਵਿੱਚ ਟੂਲ ਕਾਲਿੰਗ**: ਮਾਡਲ-ਚਾਲਤ ਟੂਲ ਆਹੁਤੀਆਂ ਲਈ `tools` ਅਤੇ `toolChoice` ਪੈਰਾਮੀਟਰ ਸਮੇਤ ਸੈਂਪਲਿੰਗ ਦਸਤਾਵੇਜ਼ ਅੱਪਡੇਟ ਕੀਤਾ
- **URL ਮੋਡ ਇਲਿਸਿਟੇਸ਼ਨ**: ਸਰਵਰ-ਆਰੰਭਿਤ ਬਾਹਰੀ ਵੈੱਬ ਪ੍ਰਕਿਰਿਆਵਾਂ ਲਈ URL-ਆਧਾਰਿਤ ਇਲਿਸਿਟੇਸ਼ਨ ਬਾਰੇ ਦਸਤਾਵੇਜ਼ ਜੋੜੀ
- **ਟਾਸਕ (ਪ੍ਰਯੋਗਾਤਮਕ)**: ਦੂਰਗામી ਐਗਜ਼ਿਕਿਊਸ਼ਨ ਵ੍ਰੈਪਰਾਂ ਅਤੇ ਮੁਲਤਵੀ ਨਤੀਜੇ ਪ੍ਰਾਪਤੀ ਲਈ ਪ੍ਰਯੋਗਾਤਮਕ ਟਾਸਕ ਫੀਚਰ ਬਾਰੇ ਨਵਾਂ ਸੈਕਸ਼ਨ
- **ਆਈਕਾਨ ਸਪੋਰਟ**: ਟੂਲ, ਸਰੋਤ, ਸਰੋਤ ਟੈਂਪਲੇਟ, ਅਤੇ ਪ੍ਰੰਪਟਸ ਹੁਣ ਵਾਧੂ ਮੈਟੈਡੇਟਾ ਵਜੋਂ ਆਈਕਾਨ ਸ਼ਾਮਿਲ ਕਰ ਸਕਦੇ ਹਨ

#### ਦਸਤਾਵੇਜ਼ ਅੱਪਡੇਟ
- **README.md**: MCP ਵਿਸ਼ੇਸ਼ਣ 2025-11-25 ਵਰਜ਼ਨ ਦਾ ਹਵਾਲਾ ਅਤੇ ਮਿਤੀ-ਆਧਾਰਿਤ ਵਰਜ਼ਨਿੰਗ ਵਿਆਖਿਆ ਸ਼ਾਮਿਲ ਕੀਤੀ
- **study_guide.md**: ਕੰਰੀਕੁਲਮ ਨਕਸ਼ੇ ਵਿੱਚ ਟਾਸਕ ਅਤੇ ਟੂਲ ਐਨੋਟੇਸ਼ਨਜ਼ ਨੂੰ ਕੋਰ ਸੰਕਲਪ ਸੈਕਸ਼ਨ ਵਿੱਚ ਸ਼ਾਮਿਲ ਕੀਤਾ; ਦਸਤਾਵੇਜ਼ ਟਾਈਮਸਟੀੰਪ ਅੱਪਡੇਟ ਕੀਤਾ

#### ਵਿਸ਼ੇਸ਼ਣ ਅਨੁਕੂਲਤਾ ਜाँच
- **ਪ੍ਰੋਟੋਕੋਲ ਵਰਜ਼ਨ**: ਸਾਰੇ ਦਸਤਾਵੇਜ਼ ਮੌਜੂਦਾ MCP ਵਿਸ਼ੇਸ਼ਣ 2025-11-25 ਨੂੰ ਦਰਸਾਉਂਦੇ ਹਨ
- **ਆਰਕੀਟੈਕਚਰ ਏਲਾਈਨਮੈਂਟ**: ਦੋ-ਪਰਤ ਵਾਲੀ ਆਰਕੀਟੈਕਚਰ (ਡੇਟਾ ਲੇਅਰ + ਟ੍ਰਾਂਸਪੋਰਟ ਲੇਅਰ) ਦਸਤਾਵੇਜ਼ੀ ਸਹੀਤਾ
- **ਪ੍ਰਿਮਿਟਿਵਸ ਦਸਤਾਵੇਜ਼ੀ**: ਸਰਵਰ ਪ੍ਰਿਮਿਟਿਵਸ (Resources, Prompts, Tools) ਅਤੇ ਕਲਾਇੰਟ ਪ੍ਰਿਮਿਟਿਵਸ (Sampling, Elicitation, Logging, Roots) ਦੀ ਪੁਸ਼ਟੀ
- **ਟ੍ਰਾਂਸਪੋਰਟ ਮਕੈਨਿਜ਼ਮ**: STDIO ਅਤੇ Streamable HTTP ਟ੍ਰਾਂਸਪੋਰਟ ਦਸਤਾਵੇਜ਼ ਸਹੀ
- **ਸੁਰੱਖਿਆ ਮਾਰਗਦਰਸ਼ਨ**: ਮੌਜੂਦਾ MCP ਸੁਰੱਖਿਆ ਵਧੀਆ ਅਮਲਾਂ ਨਾਲ ਸੰਗਤਿਤ

#### ਮੁੱਖ MCP 2025-11-25 ਫੀਚਰਾਂ ਦਾ ਦਸਤਾਵੇਜ਼ੀਕਰਨ
- **OpenID Connect ਡਿਸਕਵਰੀ**: OIDC ਰਾਹੀਂ ਪ੍ਰਮਾਣਿਕਤਾ ਸਰਵਰ ਖੋਜ
- **OAuth ਕਲਾਇੰਟ ID ਮੇਟਾ ਡਾਟਾ ਦਸਤਾਵੇਜ਼**: ਸਿਫਾਰਸ਼ੀ ਕਲਾਇੰਟ ਰਜਿਸਟ੍ਰੇਸ਼ਨ ਪ੍ਰਣਾਲੀ
- **JSON ਸਕੀਮਾ 2020-12**: MCP ਸਕੀਮਾ ਪਰਿਭਾਸ਼ਾਵਾਂ ਲਈ ਡੀਫਾਲਟ ਬੋਲੀ
- **SDK ਟੀਅਰਿੰਗ ਸਿਸਟਮ**: SDK ਫੀਚਰ ਸਹਾਇਤਾ ਅਤੇ ਰੱਖ-ਰਖਾਵ ਲਈ ਅਧਿਕਾਰਿਤ ਲੋੜਾਂ
- **ਸਰਕਾਰੀ ਸਮਿਤੀਆਂ**: MCP ਗਵਰਨੈਂਸ ਵਿੱਚ ਵਰਕਿੰਗ ਗਰੁੱਪ ਅਤੇ ਰੁਚੀ ਗਰੁੱਪ ਦਾ ਅਧਿਕਾਰਤ ਡਾਂਚਾ

  - ਸਰਵਰ ਪ੍ਰਿਮਿਟਿਵਸ: ਡਾਟਾ ਸ੍ਰੋਤ, ਟੈਂਪਲੇਟ, ਚਲਾਏ ਜਾ ਸਕਦੇ ਫੰਕਸ਼ਨ ਜਿਵੇਂ ਟੂਲਾਂ ਦੀ ਮੁਹੱਈਆ ਅਤੇ ਵੇਰਵਾ ਸਹਿਤ ਉਦਾਹਰਣ
  - ਕਲਾਇੰਟ ਪ੍ਰਿਮਿਟਿਵਸ: ਸੈਂਪਲਿੰਗ (LLM ਪੂਰਨਤਾ), ਪ੍ਰੇਰਣਾ (ਉਪਭੋਗਤਾ ਇਨਪੁੱਟ), ਲੌਗਿੰਗ (ਡਿਬੱਗ/ਨਿਗਰਾਨੀ)
  - ਮੌਜੂਦਾ ਖੋਜ (`*/list`), ਰੀਟਰੀਵਲ (`*/get`), ਅਤੇ ਚਲਾਉਣ (`*/call`) ਮੈਥਡ ਪੈਟਰਨਾਂ ਨਾਲ ਅਪਡੇਟ ਕੀਤਾ
- **ਪ੍ਰੋਟੋਕੋਲ ਆਰਕੀਟੈਕਚਰ**: ਦੋ-ਪੜ੍ਹਾਈ ਵਾਲਾ ਆਰਕੀਟੈਕਚਰ ਮਾਡਲ ਪੇਸ਼ ਕੀਤਾ
  - ਡਾਟਾ ਲੇਅਰ: JSON-RPC 2.0 ਅਧਾਰ ਦੇ ਨਾਲ ਜੀਵਨਚੱਕਰ ਪ੍ਰਬੰਧਨ ਅਤੇ ਪ੍ਰਿਮਿਟਿਵਸ
  - ਟ੍ਰਾਂਸਪੋਰਟ ਲੇਅਰ: STDIO (ਸਥਾਨਕ) ਅਤੇ Streamable HTTP ਨਾਲ SSE (ਦੂਰੀਵਾਲਾ) ਟ੍ਰਾਂਸਪੋਰਟ ਪ੍ਰਣਾਲੀ
- **ਸੁਰੱਖਿਆ ਫਰੇਮਵਰਕ**: ਸਵੈਛਿਕ ਉਪਭੋਗਤਾ ਸਹਿਮਤੀ, ਡਾਟਾ ਪ੍ਰਾਈਵੇਸੀ ਰੱਖਿਆ, ਟੂਲ ਚਲਾਉਣਾ ਸੁਰੱਖਿਆ, ਅਤੇ ਟ੍ਰਾਂਸਪੋਰਟ ਲੇਅਰ ਸੁਰੱਖਿਆ ਸਮੇਤ ਵਿਸਥਾਰਤ ਨੀਤੀਆਂ
- **ਸੰਚਾਰ ਪੈਟਰਨ**: ਸ਼ੁਰੂਆਤ, ਖੋਜ, ਚਲਾਉਣ ਅਤੇ ਸੂਚਨਾ ਪ੍ਰਵਾਹ ਦਰਸਾਉਂਦੇ ਪ੍ਰੋਟੋਕੋਲ ਸੁਨੇਹੇ ਅਪਡੇਟ ਕੀਤੇ
- **ਕੋਡ ਉਦਾਹਰਣ**: ਮੌਜੂਦਾ MCP SDK ਪੈਟਰਨਾਂ ਨੂੰ ਦਰਸਾਉਂਦੀਆਂ ਬਹੁ-ਭਾਸ਼ਾਈ ਉਦਾਹਰਣਾਂ (.NET, ਜਾਵਾ, ਪਾਇਥਾਨ, ਜਾਵਾਸਕ੍ਰਿਪਟ) ਨਵੀਨਤਮ ਕੀਤੀਆਂ

#### ਸੁਰੱਖਿਆ (02-Security/) - ਵਿਆਪਕ ਸੁਰੱਖਿਆ ਮੁੜ-ਨਿਰਮਾਣ
- **ਮਿਆਰ ਅਨੁਕੂਲਤਾ**: MCP Specification 2025-06-18 ਸੁਰੱਖਿਆ ਲੋੜਾਂ ਨਾਲ ਪੂਰੀ ਤਰ੍ਹਾਂ ਅਨੁਕੂਲ ਕੀਤਾ
- **ਪ੍ਰਮਾਣਿਕਤਾ ਦਾ ਵਿਕਾਸ**: ਕਸਟਮ OAuth ਸਰਵਰਾਂ ਤੋਂ ਬਾਹਰੀ ਪਹਿਚਾਣ ਪ੍ਰਦਾਤਾ ਡੇਲਿਗੇਸ਼ਨ (Microsoft Entra ID) ਤੱਕ ਵਿਕਾਸ ਦਰਸਾਇਆ
- **AI-ਵਿਸ਼ੇਸ਼ ਝੂਠੇ ਖਤਰੇ ਦੀ ਵਿਸ਼ਲੇਸ਼ਣ**: ਆਧੁਨਿਕ AI ਹਮਲਿਆਂ ਦੀ ਵਧੀਕ ਕਵਰੇਜ
  - ਅਸਲੀ ਝੂਠੇ ਹਮਲੇ ਦੇ ਪ੍ਰੈਪਟ ਇੰਜੈਕਸ਼ਨ ਦ੍ਰਿਸ਼ਟੀਕੋਣ ਅਤੇ ਉਦਾਹਰਣ
  - ਟੂਲ ਵਿਸ਼ਾਕਤ ਕਰਨ ਦੇ ਤਰੀਕੇ ਅਤੇ "ਰੱਗ ਪੁੱਲ" ਹਮਲੇ ਦੇ ਪੈਟਰਨ
  - ਸੰਦਰਭ ਵਿੰਡੋ ਜਹਿਰੀਲਾ ਕਰਨ ਅਤੇ ਮਾਡਲ ਗੁੰਝਲਤਾ ਹਮਲੇ
- **Microsoft AI ਸੁਰੱਖਿਆ ਹੱਲ**: Microsoft ਸੁਰੱਖਿਆ ਪਰਿਸ਼ਰ ਦੀ ਵਿਆਪਕ ਕਵਰੇਜ
  - AI Prompt Shields ਵਧੀਆ ਪਛਾਣ, ਸਪੌਟਲਾਈਟਿੰਗ ਅਤੇ ਡੈਲੀਮੀਟਰ ਤਕਨੀਕਾਂ ਨਾਲ
  - Azure Content Safety ਏਕੀਕਰਨ ਪੈਟਰਨ
  - ਸਪਲਾਈ ਚੇਨ ਸੁਰੱਖਿਆ ਲਈ GitHub Advanced Security
- **ਆਧੁਨਿਕ ਖ਼ਤਰੇ ਦਾ ਨਿਵਾਰਨ**: ਵੇਰਵਾ ਸਹਿਤ ਸੁਰੱਖਿਆ ਨਿਯੰਤਰਣ
  - MCP-ਵਿਸ਼ੇਸ਼ ਹਮਲਿਆਂ ਅਤੇ ਗੁਪਤ ਜੋੜਨ ਲਈ ਸੈਸ਼ਨ ਹਾਈਜੈਕਿੰਗ
  - MCP ਪ੍ਰਾਕਸੀ ਹਾਲਾਤਾਂ ਵਿੱਚ ਗੁੰਝਲਦਾਰ ਡਿਪਿਊ ਸਮੱਸਿਆਵਾਂ ਲਈ ਸਪਸ਼ਟ ਸਹਿਮਤੀ ਲੋੜਾਂ
  - ਟੋਕਨ ਪਾਸਥਰੂ ਵਿੱਚ ਜ਼ਰੂਰੀ ਵੈਧਤਾ ਨਿਯੰਤਰਣ
- **ਸਪਲਾਈ ਚੇਨ ਸੁਰੱਖਿਆ**: ਫਾਊਂਡੇਸ਼ਨ ਮਾਡਲ, ਐਂਬੈਡਡਿੰਗ ਸੇਵਾਵਾਂ, ਸੰਦਰਭ ਪ੍ਰਦਾਤਾ, ਅਤੇ ਤੀਜੀ-ਪੱਖੀ APIs ਸਮੇਤ AI ਸਪਲਾਈ ਚੇਨ ਦਾ ਵਿਆਪਕ ਕਵਰੇਜ
