# ការរត់គំរូនេះ

អ្នកត្រូវបានណែនាំឱ្យតំឡើង `uv` ប៉ុន្តែមិនបាច់នោះទេ សូមមើល [សេចក្តីណែនាំ](https://docs.astral.sh/uv/#highlights)

## -1- តំឡើងការទាមទារ

```bash
npm install
```

## -3- រត់ម៉ាស៊ីនបម្រើ


```bash
npm run build
```

## -4- រត់កម្មវិធីអតិថិជន

```sh
npm run client
```

អ្នកគួរតែឃើញលទ្ធផលស្រដៀងនឹង៖
