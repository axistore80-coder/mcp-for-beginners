# រត់គំរូនេះ

> [!NOTE]
> គ្លីហ្សង់ Rust នេះសន្មតថាអ្នកបានដំឡើង dotnet 8.0 SDK ដើម្បីរត់ម៉ាសុីនគំរូ MCP ស្ថិតក្នុងថត [server](../../../../../../03-GettingStarted/02-client/solution/server) ដែលជាថតបងប្អូននៃគម្រោងនេះ។

## -1- ដំឡើងការពឹងផ្អែក និង បង្កើតគម្រោង

```bash
cargo build
```

## -2- រត់គំរូ

```bash
cargo run
```
