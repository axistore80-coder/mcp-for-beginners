# ការប្រតិបត្តិតាមគំរូ

នៅទីនេះ យើងសន្មតថាអ្នកមានកូដម៉ាស៊ីនបម្រើដែលដំណើរការបានរួចហើយ។ សូមស្វែងរកម៉ាស៊ីនបម្រើមួយពីជំពូកដើមៗណាមួយ។

## កំណត់ mcp.json

នេះជាឯកសារដែលអ្នកប្រើសម្រាប់យោង, [mcp.json](../../../../../03-GettingStarted/04-vscode/solution/mcp.json)។

ប្ដូរចំណុចម៉ាស៊ីនបម្រើតាមតម្រៀប ដើម្បីបញ្ជាក់ផ្លូវត្រឹមត្រូវចំពោះម៉ាស៊ីនបម្រើរបស់អ្នក រួមទាំងបញ្ជាផ្លូវពេញលេញដែលត្រូវការដើម្បីរត់។

នៅក្នុងឯកសារគំរូដែលបានយោងខាងលើ ចំណុចម៉ាស៊ីនបម្រើមានរាងដូចខាងក្រោម៖

<details>
<summary>node.js</summary>
```json
"hello-mcp": {
    "command": "node",
    "args": [
        "build/index.js"
    ]
}
```
</details>

<details>
<summary>.NET</summary>

អ្នកប្រហែលជាត្រូវបញ្ចូលដំណើរការរុក្ខជាតិ GitHub repository ដែលអាចទាញយកបានពីពាក្យបញ្ជា `git rev-parse --show-toplevel`។

```jsonc
{
  "inputs": [
    {
      "type": "promptString",
      "id": "repository-root",
      "description": "The absolute path to the repository root"
    }
  ],
  "servers": {
    "calculator-mcp-dotnet": {
      "type": "stdio",
      "command": "dotnet",
      "args": [
        "run",
        "--project",
        "${input:repository-root}/03-GettingStarted/02-client/solution/server/server.csproj"
      ]
    }
  }
}
```

</details>

នេះនឹងសមនឹងការរត់ពាក្យបញ្ជាដូចជាៈ `node build/index.js`។

- ប្ដូរចំណុចម៉ាស៊ីនបម្រើនេះ ឲ្យសមស្រប កន្លែងដែលឯកសារម៉ាស៊ីនបម្រើរបស់អ្នកស្ថិតឬតាមតម្រូវការដើម្បីចាប់ផ្តើមម៉ាស៊ីនបម្រើរបស់អ្នកផ្អែកលើរបៀបចាប់ផ្តើម(runtime) និងទីតាំងម៉ាស៊ីនបម្រើដែលអ្នកបានជ្រើសរើស។

## ប្រើមុខងារនៅក្នុងម៉ាស៊ីនបម្រើ

- ចុចរូបតំណាង `play` ពេលដែលអ្នកបានបញ្ចូល *mcp.json* ទៅថត *./vscode* ហើយ,

    ស្តាប់មើលការផ្លាស់ប្តូររូបតំណាងឧបករណ៍ ដើម្បីបន្ថែមចំនួនឧបករណ៍ដែលអាចប្រើបាន។ រូបតំណាងឧបករណ៍ស្ថិតនៅខាងលើប្រអប់សន្ទនា ក្នុង GitHub Copilot។

## រត់ឧបករណ៍មួយ

- វាយពាក្យបញ្ជានៅក្នុងបង្អួចសន្ទនារបស់អ្នកដែលអាចផ្គូផ្គងនឹងការពិពណ៌នារបស់ឧបករណ៍។ ឧទាហរណ៍ ដើម្បីចាប់ផ្តើមឧបករណ៍ `add` វាយអ្វីម្យ៉ាងដូចជា "add 3 to 20"។

    អ្នកគួរតែបានឃើញឧបករណ៍មួយបង្ហាញនៅលើប្រអប់អត្ថបទទាំងសន្ទនា ដើម្បីជម្រើសរត់ឧបករណ៍ ដូចរូបភាពខាងក្រោម៖

    ![VS Code indicating it wanting to run a tool](../../../../../translated_images/km/vscode-agent.d5a0e0b897331060.webp)

    ការជម្រើសឧបករណ៍គួរតែផលិតលទ្ធផលជាលេខ "23" ប្រសិនបើពាក្យបញ្ជារបស់អ្នកដូចដែលបានរៀបរាប់ពីមុន។
