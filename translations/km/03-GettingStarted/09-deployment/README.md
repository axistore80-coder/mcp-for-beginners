# бһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһҮбһјбһ“бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP

бһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһҮбһјбһ“бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһўбҹ„бһҷбһўбҹ’бһ“бһҖбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбһ…бһјбһӣбһ”бҹ’бһҡбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһ“бһ·бһ„бһ’бһ“бһ’бһ¶бһ“бһҡбһ”бһҹбҹӢбһңбһ¶бһҳбһҖбһӣбһҫбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһҖбҹ’бһ“бһ»бһ„бһҹбҹ’бһҡбһ»бһҖбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһҳбһ¶бһ“бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ–бһ бһ»бһ”бһ‘бһҠбҹ’бһӢбһ¶бһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһҮбһјбһ“бһҠбҹӮбһӣбһӮбһҪбһҡбһ–бһ·бһ…бһ¶бһҡбһҺбһ¶ бһҠбһјбһ…бһҮбһ¶бһҖбһ¶бһҡбһӣбҹҶбһ бһјбһҡ, бһ—бһ¶бһ–бһ‘бҹҖбһ„бһ‘бһ¶бһҸбҹӢ, бһ“бһ·бһ„бһ—бһ¶бһ–бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бҹ” бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһғбһҫбһүбһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһҮбһјбһ“бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP бһҖбҹ’бһ“бһ»бһ„бһҹбҹ’бһҡбһ»бһҖ, бһҖбҹ’бһ“бһ»бһ„бһҖбһ»бһ„бһҸбһәбһ“бҹҗбһҡ, бһ“бһ·бһ„бһ‘бҹ…бһҳбҹҒбһғбҹ”

## бһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһҹбһ„бҹ’бһҒбҹҒбһ”

бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҸбһ”бҹӢбһ–бһёбһҡбһ”бҹҖбһ”бһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһҮбһјбһ“бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё MCP Server бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

## бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶

бһ“бҹ…бһ…бһ»бһ„бһ”бһүбҹ’бһ…бһ”бҹӢбһ“бҹғбһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһўбһ¶бһ…бһ’бҹ’бһңбһҫбһ”бһ¶бһ“бҹ–

- бһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһ‘бҹ’бһҡбһ№бһҹбҹ’бһҠбһёбһ•бҹ’бһҹбҹҒбһ„бҹ—бһ“бҹғбһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһҮбһјбһ“бҹ”
- бһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһҮбһјбһ“бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

## бһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚ бһ“бһ·бһ„бһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһҮбһјбһ“бһҖбҹ’бһ“бһ»бһ„бһҹбҹ’бһҡбһ»бһҖ

бһ”бһҫбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҠбҹ„бһҷбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһӣбһҫбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ’бҹ’бһңбһҫбһҸбһ¶бһҳбһҮбҹҶбһ бҹҠбһ¶бһ“бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

1. **бһ‘бһ¶бһүбһҷбһҖбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ**бҹ” бһ”бһҫбһўбҹ’бһ“бһҖбһҳбһ·бһ“бһ”бһ¶бһ“бһҹбһҡбһҹбҹҒбһҡбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ бһ”бһҫбһҠбһјбһ…бҹ’бһ“бҹ„бҹҮбһ‘бһ¶бһүбһҷбһҖбһңбһ¶бһӣбһҫбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҮбһ¶бһҳбһ»бһ“бһҹбһ·бһ“бҹ”  
1. **бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ**бҹ– бһ”бһҫбһҖбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё MCP Server бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ

бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ SSE (бһҳбһ·бһ“бһ‘бһ¶бһҳбһ‘бһ¶бһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒбһ”бҹ’бһҡбһ—бҹҒбһ‘ stdio)

1. **бһҖбҹҶбһҺбһҸбҹӢбһ”бһҺбҹ’бһҸбһ¶бһү**бҹ– бһ’бҹ’бһңбһҫбһұбҹ’бһҷбһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒбһўбһ¶бһ…бһ…бһјбһӣбһ”бҹ’бһҡбһҫбһ”бһ¶бһ“бһ“бҹ…бһҖбҹҶбһ–бһ„бҹӢбһ•бҹӮбһҠбҹӮбһӣбһҡбҹҶбһ–бһ№бһ„бһ‘бһ»бһҖ  
1. **бһ—бҹ’бһҮбһ¶бһ”бҹӢбһўбһҸбһ·бһҗбһ·бһҮбһ“**бҹ– бһ”бҹ’бһҡбһҫ URL бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҖбҹ’бһ“бһ»бһ„бһҹбҹ’бһҡбһ»бһҖбһҠбһјбһ…бһҮбһ¶ `http://localhost:3000`

## бһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһҮбһјбһ“бһ‘бҹ…бһҳбҹҒбһғ

бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP бһўбһ¶бһ…бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһҮбһјбһ“бһ‘бҹ…бһңбҹҒбһ‘бһ·бһҖбһ¶бһҳбҹҒбһғбһ•бҹ’бһҹбҹҒбһ„бҹ—бһ”бһ¶бһ“бҹ–

- **бһҳбһ»бһҒбһ„бһ¶бһҡ Serverless**бҹ– бһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһҮбһјбһ“бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP бһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбһ»бһҒбһ„бһ¶бһҡ serverless бһҸбһјбһ…бҹ—  
- **бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһҖбһ»бһ„бһҸбһәбһ“бҹҗбһҡ**бҹ– бһ”бҹ’бһҡбһҫбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһҠбһјбһ…бһҮбһ¶ Azure Container Apps, AWS ECS, бһ¬ Google Cloud Run  
- **Kubernetes**бҹ– бһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһҮбһјбһ“ бһ“бһ·бһ„бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP бһҖбҹ’бһ“бһ»бһ„бһҖбҹ’бһҡбһ»бһҳ Kubernetes бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бһ¶бһ–бһҡбһ№бһ„бһҳбһ¶бҹҶбһҒбҹ’бһ–бһҹбҹӢ

### бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбҹ– Azure Container Apps

Azure Container Apps бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһҮбһјбһ“бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCPбҹ” бһңбһ¶вҖӢбһ“бҹ…бһҸбҹӮбһҮбһ¶бһҖбһ¶бһҡбһ„бһ¶бһҡбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбһёбһҖбһ…бһҳбҹ’бһҡбһҫбһ“ бһ бһҫбһҷбһ–бҹҒбһӣбһ“бҹҒбҹҮбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ SSEбҹ”

бһ“бҹҒбҹҮбһҮбһ¶бһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ’бҹ’бһңбһҫбһ”бһ¶бһ“бҹ–

1. бһ•бҹ’бһ‘бһ»бһҖбһҠбһҖбһҮбһ¶бһҡбһ”бһјбһ”бҹ–

  ```sh
  git clone https://github.com/anthonychu/azure-container-apps-mcp-sample.git
  ```

1. бһҡбһҸбҹӢбһңбһ¶бһҖбҹ’бһ“бһ»бһ„бһҹбҹ’бһҡбһ»бһҖбһҠбһҫбһҳбҹ’бһ”бһёбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бҹ–

  ```sh
  uv venv
  uv sync

  # linux/macOS
  export API_KEYS=<AN_API_KEY>
  # бһңбһёбһ“бһҠбһј
  set API_KEYS=<AN_API_KEY>

  uv run fastapi dev main.py
  ```

1. бһҠбһҫбһҳбҹ’бһ”бһёбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҖбҹ’бһ“бһ»бһ„бһҹбҹ’бһҡбһ»бһҖ бһ”бһ„бҹ’бһҖбһҫбһҸбһҜбһҖбһҹбһ¶бһҡ *mcp.json* бһҖбҹ’бһ“бһ»бһ„бһҗбһҸ *.vscode* бһ бһҫбһҷбһ”бһүбҹ’бһ…бһјбһӣбһҳбһ¶бһҸбһ·бһҖбһ¶бһҡбһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

  ```json
  {
      "inputs": [
          {
              "type": "promptString",
              "id": "weather-api-key",
              "description": "Weather API Key",
              "password": true
          }
      ],
      "servers": {
          "weather-sse": {
              "type": "sse",
              "url": "http://localhost:8000/sse",
              "headers": {
                  "x-api-key": "${input:weather-api-key}"
              }
          }
      }
  }
  ```

  бһҠбҹ„бһҷбһ–бҹҒбһӣбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ SSE бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ…бһ»бһ…бһҡбһјбһ”бһҸбҹҶбһҺбһ¶бһ„бһ”бһүбҹ’бһ…бҹҒбһүбһҖбһҳбҹ’бһҡбһ·бһҸбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҜбһҖбһҹбһ¶бһҡ JSON бһ“бҹ„бҹҮ бһңбһ¶бһ–бһ·бһҸбһҮбһ¶бһҳбһҫбһӣбһғбһҫбһүбһ§бһ”бһҖбһҡбһҺбҹҚбһ“бҹ…бһӣбһҫбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҷбһҖбһ‘бҹ…бһ”бҹ’бһҡбһҫбһҖбҹ’бһ“бһ»бһ„ GitHub Copilot бһҹбһјбһҳбһҳбһҫбһӣбһҡбһјбһ”бһҸбҹҶбһҺбһ¶бһ„бһ§бһ”бһҖбһҡбһҺбҹҚбҹ”

1. бһҠбһҫбһҳбҹ’бһ”бһёбһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһҮбһјбһ“ бһ”бҹ’бһҡбһҫбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

  ```sh
  az containerapp up -g <RESOURCE_GROUP_NAME> -n weather-mcp --environment mcp -l westus --env-vars API_KEYS=<AN_API_KEY> --source .
  ```

бһ“бҹҒбҹҮбһҮбһ¶бһ•бҹ’бһ“бҹӮбһҖбһҖбҹ’бһҡбҹ„бһҳ бһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһҮбһјбһ“бһңбһ¶бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҹбҹ’бһҡбһ»бһҖ, бһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһҮбһјбһ“бһ‘бҹ… Azure бһҸбһ¶бһҳбһҮбҹҶбһ бҹҠбһ¶бһ“бһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбҹ”
