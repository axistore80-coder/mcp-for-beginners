бһӮбһҳбҹ’бһҡбҹ„бһ„бһ“бҹҒбҹҮбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһҖбһ¶бһҡбһҹбҹ’бһ‘бҹ’бһҡбһёбһҳ HTTP бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ Server-Sent Events (SSE) бһҮбһ¶бһҳбһҪбһҷ Spring Boot WebFluxбҹ” бһңбһ¶бһҳбһ¶бһ“бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ–бһёбһҡбһҷбҹүбһ¶бһ„бҹ–

- **Calculator Server**: бһҹбҹҒбһңбһ¶бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһңбҹӮбһ”бһ”бҹ’бһҡбһҸбһ·бһҖбһҳбҹ’бһҳбһҠбҹӮбһӣбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһӮбһҺбһ“бһ¶ бһ“бһ·бһ„бһҹбҹ’бһ‘бҹ’бһҡбһёбһҳбһӣбһ‘бҹ’бһ’бһ•бһӣбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ SSE
- **Calculator Client**: бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһўбһҸбһ·бһҗбһ·бһҮбһ“бһҠбҹӮбһӣбһ”бҹ’бһҡбһҫбһ…бҹҶбһҺбһ»бһ…бһ”бһүбҹ’бһ…бҹҒбһүбһҹбҹ’бһ‘бҹ’бһҡбһёбһҳ

## бһңбһ¶вҖӢбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡвҖӢбһҠбһјбһ…бһҳбҹ’бһҸбҹҒбһ…

1. **Calculator Server** бһ”бһ„бҹ’бһ бһ¶бһүбһ…бҹҶбһҺбһ»бһ…бһ”бһүбҹ’бһ…бҹҒбһү `/calculate` бһҠбҹӮбһӣбҹ–
   - бһ‘бһ‘бһҪбһӣбһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһҹбҹҶбһҺбһҪбһҡбҹҲ `a` (бһӣбҹҒбһҒ), `b` (бһӣбҹҒбһҒ), `op` (бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡ)
   - бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҠбҹӮбһӣбһӮбһ¶бҹҶбһ‘бҹ’бһҡбҹ– `add`, `sub`, `mul`, `div`
   - бһҸбҹ’бһҡбһЎбһ”бҹӢ Server-Sent Events бһҮбһ¶бһҳбһҪбһҷбһ“бһ№бһ„бһҖбһ¶бһҡбһҡбһёбһҖбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ бһ“бһ·бһ„бһӣбһ‘бҹ’бһ’бһ•бһӣбһ“бҹғбһҖбһ¶бһҡбһӮбһҺбһ“бһ¶

2. **Calculator Client** бһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ…бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ бһ бһҫбһҷбҹ–
   - бһҠбһ¶бһҖбҹӢбһҹбҹҶбһҺбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӮбһҺбһ“бһ¶бһҡ `7 * 5`
   - бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ…бһҳбҹ’бһӣбһҫбһҷбһҹбҹ’бһ‘бҹ’бһҡбһёбһҳбһҳбһҖбһңбһ·бһү
   - бһ”бһ„бҹ’бһ бһ¶бһүбһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚбһ“бһёбһҳбһҪбһҷбҹ—бһ‘бҹ…бһ‘бҹҶбһ–бҹҗбһҡбһҖбһ»бһ„бһҹбһјбһӣ

## бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒбһҠбҹ„бһҷбһҠбҹғ

## бһӣбһ‘бҹ’бһ’бһ•бһӣбһҠбҹӮбһӣбһҡбҹҶбһ–бһ№бһ„бһ‘бһ»бһҖ

бһ–бҹҒбһӣбһҡбһҸбҹӢбһўбһҸбһ·бһҗбһ·бһҮбһ“ бһўбҹ’бһ“бһҖбһӮбһҪбһҡбһҸбҹӮбһғбһҫбһүбһӣбһ‘бҹ’бһ’бһ•бһӣбһҹбҹ’бһ‘бҹ’бһҡбһёбһҳбһҠбһјбһ…бһҮбһ¶бҹ–

```
event:info
data:Calculating: 7.0 mul 5.0

event:result
data:35.0
```

## бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҠбҹӮбһӣбһӮбһ¶бҹҶбһ‘бҹ’бһҡ

- `add` - бһҖбһ¶бһҡбһ”бһјбһҖ (a + b)
- `sub` - бһҖбһ¶бһҡбһҠбһҖ (a - b)
- `mul` - бһҖбһ¶бһҡбһӮбһ»бһҺ (a * b)
- `div` - бһҖбһ¶бһҡбһ…бҹӮбһҖ (a / b, бһҸбҹ’бһҡбһЎбһ”бҹӢ NaN бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫ b = 0)

## бһҜбһҖбһҹбһ¶бһҡбһҖбһјбһҠбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ API

### GET /calculate

**бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбҹҲ**
- `a` (бһҸбһҳбҹ’бһҡбһјбһң): бһӣбҹҒбһҒбһ‘бһёбһҳбһҪбһҷ (double)
- `b` (бһҸбһҳбҹ’бһҡбһјбһң): бһӣбҹҒбһҒбһ‘бһёбһ–бһёбһҡ (double)
- `op` (бһҸбһҳбҹ’бһҡбһјбһң): бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡ (`add`, `sub`, `mul`, `div`)

**бһ…бһҳбҹ’бһӣбһҫбһҷбҹ–**
- бһ”бҹ’бһҡбһ—бҹҒбһ‘бһҳбһ¶бһҸбһ·бһҖбһ¶: `text/event-stream`
- бһҸбҹ’бһҡбһЎбһ”бҹӢ Server-Sent Events бһҮбһ¶бһҳбһҪбһҷбһ“бһ№бһ„бһҖбһ¶бһҡбһҡбһёбһҖбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ бһ“бһ·бһ„бһӣбһ‘бҹ’бһ’бһ•бһӣ

**бһҹбҹҶбһҺбһҫбһҡбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбҹ–**
```
GET /calculate?a=7&b=5&op=mul HTTP/1.1
Host: localhost:8080
Accept: text/event-stream
```

**бһ…бһҳбҹ’бһӣбһҫбһҷбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбҹ–**
```
event: info
data: Calculating: 7.0 mul 5.0

event: result
data: 35.0
```

### бһ”бһүбҹ’бһ бһ¶бһ–бҹҒбһүбһ“бһ·бһҷбһҳ

1. **бһ…бҹ’бһҡбһҖ 8080 бһҳбһ¶бһ“бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҡбһҪбһ…бһ бһҫбһҷ**
   - бһ”бһүбҹ’бһҲбһ”бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбһҠбҹӮбһӣбһҖбҹҶбһ–бһ»бһ„бһ”бҹ’бһҡбһҫбһ…бҹ’бһҡбһҖ 8080
   - бһ¬бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҠбһјбһҡбһ…бҹ’бһҡбһҖбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒбһҖбҹ’бһ“бһ»бһ„ `calculator-server/src/main/resources/application.yml`

   - бһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒбһҖбҹҶбһ–бһ»бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҳбһ»бһ“бһ–бҹҒбһӣбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҠбһҫбһҳбһўбһҸбһ·бһҗбһ·бһҮбһ“
   - бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҗбһ¶бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒбһ”бһҫбһҖбһҠбҹ„бһҷбһҮбҹ„бһӮбһҮбҹҗбһҷбһӣбһҫбһ…бҹ’бһҡбһҖ 8080

3. **бһ”бһүбҹ’бһ бһ¶бһҲбҹ’бһҳбҹ„бҹҮбһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡ**
