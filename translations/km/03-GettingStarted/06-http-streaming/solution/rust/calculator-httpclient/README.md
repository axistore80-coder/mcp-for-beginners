# ការរត់ឧទាហរណ៍នេះ

## -1- បញ្ជ запуска HTTP server Streamable

បើក terminal ពីមូលដ្ឋានគម្រោងនេះ ហើយរត់ពាក្យបញ្ជាខាងក្រោម៖

```bash
cd ../calculator-httpserver
cargo run
```

រក្សាទុក terminal នេះទុកព្រោះវានឹងរត់ HTTP server ដែល client នឹងភ្ជាប់ទៅ។ បើក terminal ថ្មីសម្រាប់ជំហ៊ានបន្ទាប់។

## -2- តំឡើង dependencies និង សាងសង់គម្រោង

```bash
cargo build
```

## -3- រត់ឧទាហរណ៍

```bash
cargo run
```
