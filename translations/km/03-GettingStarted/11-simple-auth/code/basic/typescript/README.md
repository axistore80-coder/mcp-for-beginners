# បើករត់គំរូ

## ដំឡើងការពឹងផ្អែក

```bash
npm install
```

## សង់បង្កើត

```bash
npm run build
```

## ចាប់ផ្តើមម៉ាស៊ីនបម្រើ

```bash
npm start
```

នេះគួរតែចាប់ផ្តើមម៉ាស៊ីនបម្រើ

### ចាប់ផ្តើមអតិថិជន

នៅក្នុងផ្ទាំងបញ្ជាផ្សេងទៀត ចាក់បញ្ជា៖

```bash
node run client
```
