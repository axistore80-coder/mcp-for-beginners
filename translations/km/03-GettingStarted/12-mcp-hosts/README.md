# бһҖбһ¶бһҡбһҠбҹҶбһЎбһҫбһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP бһҠбҹӮбһӣбһ–бҹҒбһүбһ“бһ·бһҷбһҳ

бһҳбһӮбҹ’бһӮбһ»бһ‘бҹҒбһҹбһҖбҹҚбһ“бҹҒбҹҮбһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҸбһ”бҹӢбһ–бһёбһҡбһ”бҹҖбһ”бһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ бһ“бһ·бһ„бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP бһҮбһ¶бһҳбһҪбһҷбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ AI бһҠбҹӮбһӣбһ–бҹҒбһүбһ“бһ·бһҷбһҳбҹ” бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒбһҳбһҪбһҷбҹ—бһҳбһ¶бһ“бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһ–бҹҒбһӣбһ”бһ¶бһ“бһҠбҹҶбһЎбһҫбһ„бһҡбһҪбһ… бһ–бһҪбһҖбһңбһ¶бһ–бһ·бһ—бһ¶бһҖбҹ’бһҹбһ¶бһӮбҹ’бһ“бһ¶бһҮбһ¶бһҳбһҪбһҷбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһ–бһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбҹ”

## бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP бһҮбһ¶бһўбҹ’бһңбһё?

**бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP** бһӮбһәбһҮбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё AI бһҳбһҪбһҷбһҠбҹӮбһӣбһўбһ¶бһ…бһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ…бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP бһҠбһҫбһҳбҹ’бһ”бһёбһ–бһ„бҹ’бһҡбһёбһҖбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҡбһ”бһҹбҹӢбһңбһ¶бҹ” бһӮбһ·бһҸбһҗбһ¶бһңбһ¶бһҮбһ¶ "бһ•бҹ’бһ“бҹӮбһҖбһҳбһ»бһҒ" бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһўбһ¶бһ…бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһ…бһјбһӣбһҡбһҪбһҳ бһ”бһ¶бһ“бһҒбһҺбҹҲбһҠбҹӮбһӣбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP бһ•бҹ’бһҸбһӣбҹӢбһ§бһ”бһҖбһҡбһҺбҹҚ бһ“бһ·бһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ "бһ•бҹ’бһ“бҹӮбһҖбһҖбҹ’бһҡбҹ„бһҷ"бҹ”

## 1. Claude Desktop

**Claude Desktop** бһӮбһәбһҮбһ¶вҖӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё official бһҠбҹҒмҠӨнҒ¬бһҸбһ»бһ”бһҡбһ”бһҹбҹӢ Anthropic бһҠбҹӮбһӣбһӮбһ¶бҹҶбһ‘бҹ’бһҡ MCP бһҠбҹ„бһҷвҖӢбһҠбһҫбһҡбһҸбһҪбһҮбһ¶вҖӢбһҸбһ¶бһҳбһҠбһҫбһҳбҹ”

### бһҖбһ¶бһҡбһҸбҹҶбһЎбһҫбһ„

1. бһ‘бһ¶бһүбһҷбһҖ Claude Desktop бһ–бһё [claude.ai/download](https://claude.ai/download)
2. бһҠбҹҶбһЎбһҫбһ„ бһ бһҫбһҷбһ…бһјбһӣбһ”бҹ’бһҡбһҫбһҮбһ¶бһҳбһҪбһҷбһӮбһҺбһ“бһё Anthropic бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ

### бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’

Claude Desktop бһ”бҹ’бһҡбһҫбһҜбһҖбһҹбһ¶бһҡ JSON бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCPбҹ”

**бһ‘бһёбһҸбһ¶бҹҶбһ„бһҜбһҖбһҹбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бҹ–**
- **macOS**: `~/Library/Application Support/Claude/claude_desktop_config.json`
- **Windows**: `%APPDATA%\Claude\claude_desktop_config.json`
- **Linux**: `~/.config/Claude/claude_desktop_config.json`

**бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бҹ–**

```json
{
  "mcpServers": {
    "calculator": {
      "command": "python",
      "args": ["-m", "mcp_calculator_server"],
      "env": {
        "PYTHONPATH": "/path/to/your/server"
      }
    },
    "weather": {
      "command": "node",
      "args": ["/path/to/weather-server/build/index.js"]
    },
    "database": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-postgres"],
      "env": {
        "DATABASE_URL": "postgresql://user:pass@localhost/mydb"
      }
    }
  }
}
```

### бһҮбһҳбҹ’бһҡбһҫбһҹбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’

| бһңбһ¶бһӣ | бһ–бһ·бһ–бһҺбҹҢбһ“бһ¶ | бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ |
|-------|-------------|---------|
| `command` | бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҡбһҸбҹӢ | `"python"`, `"node"`, `"npx"` |
| `args` | бһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶бһҶбҹ’бһӣбһ„бһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһ”бһүбҹ’бһҮбһ¶ | `["-m", "my_server"]` |
| `env` | бһ”бһҡбһ·бһҹбҹ’бһҗбһ¶бһ“бһўбһҗбҹҒбһҡ | `{"API_KEY": "xxx"}` |
| `cwd` | бһҡбһ¶бһҷбһҖбһ¶бһҡбһҺбҹҚбһҗбһҸбһҖбһ¶бһҡбһ„бһ¶бһҡ | `"/path/to/server"` |

### бһҖбһ¶бһҡбһ”бҹ’бһҡбһЎбһ„бһҸбҹҶбһЎбһҫбһ„бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ

1. бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҜбһҖбһҹбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’
2. бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһЎбһҫбһ„бһңбһ·бһүбһ‘бһ¶бҹҶбһ„бһҹбҹ’бһҡбһ»бһ„ Claude Desktop (бһ”бһ·бһ‘бһ бһҫбһҷбһ”бһҫбһҖбһЎбһҫбһ„бһңбһ·бһү)
3. бһ”бһҫбһҖбһҖбһ·бһ…бҹ’бһ…бһҹбһ“бҹ’бһ‘бһ“бһ¶бһҗбҹ’бһҳбһёбһҳбһҪбһҷ
4. бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҡбһјбһ”бһҸбҹҶбһҺбһ¶бһ„ рҹ”Ң бһ”бһ„бҹ’бһ бһ¶бһүбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒбһ”бһ¶бһ“бһ—бҹ’бһҮбһ¶бһ”бҹӢ
5. бһ–бҹ’бһҷбһ¶бһҷбһ¶бһҳбһҹбһҪбһҡ Claude бһҠбһҫбһҳбҹ’бһ”бһёбһ”бҹ’бһҡбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһҳбһҪбһҷбһҺбһ¶бһҳбһҪбһҷбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ

### бһҖбһ¶бһҡбһҮбһҪбһҷбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶ Claude Desktop

**бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒбһҳбһ·бһ“бһ”бһ„бҹ’бһ бһ¶бһүбҹ–**
- бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһңбһ·бһ“бһ·бһҷбҹ„бһӮбһҜбһҖбһҹбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҮбһ¶бһҳбһҪбһҷбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢ JSON
- бһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶бһ”бҹ’бһҡбһ—бҹҒбһ‘ command бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң
- бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбҹҶбһҺбһҸбҹӢбһ бҹҒбһҸбһ» Claude Desktop: бһҮбҹҶбһ“бһҪбһҷ вҶ’ бһ”бһ„бҹ’бһ бһ¶бһүбһҖбҹҶбһҺбһҸбҹӢбһ бҹҒбһҸбһ»

**бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒбһ’бҹ’бһңбһҫбһІбҹ’бһҷбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҒбһјбһ…бһҒбһ¶бһҸбһ–бҹҒбһӣбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбҹ–**
- бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒбһҠбҹ„бһҷбһҠбҹғбһ“бҹ…бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһ‘бһёбһҸбһ¶бҹҶбһ„ Terminal бһҮбһ¶бһӣбҹҶбһ”бһ¶бҹҶ
- бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҗбһ¶бһ”бһҡбһ·бһҹбҹ’бһҗбһ¶бһ“бһўбһҗбҹҒбһҡбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң
- бһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶бһҳбһ»бһҒбһ„бһ¶бһҡбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡ бһ”бһ¶бһ“бһҠбҹҶбһЎбһҫбһ„бһҡбһҪбһ…

## 2. VS Code бһҮбһ¶бһҳбһҪбһҷ GitHub Copilot

VS Code бһӮбһ¶бҹҶбһ‘бҹ’бһҡ MCP бһҸбһ¶бһҳбһҡбһҷбҹҲбһ”бһ“бҹ’бһҗбҹӮбһҳ GitHub Copilot Chatбҹ”

### бһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢбһҳбһ»бһ“

1. бһҳбһ¶бһ“ VS Code 1.99+ бһ”бһ¶бһ“бһҠбҹҶбһЎбһҫбһ„
2. бһ”бһ¶бһ“бһҠбҹҶбһЎбһҫбһ„бһ”бһ“бҹ’бһҗбҹӮбһҳ GitHub Copilot
3. бһ”бһ¶бһ“бһҠбҹҶбһЎбһҫбһ„бһ”бһ“бҹ’бһҗбҹӮбһҳ GitHub Copilot Chat

### бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’

VS Code бһ”бҹ’бһҡбһҫ `.vscode/mcp.json` бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҹбҹ’бһҗбһ¶бһ“бһёбһҷбҹҚбһҖбһ¶бһҡбһ„бһ¶бһҡбһ¬бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбҹ”

**бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҹбҹ’бһҗбһ¶бһ“бһёбһҷбҹҚбһҖбһ¶бһҡбһ„бһ¶бһҡ** (`.vscode/mcp.json`):

```json
{
  "servers": {
    "my-calculator": {
      "type": "stdio",
      "command": "python",
      "args": ["-m", "mcp_calculator_server"]
    },
    "my-database": {
      "type": "sse",
      "url": "http://localhost:8080/sse"
    }
  }
}
```

**бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ** (`settings.json`):

```json
{
  "mcp.servers": {
    "global-server": {
      "type": "stdio",
      "command": "npx",
      "args": ["-y", "@anthropic/mcp-server-memory"]
    }
  },
  "mcp.enableLogging": true
}
```

### бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫ MCP бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ VS Code

1. бһ”бһҫбһҖбһ•бҹ’бһ‘бһ¶бҹҶбһ„ Copilot Chat (Ctrl+Shift+I / Cmd+Shift+I)
2. бһңбһ¶бһҷ `@` бһҠбһҫбһҳбҹ’бһ”бһёбһҳбһҫбһӣбһ§бһ”бһҖбһҡбһҺбҹҚ MCP бһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбҹ’бһҡбһ¶бһ”бҹӢ
3. бһ”бҹ’бһҡбһҫбһ—бһ¶бһҹбһ¶бһ’бһҳбҹ’бһҳбһҮбһ¶бһҸбһ·бһҠбһҫбһҳбҹ’бһ”бһёбһ бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚбҹ– "Calculate 25 * 48 using the calculator"

### бһҖбһ¶бһҡбһҮбһҪбһҷбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶ VS Code

**бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP бһҳбһ·бһ“бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбҹ–**
- бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ•бҹ’бһ‘бһ¶бҹҶбһ„ Output вҶ’ "MCP" бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбҹҶбһҺбһҸбҹӢбһ бҹҒбһҸбһ»бһҖбҹҶбһ бһ»бһҹ
- бһ•бҹ’бһ‘бһ»бһҖбһЎбһҫбһ„бһңбһ·бһүбһ•бҹ’бһ‘бһ¶бҹҶбһ„бһҖбһ¶бһҡбһ„бһ¶бһҡ: Ctrl+Shift+P вҶ’ "Developer: Reload Window"
- бһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҠбҹ„бһҷбһЎбҹӮбһҖбһҳбҹ’бһҠбһ„бһҳбһҪбһҷбһ”бһ¶бһ“бһҹбһҳбҹ’бһҡбҹҒбһ…

## 3. Cursor

**Cursor** бһӮбһәбһҮбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҖбһјбһҠбһҠбҹӮбһӣбһ•бҹ’бһҸбҹ„бһҸбһӣбһҫ AI бһҠбҹ„бһҷбһҳбһ¶бһ“бһӮбһ¶бҹҶбһ‘бҹ’бһҡ MCP бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбҹ”

### бһҖбһ¶бһҡбһҸбҹҶбһЎбһҫбһ„

1. бһ‘бһ¶бһүбһҷбһҖ Cursor бһ–бһё [cursor.sh](https://cursor.sh)
2. бһҠбҹҶбһЎбһҫбһ„ бһ“бһ·бһ„бһ…бһјбһӣбһ”бҹ’бһҡбһҫ

### бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’

Cursor бһ”бҹ’бһҡбһҫбһҡбҹҖбһ”бһ…бҹҶбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҠбһјбһ…бһҮбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Claude Desktopбҹ”

**бһ‘бһёбһҸбһ¶бҹҶбһ„бһҜбһҖбһҹбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бҹ–**
- **macOS**: `~/.cursor/mcp.json`
- **Windows**: `%USERPROFILE%\.cursor\mcp.json`
- **Linux**: `~/.cursor/mcp.json`

**бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бҹ–**

```json

    },
    "github": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-github"],
      "env": {
        "GITHUB_TOKEN": "ghp_your_token_here"
      }
    }
  }
}
```

### бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫ MCP бһ“бҹ… Cursor

1. бһ”бһҫбһҖбһҖбһ¶бһҡбһҹбһ“бҹ’бһ‘бһ“бһ¶ AI бһҡбһ”бһҹбҹӢ Cursor (Ctrl+L / Cmd+L)
2. бһҳбһҫбһӣбһ§бһ”бһҖбһҡбһҺбҹҚ MCP бһ”бһ„бҹ’бһ бһ¶бһүбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ•бҹ’бһҸбһӣбҹӢбһўбһ“бһ»бһҹбһ¶бһҹбһ“бҹҚ
3. бһҹбһ»бҹҶбһІбҹ’бһҷ AI бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒбһҠбҹӮбһӣбһ”бһ¶бһ“бһ—бҹ’бһҮбһ¶бһ”бҹӢ

## 4. Cline (бһ•бҹ’бһўбҹӮбһҖбһӣбһҫ Terminal)

**Cline** бһӮбһәбһҮбһ¶вҖӢ MCP client бһ”бҹ’бһҡбһҫбһҸбһ¶бһҳ Terminal бһӮбһҪбһҡбһұбҹ’бһҷбһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһ¶вҖӢбһҳбһ¶бһҸбһ·бһҖбһ¶бҹ”

### бһҖбһ¶бһҡбһҸбҹҶбһЎбһҫбһ„

### бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’

Cline бһ”бҹ’бһҡбһҫбһўбһҗбҹҒбһҡбһ”бһҡбһ·бһҹбҹ’бһҗбһ¶бһ“ бһ“бһ·бһ„бһўбһ¶бһҖбҹ’бһҹбһҡбһ”бһүбҹ’бһҮбһ¶вҖӢбһ“бһ¶бһ“бһ¶бҹ”

**бһ”бҹ’бһҡбһҫбһўбһҗбҹҒбһҡбһ”бһҡбһ·бһҹбҹ’бһҗбһ¶бһ“бҹ–**

      "command": "python",

    }
  }
}
```

## бһҖбһ¶бһҡбһҮбһҪбһҷбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶бһҮбһ¶бһ‘бһјбһ‘бҹ…

### бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒбһҳбһ·бһ“бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ

   # бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Node.js
   node /path/to/server/index.js
   ```

2. **бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ‘бһёбһҸбһ¶бҹҶбһ„бһ”бһүбҹ’бһҮбһ¶бҹ–**
   - бһ”бҹ’бһҡбһҫбһ•бҹ’бһӣбһјбһңбһҸбҹҶбһҺбһҠбҹӮбһӣбһ–бһ·бһҸбһ”бҹҶбһ•бһ»бһҸбһ–бҹҒбһӣбһўбһ¶бһ…бһ’бҹ’бһңбһҫбһ‘бҹ…бһ”бһ¶бһ“
   - бһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһўбһ¶бһ…бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһ¶бһ“бһҖбҹ’бһ“бһ»бһ„ PATH бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ

3. **бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҳбһ»бһҒбһ„бһ¶бһҡбһҠбҹҸбһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбҹ–**

### бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ”бһ¶бһ“бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһ§бһ”бһҖбһҡбһҺбҹҚбһҳбһ·бһ“бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ

1. **бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбҹҶбһҺбһҸбҹӢбһ бҹҒбһҸбһ»бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ** - бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒбһ—бһ¶бһӮбһ…бҹ’бһҡбһҫбһ“бһҳбһ¶бһ“бһҮбһҳбҹ’бһҡбһҫбһҹбһҖбҹҶбһҺбһҸбҹӢбһ бҹҒбһҸбһ»
2. **бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҖбһ¶бһҡбһ…бһ»бҹҮбһ”бһүбҹ’бһҮбһёбһ§бһ”бһҖбһҡбһҺбҹҚ** - бһ”бҹ’бһҡбһҫ MCP Inspector бһҠбһҫбһҳбҹ’бһ”бһёбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„
3. **бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбһ¶бһҡбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸ** - бһ§бһ”бһҖбһҡбһҺбҹҚбһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ…бһјбһӣбһ”бҹ’бһҡбһҫбһҜбһҖбһҹбһ¶бһҡ/бһ”бһҺбҹ’бһҸбһ¶бһү

### бһўбһҗбҹҒбһҡбһ”бһҡбһ·бһҹбҹ’бһҗбһ¶бһ“бһҳбһ·бһ“бһ”бһ¶бһ“бһ•бҹ’бһүбһҫ

- бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒбһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бһҹбһҳбҹ’бһўбһ¶бһҸбһўбһҗбҹҒбһҡбһ”бһҡбһ·бһҹбҹ’бһҗбһ¶бһ“
- бһ”бҹ’бһҡбһҫбһңбһ¶бһӣ `env` бһҖбҹ’бһ“бһ»бһ„бһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҷбҹүбһ¶бһ„бһ…бҹ’бһ”бһ¶бһҹбҹӢ
- бһҮбҹҖбһҹбһңбһ¶бһ„бһҠбһ¶бһҖбҹӢбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбҹҶбһ„бһ¶бһҸбҹӢбһҖбҹ’бһ“бһ»бһ„бһҜбһҖбһҹбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ (бһ”бҹ’бһҡбһҫбһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢ)

3. **бһҖбҹҶбһҺбһҸбҹӢбһҹбһ·бһ‘бҹ’бһ’бһ·бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ** бһҠбһӣбҹӢбһҸбҹӮбһўбҹ’бһңбһёбһҠбҹӮбһӣбһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһҸбҹӮбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮ
4. **бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбһјбһҠбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ** бһҳбһ»бһ“бһ•бҹ’бһҠбһӣбҹӢбһҖбһ¶бһҡбһ…бһјбһӣбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ
