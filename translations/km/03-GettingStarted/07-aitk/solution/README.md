## бһҹбһ„бҹ’бһҒбҹҒбһ”
бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ„бһ¶бһҡбһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһ–бһ„бҹ’бһҡбһёбһҖбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһӮбһҺбһ“бһ¶бһҡбһ”бһҹбҹӢ MCP server бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҠбҹ„бһҷбһ”бһ“бҹ’бһҗбҹӮбһҳбһ§бһ”бһҖбһҡбһҺбҹҚбһҗбҹ’бһҳбһёбһҳбһҪбһҷбһҠбҹӮбһӣбһӮбһҺбһ“бһ¶бһҹбҹ„бһӮбһјбһ•бҹ’бһӣбһ¶бһүбһҡбһ”бһҹбҹӢбһӣбҹҒбһҒбһҳбһҪбһҷбҹ” бһҖбһ¶бһҡбһ”бһ“бҹ’бһҗбҹӮбһҳбһ“бҹҒбҹҮбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ”бһүбҹ’бһүбһ¶бһҹбһ·бһ”бҹ’бһ”бһ“бһ·бһҳбҹ’бһҳбһ·бһҸбҹ’бһҸбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҹбҹҶбһҺбһҪбһҡбһўбһ¶бһӮбһ»бһҷбһҳбҹүбһ¶бһ‘бһ·бһ…бһҡбҹҶбһҒбһ¶бһ“, бһҠбһјбһ…бһҮбһ¶ "бһҹбҹ„бһӮбһјбһ•бҹ’бһӣбһ¶бһүбһҡбһ”бһҹбҹӢ 16 бһҮбһ¶вҖӢбһўбҹ’бһңбһё?" бһ¬ "бһӮбһҺбһ“бһ¶ вҲҡ49" бһҠбҹ„бһҷвҖӢбһ”бҹ’бһҡбһҫвҖӢбһ”бҹ’бһҡбһ¶бһҹбҹӢвҖӢбһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡвҖӢбһ—бһ¶бһҹбһ¶бһ’бһҳбҹ’бһҳбһҮбһ¶бһҸбһ·бҹ”

## рҹӣ пёҸ бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ§бһ”бһҖбһҡбһҺбҹҚбһҹбҹ„бһӮбһјбһ•бҹ’бһӣбһ¶бһү
бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ“бҹ’бһҗбҹӮбһҳбһҳбһ»бһҒбһ„бһ¶бһҡбһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҠбһ¶бһҖбҹӢбһ–бһ¶бһҖбҹ’бһҷбһҖбҹҶбһҺбһҸбҹӢбһҳбһ»бһҒбһ„бһ¶бһҡвҖӢбһ§бһ”бһҖбһҡбһҺбҹҚбһҗбҹ’бһҳбһёбһҳбһҪбһҷбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҜбһҖбһҹбһ¶бһҡ server.py бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһ“бҹҒбҹҮбһҮбһ¶бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбҹ–

```python
"""
Sample MCP Calculator Server implementation in Python.

This module demonstrates how to create a simple MCP server with calculator tools
that can perform basic arithmetic operations (add, subtract, multiply, divide).
"""

from mcp.server.fastmcp import FastMCP
import math

server = FastMCP("calculator")

@server.tool()
def add(a: float, b: float) -> float:
    """Add two numbers together and return the result."""
    return a + b

@server.tool()
def subtract(a: float, b: float) -> float:
    """Subtract b from a and return the result."""
    return a - b

@server.tool()
def multiply(a: float, b: float) -> float:
    """Multiply two numbers together and return the result."""
    return a * b

@server.tool()
def divide(a: float, b: float) -> float:
    """
    Divide a by b and return the result.
    
    Raises:
        ValueError: If b is zero
    """
    if b == 0:
        raise ValueError("Cannot divide by zero")
    return a / b

@server.tool()
def sqrt(a: float) -> float:
    """
    Return the square root of a.

    Raises:
        ValueError: If a is negative.
    """
    if a < 0:
        raise ValueError("Cannot compute the square root of a negative number.")
    return math.sqrt(a)
```

## рҹ”Қ бһңбһ¶бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҷбҹүбһ¶бһ„бһҠбһјбһ…бһҳбҹ’бһҸбҹҒбһ…

- **бһ“бһ¶бҹҶбһ…бһјбһӣбһҳбҹүбһјбһҢбһ»бһӣ `math`**бҹ– бһҠбһҫбһҳбҹ’бһ”бһёбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһўбһ¶бһӮбһ»бһҷбһҳбҹүбһ¶бһ‘бһ·бһ…бһҠбҹӮбһӣбһӣбһҫбһҹбһ–бһёбһӮбһҺбһ“бһ¶бһӮбһҺбһ·бһҸбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ Python бһ•бҹ’бһҸбһӣбҹӢбһҳбҹүбһјбһҢбһ»бһӣ `math` бһҡбһҪбһҳбһҳбһ¶бһ“бһҳбһ»бһҒбһ„бһ¶бһҡбһўбһ¶бһӮбһ»бһҷбһҳбҹүбһ¶бһ‘бһ·бһ… бһ“бһ·бһ„бһўбһҗбҹҒбһҡбһ…бһҳбҹ’бһҡбһ„бҹӢбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бҹ” бһҖбһ¶бһҡбһ“бһ¶бҹҶбһ…бһјбһӣбһңбһ¶вҖӢбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ `import math` бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһҳбһ»бһҒбһ„бһ¶бһҡбһҠбһјбһ…бһҮбһ¶ `math.sqrt()` бһҠбҹӮбһӣбһӮбһҺбһ“бһ¶бһҹбҹ„бһӮбһјбһ•бҹ’бһӣбһ¶бһүбһҡбһ”бһҹбҹӢбһӣбҹҒбһҒбһҳбһҪбһҷбҹ”
- **бһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһҳбһ»бһҒбһ„бһ¶бһҡ**бҹ– бһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһҸбһҪбһўбһҖбҹ’бһҹбһҡ `@server.tool()` бһҠбһҫбһҳбҹ’бһ”бһёбһ…бһ»бҹҮбһҲбҹ’бһҳбҹ„бҹҮбһҳбһ»бһҒбһ„бһ¶бһҡ `sqrt` бһҮбһ¶бһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ AI бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бһ¶бһ“бҹ”
- **бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡвҖӢбһҷбһҖбһ…бһјбһӣ**бҹ– бһҳбһ»бһҒбһ„бһ¶бһҡбһ“бҹҒбҹҮбһ‘бһ‘бһҪбһӣбһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡвҖӢбһҸбҹӮбһҳбһҪбһҷбһҲбҹ’бһҳбҹ„бҹҮ `a` бһҠбҹӮбһӣвҖӢбһҳбһ¶бһ“бһ”бҹ’бһҡбһ—бҹҒбһ‘бһҮбһ¶ `float`бҹ”
- **бһҖбһ¶бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҖбҹҶбһ бһ»бһҹ**бҹ– бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫ `a` бһҸбһјбһ…бһҮбһ¶бһ„бһҹбһјбһ“бҹ’бһҷ бһҳбһ»бһҒбһ„бһ¶бһҡбһ“бҹҒбҹҮвҖӢбһ“бһ№бһ„бһҖбһҫбһҸбһ”бһүбҹ’бһ бһ¶ `ValueError` бһҠбһҫбһҳбҹ’бһ”бһёбһҮбҹҶбһҡбһ»бһүбһҳбһ·бһ“бһІбҹ’бһҷбһӮбһҺбһ“бһ¶бһҹбҹ„бһӮбһјбһ•бҹ’бһӣбһ¶бһүбһ–бһёбһӣбҹҒбһҒвҖӢбһўбһңбһ·бһҮбҹ’бһҮбһҳбһ¶бһ“ бһҠбҹӮбһӣбһҳбһ»бһҒбһ„бһ¶бһҡ `math.sqrt()` бһҳбһ·бһ“бһӮбһ¶бҹҶбһ‘бҹ’бһҡбҹ”
- **бһҸбһҳбҹ’бһӣбҹғбһҸбҹ’бһҡбһЎбһ”бҹӢ**бҹ– бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһүбҹ’бһ…бһјбһӣбһҳбһ·бһ“бһўбһңбһ·бһҮбҹ’бһҮбһҳбһ¶бһ“ бһҳбһ»бһҒбһ„бһ¶бһҡбһ“бҹҒбҹҮбһ“бһ№бһ„бһҸбҹ’бһҡбһЎбһ”бҹӢбһҸбһҳбҹ’бһӣбҹғбһҹбҹ„бһӮбһјбһ•бҹ’бһӣбһ¶бһүбһҡбһ”бһҹбҹӢ `a` бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһңбһ·бһ’бһё `math.sqrt()` бһҡбһ”бһҹбҹӢ Python бҹ”

## рҹ”„ бһҖбһ¶бһҡбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҠбһҫбһҳбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҗбҹ’бһҳбһё
бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһ”бһ“бҹ’бһҗбҹӮбһҳбһ§бһ”бһҖбһҡбһҺбҹҚ `sqrt` бһҗбҹ’бһҳбһёбһ“бҹҒбҹҮ бһңбһ¶вҖӢбһҳбһ¶бһ“вҖӢбһҹбһ¶бһҡбҹҲбһҹбҹҶбһҒбһ¶бһ“бҹӢбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҠбһҫбһҳбһҳбҹүбһ¶бһҹбҹҠбһёбһ“ MCP бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҳбҹ’бһҸбһ„бһ‘бҹҖбһҸ бһҠбһҫбһҳбҹ’бһ”бһёбһ’бһ¶бһ“бһ¶бһҗбһ¶бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ‘бһ‘бһҪбһӣбһҹбҹ’бһӮбһ¶бһӣбҹӢ бһ“бһ·бһ„бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһҳбһ»бһҒбһ„бһ¶бһҡбһҗбҹ’бһҳбһёбһ”бһ¶бһ“бҹ”

## рҹ’¬ бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ”бһүбҹ’бһ…бһјбһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһ§бһ”бһҖбһҡбһҺбҹҚбһҗбҹ’бһҳбһё
бһ“бҹҒбҹҮбһҮбһ¶бһ”бһүбҹ’бһ…бһјбһӣбһ—бһ¶бһҹбһ¶бһ’бһҳбҹ’бһҳбһҮбһ¶бһҸбһ·бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҳбһ»бһҒбһ„бһ¶бһҡвҖӢбһҹбҹ„бһӮбһјбһ•бҹ’бһӣбһ¶бһүбҹ–

- "бһҹбҹ„бһӮбһјбһ•бҹ’бһӣбһ¶бһүбһҡбһ”бһҹбҹӢ 25 бһҮбһ¶вҖӢбһўбҹ’бһңбһё?"
- "бһӮбһҺбһ“бһ¶бһҹбҹ„бһӮбһјбһ•бҹ’бһӣбһ¶бһүбһҡбһ”бһҹбҹӢ 81бҹ”"
- "бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҹбҹ„бһӮбһјбһ•бҹ’бһӣбһ¶бһүбһҡбһ”бһҹбҹӢ 0бҹ”"
- "бһҹбҹ„бһӮбһјбһ•бҹ’бһӣбһ¶бһүбһҡбһ”бһҹбҹӢ 2.25 бһҮбһ¶вҖӢбһўбҹ’бһңбһё?"

бһ”бһүбҹ’бһ…бһјбһӣбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһӮбһҪбһҡбһҡбҹҶбһҮбһҫбһ”бһІбҹ’бһҷбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡа№ҖаёЈаёөаёўаёҒбһ§бһ”бһҖбһҡбһҺбҹҚ `sqrt` бһ“бһ·бһ„бһҸбҹ’бһҡбһЎбһ”бҹӢбһӣбһ‘бҹ’бһ’бһ•бһӣвҖӢбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһҳбһҖбһңбһ·бһүбҹ”

- бһ бһ¶бһҸбҹӢбһ”бҹ’бһҡбһ¶бһҺбһ”бһ“бҹ’бһҗбҹӮбһҳбһ§бһ”бһҖбһҡбһҺбҹҚбһҗбҹ’бһҳбһё бһ“бһ·бһ„бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҠбһҫбһҳбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҳбһ»бһҒбһ„бһ¶бһҡбһҗбҹ’бһҳбһёбҹ”
