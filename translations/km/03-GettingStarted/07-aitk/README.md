# ការប្រើម៉ាស៊ីនមេពីកម្មវិធីបន្ថែម AI Toolkit សម្រាប់ Visual Studio Code

ពេលដែលអ្នកកំពុងបង្កើតតំណាង AI មួយ វាពុំមែនត្រឹមតែបង្កើតចម្លើយឆ្លាតវៃទេ ប៉ុន្តែជាការផ្តល់សមត្ថភាពឱ្យតំណាងរបស់អ្នកអាចអនុវត្តសកម្មភាពបានផងដែរ។ នោះជាកន្លែងដែល Model Context Protocol (MCP) ចូលមក។ MCP ធ្វើឱ្យតំណាងអាចចូលដំណើរការឧបករណ៍និងសេវាកម្មខាងក្រៅបានយ៉ាងរឹងមាំ។ គួរតែគិតថា វាដូចជាការតភ្ជាប់តំណាងរបស់អ្នកទៅឧបករណ៍ដែលវាអាចប្រើប្រាស់បានពិតប្រាកដ។

ឧទាហរណ៍ សន្មត់ថាអ្នកភ្ជាប់តំណាងទៅម៉ាស៊ីនមេ MCP គណនាតម្លៃ។ ភ្លាមៗ អ្នកតំណាងរបស់អ្នកអាចធ្វើប្រតិបត្តិការគណនា ដោយគ្រាន់តែទទួលបានសេចក្តីថ្លែងសំណួរដូចជា “តើ 47 គុណ 89 បាននៅប៉ុនណា?” — មិនចាំបាច់ដាក់​សេចក្តី​ណែនាំ MOQ ឬ​សាងសង់ API ផ្ទាល់ខ្លួនឡើយ។

## យ៉ាងទូលំទូលាយ

មេរៀននេះគ្របដណ្ដប់ពីរបៀបភ្ជាប់ម៉ាស៊ីនមេ MCP គណនាតម្លៃទៅតំណាងជាមួយកម្មវិធីបន្ថែម [AI Toolkit](https://aka.ms/AIToolkit) ក្នុង Visual Studio Code ដែលអនុញ្ញាតឲ្យតំណាងរបស់អ្នកអាចអនុវត្តប្រតិបត្តិការគណនាដូចជាការបូក, ដក, គុណ និងចែក តាមរយៈភាសាបានយល់។

AI Toolkit គឺជាកម្មវិធីបន្ថែមដែលមានពលការរឹងមាំសម្រាប់ Visual Studio Code ធ្វើឲ្យការអភិវឌ្ឍតំណាងកាន់តែងាយស្រួល។ វាជួយឲ្យវិស្វករជំនាញ AI អាចបង្កើតកម្មវិធី AI បានយ៉ាងងាយស្រួលដោយបង្កើតនិងសាកល្បងម៉ូឌែល AI បង្កើត — លើម៉ាស៊ីនភ្លើងឬនៅក្នុង cloud។ កម្មវិធីបន្ថែមនេះគាំទ្រម៉ូឌែលបង្កើតធំៗភាគច្រើនដែលមានស្រាប់ធ្វើឡើងថ្ងៃនេះ។

*ចំណាំ*: AI Toolkit បច្ចុប្បន្នគាំទ្រពី Python និង TypeScript ។

1. ពិនិត្យសេចក្តីថ្លែងប្រព័ន្ធ ហើយកែប្រែលើកទីប្រសិនបើចាំបាច់។

### -3- បង្កើតម៉ាស៊ីនមេ MCP

!["រូបថតផ្នែកក្រោមនៃផ្ទាំង Calculator Agent ក្នុង AI Toolkit សម្រាប់ Visual Studio Code បង្ហាញម៉ឺនុយសាកល្បងដែលអាចបង្រួមបានសម្រាប់ “Tools” និង “Structure output,” ជាមួយម៉ឺនុយបោះជ្រើស "Choose output format" បានកំណត់ជា "text"។ នៅខាងស្តាំ មានប៊ូតុង “+ MCP Server” សម្រាប់បន្ថែមម៉ាស៊ីនមេ Model Context Protocol។ មានរូបតំណាងរូបភាពគំរូនៅលើបរិវេណ Tools។](../../../../translated_images/km/aitk-add-mcp-server.9742cfddfe808353.webp)

1. នៅក្នុង terminal, បើកបរិយាកាស virtual ៖
    1. Windows - `.venv\Scripts\activate`
    2. macOS/Linux - `source .venv/bin/activate`
1. នៅក្នុង terminal, ដំឡើងការពឹងផ្អែក៖ `pip install -e .[dev]`

1. ជំនួសកូដក្នុងឯកសារ **server.py** ជាមួយខាងក្រោម និងរក្សាទុក៖

    ```python
    """
    Sample MCP Calculator Server implementation in Python.

    
    This module demonstrates how to create a simple MCP server with calculator tools
    that can perform basic arithmetic operations (add, subtract, multiply, divide).
    """
    
    from mcp.server.fastmcp import FastMCP
    
    server = FastMCP("calculator")
    
    @server.tool()
    def add(a: float, b: float) -> float:
        """Add two numbers together and return the result."""
        return a + b
    
    @server.tool()
    def subtract(a: float, b: float) -> float:
        """Subtract b from a and return the result."""
        return a - b
    
    @server.tool()
    def multiply(a: float, b: float) -> float:
        """Multiply two numbers together and return the result."""
        return a * b
    
    @server.tool()
    def divide(a: float, b: float) -> float:
        """
        Divide a by b and return the result.
        
        Raises:
            ValueError: If b is zero
        """
        if b == 0:
            raise ValueError("Cannot divide by zero")
        return a / b
    ```
