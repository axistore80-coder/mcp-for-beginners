# ម៉ាស៊ីនគណនាប្រភេទ MCP (Python)



ការអនុវត្តម៉ាស៊ីនស៊ែរម៉ូដែល Context Protocol (MCP) សាមញ្ញមួយក្នុង Python ដែលផ្ដល់មុខងារគណនាប základ។


## ការតំឡើង

តំឡើងបណ្ណាសម្រាប់ការទាមទារ៖

```bash
pip install -r requirements.txt
```

ឬតំឡើង MCP Python SDK ដោយផ្ទាល់៖

```bash
pip install mcp>=1.18.0
```

## ការប្រើប្រាស់

### ការរត់ម៉ាស៊ីនស៊ែរ

ម៉ាស៊ីនស៊ែរ ត្រូវបានរចនាឡើងសម្រាប់រើសអតិថិជន MCP (ដូចជា Claude Desktop)។ ដើម្បីចាប់ផ្តើមម៉ាស៊ីនស៊ែរ៖

```bash
python mcp_calculator_server.py
```

**ចំណាំ**៖ នៅពេលរត់ដំណើរការផ្ទាល់ក្នុង​ terminal អ្នកនឹងឃើញកំហុសផ្ទៀងផ្ទាត់ JSON-RPC។ នេះគឺជាវិធានការស្មើរមាន - ម៉ាស៊ីនស៊ែរ កំពុងរងចាំសារចេញពីអតិថិជន MCP ដែលមានទ្រង់ទ្រាយត្រឹមត្រូវ។

### ការធ្វើតេស្តមុខងារ

ដើម្បីធានាថា មុខងារគណនាត្រូវដំណើរការល្អ៖

```bash
python test_calculator.py
```

## ការដោះស្រាយបញ្ហា

### កំហុសការនាំចូល

បើអ្នកឃើញ `ModuleNotFoundError: No module named 'mcp'` សូមតំឡើង MCP Python SDK៖

```bash
pip install mcp>=1.18.0
```

### កំហុស JSON-RPC នៅពេលរត់ផ្ទាល់

កំហុសដូចជា "Invalid JSON: EOF while parsing a value" នៅពេលរត់ម៉ាស់ឆ្នៃផ្ទាល់ គឺជារឿងធម្មតា។ ម៉ាស៊ីនស៊ែរ ត្រូវការសារអតិថិជន MCP មិនមែន បញ្ចូលពី terminal ផ្ទាល់។
