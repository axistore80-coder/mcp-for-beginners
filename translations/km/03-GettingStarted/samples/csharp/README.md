# бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһӮбһҺбһ“бһ¶бһ•бҹ’бһ“бҹӮбһҖбһҳбһјбһӣ MCP

бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһ“бҹҒбҹҮбһ•бҹ’бһҸбһӣбҹӢбһ“бһјбһңбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡ бһӮбһҺбһ“бһ¶бһ•бҹ’бһ“бҹӮбһҖбһҳбһјбһӣбһҸбһ¶бһҳбһҡбһҷбҹҲ Model Context Protocol (MCP)бҹ” бһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбһ…бһ“бһ¶бһҮбһ¶бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҹбһ¶бһҳбһүбҹ’бһүбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһҠбҹҶбһ”бһјбһ„бһҠбҹӮбһӣбһҖбҹҶбһ–бһ»бһ„бһҡбҹҖбһ“бһўбҹҶбһ–бһёбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸ MCPбҹ”

бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ”бһ“бҹ’бһҗбҹӮбһҳ бһҹбһјбһҳбһҳбһҫбһӣ [C# SDK](https://github.com/modelcontextprotocol/csharp-sdk)

## бһӣбһҖбҹ’бһҒбһҺбҹҲбһ–бһ·бһҹбҹҒбһҹ

бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһӮбһҺбһ“бһ¶бһ“бҹҒбҹҮбһ•бҹ’бһҸбһӣбҹӢбһ“бһјбһңбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

1. **бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһӮбһҺбһ“бһ¶бһ•бҹ’бһ“бҹӮбһҖбһҳбһјбһӣ** :
   - бһ”бһјбһҖбһӣбҹҒбһҒбһ–бһёбһҡбһҠбҹӮбһӣбһҳбһ¶бһ“
   - бһҠбһҖбһӣбҹҒбһҒбһҳбһҪбһҷбһ…бҹҒбһүбһ–бһёбһӣбҹҒбһҒбһҳбһҪбһҷ
   - бһӮбһ»бһҺбһӣбҹҒбһҒбһ–бһёбһҡбһҠбҹӮбһӣбһҳбһ¶бһ“
   - бһ…бҹӮбһҖбһӣбҹҒбһҒбһҳбһҪбһҷбһҮбһ¶бһӣбҹҒбһҒбһҳбһҪбһҷбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸ (бһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ…бҹӮбһҖбһҠбҹ„бһҷбһҹбһјбһ“бҹ’бһҷ)

## бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ `stdio` бһ”бҹ’бһҡбһ—бҹҒбһ‘
  
## бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’

1. **бһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP** :
   - бһ”бһҫбһҖвҖӢбһҖбһ“бҹ’бһӣбҹӮбһ„вҖӢбһ’бҹ’бһңбһҫбһҖбһ¶бһҡвҖӢбһҡбһ”бһҹбҹӢвҖӢбһўбҹ’бһ“бһҖвҖӢбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ VS Codeбҹ”
   - бһ”бһ„бҹ’бһҖбһҫбһҸвҖӢбһҜбһҖбһҹбһ¶бһҡ `.vscode/mcp.json` бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҗбһҸвҖӢбһҖбһ“бҹ’бһӣбҹӮбһ„вҖӢбһ’бҹ’бһңбһҫбһҖбһ¶бһҡвҖӢбһҡбһ”бһҹбҹӢвҖӢбһўбҹ’бһ“бһҖ бһҠбһҫбһҳбҹ’бһ”бһёбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCPбҹ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бҹ–

     ```jsonc
     {
       "inputs": [
         {
           "type": "promptString",
           "id": "repository-root",
           "description": "The absolute path to the repository root"
         }
       ],
       "servers": {
         "calculator-mcp-dotnet": {
           "type": "stdio",
           "command": "dotnet",
           "args": [
             "run",
             "--project",
             "${input:repository-root}/03-GettingStarted/samples/csharp/src/calculator.csproj"
           ]
         }
       }
     }
     ```

   - бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҹбҹ’бһ“бһҫбһІбҹ’бһҷбһ”бһүбҹ’бһ…бһјбһӣбһ«бһҹбһӮбһҳбҹ’бһҡбҹ„бһ„ GitHub бһҠбҹӮбһӣбһўбһ¶бһ…бһ‘бһ¶бһүбһҷбһҖбһ–бһёбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶ `git rev-parse --show-toplevel`бҹ”

## бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳ

бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһ“бҹҒбҹҮбһ”бһ„бҹ’бһ бһ¶бһүбһ…бһ»бһ„бһ”бһүбҹ’бһҮбһјбһ“ API бһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҸбһ¶бһҳбһҡбһҷбҹҲбһ–бһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸ MCPбҹ–

- `add(a, b)`: бһ”бһјбһҖбһӣбҹҒбһҒбһ–бһёбһҡбһҮбһ¶бһҳбһҪбһҷбһӮбҹ’бһ“бһ¶
- `subtract(a, b)`: бһҠбһҖбһӣбҹҒбһҒбһ‘бһёбһ–бһёбһҡбһ…бҹҒбһүбһ–бһёбһӣбҹҒбһҒбһ‘бһёбһҳбһҪбһҷ
- `multiply(a, b)`: бһӮбһ»бһҺбһӣбҹҒбһҒбһ–бһёбһҡбһҮбһ¶бһҳбһҪбһҷбһӮбҹ’бһ“бһ¶
- `divide(a, b)`: бһ…бҹӮбһҖбһӣбҹҒбһҒбһ‘бһёбһҳбһҪбһҷбһҠбҹ„бһҷбһӣбҹҒбһҒбһ‘бһёбһ–бһёбһҡ (бһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҹбһјбһ“бҹ’бһҷ)
- isPrime(n): бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҗбһ¶бһӣбҹҒбһҒбһҳбһҪбһҷбһӮбһәбһҮбһ¶бһ”бҹ’бһҡбҹҠбһёбһҳбһ¬бһ‘бҹҒ

## бһҹбһҳбһ¶бһҹвҖӢбһ—бһ¶бһӮвҖӢбһҖбҹҶбһҺбһҸбҹӢвҖӢбһҮбһ¶ Container

бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҳбһ»бһ“бһӮбһәбһӣбҹ’бһўбһ”бһҫбһўбҹ’бһ“бһҖбһҳбһ¶бһ“ .NET SDK бһҠбҹҶбһЎбһҫбһ„бһҡбһҪбһ… бһ“бһ·бһ„бһңбһ¶бһҮбһ¶бһҹбҹ’бһҗбһ·бһҸбһҖбҹ’бһ“бһ»бһ„бһ”бһҡбһ·бһ”бһ‘бһҒбһ¶бһ„бһҖбҹ’бһ“бһ»бһ„бҹ” бһ‘бҹ„бҹҮбһҷбҹүбһ¶бһ„бһҺбһ¶ бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһ…бҹӮбһҖбһҡбҹҶбһӣбҹӮбһҖбһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ¬бһҡбһҸбҹӢбһңбһ¶бһҖбҹ’бһ“бһ»бһ„бһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһҹбһҳбһ¶бһҹбһ—бһ¶бһӮбһҖбҹҶбһҺбһҸбҹӢбһҮбһ¶бһҖбһ»бһ„бһҸбһәбһ“бҹҗбһҡбҹ”

1. бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ Docker бһ бһҫбһҷбһ’бҹ’бһңбһҫбһІбҹ’бһҷбһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶бһңбһ¶бһҖбҹҶбһ–бһ»бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбҹ”
1. бһ–бһёбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһ”бһүбҹ’бһҮбһ¶ бһ…бһјбһӣбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһҗбһҸ `03-GettingStarted\samples\csharp\src` 
1. бһҠбһҫбһҳбҹ’бһ”бһёбһҖбһҹбһ¶бһ„бһҡбһјбһ”бһ—бһ¶бһ– Docker бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһӮбһҺбһ“бһ¶ бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡ бҹ– (бһ”бҹ’бһҠбһјбһҡ `<YOUR-DOCKER-USERNAME>` бһҮбһ¶бһҲбҹ’бһҳбҹ„бҹҮбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ Docker Hub бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ):
   ```bash
   docker build -t <YOUR-DOCKER-USERNAME>/mcp-calculator .
   ``` 
1. бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһҡбһјбһ”бһ—бһ¶бһ–бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбһҹбһ¶бһ„бһҡбһҪбһ… бһўбҹ„бһҷбһ”бһүбҹ’бһ…бһјбһ“бһңбһ¶бһ‘бҹ… Docker Hubбҹ” бһ”бҹ’бһҡбһҫбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶бһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–
   ```bash
    docker push <YOUR-DOCKER-USERNAME>/mcp-calculator
  ```
