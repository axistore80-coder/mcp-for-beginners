# бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһҖбһ¶бһӣбһӮбһҺбһ“бҹҒбһҷбҹ’бһҷбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ MCP

бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһ“бҹҒбҹҮбһ•бҹ’бһҸбһӣбҹӢбһ“бһјбһңбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡвҖӢбһҖбһ¶бһӣвҖӢбһӮбһҺбһ“бҹҒбһҷбҹ’бһҷбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҸбһ¶бһҳбһҡбһҷбҹҲвҖӢбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ Model Context Protocol (MCP) бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ Spring Boot бһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһҠбһ№бһҖбһҮбһүбҹ’бһҮбһјбһ“ WebFluxбҹ” бһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбһ…бһ“бһ¶бһЎбһҫбһ„бһҮбһ¶бһӮбҹҶбһҡбһјбһҹбһ¶бһҳбһүбҹ’бһүбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһҠбҹҶбһ”бһјбһ„бһҠбҹӮбһӣбһҖбҹҶбһ–бһ»бһ„бһҹбһ·бһҖбҹ’бһҹбһ¶бһ–бһёбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸ MCPбҹ”

бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ”бһ“бҹ’бһҗбҹӮбһҳ бһҹбһјбһҳбһҳбһҫбһӣбһҜбһҖбһҹбһ¶бһҡбһҷбҹ„бһ„ [MCP Server Boot Starter](https://docs.spring.io/spring-ai/reference/api/mcp/mcp-server-boot-starter-docs.html)бҹ”

## бһ‘бһ·бһҠбҹ’бһӢбһ—бһ¶бһ–бһ‘бһјбһ‘бҹ…

бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһ“бҹҒбҹҮбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбҹ–
- бһҖбһ¶бһҡбһӮбһ¶бҹҶбһ‘бҹ’бһҡ SSE (Server-Sent Events)
- бһҖбһ¶бһҡбһ…бһ»бҹҮбһ”бһүбҹ’бһҮбһёбһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҹ_using`@Tool` annotation бһҡбһ”бһҹбҹӢ Spring AI
- бһҳбһ»бһҒбһ„бһ¶бһҡбһҖбһ¶бһӣбһӮбһҺбһ“бҹҒбһҷбҹ’бһҷбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бҹ–
  - бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ бһҖбһ¶бһҡбһҠбһҖ бһ•бһӣбһӮбһ»бһҺ бһ бһҫбһҷбһ”бҹӮбһ„
  - бһҖбһ¶бһҡбһӮбһҺбһ“бһ¶бһҗбһ¶бһҳбһ–бһӣ бһ“бһ·бһ„бһ«бһҹбһҖбһ¶бһҡбҹүбҹҒ
  - бһҖбһ¶бһҡбһӮбһҺбһ“бһ¶ modulus (бһ“бҹ…бһҹбҹҶбһҺбһӣбҹӢ) бһ“бһ·бһ„бһҸбһҳбҹ’бһӣбҹғбһ—бһ¶бһ– Щ…Ш·Щ„ЩӮ
  - бһҳбһ»бһҒбһ„бһ¶бһҡбһҮбҹҶбһ“бһҪбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ–бһ·бһ–бһҺбҹҢбһ“бһ¶бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡ

## бһҳбһ»бһҒбһ„бһ¶бһҡ

бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһҖбһ¶бһӣвҖӢбһӮбһҺбһ“бҹҒбһҷбҹ’бһҷбһ“бҹҒбҹҮбһ•бҹ’бһҸбһӣбҹӢбһ“бһјбһңбһҳбһ»бһҒбһ„бһ¶бһҡбһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

1. **бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһўбһҡбһјбһ”бһ·бһ…бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“**бҹ–
   - бһ”бһјбһҖбһӣбҹҒбһҒбһ–бһёбһҡбһ…бһјбһӣбһӮбҹ’бһ“бһ¶
   - бһҠбһҖбһӣбҹҒбһҒбһҳбһҪбһҷбһ–бһёбһӣбҹҒбһҒбһҳбһҪбһҷбһ‘бҹҖбһҸ
   - бһӮбһ»бһҺбһӣбҹҒбһҒбһ–бһёбһҡбһ…бһјбһӣбһӮбҹ’бһ“бһ¶
   - бһ…бҹӮбһҖбһӣбҹҒбһҒбһҳбһҪбһҷбһҠбҹ„бһҷбһӣбҹҒбһҒбһҳбһҪбһҷ (бһҳбһ¶бһ“бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ…бҹӮбһҖбһҠбҹ„бһҷбһҹбһјбһ“бҹ’бһҷ)

2. **бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһңбһ·бһҮбҹ’бһҮбһҳбһ¶бһ“**бҹ–
   - бһӮбһҺбһ“бһ¶бһҗбһ¶бһҳбһ–бһӣ (бһӣбһҫбһҖбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бҹ…бһҖбһҳбҹ’бһҡбһ·бһҸбһҹбһҺбҹ’бһӢбһ¶бһ“)
   - бһӮбһҺбһ“бһ¶бһ«бһҹбһҖбһ¶бһҡбҹүбҹҒ (бһҳбһ¶бһ“бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһӣбҹҒбһҒбһўбһңбһ·бһҮбҹ’бһҮбһҳбһ¶бһ“)
   - бһӮбһҺбһ“бһ¶бһҳбҹүбһјбһҢбһ»бһӣбһҹ (бһ“бҹ…бһҹбҹҶбһҺбһӣбҹӢ)
   - бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ—бһ¶бһ– Щ…Ш·Щ„ЩӮ

3. **бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҮбҹҶбһ“бһҪбһҷ**бҹ–
   - бһҳбһ»бһҒбһ„бһ¶бһҡбһҮбҹҶбһ“бһҪбһҷбһҠбҹҶбһ”бһјбһ„бһҠбҹӮбһӣбһ–бһ“бҹ’бһҷбһӣбҹӢбһ–бһёбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҠбҹӮбһӣбһҳбһ¶бһ“

## бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳ

бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһ“бҹҒбҹҮбһ”бһ„бҹ’бһ бһ¶бһүбһ…бҹҒбһү API endpoints бһҸбһ¶бһҳбһҡбһҷбҹҲвҖӢбһ–бһ·бһ’бһёбһҖбһҡ MCPбҹ–

- `add(a, b)`: бһ”бһјбһҖбһӣбҹҒбһҒбһ–бһёбһҡбһ…бһјбһӣбһӮбҹ’бһ“бһ¶
- `subtract(a, b)`: бһҠбһҖбһӣбҹҒбһҒбһ‘бһёбһ–бһёбһҡбһ–бһёбһӣбҹҒбһҒбһ‘бһёбһҳбһҪбһҷ
- `multiply(a, b)`: бһӮбһ»бһҺбһӣбҹҒбһҒбһ–бһёбһҡбһ…бһјбһӣбһӮбҹ’бһ“бһ¶
- `divide(a, b)`: бһ…бҹӮбһҖбһӣбҹҒбһҒбһ‘бһёбһҳбһҪбһҷбһҠбҹ„бһҷбһӣбҹҒбһҒбһ‘бһёбһ–бһёбһҡ (бһҳбһ¶бһ“бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ…бҹӮбһҖбһҠбҹ„бһҷбһҹбһјбһ“бҹ’бһҷ)
- `power(base, exponent)`: бһӮбһҺбһ“бһҗбһ¶бһҳбһ–бһӣ
- `squareRoot(number)`: бһӮбһҺбһ“бһ¶бһ«бһҹбһҖбһ¶бһҡбҹүбҹҒ (бһҳбһ¶бһ“бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһӣбҹҒбһҒбһўбһңбһ·бһҮбҹ’бһҮбһҳбһ¶бһ“)
- `modulus(a, b)`: бһӮбһҺбһ“бһ¶бһҹбҹҶбһҺбһӣбҹӢ
- `absolute(number)`: бһӮбһҺбһ“бһ¶бһҸбһҳбҹ’бһӣбҹғбһ—бһ¶бһ– Щ…Ш·Щ„ЩӮ
- `help()`: бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһўбҹҶбһ–бһёбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҠбҹӮбһӣбһҳбһ¶бһ“

## бһўбһҸбһ·бһҗбһ·бһҮбһ“бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„

бһўбһҸбһ·бһҗбһ·бһҮбһ“бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҹбһ¶бһҳбһүбҹ’бһүбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһүбҹ’бһ…бһјбһӣбһҖбҹ’бһ“бһ»бһ„бһҖбһүбҹ’бһ…бһ”бҹӢ `com.microsoft.mcp.sample.client`бҹ” бһҗбҹ’бһ“бһ¶бһҖбҹӢ `SampleCalculatorClient` бһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҠбҹӮбһӣбһҳбһ¶бһ“бһҖбҹ’бһ“бһ»бһ„бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһҖбһ¶бһӣбһӮбһҺбһ“бҹҒбһҷбҹ’бһҷбҹ”

## бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһўбһҸбһ·бһҗбһ·бһҮбһ“ LangChain4j

бһӮбһҳбҹ’бһҡбҹ„бһ„бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһўбһҸбһ·бһҗбһ·бһҮбһ“бһӮбҹҶбһҡбһј LangChain4j бһҖбҹ’бһ“бһ»бһ„ `com.microsoft.mcp.sample.client.LangChain4jClient` бһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһҖбһ¶бһӣвҖӢбһӮбһҺбһ“бҹҒбһҷбҹ’бһҷбһҮбһ¶бһҳбһҪбһҷ LangChain4j бһ“бһ·бһ„бһҳбҹүбһјбһҠбҹӮбһӣ GitHubбҹ–

### бһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢбһҳбһ»бһ“

1. **бһҖбһ¶бһҡбһҸбҹҶбһЎбһҫбһ„Token GitHub**:

   бһҠбһҫбһҳбҹ’бһ”бһёбһ”бҹ’бһҡбһҫбһҳбҹүбһјбһҠбҹӮбһӣ AI бһҡбһ”бһҹбҹӢ GitHub (бһҠбһјбһ…бһҮбһ¶ phi-4) бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ‘бһ·бһҠбҹ’бһӢбһ—бһ¶бһ–бһ…бһјбһӣбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“ GitHubбҹ–

   a. бһ…бһјбһӣбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһӮбһҺбһ“бһё GitHub бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ– https://github.com/settings/tokens
   
   b. бһ…бһ»бһ… "Generate new token" вҶ’ "Generate new token (classic)"
   
   c. бһ•бҹ’бһҠбһӣбҹӢбһҲбҹ’бһҳбҹ„бҹҮбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ token бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ
   
   d. бһҮбҹ’бһҡбһҫбһҹ scope бһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–
      - `repo` (бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„вҖӢбһ–бҹҒбһүбһӣбҹҒбһүбһ“бҹғбһғбҹ’бһӣбһ¶бҹҶбһ„бһҜбһҖбһҮбһ“)
      - `read:org` (бһўбһ¶бһ“бһҖбҹ’бһҡбһ»бһҳ бһ“бһ·бһ„бһҹбһҳбһ¶бһҮбһ·бһҖбһҖбҹ’бһҡбһ»бһҳ бһ–бҹ’бһҡбһҳбһ‘бһ¶бҹҶбһ„бһӮбһҳбҹ’бһҡбҹ„бһ„бһҖбҹ’бһ“бһ»бһ„бһўбһ„бҹ’бһӮбһ—бһ¶бһ–)
      - `gist` (бһ”бһ„бҹ’бһҖбһҫбһҸ gists)
      - `user:email` (бһ…бһјбһӣбһ”бҹ’бһҡбһҫбһўбһ¶бһҹбһҷбһҠбҹ’бһӢбһ¶бһ“бһўбҹҠбһёбһҳбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ (бһўбһ¶бһ“бһ”бһ¶бһ“бһҸбҹӮ))
   
   e. бһ…бһ»бһ… "Generate token" бһ бһҫбһҷбһ…бһҳбҹ’бһӣбһ„ token бһҗбҹ’бһҳбһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ
   
   f. бһҖбҹҶбһҺбһҸбҹӢбһңбһ¶бһҮбһ¶бһҸбһҳбҹ’бһӣбҹғбһ”бһҡбһ·бһҹбҹ’бһҗбһ¶бһ“бҹ–
      
      бһ“бҹ…бһӣбһҫ Windows:
      ```
      set GITHUB_TOKEN=your-github-token
      ```
      
      бһ“бҹ…бһӣбһҫ macOS/Linux:
      ```bash
      export GITHUB_TOKEN=your-github-token
      ```

### бһ”бһүбҹ’бһ бһ¶бһ‘бһјбһ‘бҹ…бһҮбһ¶бһҳбһҪбһҷ Token GitHub

1. **бһ”бһүбҹ’бһ бһ¶бһҹбһ·бһ‘бҹ’бһ’бһ· Token**бҹ– бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһҖбҹҶбһ бһ»бһҹ 403 Forbidden бһҹбһјбһҳбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҗбһ¶ token бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһҹбһ·бһ‘бҹ’бһ’бһ·бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһҠбһјбһ…бһ”бһ¶бһ“бһ”бһ„бҹ’бһ бһ¶бһүбһҖбҹ’бһ“бһ»бһ„бһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢбһҳбһ»бһ“бҹ”

2. **бһҡбһҖбһҳбһ·бһ“бһғбһҫбһү Token**бҹ– бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһҖбҹҶбһ бһ»бһҹ "No API key found" бһҹбһјбһҳбһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶ environment variable GITHUB_TOKEN бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбҹ”

3. **бһҖбҹҶбһҺбһҸбҹӢбһўбһҸбҹ’бһҡбһ¶бһ…бһјбһӣбһ”бҹ’бһҡбһҫ**бҹ– GitHub API бһҳбһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһўбһҸбҹ’бһҡбһ¶бҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһ‘бҹҮбһҖбҹҶбһ бһ»бһҹбһҖбҹҶбһҺбһҸбҹӢбһўбһҸбҹ’бһҡбһ¶ (бһҹбҹ’бһҗбһ¶бһ“бһ’бһ¶бһ“бһӣбҹҒбһҒ 429) бһҹбһјбһҳбһҡбһ„бҹӢбһ…бһ¶бҹҶбһ”бҹүбһ»бһ“бҹ’бһҳбһ¶бһ“бһ“бһ¶бһ‘бһёбһҳбһ»бһ“бһ–бҹ’бһҷбһ¶бһҷбһ¶бһҳбһҳбҹ’бһҸбһ„бһ‘бҹҖбһҸбҹ”

4. **бһ—бһ¶бһ–бһ•бһ»бһҸбһҖбҹҶбһҺбһҸбҹӢбһҡбһ”бһҹбҹӢ Token**бҹ– Token GitHub бһўбһ¶бһ…бһ•бһ»бһҸбһҖбҹҶбһҺбһҸбҹӢбҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһҖбҹҶбһ бһ»бһҹбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһҹбһ»бһ–бһӣбһ—бһ¶бһ–бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһҡбһҷбҹҲбһ–бҹҒбһӣ бһҹбһјбһҳбһ”бһ„бҹ’бһҖбһҫбһҸ token бһҗбҹ’бһҳбһё бһ“бһ·бһ„бһ”бһ“бҹ’бһ‘бһ¶бһ“бҹӢбһҹбһҳбҹҗбһҷ environment variable бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

бһ”бһҫбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҮбҹҶбһ“бһҪбһҷбһ”бһ“бҹ’бһҗбҹӮбһҳ бһҹбһјбһҳбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ [бһҜбһҖбһҹбһ¶бһҡ LangChain4j](https://github.com/langchain4j/langchain4j) бһ¬ [бһҜбһҖбһҹбһ¶бһҡ GitHub API](https://docs.github.com/en/rest)бҹ”
