# бһӣбҹҶбһ“бһ¶бҹҶ

бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҳбһ»бһ“бһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһңбһ·бһ’бһёбһ”бҹ’бһҡбһҫбһ•<Project> бһҡбһ”бһҹбҹӢ .NET бһҮбһ¶бһ”бҹ’бһҡбһ—бҹҒбһ‘ `stdio` бһ бһҫбһҷбһңбһ·бһ’бһёбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бҹ’бһүбһҹбҹҗбһҡбһҮбһ¶бһҳбһҪбһҷбһҗбһ¶бһҹбһ’бһ¶бһҸбһ»бһҸбҹҶбһ”бһ“бҹӢбҹ” бһңбһ¶бһҮбһ¶бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһӣбҹ’бһўбһ“бҹ…бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҹбҹ’бһҗбһ¶бһ“бһҖбһ¶бһҡбһҺбҹҚбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бҹ” бһ‘бҹ„бҹҮбһҮбһ¶бһҷбҹүбһ¶бһ„бһҺбһ¶ бһңбһ¶бһҖбҹҸбһўбһ¶бһ…бһҳбһ¶бһ“бһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҡбһҸбҹӢбһ”бһүбҹ’бһҮбһ¶бһ”бҹ’бһүбһҹбҹҗбһҡбһ–бһёбһ…бһҳбҹ’бһ„бһ¶бһҷ бһҠбһјбһ…бһҮбһ¶бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҡбһ„бҹ’бһңбһ»бһ„бһ–бһ–бһҖбҹ” бһ“бҹҒбҹҮбһҮбһ¶бһҡбһҝбһ„бһҠбҹӮбһӣбһ”бҹ’бһҡбһ—бҹҒбһ‘ `http` бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбҹ”

бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣбһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҖбҹ’бһ“бһ»бһ„бһҗбһҸ `04-PracticalImplementation` бһңбһ¶бһўбһ¶бһ…бһ”бһ„бҹ’бһ бһ¶бһүбһҗбһ¶бһҳбһ¶бһ“бһ—бһ¶бһ–бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбһҮбһ¶бһ„бһҳбһ»бһ“бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бҹ” бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҖбһ¶бһҡбһ–бһ·бһҸбһңбһ¶бһҳбһ·бһ“бһҳбҹӮбһ“бһҮбһ¶бһ”бҹӢбһӣбҹҶбһ”бһ¶бһҖбһ‘бҹҒбҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһӣбҹ„бһҖбһўбҹ’бһ“бһҖбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҷбҹүбһ¶бһ„бһҮбһ·бһҸбһҹбҹ’бһ“бһ·бһ‘бҹ’бһ’бһ‘бҹ…бһӣбһҫбһӮбһҳбҹ’бһҡбҹ„бһ„ `src/Calculator` бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҳбһҫбһӣбһғбһҫбһүбһҗбһ¶бһңбһ¶вҖӢбһӮбһәвҖӢбһҮбһ¶вҖӢбһҖбһјбһҠвҖӢбһҠбһҠбҹӮбһӣвҖӢбһҮбһ¶бһ„вҖӢбһҳбһ»бһ“бҹ” бһ…бҹҶбһҺбһ»бһ…бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһўбҹ’бһ“бһҖбһҖбҹҶбһ–бһ»бһ„бһ”бҹ’бһҡбһҫбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһ•бҹ’бһҹбҹҒбһ„ `ModelContextProtocol.AspNetCore` бһҠбһҫбһҳбҹ’бһ”бһёбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһ¶бһҡбһҹбҹ’бһ“бһҫбһҹбһ»бҹҶбһ“бҹғ HTTPбҹ” бһ бһҫбһҷбһҷбһҫбһ„бһ”бһ¶бһ“бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҳбҹҒбһ’бһҸбҹ’бһҷбһјбһҸ `IsPrime` бһІбҹ’бһҷбһ‘бҹ…бһҮбһ¶бһҡбһ”бҹҖбһ”бһҜбһҖбһҮбһ“ бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһ бһ¶бһүбһҗбһ¶бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҳбһ¶бһ“бһҳбҹҒбһ’бһҸбҹ’бһҷбһјбһҜбһҖбһҮбһ“бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһјбһҠбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһҖбһјбһҠбһҠбҹӮбһӣбһ“бҹ…бһҹбһӣбҹӢбһӮбһәбһҠбһјбһ…бһҠбһҫбһҳбҹ”

бһӮбһҳбҹ’бһҡбҹ„бһ„бһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбһҳбһҖбһ–бһё [.NET Aspire](https://learn.microsoft.com/dotnet/aspire/get-started/aspire-overview)бҹ” бһҖбһ¶бһҡбһҳбһ¶бһ“ .NET Aspire бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ“бһ№бһ„бһ”бһ„бҹ’бһҖбһҫбһ“бһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҖбҹ’бһ“бһ»бһ„бһ–бҹҒбһӣбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚ бһ“бһ·бһ„бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„ бһ“бһ·бһ„бһҮбһҪбһҷбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһ“бһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бҹ” бһңбһ¶бһҳбһ·бһ“бһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҠбһҫбһҳбҹ’бһ”бһёбһҡбһҸбҹӢбһ”бһүбҹ’бһҮбһ¶бһ”бҹ’бһүбһҹбҹҗбһҡбһ‘бҹҒ бһ”бҹүбһ»бһ“бҹ’бһҸбҹ’бһҸбһңбһ¶бһҮбһ¶бһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚбһӣбҹ’бһўбһҖбҹ’бһ“бһ»бһ„бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

## бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ”бһүбҹ’бһҮбһ¶бһ”бҹ’бһүбһҹбҹҗбһҡбһ–бһёбһҖбҹ’бһ“бһ»бһ„бһҸбҹҶбһ”бһ“бҹӢ

1. бһ–бһё Visual Studio Code (бһҮбһ¶бһҳбһҪбһҷбһ•бҹ’бһ“бҹӮбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳ C# DevKit) бһ…бһјбһӣбһ‘бҹ…бһҗбһҸ `04-PracticalImplementation/samples/csharp`бҹ”
1. бһҡбһҸбҹӢбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶бһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҠбһҫбһҳбҹ’бһ”бһёбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ”бһүбҹ’бһҮбһ¶бһ”бҹ’бһүбһҹбҹҗбһҡбҹ–


## бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„ Streamable HTTP бһҮбһ¶бһҳбһҪбһҷ MCP Inspector

бһ”бһҫбһӣбҹ„бһҖбһўбҹ’бһ“бһҖбһҳбһ¶бһ“ Node.js 22.7.5 бһ«бһҒбҹ’бһ–бһҹбҹӢбһҮбһ¶бһ„бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹ’бһҡбһҫ MCP Inspector бһҠбһҫбһҳбҹ’бһ”бһёбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһ”бһүбҹ’бһҮбһ¶бһ”бҹ’бһүбһҹбҹҗбһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ”бһүбҹ’бһҮбһ¶бһ”бҹ’бһүбһҹбҹҗбһҡ бһ бһҫбһҷбһҡбһҸбҹӢбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶бһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бһүбҹ’бһүбһ¶бһҹбһ¶бһҡбһҳбһҪбһҷбҹ–

```bash
npx @modelcontextprotocol/inspector http://localhost:5058
```

![MCP Inspector](../../../../../translated_images/km/mcp-inspector.c223422b9b494fb4.webp)

- бһҮбҹ’бһҡбһҫбһҹбһ”бҹ’бһҡбһ—бҹҒбһ‘бһ•бҹ’бһӣбһјбһңбһҠбһ№бһҖбһҗбһ¶ `Streamable HTTP`бҹ”
- бһ“бҹ…бһңбһ¶бһӣ Url бһ”бһүбҹ’бһ…бһјбһӣ URL бһҡбһ”бһҹбҹӢбһ”бһүбҹ’бһҮбһ¶бһ”бҹ’бһүбһҹбҹҗбһҡбһҠбҹӮбһӣбһ”бһ¶бһ“бһ…бҹҶбһҺбһ¶бҹҶбһҳбһ»бһ“ бһ бһҫбһҷбһ”бһ“бҹ’бһҗбҹӮбһҳ `/mcp` бһ“бҹ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҷбҹ” бһңбһ¶бһӮбһҪбһҡбһҸбҹӮбһҳбһ¶бһ“бһ”бҹ’бһҡбһ—бҹҒбһ‘ `http` (бһҳбһ·бһ“бһҳбҹӮбһ“ `https`) бһ бһҫбһҷбһҡбһјбһ”бһҡбһ¶бһ„бһҠбһјбһ…бһҮбһ¶ `http://localhost:5058/mcp`бҹ”
- бһ…бһ»бһ…бһ”бҹҠбһјбһҸбһ»бһ„ Connectбҹ”

бһўбҹ’бһңбһёбһҠбҹӮбһӣбһӣбҹ’бһўбһўбҹҶбһ–бһё Inspector бһӮбһәбһңбһ¶бһ”бһ„бҹ’бһҖбһҫбһҸбһӮбҹҶбһ“бһ·бһҸбһҳбһҫбһӣбһғбһҫбһүбһӣбҹ’бһўбһӣбһҫбһўбҹ’бһңбһёбһҠбҹӮбһӣбһҖбҹҶбһ–бһ»бһ„бһҖбһҫбһҸбһҳбһ¶бһ“бҹ”

- бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһ”бһ„бҹ’бһ бһ¶бһүбһ”бһҡбһ·бһҖбҹ’бһҒбһ¶бһҡбһҠбҹӮбһӣбһҳбһ¶бһ“
- бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҳбһ»бһҒбһ„бһ¶бһҡбһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“ бһңбһ¶бһӮбһҪбһҡбһҸбҹӮбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҠбһјбһ…бһҳбһ»бһ“бҹ”

## бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„ MCP Server бһҮбһ¶бһҳбһҪбһҷ GitHub Copilot Chat бһҖбҹ’бһ“бһ»бһ„ VS Code

бһҠбһҫбһҳбҹ’бһ”бһёбһ”бҹ’бһҡбһҫбһ•бҹ’бһӣбһјбһңбһҠбһ№бһҖ Streamable HTTP бһҮбһ¶бһҳбһҪбһҷ GitHub Copilot Chat бһ”бҹ’бһҡбҹӮбһ”бҹ’бһҡбһҪбһӣбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһ”бһҹбҹӢбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ `calc-mcp` бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһ»бһ“бһ“бҹҒбҹҮбһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

```jsonc
// .vscode/mcp.json
{
  "servers": {
    "calc-mcp": {
      "type": "http",
      "url": "http://localhost:5058/mcp"
    }
  }
}
```

бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҒбҹ’бһӣбҹҮбҹ–

- бһҹбҹ’бһ“бһҫбһҹбһ»бҹҶ "3 prime numbers after 6780"бҹ” бһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһҗбһ¶ Copilot бһ“бһ№бһ„бһ”бҹ’бһҡбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһҗбҹ’бһҳбһё `NextFivePrimeNumbers` бһ бһҫбһҷбһ”бһ„бҹ’бһ бһ¶бһүбһҸбҹӮ 3 бһӣбҹҒбһҒбһӮбҹ„бһӣбһ”бһҸбҹ’бһҡбһҠбҹҶбһ”бһјбһ„бҹ”
- бһҹбҹ’бһ“бһҫбһҹбһ»бҹҶ "7 prime numbers after 111", бһҠбһҫбһҳбҹ’бһ”бһёбһҳбһҫбһӣбһўбҹ’бһңбһёбһҖбһҫбһҸбһЎбһҫбһ„бҹ”
