## бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’

бһӮбһҳбҹ’бһҡбҹ„бһ„бһ“бҹҒбҹҮбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһңбҹӮбһ”бһҳбһҪбһҷ бһҠбҹӮбһӣбһ”бҹ’бһҡбһҫбһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҳбһ¶бһҸбһ·бһҖбһ¶бһҳбһ»бһ“бһ–бҹҒбһӣбһ•бҹ’бһүбһҫбһҹбҹҶбһҺбһҪбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ‘бҹ…бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһӮбһҺбһ“бһ¶бһӮбҹ’бһҡбһ№бҹҮбһҸбһ¶бһҳбһҡбһҷбҹҲбһ– Protocol Model Context (MCP)бҹ”

### бһңбһ¶бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ”бҹӮбһ”бһҺбһ¶

1. **бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһӣбһ–бһёбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ**бҹ– бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бһүбҹ’бһ…бһјбһӣбһҹбҹҶбһҺбһҪбһҡбһҖбҹ’бһ“бһ»бһ„бһ•бҹ’бһ‘бҹғбһҳбһ»бһҒбһңбҹӮбһ”
2. **бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҳбһ¶бһҸбһ·бһҖбһ¶ (бһ”бһүбҹ’бһ…бһјбһӣ)**бҹ– бһҹбҹҶбһҺбһҪбһҡбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһңбһ·бһ—бһ¶бһӮбһҠбҹ„бһҷ Azure Content Safety API
3. **бһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбҹҒбһ…бһ…бһ·бһҸбҹ’бһҸбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– (бһ”бһүбҹ’бһ…бһјбһӣ)**бҹ–
   - бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҳбһ¶бһҸбһ·бһҖбһ¶бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– (бһҖбһҳбҹ’бһҡбһ·бһҸбһ’бҹ’бһ„бһ“бҹӢбһ’бҹ’бһ„бһҡ < 2 бһҖбҹ’бһ“бһ»бһ„бһӮбҹ’бһҡбһ”бҹӢбһ”бҹ’бһҡбһ—бҹҒбһ‘) бһңбһ¶бһ”бһ“бҹ’бһҸбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһӮбһҺбһ“бһ¶бһӮбҹ’бһҡбһ№бҹҮ
   - бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҳбһ¶бһҸбһ·бһҖбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҹбһ„бҹ’бһҹбҹҗбһҷбһҗбһ¶бһўбһ¶бһ…бһҳбһ¶бһ“бһӮбҹ’бһҡбҹ„бҹҮбһҗбҹ’бһ“бһ¶бһҖбҹӢ бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҮбһ”бҹӢбһҹбҹ’бһ„бһ¶бһҸбҹӢбһ бһҫбһҷбһ”бһ„бҹ’бһҡбһҪбһҳбһҹбһ¶бһҡбһ–бҹ’бһҡбһҳбһ¶бһ“
4. **бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһӮбһҺбһ“бһ¶бһӮбҹ’бһҡбһ№бҹҮ**бҹ– бһҳбһ¶бһҸбһ·бһҖбһ¶бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҠбҹ„бһҷ LangChain4j бһҠбҹӮбһӣбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҮбһ¶бһҳбһҪбһҷбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫбһӮбһҺбһ“бһ¶бһӮбҹ’бһҡбһ№бҹҮ MCP
5. **бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҳбһ¶бһҸбһ·бһҖбһ¶ (бһ”бһүбҹ’бһ…бһјбһӣ)**бҹ– бһ…бһҳбҹ’бһӣбһҫбһҷбһҡбһ”бһҹбҹӢбһ”бһ»бһҸбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһңбһ·бһ—бһ¶бһӮбһҠбҹ„бһҷ Azure Content Safety API
6. **бһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбҹҒбһ…бһ…бһ·бһҸбҹ’бһҸбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– (бһ…бҹҒбһү)**бҹ–
   - бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһ…бһҳбҹ’бһӣбһҫбһҷбһ”бһ»бһҸбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– бһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ„бҹ’бһ бһ¶бһүбһ‘бҹ…бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ
   - бһ”бҹ’бһҡбһҹбһҫбһ”бһҫбһ…бһҳбҹ’бһӣбһҫбһҷбһ”бһ»бһҸбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҹбһ„бҹ’бһҹбҹҗбһҷбһҗбһ¶бһўбһ¶бһ…бһҳбһ¶бһ“бһӮбҹ’бһҡбҹ„бҹҮбһҗбҹ’бһ“бһ¶бһҖбҹӢ бһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҮбҹҶбһ“бһҪбһҹбһҠбҹ„бһҷбһҹбһ¶бһҡбһ–бҹ’бһҡбһҳбһ¶бһ“
7. **бһ…бһҳбҹ’бһӣбһҫбһҷ**бҹ– бһӣбһ‘бҹ’бһ’бһ•бһӣ (бһ”бһҫбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–) бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһ бһ¶бһүбһ‘бҹ…бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһҮбһ¶бһҳбһҪбһҷбһ“бһ№бһ„бһҖбһ¶бһҡбһңбһ·бһ—бһ¶бһӮбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһ‘бҹ’бһңбҹҒбһ•бҹ’бһ‘бҹғ

## бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ Model Context Protocol (MCP) бһҮбһ¶бһҳбһҪбһҷбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһӮбһҺбһ“бһ¶бһӮбҹ’бһҡбһ№бҹҮ

бһӮбһҳбҹ’бһҡбҹ„бһ„бһ“бҹҒбҹҮбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһңбһ·бһ’бһёбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ Model Context Protocol (MCP) бһҠбһҫбһҳбҹ’бһ”бһёбһ бҹ…бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһӮбһҺбһ“бһ¶бһӮбҹ’бһҡбһ№бҹҮ MCP бһ–бһё LangChain4jбҹ” бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ”бҹ’бһҡбһҫбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ MCP бһҖбҹ’бһ“бһ»бһ„бһҸбҹҶбһ”бһ“бҹӢбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҠбҹӮбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ“бҹ…бһ…бҹ’бһҡбһҖ бҹЁбҹ бҹЁбҹ  бһҠбһҫбһҳбҹ’бһ”бһёбһ•бҹ’бһҠбһӣбҹӢбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡ бһӮбһҺбһ“бһ¶бһӮбҹ’бһҡбһ№бҹҮбҹ”

### бһҖбһ¶бһҡбһҸбҹҶбһЎбһҫбһ„бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳ Azure Content Safety

бһҳбһ»бһ“бһ–бҹҒбһӣбһ”бҹ’бһҡбһҫбһӣбһҖбҹ’бһҒбһҺбҹҲбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҳбһ¶бһҸбһ·бһҖбһ¶ бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһ”бһ„бҹ’бһҖбһҫбһҸбһ’бһ“бһ’бһ¶бһ“бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳ Azure Content Safety:

1. бһ…бһјбһӣбһ‘бҹ…бһҖбһ¶бһ“бҹӢ [Azure Portal](https://portal.azure.com)
2. бһ…бһ»бһ… "Create a resource" бһ бһҫбһҷбһҹбҹ’бһңбҹӮбһ„бһҡбһҖ "Content Safety"
3. бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ "Content Safety" бһ бһҫбһҷбһ…бһ»бһ… "Create"
4. бһ”бһүбҹ’бһ…бһјбһӣбһҲбҹ’бһҳбҹ„бҹҮбһҳбһҪбһҷбһҠбҹӮбһӣбһҳбһ·бһ“бһҳбһ¶бһ“бһҳбһ»бһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ’бһ“бһ’бһ¶бһ“бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ
5. бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҖбһ»бһ„бһҸбҹ’бһҡбһјбһңбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ“бһ·бһ„бһҖбҹ’бһҡбһ»бһҳбһ’бһ“бһ’бһ¶бһ“ (бһ¬бһ”бһ„бҹ’бһҖбһҫбһҸбһҗбҹ’бһҳбһё)
6. бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҸбҹҶбһ”бһ“бҹӢбһҠбҹӮбһӣбһӮбһ¶бҹҶбһ‘бҹ’бһҡ (бһ–бһ·бһ“бһ·бһҸбҹ’бһҷ [Region availability](https://azure.microsoft.com/en-us/global-infrastructure/services/?products=cognitive-services) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһӣбһҳбҹ’бһўбһ·бһҸ)
7. бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҖбһҳбҹ’бһҡбһ·бһҸбһҸбһҳбҹ’бһӣбҹғбһҠбҹӮбһӣбһҹбһҳбһҡбһҳбҹ’бһҷ
8. бһ…бһ»бһ… "Create" бһҠбһҫбһҳбҹ’бһ”бһёбһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣбһ’бһ“бһ’бһ¶бһ“
9. бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣбһ”бһ¶бһ“ бһҹбһјбһҳбһ…бһ»бһ… "Go to resource"
10. бһ“бҹ…бһ•бҹ’бһ“бҹӮбһҖбһҒбһ¶бһ„бһҶбҹ’бһңбҹҒбһ„ бһҖбҹ’бһҡбҹ„бһҳ "Resource Management" бһҹбһјбһҳбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ "Keys and Endpoint"
11. бһ…бһҳбҹ’бһӣбһ„бһҖбһјбһ“бһҹбҹ„бһҳбһҪбһҷбһҺбһ¶бһҳбһҪбһҷ бһ“бһ·бһ„ URL бһ…бһ»бһ„бһ•бҹ’бһӣбһјбһңбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһҫбһҖбҹ’бһ“бһ»бһ„бһҮбҹҶбһ бһ¶бһ“бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢ

### бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһўбһҗбҹҒбһҡбһҹбһ бһ–бҹҗбһ“бҹ’бһ’бһ”бһҡбһ·бһҹбҹ’бһҗбһ¶бһ“

бһҖбҹҶбһҺбһҸбҹӢбһўбһҗбҹҒбһҡ `GITHUB_TOKEN` бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҳбҹүбһјбһҠбҹӮбһӣ GitHub:
```sh
export GITHUB_TOKEN=<your_github_token>
```

- бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҳбһ¶бһҸбһ·бһҖбһ¶бһ‘бҹ’бһңбҹҒбһҹбҹ’бһҡбһ‘бһ¶бһ”бҹӢбһӣбһҫбһҹбҹҶбһҺбһҪбһҡбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ бһ“бһ·бһ„бһ…бһҳбҹ’бһӣбһҫбһҷбһ”бһ»бһҸ
- бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҮбһ¶бһҳбһҪбһҷбһҳбҹүбһјбһҠбҹӮбһӣ gpt-4.1-nano бһҡбһ”бһҹбҹӢ GitHub бһҸбһ¶бһҳ LangChain4j
- бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ Server-Sent Events (SSE) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҠбһ№бһҖбһҮбһүбҹ’бһҮбһјбһ“ MCP

## бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҳбһ¶бһҸбһ·бһҖбһ¶

1. **бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ”бһүбҹ’бһ…бһјбһӣ**бҹ– бһҹбҹҶбһҺбһҪбһҡбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһңбһ·бһ—бһ¶бһӮбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҮбҹҶбһ–бһјбһҖбһҳбһ¶бһҸбһ·бһҖбһ¶бһ бһ¶бһ“бһ·бһ—бҹҗбһҷбһҠбһјбһ…бһҮбһ¶ бһ—бһ¶бһҹбһ¶бһҖбҹҶбһ бһ№бһ„ бһҖбһ¶бһҡбһҡбһҫбһҹбһўбһҫбһ„бһҒбҹ’бһӣбһҪбһ“ бһ¬бһҳбһ¶бһҸбһ·бһҖбһ¶бһ•бҹ’бһӣбһјбһңбһ—бҹҒбһ‘ бһҳбһ»бһ“бһ–бҹҒбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбҹ”

## бһўбһҸбһ·бһҗбһ·бһҮбһ“бһңбҹӮбһ”

### бһӣбһҖбҹ’бһҒбһҺбҹҲбһ•бҹ’бһ‘бҹғбһҳбһ»бһҒбһңбҹӮбһ”

   - бһӣбһ‘бҹ’бһ’бһ•бһӣбһӮбһҺбһ“бһ¶бһӣбһ‘бҹ’бһ’бһ•бһӣ (бһ”бһҫбһҹбҹҶбһҺбһҪбһҡбһ–бһ·бһҸбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–)
   - бһҖбһ¶бһҡбһңбһ·бһ—бһ¶бһӮбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҳбһ¶бһҸбһ·бһҖбһ¶бһ…бһҳбҹ’бһӣбһҫбһҷбһ”бһ»бһҸ
   - бһңбһ·бһүбҹ’бһүбһ¶бһҺбһ“бҹғбһҖбһ¶бһҡбһ–бҹ’бһҡбһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһӣ бһ¬бһ…бҹҒбһүбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҹбһ„бҹ’бһҹбҹҗбһҷ
