# Weather MCP Server

бһ“бҹҒбҹҮбһӮбһәбһҮбһ¶бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ MCP Server бһҖбҹ’бһ“бһ»бһ„бһ—бһ¶бһҹбһ¶ Python бһҠбҹӮбһӣбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ§бһ”бһҖбһҡбһҺбҹҚбһўбһ¶бһҖбһ¶бһҹбһ’бһ¶бһҸбһ»бһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһҶбҹ’бһӣбһҫбһҷбһҸбһ”бһ”бҹ’бһӣбҹӮбһҖбҹ—бҹ” бһңбһ¶бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһҮбһ¶бһӮбҹҶбһҡбһјбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ MCP Server бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһңбһ¶бһўбһ¶бһ…бһҳбһ¶бһ“бһҳбһ»бһҒбһ„бһ¶бһҡбһҠбһјбһ…бһҸбһ‘бҹ…бҹ–

- **бһ§бһ”бһҖбһҡбһҺбҹҚбһўбһ¶бһҖбһ¶бһҹбһ’бһ¶бһҸбһ»**бҹ– бһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣбһ•бҹ’бһҸбһӣбҹӢбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһўбһ¶бһҖбһ¶бһҹбһ’бһ¶бһҸбһ»бһ”бҹ’бһӣбҹӮбһҖбҹ—бһҳбһҪбһҷбһҹбҹ’бһҸбһёбһ–бһёбһ‘бһёбһҸбһ¶бҹҶбһ„бһҠбҹӮбһӣбһ”бһ¶бһ“бһ•бҹ’бһҸбһӣбҹӢбҹ”
- **бһ§бһ”бһҖбһҡбһҺбҹҚ Git Clone**бҹ– бһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ…бһҳбҹ’бһӣбһ„бһ—бһ¶бһҹ Git бһҳбһҪбһҷбһ‘бҹ…бһҗбһҸбһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбҹ”
- **бһ§бһ”бһҖбһҡбһҺбҹҚбһ”бһҫбһҖ VS Code**бҹ– бһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣбһ”бһҫбһҖбһҗбһҸбһҳбһҪбһҷбһҖбҹ’бһ“бһ»бһ„ VS Code бһ¬ VS Code Insidersбҹ”
- **бһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ…бһҖбһ¶бһ“бҹӢ Agent Builder**бҹ– бһҳбһ»бһҒбһ„бһ¶бһҡбһҠбҹӮбһӣбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһ—бҹ’бһҮбһ¶бһ”бҹӢ MCP Server бһ‘бҹ… Agent Builder бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҸбҹҒбһҹбҹ’бһҸ бһ“бһ·бһ„бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ”бһүбҹ’бһ бһ¶бҹ”
- **бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ”бһүбҹ’бһ бһ¶бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ [MCP Inspector](https://github.com/modelcontextprotocol/inspector)**бҹ– бһҳбһ»бһҒбһ„бһ¶бһҡбһҠбҹӮбһӣбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ”бһүбҹ’бһ бһ¶ MCP Server бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ MCP Inspectorбҹ”
