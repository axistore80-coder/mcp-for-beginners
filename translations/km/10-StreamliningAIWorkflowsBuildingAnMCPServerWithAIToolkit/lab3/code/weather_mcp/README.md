# Weather MCP Server

бһ“бҹҒбҹҮвҖӢбһӮбһәвҖӢбһҮбһ¶вҖӢбһҳбҹүбһ¶бһҹбҹҠбһёбһ“вҖӢбһҳбҹүбһ¶бһҹбҹҠбһёбһ“ MCP бһҖбҹ’бһ“бһ»бһ„вҖӢбһ—бһ¶бһҹбһ¶ Python бһҠбҹӮбһӣвҖӢбһўбһ“бһ»бһңбһҸбҹ’бһҸвҖӢбһ§бһ”бһҖбһҡбһҺбҹҚвҖӢбһўбһ¶бһҖбһ¶бһҹбһ’бһ¶бһҸбһ»вҖӢбһҮбһ¶бһҳбһҪбһҷвҖӢбһ“бһ№бһ„вҖӢбһ…бһҳбҹ’бһӣбһҫбһҷвҖӢбһӮбҹҶбһ“бһјбһҡбҹ” бһңбһ¶бһўбһ¶бһ…вҖӢбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“вҖӢбһ”бҹ’бһҡбһҫвҖӢбһҮбһ¶бһӮбҹҶбһҡбһјвҖӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢвҖӢбһҳбҹүбһ¶бһҹбҹҠбһёбһ“ MCP бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“вҖӢбһҡбһ”бһҹбҹӢвҖӢбһўбҹ’бһ“бһҖбҹ” бһңбһ¶бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣвҖӢбһҳбһ»бһҒбһ„бһ¶бһҡвҖӢбһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

- **бһ§бһ”бһҖбһҡбһҺбҹҚбһўбһ¶бһҖбһ¶бһҹбһ’бһ¶бһҸбһ»**бҹ– бһ§бһ”бһҖбһҡбһҺбҹҚвҖӢбһҠбҹӮбһӣвҖӢбһ•бҹ’бһҸбһӣбҹӢвҖӢбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“вҖӢбһўбһ¶бһҖбһ¶бһҹбһ’бһ¶бһҸбһ»вҖӢбһӮбҹҶбһ“бһјбһҡвҖӢбһ•бҹ’бһўбҹӮбһҖвҖӢбһӣбһҫбһ‘бһёбһҸбһ¶бҹҶбһ„вҖӢбһҠбҹӮбһӣбһ•бҹ’бһҠбһӣбҹӢбҹ”
- **бһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ…вҖӢбһҖбһ¶бһ“бҹӢ Agent Builder**бҹ– бһҳбһ»бһҒбһ„бһ¶бһҡвҖӢбһҠбҹӮбһӣвҖӢбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸвҖӢбһІбҹ’бһҷвҖӢбһўбҹ’бһ“бһҖвҖӢбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҳбҹүбһ¶бһҹбҹҠбһёбһ“ MCP бһ‘бҹ…вҖӢбһҖбһ¶бһ“бҹӢ Agent Builder бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢвҖӢбһҖбһ¶бһҡвҖӢбһҸбҹҒбһҹбҹ’бһҸ бһ“бһ·бһ„вҖӢбһ…бһ¶бһ”бҹӢбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷвҖӢбһҖбҹҶбһ бһ»бһҹбҹ”
- **бһ…бһ¶бһ”бҹӢбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷвҖӢбһҖбҹ’бһ“бһ»бһ„ [MCP Inspector](https://github.com/modelcontextprotocol/inspector)**бҹ– бһҳбһ»бһҒбһ„бһ¶бһҡвҖӢбһҠбҹӮбһӣвҖӢбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸвҖӢбһІбҹ’бһҷвҖӢбһўбҹ’бһ“бһҖвҖӢбһ…бһ¶бһ”бҹӢбһҸбҹ’бһҡбһҪбһҸвҖӢбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбҹүбһ¶бһҹбҹҠбһёбһ“ MCP бһҠбҹ„бһҷвҖӢбһ”бҹ’бһҡбһҫ MCP Inspectorбҹ”

## бһҡбҹҖбһ”бһ…бҹҶвҖӢбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹ

бһҳбһ¶бһ“вҖӢбһңбһ·бһ’бһёвҖӢбһ–бһёбһҡвҖӢбһҖбҹ’бһ“бһ»бһ„вҖӢбһҖбһ¶бһҡбһҡбҹҖбһ”бһ…бҹҶвҖӢбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹвҖӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӮбһҳбҹ’бһҡбҹ„бһ„вҖӢбһ“бҹҒбҹҮбҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҳбһҪбһҷбһҺбһ¶бһҳбһҪбһҷбһ•бҹ’бһўбҹӮбһҖвҖӢбһӣбһҫбһ…бҹҶбһҺбһ„бҹӢбһ…бҹҶбһҺбһјбһӣбһ…бһ·бһҸбҹ’бһҸвҖӢбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

> бһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбҹ– бһ”бһҫбһҖбһ”бһ„бҹ’бһ бһ¶бһүбһЎбһҫбһ„бһңбһ·бһү VSCode бһ¬ terminal бһҠбһҫбһҳбҹ’бһ”бһёвҖӢбһ’бҹ’бһңбһҫвҖӢбһұбҹ’бһҷвҖӢбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹ python бһңбҹӮбһҡбһҶбһңбһёбһҸбһ…бҹҶбһ–бҹ„бҹҮ virtual env бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“вҖӢбһ”бҹ’бһҡбһҫвҖӢбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢвҖӢбһ–бһёбһ”бһ¶бһ“вҖӢбһ”бһ„бҹ’бһҖбһҫбһҸвҖӢбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹ virtual envбҹ”

| бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡ | бһҮбҹҶбһ бһ¶бһ“ |
| -------- | ----- |
| бһ”бҹ’бһҡбһҫ `uv` | 1. бһ”бһ„бҹ’бһҖбһҫбһҸвҖӢбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹ virtual env: `uv venv` <br>2. бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһ¶ VSCode "***Python: Select Interpreter***" бһ бһҫбһҷбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ python бһ–бһёбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹ virtual env бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸ <br>3. бһҠбҹҶбһЎбһҫбһ„ dependency (бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ dev dependencies): `uv pip install -r pyproject.toml --extra dev` |
| бһ”бҹ’бһҡбһҫ `pip` | 1. бһ”бһ„бҹ’бһҖбһҫбһҸвҖӢбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹ virtual env: `python -m venv .venv` <br>2. бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһ¶ VSCode "***Python: Select Interpreter***" бһ бһҫбһҷбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ python бһ–бһёбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹ virtual env бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸ<br>3. бһҠбҹҶбһЎбһҫбһ„ dependency (бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ dev dependencies): `pip install -e .[dev]` | 

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһҡбҹҖбһ”бһ…бҹҶвҖӢбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһҡбһҪбһ… бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бһҫбһҖбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҳбҹүбһ¶бһҹбҹҠбһёбһ“вҖӢбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„вҖӢбһҖбҹ’бһҡбһ»бһҳбһ бҹҠбһ»бһ“вҖӢбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚвҖӢбһ•бҹ’бһ‘бһ¶бһӣбҹӢвҖӢбһҡбһ”бһҹбҹӢвҖӢбһўбҹ’бһ“бһҖбһҠбҹ„бһҷвҖӢбһ”бҹ’бһҡбһҫ Agent Builder бһҮбһ¶ MCP Client бһҠбһҫбһҳбҹ’бһ”бһёвҖӢбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбҹ–
1. бһ”бһҫбһҖвҖӢбһ•бҹ’бһ‘бһ¶бҹҶбһ„ Debug бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ VS Codeбҹ” бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ `Debug in Agent Builder` бһ¬бһ…бһ»бһ… `F5` бһҠбһҫбһҳбҹ’бһ”бһёвҖӢбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ debugging бһҳбҹүбһ¶бһҹбҹҠбһёбһ“ MCPбҹ”
2. бһ”бҹ’бһҡбһҫ AI Toolkit Agent Builder бһҠбһ¶бһҖбҹӢвҖӢбһҸбҹҒбһҹбҹ’бһҸвҖӢбһҳбҹүбһ¶бһҹбҹҠбһёбһ“вҖӢбһҮбһ¶бһҳбһҪбһҷ [prompt бһ“бҹҒбҹҮ](../../../../../../../../../../../open_prompt_builder)бҹ” бһҳбҹүбһ¶бһҹбҹҠбһёбһ“вҖӢбһ“бһ№бһ„вҖӢбһҸбҹ’бһҡбһјбһңбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҖбҹ’бһ“бһ»бһ„ Agent Builderбҹ”
3. бһ…бһ»бһ… `Run` бһҠбһҫбһҳбҹ’бһ”бһёвҖӢбһҸбҹҒбһҹбҹ’бһҸвҖӢбһҳбҹүбһ¶бһҹбҹҠбһёбһ“вҖӢбһҮбһ¶бһҳбһҪбһҷ prompt бһ“бҹҒбҹҮбҹ”
