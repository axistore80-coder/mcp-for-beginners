## бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ‘бһјбһ‘бҹ…

бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбһ•бҹ’бһҠбҹ„бһҸбһҹбҹҶбһҒбһ¶бһ“бҹӢбһӣбһҫбһңбһ·бһ’бһ¶бһ“бһҹбһҳбҹ’бһҡбһҪбһӣбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҒбҹ’бһ–бһҹбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚ, бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„, бһ“бһ·бһ„бһ…бҹӮбһҖбһ…бһ¶бһҷбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP бһ“бһ·бһ„бһҳбһ»бһҒбһ„бһ¶бһҡбһ“бһ¶бһ“бһ¶бһҖбҹ’бһ“бһ»бһ„бһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳбҹ” бһ“бҹ…бһ–бҹҒбһӣбһҠбҹӮбһӣбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһўбҹҒбһҖбһјвҖӢбһҹбҹҠбһёбһҹбҹ’бһ‘бһҳ MCP бһҖбһҫбһ“бһЎбһҫбһ„бһҠбҹ„бһҷбһҹбһ¶бһҡбһҮбһ¶бһҹбҹҶбһҒбһ¶бһ“бҹӢбһ“бһ·бһ„бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһү бһҖбһ¶бһҡбһҸбһ¶бһҳбһҠбһ¶бһ“бһӣбҹҶбһҠбһ¶бһ”бҹӢбһӣбҹҶбһҠбҹ„бһҷбһ“бһ№бһ„бһ’бһ¶бһ“бһ¶бһҗбһ¶бһҳбһ¶бһ“бһҖбһ¶бһҡбһ‘бһ»бһҖбһ…бһ·бһҸбҹ’бһҸбһ”бһ¶бһ“ бһ”бҹ’бһҡбҹӮбһ”бҹ’бһҡбһҪбһӣбһ”бһ¶бһ“бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣ бһ“бһ·бһ„бһўбһ¶бһ…бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҹбһ бһ”бҹ’бһҡбһҸбҹ’бһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһӮбҹ’бһ“бһ¶бҹ” бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбһ”бһ„бҹ’бһҡбһҪбһ”бһ”бһ„бҹ’бһҡбһҪбһҳбһ”бһүбҹ’бһүбһ¶бһҠбҹҸбһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„бһҠбҹӮбһӣбһ…бҹҶбһҺбһ¶бһҷбһ–бҹҒбһӣбһҷбҹүбһ¶бһ„бһ…бҹ’бһҡбһҫбһ“бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸ MCP бһҖбҹ’бһ“бһ»бһ„бһ–бһ·бһ—бһ–бһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„бһҠбһҫбһҳбҹ’бһ”бһёбһҺбҹӮбһ“бһ¶бҹҶбһўбҹ’бһ“бһҖбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒбһҡбһ№бһ„бһҳбһ¶бҹҶ бһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ– бһҮбһ¶бһҳбһҪбһҷбһ’бһ“бһ’бһ¶бһ“, бһҖбһ¶бһҡбһҮбҹҶбһҡбһ»бһү, бһ“бһ·бһ„бһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣбһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бҹ”

## бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҹбһ·бһҖбҹ’бһҹбһ¶

бһ“бҹ…бһ…бһ»бһ„бһ”бһүбҹ’бһ…бһ”бҹӢбһ“бҹғбһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһўбһ¶бһ…:

- бһ”бҹҶбһ–бһ¶бһ“бһңбһ·бһ“бҹҗбһҷбһ‘бҹҶбһ“бһҫбһ”бһ‘бһёбһ•бҹ’бһҹбһ¶бһҡбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҡбһ…бһ“бһ¶бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP бһ“бһ·бһ„бһҳбһ»бһҒбһ„бһ¶бһҡ
- бһ”бһ„бҹ’бһҖбһҫбһҸбһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһ–бҹҒбһүбһӣбҹҒбһүбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP
- бһҡбһ…бһ“бһ¶бһҡбһјбһ”бһҳбһ“бҹ’бһҸбһҖбһ¶бһҡбһ„бһ¶бһҡбһҠбҹӮбһӣбһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ– бһ“бһ·бһ„бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһЎбһҫбһ„бһңбһ·бһүбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё MCP бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһү
- бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһ¶бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҖбҹҶбһ бһ»бһҹ бһҖбһ¶бһҡбһ…бһ»бҹҮбһ”бһүбҹ’бһҮбһё бһ“бһ·бһ„бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP
- бһ”бһ„бҹ’бһҖбһҫбһҸбһўбһ“бһ»бһңбһҸбҹ’бһҸ MCP бһІбҹ’бһҷбһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҒбҹ’бһ–бһҹбҹӢ бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– бһ“бһ·бһ„бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҗбҹӮбһ‘бһ¶бҹҶ

## бһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚбһҹбҹ’бһ“бһјбһӣбһҡбһ”бһҹбҹӢ MCP

бһҳбһ»бһ“бһ–бҹҒбһӣбһ…бһјбһӣбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢ бһӮбһәбһҹбҹҶбһҒбһ¶бһ“бҹӢбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚбһҹбҹ’бһ“бһјбһӣбһҠбҹӮбһӣбһҺбҹӮбһ“бһ¶бҹҶбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбһ—бһ·бһңбһҢбҹ’бһҚ MCP бһҠбҹҸбһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–:

1. **бһҖбһ¶бһҡбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбһҮбһ¶бһҖбһ¶бһҡбһ”бҹ’бһҡбһҖбһ¶бһ“бҹӢ**: MCP бһ”бҹ’бһҡбһҫ JSON-RPC 2.0 бһҮбһ¶бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ бһ бһҫбһҷбһ•бҹ’бһҸбһӣбҹӢбһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷбһҹбҹ’бһҳбһҫбһҹбҹ’бһҳбһ¶бһүбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҹбҹ’бһ“бһҫбһҹбһ»бҹҶ бһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһҶбҹ’бһӣбһҫбһҷбһҸбһ” бһ“бһ·бһ„бһҖбһ¶бһҡбһҠбҹ„бҹҮбһҡбһҝбһ„бһҖбҹҶбһ бһ»бһҹбһӮбҹ’бһҡбһ”бҹӢбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбҹ”

2. **бһҖбһ¶бһҡбһҡбһ…бһ“бһ¶бһҠбҹӮбһӣбһ•бҹ’бһҠбҹ„бһҸбһӣбһҫбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ**: бһҸбҹӮбһ„бһҸбҹӮбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҖбһ¶бһҡбһҷбһӣбҹӢбһ–бҹ’бһҡбһҳ бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ бһ“бһ·бһ„бһ—бһ¶бһ–бһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢбһ…бҹҶбһ–бҹ„бҹҮбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸ MCP бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

3. **бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҮбһ¶бһ…бҹҶбһ”бһ„**: бһўбһ“бһ»бһңбһҸбҹ’бһҸбһңбһ·бһ’бһ¶бһ“бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҡбһ№бһ„бһҳбһ¶бҹҶ бһҡбһҪбһҳбһҳбһ¶бһ“бһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһўбһҸбҹ’бһҸбһҹбһүбҹ’бһүбһ¶бһҺ бһҖбһ¶бһҡбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸ бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ бһ“бһ·бһ„бһҖбҹҶбһҺбһҸбҹӢбһўбһҸбҹ’бһҡбһ¶бҹ”

4. **бһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳбһҳбҹүбһјбһҢбһ»бһӣ**: бһҡбһ…бһ“бһ¶бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҮбһ¶бһҳбҹүбһјбһҢбһ»бһӣ бһҠбҹӮбһӣбһ§бһ”бһҖбһҡбһҺбҹҚбһ“бһ·бһ„бһ’бһ“бһ’бһ¶бһ“бһ“бһёбһҳбһҪбһҷбҹ—бһҳбһ¶бһ“бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢ бһ“бһ·бһ„бһ•бҹ’бһҠбҹ„бһҸбһӣбһҫбҹ”

5. **бһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҳбһ¶бһ“бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–**: бһ”бҹ’бһҡбһҫбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ– MCP бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҡбһҖбҹ’бһҹбһ¶бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҹбҹ’бһ“бһҫбһҹбһ»бҹҶ бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ¶бһ“бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҠбҹӮбһӣбһҳбһ¶бһ“бһҖбһ¶бһҡбһҹбҹҠбһёбһҡбҹҮбһҗбҹӮбһҸбһ“бһ·бһ„бһҳбһ¶бһ“бһ”бһҡбһ·бһ”бһ‘бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бҹ”

## бһңбһ·бһ’бһ¶бһ“бһҹбһҳбҹ’бһҡбһҪбһӣбһўбһ—бһ·бһңбһҢбҹ’бһҚ MCP бһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸ

бһңбһ·бһ’бһ¶бһ“бһҹбһҳбҹ’бһҡбһҪбһӣбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҳбһ¶бһ“бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ–бһёбһҜбһҖбһҹбһ¶бһҡ Model Context Protocol бһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡбҹ–

### бһңбһ·бһ’бһ¶бһ“бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸ

1. **бһҖбһ¶бһҡбһҷбһӣбҹӢбһ–бҹ’бһҡбһҳ бһ“бһ·бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ**: бһҸбҹӮбһ„бһҸбҹӮбһ‘бһ¶бһҳбһ‘бһ¶бһҡбһҸбһ¶бһҳбһҖбһ¶бһҡбһҷбһӣбҹӢбһ–бҹ’бһҡбһҳбһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢбһ–бһёбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ бһҳбһ»бһ“бһ–бҹҒбһӣбһ…бһјбһӣбһ”бҹ’бһҡбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһ¬бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ“бҹҚбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚбҹ” бһ•бҹ’бһҠбһӣбҹӢбһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҗбһ¶бһҸбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҺбһ¶бһҠбҹӮбһӣбһ”бһ¶бһ“бһ…бҹӮбһҖбһҡбҹҶбһӣбҹӮбһҖ бһ“бһ·бһ„бһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһҺбһ¶бһҠбҹӮбһӣбһ”бһ¶бһ“бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбҹ”

2. **бһ—бһ¶бһ–бһҜбһҖбһҮбһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ**: бһ”бһ„бҹ’бһ бһ¶бһүбһҸбҹӮбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһҠбҹ„бһҷбһҳбһ¶бһ“бһҖбһ¶бһҡбһҷбһӣбҹӢбһ–бҹ’бһҡбһҳбһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢ бһ“бһ·бһ„бһҖбһ¶бһҡбһ–бһ¶бһҡбһҠбҹ„бһҷбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһ¶бһҡбһ…бһјбһӣбһ”бҹ’бһҡбһҫбһҠбҹӮбһӣбһҹбһҳбһҡбһҳбҹ’бһҷбҹ” бһҖбһ¶бһҡбһ–бһ¶бһҡбһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҒбһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹ„бһҷбһӮбҹ’бһҳбһ¶бһ“бһҖбһ¶бһҡбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбҹ”

3. **бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһ§бһ”бһҖбһҡбһҺбҹҚ**: бһҸбҹ’бһҡбһјбһңбһ‘бһ¶бһҳбһ‘бһ¶бһҡбһҮбһ¶бһҖбһ¶бһҡбһҷбһӣбҹӢбһ–бҹ’бһҡбһҳбһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢбһ–бһёбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһҳбһ»бһ“бһ бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚбһҺбһ¶бһҳбһҪбһҷбҹ” бһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһҷбһӣбҹӢвҖӢбһўбҹҶбһ–бһёбһҳбһ»бһҒбһ„бһ¶бһҡбһ“бҹғбһ§бһ”бһҖбһҡбһҺбҹҚбһ“бһёбһҳбһҪбһҷбҹ— бһ“бһ·бһ„бһўбһ“бһ»бһңбһҸбҹ’бһҸбһІбҹ’бһҷбһҳбһ¶бһ“бһ–бҹ’бһҡбҹҶбһҠбҹӮбһ“бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҡбһ№бһ„бһҳбһ¶бҹҶбҹ”

4. **бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбһ·бһ‘бҹ’бһ’бһ·бһ§бһ”бһҖбһҡбһҺбҹҚ**: бһҖбҹҶбһҺбһҸбҹӢбһҗбһ¶бһ§бһ”бһҖбһҡбһҺбҹҚбһҺбһ¶бһҠбҹӮбһӣбһҳбҹүбһјбһҢбҹӮбһӣбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бһ¶бһ“бһ“бҹ…бһ–бҹҒбһӣбһҹбһҳбҹҗбһҷбһ…бһјбһӣбһ”бҹ’бһҡбһҫ бһҠбһҫбһҳбҹ’бһ”бһёбһ’бһ¶бһ“бһ¶бһҗбһ¶бһӮбҹ’бһҳбһ¶бһ“бһ§бһ”бһҖбһҡбһҺбҹҚбһҺбһ¶бһҳбһҪбһҷбһҠбҹӮбһӣбһҒбһ»бһҹбһ…бҹ’бһ”бһ¶бһ”бҹӢбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ…бһјбһӣбһ”бҹ’бһҡбһҫбҹ”

5. **бһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһўбһҸбҹ’бһҸбһҹбһүбҹ’бһүбһ¶бһҺ**: бһ‘бһ¶бһҳбһ‘бһ¶бһҡбһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң бһҳбһ»бһ“бһҖбһ¶бһҡбһ…бһјбһӣбһ”бҹ’бһҡбһҫбһ§бһ”бһҖбһҡбһҺбҹҚ, бһ’бһ“бһ’бһ¶бһ“ бһ¬бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ— бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһҖбһјбһ“бһҹбҹ„ API, бһ§бһ”бһҖбһҡбһҺбҹҚ OAuth, бһ¬бһңбһ·бһ’бһёбһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бҹ”

6. **бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡ**: бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢ бһҠбһҫбһҳбҹ’бһ”бһёбһ‘бһ”бҹӢбһҹбҹ’бһҖбһ¶бһҸбҹӢбһ“бһјбһңбһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһӣбһҒбһ»бһҹбһ”бҹҶбһ–бһ¶бһ“ бһ¬бһӮбҹҶбһҡбһ¶бһҳбһҖбҹҶбһ бҹӮбһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһҳбһҖбһҖбһ¶бһ“бҹӢбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ§бһ”бһҖбһҡбһҺбҹҚбҹ”

7. **бһҖбҹҶбһҺбһҸбҹӢбһўбһҸбҹ’бһҡбһ¶бһҹбҹ’бһ“бһҫбһҹбһ»бҹҶ**: бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбҹҶбһҺбһҸбҹӢбһўбһҸбҹ’бһҡбһ¶бһҠбһҫбһҳбҹ’бһ”бһёбһ‘бһ”бҹӢбһҹбҹ’бһҖбһ¶бһҸбҹӢбһҖбһ¶бһҡбһ”бҹҶбһ–бһ¶бһ“ бһ“бһ·бһ„бһ’бһ¶бһ“бһ¶бһ“бһјбһңбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ’бһ“бһ’бһ¶бһ“бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒбһҷбҹүбһ¶бһ„бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбҹ”

### бһңбһ·бһ’бһ¶бһ“бһўбһ“бһ»бһңбһҸбҹ’бһҸбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸ

1. **бһҖбһ¶бһҡбһҮбһҮбҹӮбһҖбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–**: бһ“бҹ…бһ–бҹҒбһӣбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢ бһ”бҹ’бһҸбһјбһҡбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһўбҹҶбһ–бһёбһҳбһ»бһҒбһ„бһ¶бһҡбһӮбһ¶бҹҶбһ‘бҹ’бһҡ, бһҖбҹҶбһҺбҹӮбһ–бһ·бһ’бһёбһҖбһ¶бһҡ, бһ§бһ”бһҖбһҡбһҺбҹҚ бһ“бһ·бһ„бһ’бһ“бһ’бһ¶бһ“бһҠбҹӮбһӣбһҳбһ¶бһ“бҹ”

2. **бһҖбһ¶бһҡбһҡбһ…бһ“бһ¶бһ§бһ”бһҖбһҡбһҺбҹҚ**: бһ”бһ„бҹ’бһҖбһҫбһҸбһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣбһ•бҹ’бһҠбҹ„бһҸбһӣбһҫбһҳбһ»бһҒбһ„бһ¶бһҡбһҸбҹӮбһҳбһҪбһҷбһӣбҹ’бһў бһҮбҹҖбһҹбһңбһ¶бһ„бһ§бһ”бһҖбһҡбһҺбҹҚбһҳбҹүбһјбһҺбһјбһӣбһ»бһҷбһҗбһ·бһ…бһҠбҹӮбһӣбһҡбҹҖбһ”бһ…бҹҶбһ”бһүбҹ’бһ бһ¶бһңбһ·бһ…бһ·бһҸбҹ’бһҡбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһ…бҹ’бһҡбһҫбһ“бҹ”

3. **бһҖбһ¶бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҖбҹҶбһ бһ»бһҹ**: бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҹбһ¶бһҡбһ”бһүбҹ’бһҶбҹ„бһҸбһҖбҹҶбһ бһ»бһҹ бһ“бһ·бһ„бһӣбҹҒбһҒбһҖбһјбһҠбһҠбҹӮбһӣбһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶ бһҠбһҫбһҳбҹ’бһ”бһёбһҮбһҪбһҷбһІбҹ’бһҷбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһ”бһүбҹ’бһ бһ¶ бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҖбһҡбһҺбһёбһ”бһҡбһ¶бһҮбҹҗбһҷбһҷбҹүбһ¶бһ„бһ‘бһҳбҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷ бһ“бһ·бһ„бһ•бҹ’бһҠбһӣбҹӢбһҳбһҸбһ·бһҷбҹ„бһ”бһӣбҹӢбһҠбҹӮбһӣбһўбһ¶бһ…бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ”бһ¶бһ“бҹ”

4. **бһҖбһ¶бһҡбһ…бһ»бҹҮбһ”бһүбҹ’бһҮбһё**: бһҖбҹҶбһҺбһҸбҹӢбһҖбһ¶бһҡбһ…бһ»бҹҮбһ”бһүбҹ’бһҮбһёбһҮбһ¶бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ бһҖбһ¶бһҡбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҖбҹҶбһ бһ»бһҹ бһ“бһ·бһ„бһҸбһ¶бһҳбһҠбһ¶бһ“бһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚбһ“бҹғбһ–бһ·бһ’бһёбһҖбһ¶бһҡбһ…бһјбһӣбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚбҹ”

#### 1. бһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚбһҖбһ¶бһҸбһ–бҹ’бһңбһҖбһ·бһ…бҹ’бһ…бһҸбҹӮбһҳбһҪбһҷ

бһ§бһ”бһҖбһҡбһҺбҹҚ MCP бһ“бһёбһҳбһҪбһҷбҹ—бһҸбҹ’бһҡбһјбһңбһҳбһ¶бһ“бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢбһ•бҹ’бһҠбҹ„бһҸбһӣбһҫбһҳбһ»бһҒбһ„бһ¶бһҡбһҸбҹӮбһҳбһҪбһҷбҹ” бһҳбһ·бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһ§бһ”бһҖбһҡбһҺбҹҚбһҳбҹүбһјбһҺбһјбһӣбһ»бһҷбһҗбһ·бһ…бһҠбҹӮбһӣбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶бһ…бҹ’бһҡбһҫбһ“бһҡбһҪбһҳбһӮбҹ’бһ“бһ¶бһ‘бҹҒ бһҸбҹ’бһҡбһјбһңбһ”бһ„бҹ’бһҖбһҫбһҸбһ§бһ”бһҖбһҡбһҺбҹҚбһ–бһ·бһҹбҹҒбһҹбһҠбҹӮбһӣбһўбһ¶бһ…бһ”бҹҶбһ–бҹҒбһүбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбһ”бһ¶бһ“бһӣбҹ’бһўбҹ”

```csharp
// A focused tool that does one thing well
public class WeatherForecastTool : ITool
{
    private readonly IWeatherService _weatherService;
    
    public WeatherForecastTool(IWeatherService weatherService)
    {
        _weatherService = weatherService;
    }
    
    public string Name => "weatherForecast";
    public string Description => "Gets weather forecast for a specific location";
    
    public ToolDefinition GetDefinition()
    {
        return new ToolDefinition
        {
            Name = Name,
            Description = Description,
            Parameters = new Dictionary<string, ParameterDefinition>
            {
                ["location"] = new ParameterDefinition
                {
                    Type = ParameterType.String,
                    Description = "City or location name"
                },
                ["days"] = new ParameterDefinition
                {
                    Type = ParameterType.Integer,
                    Description = "Number of forecast days",
                    Default = 3
                }
            },
            Required = new[] { "location" }
        };
    }
    
    public async Task<ToolResponse> ExecuteAsync(IDictionary<string, object> parameters)
    {
        var location = parameters["location"].ToString();
        var days = parameters.ContainsKey("days") 
            ? Convert.ToInt32(parameters["days"]) 
            : 3;
            
        var forecast = await _weatherService.GetForecastAsync(location, days);
        
        return new ToolResponse
        {
            Content = new List<ContentItem>
            {
                new TextContent(JsonSerializer.Serialize(forecast))
            }
        };
    }
}
```

```python
            self._log_error("Unexpected error in DataQueryTool", e)
            raise ToolExecutionError(f"An unexpected error occurred: {str(e)}")
```

```java
public class SecureDataAccessTool implements Tool {
    private final AuthenticationService authService;
    private final AuthorizationService authzService;
    private final DataService dataService;
    
    // бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһӣбһўбһ¶бһҹбҹ’бһҡбҹҗбһҷбһ—бһ¶бһ–
    public SecureDataAccessTool(
            AuthenticationService authService,
            AuthorizationService authzService,
            DataService dataService) {
        this.authService = authService;
        this.authzService = authzService;
        this.dataService = dataService;
    }
    
    @Override
    public String getName() {
        return "secureDataAccess";
    }
    
    @Override
    public ToolResponse execute(ToolRequest request) {
        // 1. бһҠбһҖбһҷбһҖбһ”бһҡбһ·бһ”бһ‘бһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢ
        String authToken = request.getContext().getAuthToken();
        
        // 2. бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ
        UserIdentity user;
        try {
            user = authService.validateToken(authToken);
        } catch (AuthenticationException e) {
            return ToolResponse.error("Authentication failed: " + e.getMessage());
        }
        
        // 3. бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбҹ’бһҡбһ»бһҳбһ бҹҠбһ»бһ“бһўбһ¶бһҮбҹ’бһүбһ¶бһ”бҹҗбһҺбҹ’бһҺбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡ бһ–бһ·бһҹбҹҒбһҹ
        String dataId = request.getParameters().get("dataId").getAsString();
        String operation = request.getParameters().get("operation").getAsString();
        
        boolean isAuthorized = authzService.isAuthorized(user, "data:" + dataId, operation);
        if (!isAuthorized) {
            return ToolResponse.error("Access denied: Insufficient permissions for this operation");
        }
        
        // 4. бһ”бһ“бҹ’бһҸбһҠбҹ„бһҷбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҠбҹӮбһӣбһҳбһ¶бһ“бһўбһ¶бһҮбҹ’бһүбһ¶бһ”бҹҗбһҺбҹ’бһҺ
        try {
            switch (operation) {
                case "read":
                    Object data = dataService.getData(dataId, user.getId());
                    return ToolResponse.success(data);
                case "update":
                    JsonNode newData = request.getParameters().get("newData");
                    dataService.updateData(dataId, newData, user.getId());
                    return ToolResponse.success("Data updated successfully");
                default:
                    return ToolResponse.error("Unsupported operation: " + operation);
            }
        } catch (Exception e) {
            return ToolResponse.error("Operation failed: " + e.getMessage());
        }
    }
}
```

```csharp
public class RateLimitingMiddleware
{
    private readonly RequestDelegate _next;
    private readonly IMemoryCache _cache;
    private readonly ILogger<RateLimitingMiddleware> _logger;
    
    // Configuration options
    private readonly int _maxRequestsPerMinute;
    
    public RateLimitingMiddleware(
        RequestDelegate next,
        IMemoryCache cache,
        ILogger<RateLimitingMiddleware> logger,
        IConfiguration config)
    {
        _next = next;
        _cache = cache;
        _logger = logger;
        _maxRequestsPerMinute = config.GetValue<int>("RateLimit:MaxRequestsPerMinute", 60);
    }
    
    public async Task InvokeAsync(HttpContext context)
    {
        // 1. Get client identifier (API key or user ID)
        string clientId = GetClientIdentifier(context);
        
        // 2. Get rate limiting key for this minute
        string cacheKey = $"rate_limit:{clientId}:{DateTime.UtcNow:yyyyMMddHHmm}";
        
        // 3. Check current request count
        if (!_cache.TryGetValue(cacheKey, out int requestCount))
        {
            requestCount = 0;
        }
        
        // 4. Enforce rate limit
        if (requestCount >= _maxRequestsPerMinute)
        {
            _logger.LogWarning("Rate limit exceeded for client {ClientId}", clientId);
            
            context.Response.StatusCode = StatusCodes.Status429TooManyRequests;
            context.Response.Headers.Add("Retry-After", "60");
            
            await context.Response.WriteAsJsonAsync(new

                error = "Rate limit exceeded",
                message = "Too many requests. Please try again later.",
                retryAfterSeconds = 60
            });

        // 5. Increment request count
        _cache.Set(cacheKey, requestCount + 1, TimeSpan.FromMinutes(2));
        
        // 6. Add rate limit headers
        context.Response.Headers.Add("X-RateLimit-Limit", _maxRequestsPerMinute.ToString());
        context.Response.Headers.Add("X-RateLimit-Remaining", (_maxRequestsPerMinute - requestCount - 1).ToString());
        
        // 7. Continue with the request
        await _next(context);
    }
```

```csharp
// C# example with caching
public class CachedWeatherTool : ITool
{
    private readonly IWeatherService _weatherService;
    private readonly IDistributedCache _cache;
    private readonly ILogger<CachedWeatherTool> _logger;
    
    public CachedWeatherTool(
        IWeatherService weatherService,
        IDistributedCache cache,
        ILogger<CachedWeatherTool> logger)
    {
        _weatherService = weatherService;
        _cache = cache;
        _logger = logger;
    }
    
    public string Name => "weatherForecast";
    
    public async Task<ToolResponse> ExecuteAsync(IDictionary<string, object> parameters)
    {
        var location = parameters["location"].ToString();
        var days = Convert.ToInt32(parameters.GetValueOrDefault("days", 3));
        
        // Create cache key
        string cacheKey = $"weather:{location}:{days}";
        
        // Try to get from cache
        string cachedForecast = await _cache.GetStringAsync(cacheKey);
        if (!string.IsNullOrEmpty(cachedForecast))
        {
            _logger.LogInformation("Cache hit for weather forecast: {Location}", location);
            return new ToolResponse
            {
                Content = new List<ContentItem>
                {
                    new TextContent(cachedForecast)
                }
            };
        }
        
        // Cache miss - get from service
        _logger.LogInformation("Cache miss for weather forecast: {Location}", location);
        var forecast = await _weatherService.GetForecastAsync(location, days);
        string forecastJson = JsonSerializer.Serialize(forecast);
        
        // Store in cache (weather forecasts valid for 1 hour)
        await _cache.SetStringAsync(
            cacheKey,
            forecastJson,
            new DistributedCacheEntryOptions
            {
                AbsoluteExpirationRelativeToNow = TimeSpan.FromHours(1)
            });
        
        return new ToolResponse
        {
            Content = new List<ContentItem>
            {
                new TextContent(forecastJson)
            }
        };
    }
}
```

```python
class TokenBucketRateLimiter:
    def __init__(self, tokens_per_second, bucket_size):
        self.tokens_per_second = tokens_per_second
        self.bucket_size = bucket_size
        self.tokens = bucket_size
        self.last_refill = time.time()
        self.lock = asyncio.Lock()
    
    async def get_delay_time(self):
        async with self.lock:
            self._refill()
            if self.tokens >= 1:
                return 0
            
            # бһӮбһҺбһ“бһ¶бһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһҡбһ бһјбһҸбһҠбһӣбҹӢбһҹбҹ’бһӣбһ¶бһ”бһ–бҹ’бһҡбһ·бһӣбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһ¶бһ“
            return (1 - self.tokens) / self.tokens_per_second
    
    async def consume(self):
        async with self.lock:
            self._refill()
            self.tokens -= 1
    
    def _refill(self):
        now = time.time()
        elapsed = now - self.last_refill
        
        # бһ”бһ“бҹ’бһҗбҹӮбһҳбһҹбҹ’бһӣбһ¶бһ”бһ–бҹ’бһҡбһ·бһӣбһҗбҹ’бһҳбһёбһўбһ¶бһҹбҹ’бһҡбҹҗбһҷбһӣбһҫбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһҠбҹӮбһӣбһҖбһ“бҹ’бһӣбһ„бһ•бһ»бһҸ
        new_tokens = elapsed * self.tokens_per_second
        self.tokens = min(self.bucket_size, self.tokens + new_tokens)
        self.last_refill = now
```

```python
class SecureDataTool(Tool):
    def get_schema(self):
        return {
            "type": "object",
            "properties": {
                "userId": {"type": "string"},
                "includeSensitiveData": {"type": "boolean", "default": False}
            },
            "required": ["userId"]
        }
    
    async def execute_async(self, request):
        user_id = request.parameters["userId"]
        include_sensitive = request.parameters.get("includeSensitiveData", False)
        
        # бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ
        user_data = await self.user_service.get_user_data(user_id)
        
        # бһҡбҹҶбһӣбһ„бһңбһ¶бһӣбһ—бһ¶бһӮбһ…бҹ’бһҡбһҫбһ“бһҠбҹӮбһӣбһҳбһ¶бһ“бһўбһ¶бһҡбһҖбҹ’бһҹбһ¶ бһӣбһ»бҹҮбһҸбҹ’бһҡбһ¶бһҸбҹӮбһҳбһ¶бһ“бһҖбһ¶бһҡбһҹбҹ’бһ“бһҫбһҹбһ»бҹҶ бһ“бһ·бһ„бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢ
        if not include_sensitive or not self._is_authorized_for_sensitive_data(request):
            user_data = self._redact_sensitive_fields(user_data)
        
        return ToolResponse(result=user_data)
    
    def _is_authorized_for_sensitive_data(self, request):
        # бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбһҳбҹ’бһҡбһ·бһҸбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһҖбҹ’бһ“бһ»бһ„бһ”бһҡбһ·бһ”бһ‘бһҹбҹҶбһҺбһҫ
        auth_level = request.context.get("authorizationLevel")
        return auth_level == "admin"
    
    def _redact_sensitive_fields(self, user_data):
        # бһ”бһ„бҹ’бһҖбһҫбһҸбһ…бҹ’бһ”бһ¶бһ”бҹӢбһ…бҹҶбһӣбһ„бһҠбһҫбһҳбҹ’бһ”бһёбһҮбҹҖбһҹбһңбһ¶бһ„бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҠбһҫбһҳ
        redacted = user_data.copy()
        
        # бһ”бһ·бһ‘бһ”бһ¶бҹҶбһ„бһңбһ¶бһӣбһҠбҹӮбһӣбһҳбһ¶бһ“бһўбһ¶бһҡбһҖбҹ’бһҹбһ¶бһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢ
        sensitive_fields = ["ssn", "creditCardNumber", "password"]
        for field in sensitive_fields:
            if field in redacted:
                redacted[field] = "REDACTED"
        
        # бһ”бһ·бһ‘бһ”бһ¶бҹҶбһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһўбһ¶бһҡбһҖбҹ’бһҹбһҠбҹӮбһӣбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ nested
        if "financialInfo" in redacted:
            redacted["financialInfo"] = {"available": True, "accessRestricted": True}
        
        return redacted
```

```python
    @server.router.get("/metrics")
```

бһ—бҹ’бһҮбһ¶бһ”бҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҮбһ¶бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҠбҹҸбһӣбҹҶбһҡбһ¶бҹҶ бһҠбҹӮбһӣбһӣбһ‘бҹ’бһ’бһ•бһӣбһ“бҹғбһ§бһ”бһҖбһҡбһҺбҹҚбһҳбһҪбһҷбһҖбҹ’бһӣбһ¶бһҷбһҮбһ¶бһ”бһүбҹ’бһ…бһјбһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбһҪбһҷбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбҹ–

```python
# бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҹбһ бһ”бһ‘бһҠбҹ’бһӢбһ¶бһ“бһ§бһ”бһҖбһҡбһҺбҹҚ Python
class ChainWorkflow:
    def __init__(self, tools_chain):
        self.tools_chain = tools_chain  # бһ”бһүбҹ’бһҮбһёбһҲбҹ’бһҳбҹ„бҹҮбһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ‘бҹ…бһӣбһҫбһӣбҹҶбһҠбһ¶бһ”бҹӢ
    
    async def execute(self, mcp_client, initial_input):
        current_result = initial_input
        all_results = {"input": initial_input}
        
        for tool_name in self.tools_chain:
            # бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ§бһ”бһҖбһҡбһҺбҹҚбһ“бһ·бһҳбһҪбһҷбҹ—бһҖбҹ’бһ“бһ»бһ„бһҹбһ бһ”бһ‘бһҠбҹ’бһӢбһ¶бһ“ бһ”бһүбҹ’бһҮбһјбһ“бһӣбһ‘бҹ’бһ’бһ•бһӣбһҳбһ»бһ“
            response = await mcp_client.execute_tool(tool_name, current_result)
            
            # бһҡбһҖбҹ’бһҹбһ¶вҖӢбһӣбһ‘бҹ’бһ’бһ•бһӣ бһ бһҫбһҷбһ”бҹ’бһҡбһҫбһҮбһ¶бһҠбһҫбһҳбһ…бҹҒбһүбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢ
            all_results[tool_name] = response.result
            current_result = response.result
        
        return {
            "final_result": current_result,
            "all_results": all_results
        }

# бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ
data_processing_chain = ChainWorkflow([
    "dataFetch",
    "dataCleaner",
    "dataAnalyzer",
    "dataVisualizer"
])

result = await data_processing_chain.execute(
    mcp_client,
    {"source": "sales_database", "table": "transactions"}
)
```

### 2. бһҡбһјбһ”бһҳбһ“бҹ’бһҸбһҡбҹҖбһ”бһ…бҹҶбһ•бҹ’бһҹбһ–бҹ’бһңбһ•бҹ’бһҹбһ¶бһҷ

бһ”бҹ’бһҡбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһҖбһҺбҹ’бһҠбһ¶бһӣ бһҠбҹӮбһӣбһ”бһүбҹ’бһҮбһјбһ“бһ‘бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚбһ–бһ·бһҹбҹҒбһҹбһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһ”бһүбҹ’бһ…бһјбһӣбҹ–

```csharp
public class ContentDispatcherTool : IMcpTool
{
    private readonly IMcpClient _mcpClient;
    
    public ContentDispatcherTool(IMcpClient mcpClient)
    {
        _mcpClient = mcpClient;
    }
    
    public string Name => "contentProcessor";
    public string Description => "Processes content of various types";
    
    public object GetSchema()
    {
        return new {
            type = "object",
            properties = new {
                content = new { type = "string" },
                contentType = new { 
                    type = "string",
                    enum = new[] { "text", "html", "markdown", "csv", "code" }
                },
                operation = new { 
                    type = "string",
                    enum = new[] { "summarize", "analyze", "extract", "convert" }
                }
            },
            required = new[] { "content", "contentType", "operation" }
        };
    }
    
    public async Task<ToolResponse> ExecuteAsync(ToolRequest request)
    {
        var content = request.Parameters.GetProperty("content").GetString();
        var contentType = request.Parameters.GetProperty("contentType").GetString();
        var operation = request.Parameters.GetProperty("operation").GetString();
        
        // Determine which specialized tool to use
        string targetTool = DetermineTargetTool(contentType, operation);
        
        // Forward to the specialized tool
        var specializedResponse = await _mcpClient.ExecuteToolAsync(
            targetTool,
            new { content, options = GetOptionsForTool(targetTool, operation) }
        );
        
        return new ToolResponse { Result = specializedResponse.Result };
    }
    
    private string DetermineTargetTool(string contentType, string operation)
    {
        return (contentType, operation) switch
        {
            ("text", "summarize") => "textSummarizer",
            ("text", "analyze") => "textAnalyzer",
            ("html", _) => "htmlProcessor",
            ("markdown", _) => "markdownProcessor",
            ("csv", _) => "csvProcessor",
            ("code", _) => "codeAnalyzer",
            _ => throw new ToolExecutionException($"No tool available for {contentType}/{operation}")
        };
    }
    
    private object GetOptionsForTool(string toolName, string operation)
    {
        // Return appropriate options for each specialized tool
        return toolName switch
        {
            "textSummarizer" => new { length = "medium" },
            "htmlProcessor" => new { cleanUp = true, operation },
            // Options for other tools...
            _ => new { }
        };
    }
}
```

```java
    public WorkflowResult execute(String datasetId) {
        // бһҮбҹҶбһ бҹҠбһ¶бһ“ бҹЎбҹ– бһ‘бһ¶бһүбһҷбһҖбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһӣбһҳбҹ’бһўбһ·бһҸбһ“бҹғбһҹбҹҶбһҺбһ»бҹҶбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ (бһҮбһ¶бһҳбһҪбһҷбһӮбҹ’бһ“бһ¶)

        Map<String, Object> combinedResults = new HashMap<>();
        combinedResults.put("metadata", metadataResponse.getResult());
        combinedResults.put("statistics", statisticalAnalysis.join().getResult());
        combinedResults.put("correlations", correlationAnalysis.join().getResult());

    }
```

```python
class ResilientWorkflow:
    def __init__(self, mcp_client):
        self.client = mcp_client
    
    async def execute_with_fallback(self, primary_tool, fallback_tool, parameters):
        try:
            # бһ–бҹ’бһҷбһ¶бһҷбһ¶бһҳбһ§бһ”бһҖбһҡбһҺбҹҚбһҹбҹҶбһҒбһ¶бһ“бҹӢбһҮбһ¶бһҠбҹҶбһ”бһјбһ„
            response = await self.client.execute_tool(primary_tool, parameters)
            return {
                "result": response.result,
                "source": "primary",
                "tool": primary_tool
            }
        except ToolExecutionException as e:
            # бһҖбһҸбҹӢбһҸбҹ’бһҡбһ¶бһҖбһ¶бһҡбһ”бһҡбһ¶бһҮбҹҗбһҷ
            logging.warning(f"Primary tool '{primary_tool}' failed: {str(e)}")
            
            # бһҸбҹ’бһҡбһЎбһ”бҹӢбһ‘бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚбһҮбҹҶбһ“бһҪбһҷ
            try:
                # бһ”бҹ’бһҡбһ бҹӮбһӣбһҮбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһҳбҹ’бһӣбҹӮбһ„бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбһҸбҹ’бһҡбһЎбһ”бҹӢбһҖбҹ’бһҡбҹ„бһҷ
                fallback_params = self._adapt_parameters(parameters, primary_tool, fallback_tool)
                
                response = await self.client.execute_tool(fallback_tool, fallback_params)
                return {
                    "result": response.result,
                    "source": "fallback",
                    "tool": fallback_tool,
                    "primaryError": str(e)
                }
            except ToolExecutionException as fallback_error:
                # бһ§бһ”бһҖбһҡбһҺбҹҚбһ‘бһ¶бҹҶбһ„бһ–бһёбһҡбһ”бһҡбһ¶бһҮбҹҗбһҷ
                logging.error(f"Both primary and fallback tools failed. Fallback error: {str(fallback_error)}")
                raise WorkflowExecutionException(
                    f"Workflow failed: primary error: {str(e)}; fallback error: {str(fallback_error)}"
                )
    
    def _adapt_parameters(self, params, from_tool, to_tool):
        """Adapt parameters between different tools if needed"""
        # бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ“бҹҒбҹҮбһ“бһ№бһ„бһўбһ¶бһҹбҹ’бһҡбҹҗбһҷбһӣбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢ
        # бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ“бҹҒбҹҮ бһҷбһҫбһ„бһ“бһ№бһ„бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһ¶бһҹбҹӢбһҸбҹ’бһҡбһЎбһ”бҹӢбһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһҠбһҫбһҳ
        return params

# бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ
```

### 5. бһҡбһјбһ”бһҳбһ“бҹ’бһҸбһҠбһ¶бһҖбҹӢбһҮбһ¶бһҹбҹҶбһҺбһ»бҹҶбһ…бһҡбһ“бҹ’бһҸбһҖбһ¶бһҡбһ„бһ¶бһҡ

бһ”бһ„бҹ’бһҖбһҫбһҸбһ…бһҡбһ“бҹ’бһҸбһҖбһ¶бһҡбһ„бһ¶бһҡбһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбһҠбҹ„бһҷбһ”бһүбҹ’бһ…бһјбһӣбһ…бһҡбһ“бҹ’бһҸбһӣбҹҒбһҒбһ„бһ¶бһҷбҹ—бҹ–

```csharp
public class CompositeWorkflow : IWorkflow
{
    private readonly List<IWorkflow> _workflows;
    
    public CompositeWorkflow(IEnumerable<IWorkflow> workflows)
    {
        _workflows = new List<IWorkflow>(workflows);
    }
    
    public async Task<WorkflowResult> ExecuteAsync(WorkflowContext context)
    {
        var results = new Dictionary<string, object>();
        
        foreach (var workflow in _workflows)
        {
            var workflowResult = await workflow.ExecuteAsync(context);
            
            // Store each workflow's result
            results[workflow.Name] = workflowResult;
            
            // Update context with the result for the next workflow
            context = context.WithResult(workflow.Name, workflowResult);
        }
        
        return new WorkflowResult(results);
    }
    
    public string Name => "CompositeWorkflow";
    public string Description => "Executes multiple workflows in sequence";
}

// Example usage
var documentWorkflow = new CompositeWorkflow(new IWorkflow[] {
    new DocumentFetchWorkflow(),
    new DocumentProcessingWorkflow(),
    new InsightGenerationWorkflow(),
    new ReportGenerationWorkflow()
});

var result = await documentWorkflow.ExecuteAsync(new WorkflowContext {
    Parameters = new { documentId = "12345" }
});
```

# бһҖбһ¶бһҡбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP: бһңбһ·бһ’бһ¶бһ“бһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸ бһ“бһ·бһ„бһҠбҹҶбһҺбһ№бһ„бһӣбҹ’бһўбҹ—

## бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ‘бһјбһ‘бҹ…

бһҖбһ¶бһҡбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһӮбһәбһҮбһ¶бһ•бҹ’бһ“бҹӮбһҖбһҹбҹҶбһҒбһ¶бһ“бҹӢбһ“бҹғбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP бһҠбҹӮбһӣбһ‘бһ»бһҖбһ…бһ·бһҸбҹ’бһҸбһ”бһ¶бһ“ бһ“бһ·бһ„бһҳбһ¶бһ“бһӮбһ»бһҺбһ—бһ¶бһ–бһҒбҹ’бһ–бһҹбҹӢбҹ” бһҳбһӮбҹ’бһӮбһ»бһ‘бҹ’бһ‘бҹҒбһҹбһҖбҹҚбһ“бҹҒбҹҮбһ•бҹ’бһҠбһӣбҹӢбһ“бһјбһңбһңбһ·бһ’бһ¶бһ“бһҹбһҳбҹ’бһҡбһҪбһӣбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸ бһ“бһ·бһ„бһҠбҹҶбһҺбһ№бһ„бһӣбҹ’бһўбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҮбһ¶бһҳбһ·бһүбһҳбһҪбһҷбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢ бһҠбҹ„бһҷбһ…бһ¶бһ”бҹӢбһ–бһёбһҖбһ¶бһҡбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһӮбҹ’бһ“бһ¶ бһҠбһӣбҹӢбһҖбһ¶бһҡбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҡбһҪбһҳбһӮбҹ’бһ“бһ¶ бһ“бһ·бһ„бһ”бһүбҹ’бһ…бһ”бҹӢбһҠбҹ„бһҷбһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҷбҹүбһ¶бһ„бһ–бҹҒбһүбһӣбҹҒбһүбҹ”

## бһ бҹҒбһҸбһ»бһўбҹ’бһңбһёбһ”бһ¶бһ“бһҮбһ¶ бһҖбһ¶бһҡбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҳбһ¶бһ“бһҹбһ¶бһҡбҹҲбһҹбҹҶбһҒбһ¶бһ“бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP

бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP бһҳбһ¶бһ“бһҸбһҪбһ“бһ¶бһ‘бһёбһҮбһ¶бһҳбһ’бҹ’бһҷбһҳбһ—бһ¶бһҹбһ¶бһҡбһңбһ¶бһ„бһҳбҹүбһјбһҠбҹӮбһӣ AI бһ“бһ·бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһўбһҸбһ·бһҗбһ·бһҮбһ“бҹ” бһҖбһ¶бһҡбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҷбҹүбһ¶бһ„бһҳбҹүбһҸбҹӢбһ…бһҸбҹӢбһ’бһ¶бһ“бһ¶бһҗбһ¶:

- бһҳбһ¶бһ“бһҖбһ¶бһҡбһ‘бһ»бһҖбһ…бһ·бһҸбҹ’бһҸбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ
- бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҹбҹ’бһ“бһҫбһҹбһ»бҹҶ бһ“бһ·бһ„бһҶбҹ’бһӣбһҫбһҷбһҸбһ”
- бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҸбһ¶бһҳбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢ MCP бһҠбҹ„бһҷбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң
- бһҳбһ¶бһ“бһ—бһ¶бһ–бһҡбһ№бһ„бһҳбһ¶бҹҶбһҖбҹ’бһ“бһ»бһ„бһҖбһҡбһҺбһёбһ”бһҡбһ¶бһҮбҹҗбһҷ бһ“бһ·бһ„бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһўбһ»бһ…бһўбһ„бҹ’бһҡбһҪбһү
- бһҳбһ¶бһ“бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһ’бҹ’бһңбһҫбһҖбһ¶бһҡвҖӢбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһҖбҹ’бһҡбҹ„бһҳбһ”бһ“бҹ’бһ‘бһ»бһҖбһ•бҹ’бһҹбҹҒбһ„бҹ—бһӮбҹ’бһ“бһ¶

## бһҖбһ¶бһҡбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҠбҹ„бһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP

### бһҖбһ¶бһҡбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҠбҹ„бһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢ (бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“)

бһҖбһ¶бһҡбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҠбҹ„бһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһҗбһ¶ бһҹбһҳбһ¶бһҹбһ—бһ¶бһ–бһ“бһёбһҳбһҪбһҷбҹ—бһ“бҹғбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҠбҹ„бһҷбһҠбҹ„бһҷбһЎбҹӮбһҖбҹ”

#### бһўбҹ’бһңбһёбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„

1. **бһўбҹ’бһ“бһҖбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ’бһ“бһ’бһ¶бһ“**: бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһўбҹ’бһ“бһҖбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ’бһ“бһ’бһ¶бһ“бһ“бһёбһҳбһҪбһҷбҹ—бһҠбҹ„бһҷбһЎбҹӮбһҖ  
2. **бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ§бһ”бһҖбһҡбһҺбҹҚ**: бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһўбһ¶бһҖбһ”бҹ’бһ”бһҖбһ·бһҡбһ·бһҷбһ¶бһ§бһ”бһҖбһҡбһҺбҹҚбһҮбһ¶бһҳбһҪбһҷбһ”бһүбҹ’бһ…бһјбһӣбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“  
3. **бһӮбҹҶбһҡбһјбһҮбҹҶбһҡбһ»бһү**: бһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶бһӮбҹҶбһҡбһјбһҮбҹҶбһҡбһ»бһүбһ”бһ„бҹ’бһ бһ¶бһүбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң  
4. **бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷschema**: бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҷбһ»бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһ“бҹғбһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡ  
5. **бһҖбһ¶бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҖбҹҶбһ бһ»бһҹ**: бһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбһ¶бһҡбһҶбҹ’бһӣбһҫбһҷбһҸбһ”бһҖбҹҶбһ бһ»бһҹбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһүбҹ’бһ…бһјбһӣбһҳбһ·бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң

#### бһңбһ·бһ’бһ¶бһ“бһӣбҹ’бһўбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҠбҹ„бһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢ

бһҖбһ¶бһҡбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҖбһ¶бһҡбһ”бҹ’бһҡбһ¶бһҹбҹ’бһҡбҹҗбһҷбһ‘бһ¶бһҖбҹӢбһ‘бһ„бһҡбһңбһ¶бһ„бһҹбһҳбһ¶бһҹбһ—бһ¶бһ–бһ“бһ¶бһ“бһ¶бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

#### бһўбҹ’бһңбһёбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„

1. **бһҖбһ¶бһҡбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҠбһҫбһҳбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ**: бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҖбһ¶бһҡбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҠбһҫбһҳбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒбһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһ•бҹ’бһҹбҹҒбһ„бҹ—

3. **бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҹбҹ’бһ“бһҫбһҹбһ»бҹҶ**: бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҹбҹ’бһ“бһҫбһҹбһ»бҹҶ-бһҶбҹ’бһӣбһҫбһҷбһҸбһ”бһ–бҹҒбһүбһӣбҹҒбһү

## бһҖбһ¶бһҡбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–

```javascript
import http from 'k6/http';
import { check, sleep } from 'k6';

export const options = {
  vus: 10,  // бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ“бһ·бһҳбҹ’бһҳбһ·бһҸ бҹЎбҹ  бһ“бһ¶бһҖбҹӢ
  duration: '30s',

  });
  
  sleep(1);
}
```

7. **бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ”бһүбҹ’бһ бһ¶бһ”бһҺбҹ’бһҠбһ¶бһүбһ“бһ¶бһ“бһ¶**бҹ– бһҸбҹҒбһҹбҹ’бһҸбһ“бһјбһңбһҖбһ¶бһҡбһ”бҹ’бһҡбһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ“бҹ…бһҖбҹ’бһҡбҹ„бһҳбһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢбһ”бһҺбҹ’бһҠбһ¶бһүбһҒбҹ’бһҹбҹ„бһҷ
8. **бһҸбҹҒбһҹбҹ’бһҸбһҠбҹӮбһ“бһҖбҹҶбһҺбһҸбҹӢбһ’бһ“бһ’бһ¶бһ“**бҹ– бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһўбһ¶бһҖбһ”бҹ’бһ”бһҖбһ·бһҡбһ·бһҷбһ¶бһ–бҹҒбһӣбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһӮбҹ„бһӣбһҠбҹӮбһ“бһ¬бһҖбһҳбҹ’бһҡбһ·бһҸбһ…бҹҶбһ“бһҪбһ“

10. **бһҜбһҖбһҹбһ¶бһҡбһҖбһҡбһҺбһёбһҸбҹҒбһҹбҹ’бһҸ**бҹ– бһҡбһҖбҹ’бһҹбһ¶бһҜбһҖбһҹбһ¶бһҡбһҸбҹҒбһҹбҹ’бһҸбһҡбһ”бһҹбҹӢбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҸбҹҒбһҹбҹ’бһҸбһ”бһ“бҹ’бһҠ

## бһҖбҹҶбһ бһ»бһҹбһҮбһ¶бһүбһ№бһҖбһүбһ¶бһ”бҹӢбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҸбҹҒбһҹбҹ’бһҸ

- **бһҖбһ¶бһҡбһ‘бһ»бһҖбһ…бһ·бһҸбҹ’бһҸбһӣбһҫбһ•бҹ’бһӣбһјбһңбһҹбһ”бҹ’бһ”бһ¶бһҷбһҡбһёбһҖбһҡбһ¶бһҷбһ…бҹ’бһҡбһҫбһ“бһ–бҹҒбһҖ**бҹ– бһҸбҹ’бһҡбһјбһңбһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶бһҸбҹҒбһҹбҹ’бһҸбһҖбһҡбһҺбһёбһҖбҹҶбһ бһ»бһҹбһҷбҹүбһ¶бһ„бһӣбҹ’бһў
- **бһҳбһ·бһ“бһҷбһҖбһ…бһ·бһҸбҹ’бһҸбһ‘бһ»бһҖбһҠбһ¶бһҖбҹӢбһҖбһ¶бһҡбһҸбҹҒбһҹбҹ’бһҸбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–**бҹ– бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ…бҹҶбһҺбһ»бһ…бһҡбһ”бһ¶бҹҶбһ„бһҳбһ»бһ“бһ–бҹҒбһӣбһңбһ¶бһ”бҹүбҹҮбһ–бһ¶бһӣбҹӢбһҖбһ¶бһҡбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ
- **бһҸбҹҒбһҹбҹ’бһҸбһ“бҹ…бһ…бҹҶбһҺбһ»бһ…бһ•бҹ’бһҸбһ¶бһ…бҹӢбһҳбһ»бһҒбһҸбҹӮбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮ**бҹ– бһ”бһүбҹ’бһ…бһјбһӣбһҖбһ¶бһҡбһҸбҹҒбһҹбҹ’бһҸбһҜбһҖбһҸбҹ’бһҸбһ¶ бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ бһ“бһ·бһ„ E2E
- **бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҸбһ”бҹӢ API бһҳбһ·бһ“бһ–бҹҒбһүбһӣбҹҒбһү**бҹ– бһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶бһ…бҹҶбһҺбһ»бһ…бһ”бһүбҹ’бһ…бһ”бҹӢбһ“бһ·бһ„бһҳбһ»бһҒбһ„бһ¶бһҡбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҸбҹҒбһҹбҹ’бһҸ
- **бһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһҸбҹҒбһҹбҹ’бһҸбһҳбһ·бһ“бһҹбҹ’бһҡбһ”бһӮбҹ’бһ“бһ¶**бҹ– бһ”бҹ’бһҡбһҫ container бһҠбһҫбһҳбҹ’бһ”бһёбһ’бһ¶бһ“бһ¶бһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһҸбҹҒбһҹбҹ’бһҸбһҹбҹ’бһҡбһ”бһӮбҹ’бһ“бһ¶

## бһ”бһ‘бһ”бһүбҹ’бһүбһҸбҹ’бһҸбһ·

бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҸбҹҒбһҹбҹ’бһҸбһ‘бһјбһӣбҹҶбһ‘бһјбһӣбһ¶бһҷбһҮбһ¶бһ”бҹ’бһҡбһ—бҹҒбһ‘бһҹбҹҶбһҒбһ¶бһ“бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ MCP бһҠбҹӮбһӣбһ‘бһ»бһҖбһ…бһ·бһҸбҹ’бһҸбһ”бһ¶бһ“бһ“бһ·бһ„бһҳбһ¶бһ“бһӮбһ»бһҺбһ—бһ¶бһ–бһҒбҹ’бһ–бһҹбҹӢбҹ” бһҠбҹ„бһҷбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ“бҹҚбһҖбһ¶бһҡбһӣбҹ’бһўбһ“бһ·бһ„бһӮбһ“бҹ’бһӣбһ№бҹҮбһҠбҹӮбһӣбһ”бһ¶бһ“бһӣбһҳбҹ’бһўбһ·бһҸбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбһӮбҹ’бһӮбһ»бһ‘бҹҒбһҹбһҖбҹҚбһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ’бһ¶бһ“бһ¶бһҗбһ¶бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸ MCP бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһӮбһ»бһҺбһ—бһ¶бһ– бһҒбһӣбһ—бһ¶бһ– бһ“бһ·бһ„бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҒбҹ’бһ–бһҹбҹӢбһ”бҹҶбһ•бһ»бһҸбҹ”

## бһ…бҹҶбһҺбһ»бһ…бһҹбҹҶбһҒбһ¶бһ“бҹӢбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҷбһӣбҹӢбһғбһҫбһү

1. **бһҖбһ¶бһҡбһҡбһ…бһ“бһ¶бһ§бһ”бһҖбһҡбһҺбҹҚ**бҹ– бһ’бҹ’бһңбһҫбһҸбһ¶бһҳ PRINCIPLE бһҹбһ·бһ‘бҹ’бһ’бһҸбҹӮбһҳбһҪбһҷ, бһ”бҹ’бһҡбһҫ dependency injection, бһ“бһ·бһ„бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ
2. **бһҖбһ¶бһҡбһҡбһ…бһ“бһ¶бһҹбҹ’бһҖбһёбһҳбҹүбһ¶**бҹ– бһ”бһ„бҹ’бһҖбһҫбһҸбһҹбҹ’бһҖбһёбһҳбҹүбһ¶бһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢ бһҳбһ¶бһ“бһҜбһҖбһҹбһ¶бһҡбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢ бһ“бһ·бһ„бһҖбҹҶбһҺбһҸбҹӢбһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң
3. **бһҖбһ¶бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҖбҹҶбһ бһ»бһҹ**бҹ– бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһ¶бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҖбҹҶбһ бһ»бһҹбһҷбҹүбһ¶бһ„бһ‘бһ“бҹӢбһ—бҹ’бһӣбһ“бҹӢ, бһ…бһҳбҹ’бһӣбһҫбһҷбһҖбҹҶбһ бһ»бһҹбһҠбҹӮбһӣбһҳбһ¶бһ“бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ бһ“бһ·бһ„бһҷбһ¶бһ“бһҷбһ“бҹ’бһҸ retry
4. **бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–**бҹ– бһ”бҹ’бһҡбһҫ caching, бһҖбһ¶бһҡбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҷбҹүбһ¶бһ„бһҳбһ·бһ“бһ‘бһ¶бһ“бҹӢбһҹбҹҶбһҡбһ»бһ„, бһ“бһ·бһ„бһҖбһ¶бһҡбһҡбһ№бһҸбһ”бһ“бҹ’бһҸбһ№бһ„бһ’бһ“бһ’бһ¶бһ“
5. **бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–**бҹ– бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһӣбһҷбҹүбһ¶бһ„бһҳбҹүбһҸбҹӢбһ…бһҸбҹӢ, бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбһ¶бһҡбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸ бһ“бһ·бһ„бһҖбһ¶бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹӮбһӣбһҡбҹҶбһӣбҹ„бһ—
6. **бһҖбһ¶бһҡбһҸбҹҒбһҹбҹ’бһҸ**бҹ– бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһ¶бһҡбһҸбҹҒбһҹбҹ’бһҸбһҜбһҖбһҸбҹ’бһҸбһ¶ бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ бһ“бһ·бһ„бһ…бһ»бһ„бһ‘бҹ…бһ…бһ»бһ„бһҷбҹүбһ¶бһ„бһ‘бһјбһӣбҹҶбһ‘бһјбһӣбһ¶бһҷ
7. **бһӣбҹҶбһ“бһ¶бҹҶбһ’бҹ’бһңбһҫбһҖбһ¶бһҡ**бҹ– бһўбһ“бһ»бһңбһҸбҹ’бһҸбһӣбҹҶбһ“бһ¶бҹҶбһҠбһјбһ…бһҮбһ¶ бһҒбҹ’бһҹбҹӮбһҹбһ„бҹ’бһңбһ¶бһҖбҹӢ, бһўбҹ’бһ“бһҖбһ”бһүбҹ’бһҮбһјбһ“, бһ“бһ·бһ„бһҖбһ¶бһҡбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһӣбһҳбҹ’бһўбһ·бһҸ

## бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ“бҹҚ
