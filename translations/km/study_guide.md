# Model Context Protocol (MCP) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ - бһҳбһӮбҹ’бһӮбһ»бһ‘бҹҒбһҹбһҖбҹҚбһҹбһ·бһҖбҹ’бһҹбһ¶

бһҳбһӮбҹ’бһӮбһ»бһ‘бҹҒбһҹбһҖбҹҚбһҹбһ·бһҖбҹ’бһҹбһ¶бһ“бҹҒбҹҮбһ•бҹ’бһҸбһӣбҹӢбһ“бһјбһңбһҖбһ¶бһҡбһ–бһ·бһ—бһ¶бһҖбҹ’бһҹбһ¶бһўбҹҶбһ–бһёбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ бһ“бһ·бһ„бһҳбһ¶бһҸбһ·бһҖбһ¶бһ“бҹғбһғбҹ’бһӣбһ¶бҹҶбһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһңбһӮбҹ’бһӮбһҹбһ·бһҖбҹ’бһҹбһ¶ "Model Context Protocol (MCP) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ"бҹ” бһ”бҹ’бһҡбһҫбһҳбһӮбҹ’бһӮбһ»бһ‘бҹҒбһҹбһҖбҹҚбһ“бҹҒбҹҮбһҠбһҫбһҳбҹ’бһ”бһёбһҡбһ»бһҖбһҡбһҖбһғбҹ’бһӣбһ¶бҹҶбһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹ„бһҷбһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ– бһ“бһ·бһ„бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһўбһҸбҹ’бһҗбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһ–бһёбһ’бһ“бһ’бһ¶бһ“бһҠбҹӮбһӣбһҳбһ¶бһ“бҹ”

## бһ–бһ·бһ–бһҺбҹҢбһ“бһ¶бһғбҹ’бһӣбһ¶бҹҶбһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ

Model Context Protocol (MCP) бһӮбһәбһҮбһ¶бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҸбһҳбҹ’бһҡбһ„бҹӢбһҸбһ¶бһҳбһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбһ“бҹ’бһҸбһҡбһҖбһҳбҹ’бһҳбһҡбһңбһ¶бһ„бһҳбҹүбһјбһҢбҹӮбһӣ AI бһ“бһ·бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһўбһҸбһ·бһҗбһ·бһҮбһ“бҹ” бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһҠбҹҶбһ”бһјбһ„бһҠбҹ„бһҷ Anthropic, MCP бһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҗбҹӮбһ‘бһ¶бҹҶбһҠбҹ„бһҷбһҹбһ бһӮбһҳбһ“бҹҚ MCP бһңбһ·бһҸбһҗбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҸбһ¶бһҳбһҡбһҷбҹҲбһўбһ„бҹ’бһӮбһҖбһ¶бһҡбһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡбһӣбһҫ GitHubбҹ” бһғбҹ’бһӣбһ¶бҹҶбһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ“бҹҒбҹҮбһ•бҹ’бһҸбһӣбҹӢбһ“бһјбһңбһңбһӮбҹ’бһӮбһҹбһ·бһҖбҹ’бһҹбһ¶бһҡбһҪбһҳбһҮбһ¶бһҳбһҪбһҷбһӮбҹҶбһҡбһјбһҖбһјбһҠбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбһҮбһ¶бһ—бһ¶бһҹбһ¶ C#, Java, JavaScript, Python, бһ“бһ·бһ„ TypeScript бһҠбҹӮбһӣбһҳбһ¶бһ“бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚ AI, бһўбҹ’бһ“бһҖбһҡбһ…бһ“бһ¶бһ”бһ‘бһ”бһҳбҹ’бһӣбҹӮбһ„бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ бһ“бһ·бһ„бһңбһ·бһҹбҹ’бһңбһҖбһҡбһҗбҹ’бһ“бһ¶бҹҶбһ”бһ„бҹ’бһҖбҹҒбһёбһҸбҹ”

## бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһғбҹ’бһӣбһ¶бҹҶбһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ

бһғбҹ’бһӣбһ¶бҹҶбһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбҹҖбһ”бһ…бҹҶбһ‘бҹ…бһҮбһ¶бһ•бҹ’бһ“бҹӮбһҖбһ…бһҳбҹ’бһ”бһ„ бҹЎбҹЎ бһ•бҹ’бһ“бҹӮбһҖ бһҠбҹӮбһӣбһ“бһ·бһ…бһҳбҹ’бһҸбһ„бһҳбһҪбһҷбһ•бҹ’бһҠбҹ„бһҸбһӣбһҫбһ”бҹ’бһҡбһ’бһ¶бһ“бһ”бһ‘бһ•бҹ’бһҹбҹҒбһ„бҹ—бһ“бҹғ MCPбҹ–

1. **бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶ (00-Introduction/)**
   - бһ‘бһ·бһҠбҹ’бһӢбһ—бһ¶бһ–бһ‘бһјбһ‘бҹ…бһўбҹҶбһ–бһё Model Context Protocol
   - бһ бҹҒбһҸбһ»бһўбҹ’бһңбһёбһ”бһ¶бһ“бһҮбһ¶бһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбһҹбҹҶбһҒбһ¶бһ“бҹӢбһҖбҹ’бһ“бһ»бһ„бһ”бһҺбҹ’бһҠбһ¶бһү AI
   - бһҖбһҡбһҺбһёбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„ бһ“бһ·бһ„бһўбһҸбҹ’бһҗбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚ

2. **бһӮбҹҶбһ“бһ·бһҸбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ (01-CoreConcepts/)**
   - бһҡбҹҖбһ”бһ…бҹҶбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ•бҹ’бһ“бҹӮбһҖбһўбһҸбһ·бһҗбһ·бһҮбһ“-бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ•бҹ’бһ“бҹӮбһҖбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ
   - бһҹбһҳбһ¶бһҹбһ’бһ¶бһҸбһ»бһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—бһ“бҹғбһ–бһ·бһ’бһёбһҖбһ¶бһҡбһҺбҹҚ
   - бһ”бҹ’бһҡбһ—бҹҒбһ‘бһҹбһ¶бһҡвҖӢбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ MCP

3. **бһҹбһ“бҹ’бһҸбһ·бһҹбһ»бһҒ (02-Security/)**
   - бһӮбҹҶбһҡбһ¶бһҳбһҖбҹҶбһ бҹӮбһ„бһҹбһ“бҹ’бһҸбһ·бһҹбһ»бһҒбһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ•бҹ’бһўбҹӮбһҖбһӣбһҫ MCP
   - бһҮбҹҶбһ“бһ¶бһүбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҖбһ¶бһҡбһ–бһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ“бҹҚ
   - бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢ бһ“бһ·бһ„бһўбҹ„бһҷбһҹбһ·бһ‘бҹ’бһ’бһ·
   - **бһҜбһҖбһҹбһ¶бһҡбһҹбһ“бҹ’бһҸбһ·бһҹбһ»бһҒбһ‘бһјбһӣбҹҶбһ‘бһјбһӣбһ¶бһҷ**бҹ–
     - MCP Security Best Practices 2025
     - Azure Content Safety Implementation Guide
     - MCP Security Controls and Techniques
     - MCP Best Practices Quick Reference
   - **бһ”бҹ’бһҡбһ’бһ¶бһ“бһ”бһ‘бһҹбһ“бҹ’бһҸбһ·бһҹбһ»бһҒбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—**бҹ–
     - бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһӣбһҹбһҳбҹ’бһҡбһ„бҹӢ бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бҹҶбһ–бһ»бһӣбһ§бһ”бһҖбһҡбһҺбҹҚ
     - бһҖбһ¶бһҡбһ…бһ¶бһ”бҹӢбһҷбһҖбһҹбһҳбҹҗбһҷ бһ“бһ·бһ„бһ”бһүбҹ’бһ бһ¶бһўбҹ’бһ“бһҖбһҸбҹҶбһҺбһ¶бһ„бһ…бҹ’бһҡбһ”бһјбһҖбһ…бҹ’бһҡбһ”бһӣбҹӢ
     - бһҖбһ¶бһҡвҖӢбһҖбҹӮбһ”бҹ’бһҡбҹӮвҖӢбһҹбһҳбҹ’бһ—бһ¶бһҡбҹҲвҖӢбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷвҖӢбһҖбҹ’бһ“бһ»бһ„ token
     - бһҹбһ·бһ‘бҹ’бһ’бһ·бһ…бҹ’бһҡбһҫбһ“бһ–бҹҒбһҖ бһ“бһ·бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһ¶бһҡбһ…бһјбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ
     - бһҹбһ“бҹ’бһҸбһ·бһҹбһ»бһҒбһҒбҹ’бһҹбҹӮбһ•бҹ’бһӮбһҸбҹӢбһ•бҹ’бһӮбһ„бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҮбһ„бҹ’бһӮбһ„бҹӢ AI
     - бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ Microsoft Prompt Shields

4. **бһҖбһ¶бһҡбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ (03-GettingStarted/)**
   - бһҖбһ¶бһҡбһҸбҹҶбһЎбһҫбһ„бһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹ бһ“бһ·бһ„бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’
   - бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ MCP бһ“бһ·бһ„бһўбһҸбһ·бһҗбһ·бһҮбһ“бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“
   - бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҮбһ¶бһҳбһҪбһҷбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҳбһ¶бһ“бһҹбҹ’бһҡбһ¶бһ”бҹӢ
   - бһҡбһҪбһҳбһҳбһ¶бһ“бһ•бҹ’бһ“бҹӮбһҖбҹ–
     - бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫбһҠбҹҶбһ”бһјбһ„
     - бһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһўбһҸбһ·бһҗбһ·бһҮбһ“
     - бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһўбһҸбһ·бһҗбһ·бһҮбһ“ LLM

     - бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ AI Toolkit
     - бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢ
     - бһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚбһ…бҹӮбһҖбһ…бһ¶бһҷ

5. **бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ“бҹҚбһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„ (04-PracticalImplementation/)**
   - бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫ SDK бһҮбһ¶бһ‘бһјбһ‘бҹ…бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ—бһ¶бһҹбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ•бҹ’бһҹбҹҒбһ„бҹ—
   - бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бҹҒбһүбһҖбҹҶбһ бһ»бһҹ, бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҸбҹҒбһҹбҹ’бһҸ бһ“бһ·бһ„бһңбһ·бһ’бһ¶бһ“бһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢ
   - бһ”бһ„бҹ’бһҖбһҫбһҸбһ”бҹ’бһӣбһ„бҹӢбһ”бһүбҹ’бһ…бһјбһӣбһҮбһ¶бһҡбһҷбҹҲбһ‘бһјбһ‘бҹ… бһ“бһ·бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбһ»бһҒбһ„бһ¶бһҡ
   - бһӮбҹҶбһҡбһјбһӮбһҳбҹ’бһҡбҹ„бһ„бһҮбһ¶бһҳбһҪбһҷбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһўбһ“бһ»бһңбһҸбҹ’бһҸ

6. **бһ”бҹ’бһҡбһ’бһ¶бһ“бһ”бһ‘бһҮбһёбһңбһ·бһҸбһҒбҹ’бһ–бһҹбҹӢ (05-AdvancedTopics/)**
   - бһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹбһңбһ·бһҹбҹ’бһңбһҖбһҳбҹ’бһҳбһ”бһҡбһ·бһ”бһ‘
   - бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ Foundry
   - бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҮбһ¶бһ•бҹ’бһӣбһјбһңбһҳбһҪбһҷбһ…бҹ’бһҡбһҫбһ“бһӣбһҖбҹ’бһҒбһҺбҹҲ AI 
   - бһ”бһ„бҹ’бһ бһ¶бһү OAuth2 бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢ
   - бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ–бҹҒбһӣбһҸбҹӮбһҳбҹ’бһҸбһ„
   - бһҹбҹ’бһ‘бҹ’бһҡбһёбһҳбһ–бҹҒбһӣбһҸбҹӮбһҳбҹ’бһҸбһ„
   - бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ”бһҡбһ·бһ”бһ‘бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“
   - бһңбһ·бһ’бһёбһңбһ·бһүбһҮбһ¶бһҸбһ·
   - бһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹбһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢ
   - бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһӣбһ¶бһҷбһЎбҹҶ
   - бһҖбһ¶бһҡбһ–бһ·бһ…бһ¶бһҡбһҺбһ¶бһҹбһ“бҹ’бһҸбһ·бһҹбһ»бһҒ
   - бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҹбһ“бҹ’бһҸбһ·бһҹбһ»бһҒ Entra ID
   - бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһңбҹҒбһ”бһҹбһ¶бһҷ
   - бһҖбһ¶бһҡбһӮбһ·бһҸбһҠбҹ„бһҷбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ…бҹ’бһҡбһҫбһ“бһ”бҹ’бһҡбһҶбһ¶бҹҶбһ„бһӮбҹ’бһ“бһ¶ (бһӣбҹҶбһ“бһ¶бҹҶбһңбһ¶бһҷбһ”бҹ’бһҡбһ бһ¶бһҡ)

7. **бһҖбһ¶бһҡбһҡбһҪбһҳбһ…бҹҶбһҺбҹӮбһҖбһҹбһ бһӮбһҳбһ“бҹҚ (06-CommunityContributions/)**
   - бһҡбһ”бҹҖбһ”бһҡбһҪбһҳбһ…бҹҶбһҺбҹӮбһҖбһҖбһјбһҠ бһ“бһ·бһ„бһҜбһҖбһҹбһ¶бһҡ
   - бһҖбһ¶бһҡбһҹбһ бһҖбһ¶бһҡбһҸбһ¶бһҳ GitHub
   - бһҖбһ¶бһҡбһҖбҹӮбһӣбһҳбҹ’бһўбһҠбҹ„бһҷбһҹбһ бһӮбһҳбһ“бҹҚ бһ“бһ·бһ„бһҳбһҸбһ·бһҷбҹ„бһ”бһӣбҹӢ
   - бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһўбһҸбһ·бһҗбһ·бһҮбһ“ MCP бһҸбһ¶бһҳбһҡбһҷбҹҲ Claude Desktop, Cline, VSCode
   - бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ MCP бһ–бҹҒбһүбһ“бһ·бһҷбһҳ бһҡбһҪбһҳбһҳбһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһјбһ”бһ—бһ¶бһ–

8. **бһҳбҹҒбһҡбҹҖбһ“бһ–бһёбһҖбһ¶бһҡбһ‘бһ‘бһҪбһӣбһҷбһҖбһҠбҹҶбһ”бһјбһ„ (07-LessonsfromEarlyAdoption/)**
   - бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ“бҹҚбһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠ бһ“бһ·бһ„бһҡбһҝбһ„бһҮбҹ„бһӮбһҮбҹҗбһҷ
   - бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһ“бһ·бһ„бһ…бҹӮбһҖбһ…бһ¶бһҷбһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫ MCP
   - бһ“бһ·бһ“бҹ’бһ“бһ¶бһҖбһ¶бһҡбһ“бһ·бһ„бһ•бҹӮбһ“бһ‘бһёбһ•бҹ’бһӣбһјбһңбһўбһ“бһ¶бһӮбһҸ
   - **бһҳбһӮбҹ’бһӮбһ»бһ‘бҹҒбһҹбһҖбҹҚбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ Microsoft MCP**бҹ– бһҳбһӮбҹ’бһӮбһ»бһ‘бҹҒбһҹбһҖбҹҚбһ‘бһјбһӣбҹҶбһ‘бһјбһӣбһ¶бһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ Microsoft MCP бҹЎбҹ  бһӮбҹ’бһҡбһҝбһ„бһҡбһҪбһҳбһҳбһ¶бһ“бҹ–
     - Microsoft Learn Docs MCP Server
     - Azure MCP Server (бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҮбһ¶бһ„ бҹЎбҹҘ)
     - GitHub MCP Server
     - Azure DevOps MCP Server
     - MarkItDown MCP Server
     - SQL Server MCP Server
     - Playwright MCP Server
     - Dev Box MCP Server
     - Azure AI Foundry MCP Server
     - Microsoft 365 Agents Toolkit MCP Server

9. **бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸ (08-BestPractices/)**
   - бһҖбһ¶бһҡбһҸбҹҶбһЎбһҫбһ„бһҖбһҳбҹ’бһҡбһ·бһҸбһҖбһҳбҹ’бһҡбһ·бһҸ бһ“бһ·бһ„бһ”бһ“бҹ’бһҗбҹӮбһҳбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–
   - бһҖбһ¶бһҡбһҡбһ…бһ“бһ¶бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ MCP бһўбҹ„бһҷбһўбһ¶бһ…бһ’бһ“бҹӢбһ‘бҹ’бһҡбһ¶бҹҶбһ”бһҡбһ¶бһҮбҹҗбһҷ
   - бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ’бҹ’бһңбһҫбһҸбҹҒбһҹбҹ’бһҸ бһ“бһ·бһ„бһҖбҹҶбһҺбһҸбҹӢбһ—бһ¶бһ–бһ‘бһ“бҹӢбһҒбҹ’бһҹбҹ„бһҷ

10. **бһҖбһҡбһҺбһёбһҹбһ·бһҖбҹ’бһҹбһ¶ (09-CaseStudy/)**
    - **бһҖбһҡбһҺбһёбһҹбһ·бһҖбҹ’бһҹбһ¶бҹ§бһҖбһҡбһҺбһёбһ‘бһјбһӣбҹҶбһ‘бһјбһӣбһ¶бһҷ**бһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһ—бһ¶бһ–бһ…бһҳбҹ’бһҡбһ»бҹҮбһҡбһ”бһҹбҹӢ MCP бһҖбҹ’бһ“бһ»бһ„бһҹбҹ’бһҗбһ¶бһ“бһҖбһ¶бһҡбһҺбҹҚбһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ–
    - **бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҹбҹҒбһҹбҹ’бһҹбһ¶ AI Azure**бҹ– бһҖбһ¶бһҡбһўбһ»бһҸбһ…бһ„бҹ’бһўбһ»бһӣбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ…бҹ’бһҡбһҫбһ“бһҮбһ¶бһҳбһҪбһҷ Azure OpenAI бһ“бһ·бһ„ AI Search
    - **бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ Azure DevOps**бҹ– бһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ YouTube
    - **бһҖбһ¶бһҡбһ‘бһ¶бһүбһҷбһҖбһҜбһҖбһҹбһ¶бһҡбһ–бҹҒбһӣбһҸбһ¶бһҳбһ–бһ·бһҸ**бҹ– бһўбһҸбһ·бһҗбһ·бһҮбһ“бһ”бҹ’бһҡбҹҒбһҖбһ„бҹӢ Python бһҮбһ¶бһҳбһҪбһҷбһҹбҹ’бһ‘бҹ’бһҡбһёбһҳ HTTP
    - **бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһ•бҹӮбһ“бһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶бһӮбһ“бҹ’бһӣбһ„бһҳбһ¶бһ“бһҖбһ¶бһҡбһҹбһ“бҹ’бһ‘бһ“бһ¶**бҹ– бһңбҹҒбһ”бһҹбһ¶бһҷ Chainlit бһҮбһ¶бһҳбһҪбһҷ AI бһҹбһ“бҹ’бһ‘бһ“бһ¶
    - **бһҜбһҖбһҹбһ¶бһҡбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһҡбһ·бһҷбһӣбһҫбһҸбһҮбһҪбһҡбһҒбһ¶бһ„бһҖбҹ’бһ“бһ»бһ„**бҹ– бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ VS Code бһҮбһ¶бһҳбһҪбһҷ GitHub Copilot
    - **бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ API Azure**бҹ– бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ API бһҹбһ бһӮбҹ’бһҡбһ¶бһҹбһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ MCP
    - **бһ”бһүбҹ’бһҮбһё MCP бһҡбһ”бһҹбҹӢ GitHub**бҹ– бһңбҹҒбһ‘бһ·бһҖбһ¶бһ”бһ„бҹ’бһҖбһҫбһҸбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ бһ“бһ·бһ„бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ…бһҳбҹ’бһҡбһ»бҹҮ
    - бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһўбһ¶бһҮбһёбһңбһҖбһҳбҹ’бһҳ, бһ•бһӣбһ·бһҸбһ—бһ¶бһ–бһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚ бһ“бһ·бһ„бһҖбһ¶бһҡбһҡбһёбһҖбһ…бһҳбҹ’бһҡбһҫбһ“бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһўбҹҒбһҖбһјбһҹбҹҠбһёбһҹбҹ’бһ‘бһ№бһҳ

11. **бһҹбһ·бһҖбҹ’бһҒбһ¶бһ¶бһӣбҹҗбһҷбһ”бҹ’бһҡбҹӮбһ”бҹ’бһҡбһҪбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ AI бһ“бһ·бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ MCP бһҮбһ¶бһҳбһҪбһҷ AI Toolkit (10-StreamliningAIWorkflowsBuildingAnMCPServerWithAIToolkit/)**
    - бһҹбһ·бһҖбҹ’бһҒбһ¶бһ¶бһӣбҹҗбһҷбһ”бҹ’бһҡбҹӮбһ”бҹ’бһҡбһҪбһӣбһҠбҹҶбһ”бһјбһ„бһ…бһјбһӣбһҡбһҪбһҳ MCP бһҮбһ¶бһҳбһҪбһҷ AI Toolkit
    - бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҶбҹ’бһӣбһ¶бһҸбһңбҹғбһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһҡбһңбһ¶бһ„бһҳбҹүбһјбһҢбҹӮбһӣ AI бһ“бһ·бһ„бһ§бһ”бһҖбһҡбһҺбҹҚбһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠ
    - бһҳбҹүбһјбһҢбһ»бһӣбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҠбһ”бҹӢбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“, бһўбһ—бһ·бһңбһҢбҹ’бһҚбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“ бһ“бһ·бһ„бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ…бҹӮбһҖбһ…бһ¶бһҷбһ•бһӣбһ·бһҸбһ•бһӣ
    - **бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҹбһ·бһҖбҹ’бһҒбһ¶бһ¶бһӣбҹҗбһҷ**бҹ–
      - бһҹбһ·бһҖбҹ’бһҒбһ¶бһ¶бһӣбҹҗбһҷбһ‘бһё бҹЎбҹ– бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ MCP
      - бһҹбһ·бһҖбҹ’бһҒбһ¶бһ¶бһӣбҹҗбһҷбһ‘бһё бҹўбҹ– бһўбһ—бһ·бһңбһҢбҹ’бһҚбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ MCP бһҖбһҳбҹ’бһҡбһ·бһҸбһҒбҹ’бһ–бһҹбҹӢ
      - бһҹбһ·бһҖбҹ’бһҒбһ¶бһ¶бһӣбҹҗбһҷбһ‘бһё бҹЈбҹ– бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ AI Toolkit
      - бһҹбһ·бһҖбҹ’бһҒбһ¶бһ¶бһӣбҹҗбһҷбһ‘бһё бҹӨбҹ– бһ…бҹӮбһҖбһ…бһ¶бһҷбһ•бһӣбһ·бһҸбһ•бһӣ бһ“бһ·бһ„бһӣбһ¶бһҷбһЎбҹҶ
    - бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶бһҠбҹ„бһҷбһҹбһ·бһҖбҹ’бһҒбһ¶бһ¶бһӣбҹҗбһҷбһҮбһ¶бһҳбһҪбһҷбһ“бҹғбһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһҮбһ¶бһӣбҹҶбһҠбһ¶бһ”бҹӢбһҮбҹҶбһ бһ¶бһ“

12. **бһҹбһ·бһҖбҹ’бһҒбһ¶бһ¶бһӣбҹҗбһҷбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ MCP (11-MCPServerHandsOnLabs/)**
    - **бһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶бһ…бҹҶбһ“бһҪбһ“ бҹЎбҹЈ бһҹбһ·бһҖбҹ’бһҒбһ¶бһ¶бһӣбҹҗбһҷ** бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ„бҹ’бһҖбһҫбһҸбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ MCP бһҸбҹ’бһҡбҹҖбһҳбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳбһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ PostgreSQL
    - **бһўбһ“бһ»бһңбһҸбҹ’бһҸбһңбһ·бһ—бһ¶бһӮбһӣбһҖбҹӢбһҡбһ¶бһҷбһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠ**бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһҖбһҡбһҺбһёбһ”бҹ’бһҡбһҫ Zava Retail
    - **бһӣбҹҶбһ“бһ¶бҹҶбһҠбҹӮбһӣбһҳбһ¶бһ“бһӮбһ»бһҺбһ—бһ¶бһ–бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһ бһӮбҹ’бһҡбһ¶бһҹ** бһҡбһҪбһҳбһҳбһ¶бһ“ Row Level Security (RLS), бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ бҹҒбһҸбһ»бһ•бһӣ бһ“бһ·бһ„бһҖбһ¶бһҡбһ…бһјбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһӣбһҫбһ—бҹ’бһүбҹҖбһңбһ…бҹ’бһҡбһҫбһ“
    - **бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҹбһ·бһҖбҹ’бһҒбһ¶бһ¶бһӣбҹҗбһҷбһ–бҹҒбһүбһӣбҹҒбһү**бҹ–
      - **бһҹбһ·бһҖбҹ’бһҒбһ¶бһ¶бһӣбҹҗбһҷ бҹ бҹ -бҹ бҹЈбҹ– бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“** - бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶ, бһҡбҹҖбһ”бһ…бҹҶ, бһҹбһ“бҹ’бһҸбһ·бһҹбһ»бһҒ, бһҖбһ¶бһҡбһҸбҹҶбһЎбһҫбһ„бһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹ
      - **бһҹбһ·бһҖбҹ’бһҒбһ¶бһ¶бһӣбҹҗбһҷ бҹ бҹӨ-бҹ бҹҰбҹ– бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ MCP** - бһҖбһ¶бһҡбһҡбһ…бһ“бһ¶бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ, бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ MCP, бһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ§бһ”бһҖбһҡбһҺбҹҚ
      - **бһҹбһ·бһҖбҹ’бһҒбһ¶бһ¶бһӣбҹҗбһҷ бҹ бҹ§-бҹ бҹ©бҹ– бһӣбһҖбҹ’бһҒбһҺбҹҲбһҒбҹ’бһ–бһҹбҹӢ** - бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ бҹҒбһҸбһ»бһ•бһӣ, бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҸбҹҒбһҹбҹ’бһҸ бһ“бһ·бһ„бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҖбҹҶбһ бһ»бһҹ, бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ VS Code
      - **бһҹбһ·бһҖбҹ’бһҒбһ¶бһ¶бһӣбҹҗбһҷ бҹЎбҹ -бҹЎбҹўбҹ– бһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ бһ“бһ·бһ„бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸ** - бһ…бҹӮбһҖбһ…бһ¶бһҷ, бһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ, бһ”бһ“бҹ’бһҗбҹӮбһҳбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–
    - **бһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһҠбҹӮбһӣбһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҸбһ”бҹӢ**бҹ– FastMCP framework, PostgreSQL, Azure OpenAI, Azure Container Apps, Application Insights
    - **бһӣбһ‘бҹ’бһ’бһ•бһӣбһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶**бҹ– бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ MCP бһҸбҹ’бһҡбҹҖбһҳбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ, бһӣбҹҶбһ“бһ¶бҹҶбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ, бһңбһ·бһ—бһ¶бһӮбһҠбҹ„бһҷ AI, бһҹбһ“бҹ’бһҸбһ·бһҹбһ»бһҒбһҹбһ бһӮбҹ’бһҡбһ¶бһҹ

## бһ’бһ“бһ’бһ¶бһ“бһ”бһ“бҹ’бһҗбҹӮбһҳ

бһғбҹ’бһӣбһ¶бҹҶбһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһ’бһ“бһ’бһ¶бһ“бһӮбһ¶бҹҶбһ‘бҹ’бһҡбҹ–

- **бһҗбһҸбһҡбһјбһ”**бҹ– бһҡбһҪбһҳбһҳбһ¶бһ“ бһӮбҹҶбһ“бһјбһҡ бһ“бһ·бһ„бһҡбһјбһ”бһ—бһ¶бһ–бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһңбһӮбҹ’бһӮбһҹбһ·бһҖбҹ’бһҹбһ¶
- **бһҖбһ¶бһҡбһ”бҹ’бһҡбҹӮбһ”бҹ’бһҡбҹӮ**бҹ– бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһ—бһ¶бһҹбһ¶бһ…бҹ’бһҡбһҫбһ“бһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһ”бҹ’бһҡбҹӮбһҹбҹҒбһ…бһҖбҹ’бһҠбһёбһўбһҸбһјбһҳбҹүбһ¶бһ‘бһ·бһҖбһ“бҹғбһҜбһҖбһҹбһ¶бһҡ
- **бһ’бһ“бһ’бһ¶бһ“ MCP бһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡ**бҹ–

## бһҡбһ”бҹҖбһ”бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһғбҹ’бһӣбһ¶бҹҶбһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ“бҹҒбҹҮ

1. **бһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶бһҠбҹ„бһҷбһӣбҹҶбһҠбһ¶бһ”бҹӢ**бҹ– бһҸбһ¶бһҳбһҠбһ¶бһ“бһҮбҹҶбһ–бһјбһҖбһҸбҹӮбһҳбһҪбһҷбһҳбһ»бһҒ (00 бһҠбһӣбҹӢ 11) бһҠбһҫбһҳбҹ’бһ”бһёбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһҹбһ·бһҖбҹ’бһҹбһ¶бһҠбҹӮбһӣбһҳбһ¶бһ“бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бҹ”
