## бһ‘бһ·бһҠбҹ’бһӢбһ—бһ¶бһ–д»“еӮЁ

Model Context Protocol (MCP) бһӮбһәбһҮбһ¶бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ”бҹ’бһҡбһ¶бһҹбҹ’бһҡбҹҗбһҷбһҡбһңбһ¶бһ„бһҳбҹүбһјбһҠбҹӮбһӣ AI бһ“бһ·бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһўбһҸбһ·бһҗбһ·бһҮбһ“бҹ” бһҠбҹҶбһ”бһјбһ„бһӮбһәбһ”бһ„бҹ’бһҖбһҫбһҸбһҠбҹ„бһҷ Anthropic бһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“ MCP бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҗбҹӮбһҡбһҖбҹ’бһҹбһ¶бһҠбҹ„бһҷбһҹбһ бһӮбһҳбһ“бҹҚ MCP бһҹбһҖбһӣбһҸбһ¶бһҳбһҡбһҷбҹҲбһўбһ„бҹ’бһӮбһ—бһ¶бһ– GitHub бһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡбһҳбһҪбһҷбҹ”д»“еӮЁбһ“бҹҒбҹҮбһ•бҹ’бһҠбһӣбҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҹбһ·бһҖбҹ’бһҹбһ¶бһ‘бһјбһӣбҹҶбһ‘бһјбһӣбһ¶бһҷбһҮбһ¶бһҳбһҪбһҷбһҖбһјбһҠбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҖбһ¶бһ“бҹӢбһҸбҹӮбһўбһ¶бһ…бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ”бһ¶бһ“бһҖбҹ’бһ“бһ»бһ„бһ—бһ¶бһҹбһ¶ C#, Java, JavaScript, Python бһ“бһ·бһ„ TypeScript бһҠбҹӮбһӣбһ…бһ„бҹӢбһ”бһ„бҹ’бһҖбһҫбһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚ AI, бһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҡбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ бһ“бһ·бһ„бһңбһ·бһҹбҹ’бһңбһҖбһҡбһ•бҹ’бһ“бҹӮбһҖбһ‘бһ“бҹӢбҹ”

## бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’д»“еӮЁ

д»“еӮЁбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбҹҖбһ”бһ…бҹҶбһҮбһ¶бһ”бҹ’бһҡбһ’бһ¶бһ“бһ”бһ‘бһҠбҹҸбһҹбҹҶбһҒбһ¶бһ“бҹӢ бҹЎбҹЎ бһҠбһ»бҹҶ бһҠбҹ„бһҷбһ•бҹ’бһҠбҹ„бһҸбһ‘бҹ…бһӣбһҫбһ‘бһ¶бҹҶбһ„бһҮбҹ’бһҡбһ»бһ„бһ•бҹ’бһҹбҹҒбһ„бҹ—бһ“бҹғ MCP бҹ”

1. **бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶ (00-Introduction/)**
   - бһ‘бһ·бһҠбҹ’бһӢбһ—бһ¶бһ–бһ‘бһјбһ‘бҹ…бһ“бҹғ Model Context Protocol
   - бһ бҹҒбһҸбһ»бһўбҹ’бһңбһёбһ”бһ¶бһ“бһҮбһ¶бһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбһҳбһ¶бһ“бһҹбһ¶бһҡбҹҲбһҹбҹҶбһҒбһ¶бһ“бҹӢбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ•бҹ’бһӣбһјбһңбһ”бһүбҹ’бһ…бһјбһ“ AI
   - бһҖбһҡбһҺбһёбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ“бҹҚ бһ“бһ·бһ„бһўбһҸбҹ’бһҗбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚ

2. **бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһӮбҹҶбһ“бһ·бһҸ (01-CoreConcepts/)**
   - бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһўбһҸбһ·бһҗбһ·бһҮбһ“-бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ
   - бһ§бһ”бһҖбһҡбһҺбҹҚбһҹбҹҶбһҒбһ¶бһ“бҹӢбһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’
   - бһӮбҹҶбһҡбһјбһҹбһ¶бһҡбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ MCP

3. **бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– (02-Security/)**
   - бһӮбҹҶбһҡбһ¶бһҳбһҖбҹҶбһ бҹӮбһ„бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҠбҹӮбһӣбһ•бҹ’бһўбҹӮбһҖбһӣбһҫ MCP
   - бһӣбҹ’бһ”бһ„бһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ–бһ¶бһҡбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸ
   - бһӣбҹ’бһ”бһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢ бһ“бһ·бһ„бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸ
   - **бһҜбһҖбһҹбһ¶бһҡвҖӢбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–вҖӢбһҷбҹүбһ¶бһ„бһ–бҹҒбһүбһӣбҹҒбһү**бҹ–
     - бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– MCP бһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸ бһҶбҹ’бһ“бһ¶бҹҶ бҹўбҹ бҹўбҹҘ
     - бһҹбҹҖбһңбһ—бҹ…бһҺбҹӮбһ“бһ¶бҹҶбһўбҹҶбһ–бһёбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҒбҹ’бһҹбҹӮбһ—бһ¶бһ–бһҷбһ“бҹ’бһҸ Azure Content Safety
     - бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ бһ“бһ·бһ„бһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– MCP
     - бһ…бҹҶбһҺбһ¶бҹҶбһӣбҹ’бһ”бһ„бһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸ MCP
   - **бһ”бҹ’бһҡбһ’бһ¶бһ“бһ”бһ‘бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—**бҹ–
     - бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһӣбһ–бһ¶бһҖбҹ’бһҷбһҮбһ¶бһҷбһ“бһ·бһ„бһҖбһ¶бһҡбһҡбҹҶбһҒбһ¶бһ“бһ§бһ”бһҖбһҡбһҺбҹҚ
     - бһҖбһ¶бһҡбһҮбһ¶бһ”бҹӢбһ”бҹҶбһҺбһ»бһӣбһҹбҹҒбһҹбҹ’бһҷбһ»бһ“ бһ“бһ·бһ„бһ”бһүбҹ’бһ бһ¶бһўбҹ’бһ“бһҖбһҮбҹҶбһ“бһҪбһҹбһ…бҹ’бһҡбһЎбҹҶ
     - бһҖбһ¶бһҡбһҮбҹ’бһҡбҹ„бҹҮбһ…бһјбһӣ token
     - бһўбһ¶бһҮбҹ’бһүбһ¶бһ”бһҺбҹ’бһҺ бһ“бһ·бһ„бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбһ¶бһҡбһ…бһјбһӣ
     - бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҒбҹ’бһҹбҹӮбһ•бҹ’бһӮбһҸбҹӢбһ•бҹ’бһӮбһ„бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ’бһ¶бһҸбһ» AI
     - бһҹбһҳбһ¶бһ бһҡбһҺбһҖбһҳбҹ’бһҳ Microsoft Prompt Shields

4. **бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ (03-GettingStarted/)**
   - бһҖбһ¶бһҡбһҸбҹ’бһҡбҹҖбһҳбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһ“бһ·бһ„бһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’
   - бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ MCP бһ“бһ·бһ„бһўбһҸбһ·бһҗбһ·бһҮбһ“бһҠбҹҶбһ”бһјбһ„
   - бһҹбһҳбһ·бһ‘бҹ’бһ’бһ•бһӣбһҮбһ¶бһҳбһҪбһҷбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбҹ’бһҡбһ¶бһ”бҹӢ
   - бһҳбһ¶бһ“бһ•бҹ’бһ“бҹӮбһҖбһ”бһ¶бһ“бһ•бҹ’бһҸбһӣбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбҹ–
     - бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫбһҠбҹҶбһ”бһјбһ„
     - бһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһўбһҸбһ·бһҗбһ·бһҮбһ“
     - бһҹбһҳбһ¶бһ бһҡбһҺбһҖбһҳбҹ’бһҳ LLM бһўбһҸбһ·бһҗбһ·бһҮбһ“
     - бһҹбһҳбһ¶бһ бһҡбһҺбһҖбһҳбҹ’бһҳ VS Code
     - бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ Server-Sent Events (SSE)
     - бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫбһҖбһҳбҹ’бһҡбһ·бһҸбһҒбҹ’бһ–бһҹбҹӢ
     - бһ…бҹҶбһ бҹҒбҹҮ HTTP
     - бһҹбһҳбһ¶бһ бһҡбһҺбһҖбһҳбҹ’бһҳбһ”бҹ’бһҡбһўбһ”бҹӢбһ§бһ”бһҖбһҡбһҺбҹҚ AI
     - бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„
     - бһҳбһӮбҹ’бһӮбһ»бһ‘бҹҒбһҹбһҖбҹҚбһ•бҹ’бһҹбһ–бҹ’бһңбһ•бҹ’бһҹбһ¶бһҷ

5. **бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ“бҹҚбһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„ (04-PracticalImplementation/)**
   - бһ”бҹ’бһҡбһҫ SDKs бһҮбһ¶бһ—бһ¶бһҹбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ•бҹ’бһҹбҹҒбһ„бҹ—
   - бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҖбҹҶбһ бһ»бһҹ, бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„ бһ“бһ·бһ„бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢ
   - бһ”бһ„бҹ’бһҖбһҫбһҸбһӮбҹҶбһҡбһјбһ”бһүбҹ’бһҮбһ¶ бһ“бһ·бһ„бһӣбҹҶбһ“бһ¶бҹҶбһ’бҹ’бһңбһҫбһҖбһ¶бһҡ бһҠбҹӮбһӣбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһЎбһҫбһ„бһңбһ·бһүбһ”бһ¶бһ“
   - бһӮбҹҶбһҡбһјбһӮбһҳбҹ’бһҡбҹ„бһ„бһҮбһ¶бһҳбһҪбһҷбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһўбһ“бһ»бһңбһҸбҹ’бһҸ

6. **бһ”бҹ’бһҡбһ’бһ¶бһ“бһ”бһ‘бһҖбһҳбҹ’бһҡбһ·бһҸбһҒбҹ’бһ–бһҹбҹӢ (05-AdvancedTopics/)**
   - бһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹбһңбһ·бһҹбҹ’бһңбһҖбһҳбҹ’бһҳбһ”бһҡбһ·бһ”бһ‘
   - бһҹбһҳбһ¶бһ бһҡбһҺбһҖбһҳбҹ’бһҳ Foundry agent
   - бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸ AI бһҳбһ¶бһҸбһ·бһҖбһ¶бһ…бһҳбҹ’бһҡбһ»бҹҮбһҳбһ»бһҒбһ„бһ¶бһҡ
   - бһ”бһ„бҹ’бһ бһ¶бһүбһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢ OAuth2
   - бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ–бҹҒбһӣбһ–бһ·бһҸ
   - бһ”бһ„бҹ’бһ бҹ„бҹҮбһ–бҹҒбһӣбһ–бһ·бһҸ
   - бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ”бһҡбһ·бһ”бһ‘бһҠбһҫбһҳ
   - бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ”бһүбҹ’бһҮбһјбһ“
   - бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһҮбҹҶбһҡбһ»бһүбһҖбһ¶бһҡбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹ
   - бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһҖбҹҶбһҺбһҸбҹӢбһҖбһҳбҹ’бһҡбһ·бһҸ
   - бһӮбһ·бһҸбһ–бһёбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–
   - бһҹбһҳбһ¶бһ бһҡбһҺбһҖбһҳбҹ’бһҳбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– Entra ID
   - бһҹбһҳбһ¶бһ бһҡбһҺбһҖбһҳбҹ’бһҳбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ”бһҺбҹ’бһҸбһ¶бһү

7. **бһҖбһ¶бһҡбһҡбһҪбһҳбһ…бҹҶбһҺбҹӮбһҖбһҹбһ бһӮбһҳбһ“бҹҚ (06-CommunityContributions/)**
   - бһҡбһ”бҹҖбһ”бһ…бһјбһӣбһҡбһҪбһҳбһҖбһјбһҠ бһ“бһ·бһ„бһҜбһҖбһҹбһ¶бһҡ
   - бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷ GitHub
   - бһҖбһ¶бһҡбһҖбҹӮбһӣбһҳбҹ’бһў бһ“бһ·бһ„бһҳбһҸбһ·бһ–бһёбһҹбһ бһӮбһҳбһ“бҹҚ
   - бһ”бҹ’бһҡбһҫбһўбһҸбһ·бһҗбһ·бһҮбһ“ MCP бһ•бҹ’бһҹбҹҒбһ„бҹ— (Claude Desktop, Cline, VSCode)
   - бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ MCP бһ–бҹҒбһүбһ“бһ·бһҷбһҳ бһҡбһҪбһҳбһҳбһ¶бһ“бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһјбһ”бһ—бһ¶бһ–

8. **бһҳбҹҒбһҡбҹҖбһ“бһ–бһёбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҠбҹҶбһ”бһјбһ„ (07-LessonsfromEarlyAdoption/)**
   - бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ“бҹҚбһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠ бһ“бһ·бһ„бһҡбһҝбһ„бһҮбҹ„бһӮбһҮбҹҗбһҷ
   - бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһ“бһ·бһ„бһ•бҹ’бһҹбһ–бҹ’бһңбһ•бҹ’бһҹбһ¶бһҷбһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҠбҹӮбһӣбһ•бҹ’бһўбҹӮбһҖбһӣбһҫ MCP
   - бһҡбһ”бҹҖбһ”бһ“бҹғбһўбһ“бһ¶бһӮбһҸбһ“бһ·бһ„бһ•бҹӮбһ“бһ‘бһёбһ•бҹ’бһӣбһјбһң
   - **бһҳбһӮбҹ’бһӮбһ»бһ‘бҹҒбһҹбһҖбҹҚбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ Microsoft MCP**бҹ– бһҳбһӮбҹ’бһӮбһ»бһ‘бҹҒбһҹбһҖбҹҚбһ–бҹҒбһүбһӣбҹҒбһүбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ MCP Microsoft бҹЎбҹ  бһҜбһҖбһҹбһ¶бһҡ бһҠбҹӮбһӣбһҡбһҪбһҳбһҳбһ¶бһ“:
     - Microsoft Learn Docs MCP Server
     - Azure MCP Server (бһўбҹ’бһ“бһҖбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҮбһ¶бһ„ бҹЎбҹҘ)

9. **бһӣбҹ’бһ”бһ„бһӣбҹ’бһ”бһёбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸ (08-BestPractices/)**
   - бһҖбһ¶бһҡбһҹбҹҶбһҡбһ»бһ„бһҹбҹҶбһҡбһҪбһӣ бһ“бһ·бһ„бһҸбһҳбҹ’бһӣбһҫбһ„
   - бһҖбһ¶бһҡбһҡбһ…бһ“бһ¶бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ MCP бһҠбҹӮбһӣбһўбһ¶бһ…бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҖбҹҶбһ бһ»бһҹбһ”бһ¶бһ“
   - бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„ бһ“бһ·бһ„бһ’бһ“бҹӢбһ’бҹ’бһ„бһ“бҹӢ

10. **бһҖбһҡбһҺбһёбһҹбһ·бһҖбҹ’бһҹбһ¶ (09-CaseStudy/)**
    - **бһҖбһҡбһҺбһёбһҹбһ·бһҖбҹ’бһҹбһ¶бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“** бһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһ—бһ¶бһ–бһ…бһҳбҹ’бһҡбһ»бҹҮбһҡбһ”бһҹбҹӢ MCP бһҖбҹ’бһ“бһ»бһ„бһҹбҹ’бһҗбһ¶бһ“аёҒаёІаёЈаё“а№Ңбһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ–
    - **бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ’бҹ’бһңбһҫбһҠбҹҶбһҺбһҫбһҡ Azure AI**бҹ– бһҖбһ¶бһҡбһҡбһҸбҹӢбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ…бҹ’бһҡбһҫбһ“бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҮбһ¶бһҳбһҪбһҷ Azure OpenAI бһ“бһ·бһ„бһҹбҹ’бһңбҹӮбһ„бһҡбһҖ AI
    - **бһҹбһҳбһ¶бһ бһҡбһҺбһҖбһҳбҹ’бһҳ Azure DevOps**бҹ– бһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһӣбҹҶбһ бһјбһҡбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҮбһ¶бһҳбһҪбһҷбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ YouTube
    - **бһ‘бһ‘бһҪбһӣбһҷбһҖбһҜбһҖбһҹбһ¶бһҡбһ–бҹҒбһӣбһ–бһ·бһҸ**бҹ– бһўбһҸбһ·бһҗбһ·бһҮбһ“ Python console бһҮбһ¶бһҳбһҪбһҷ HTTP streaming
    - **бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһ•бҹӮбһ“бһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶бһҳбһ¶бһ“бһўбһ“бҹ’бһҸбһҡбһҖбһҳбҹ’бһҳ**бҹ– бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһңбҹҒбһ” Chainlit бһҮбһ¶бһҳбһҪбһҷ AI бһҡбһ”бҹҖбһ”бһҮбһҮбҹӮбһҖ
    - **бһҜбһҖбһҹбһ¶бһҡбһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ—бһ¶бһҹбһ¶бһҖбһјбһҠ**бҹ– бһҹбһҳбһ¶бһ бһҡбһҺбһҖбһҳбҹ’бһҳ VS Code бһҮбһ¶бһҳбһҪбһҷ GitHub Copilot workflow
    - **бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ Azure API**бҹ– бһҹбһҳбһ¶бһ бһҡбһҺбһҖбһҳбҹ’бһҳ API бһҹбһ бһӮбҹ’бһҡбһ¶бһҹбһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ MCP
    - **бһ”бһүбҹ’бһҮбһёбһҡбҹүбҹҒбһҹбҹҠбһёбһҹбҹ’бһ‘бҹ’бһҡбһё MCP GitHub**бҹ– бһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһўбҹҒбһҖбһјбһҹбҹҠбһёбһҹбҹ’бһҸбһ№бһҳ бһ“бһ·бһ„бһҖбһ¶бһҡбһҹбһҳбһ¶бһ бһҡбһҺбһҖбһҳбҹ’бһҳбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ
    - бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһўбһ“бһ»бһңбһҸбҹ’бһҸбһӮбҹ’бһҡбһ”бҹӢбһ”бҹӮбһ”бһҷбҹүбһ¶бһ„ бһҡбһҪбһҳбһҳбһ¶бһ“бһҖбһ¶бһҡбһҹбһҳбһ¶бһ бһҡбһҺбһҖбһҳбҹ’бһҳбһҹбһ бһӮбҹ’бһҡбһ¶бһҹ, бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚ бһ“бһ·бһ„бһўбһ—бһ·бһңбһҢбҹ’бһҚбһўбҹҒбһҖбһјбһҹбҹҠбһёбһҹбҹ’бһҸбһ№бһҳ

11. **бһңбһӮбҹ’бһӮбһ бһ¶бһҸбҹӢбһ–бҹ’бһңбһҫбһҠбҹҶбһҺбһҫбһҡбһ•бҹ’бһ‘бһ¶бһӣбҹӢ (10-StreamliningAIWorkflowsBuildingAnMCPServerWithAIToolkit/)**
    - бһңбһӮбҹ’бһӮбһ бһ¶бһҸбҹӢбһ–бҹ’бһҡбҹғбһ’бһ“бҹӢбһҷбҹүбһ¶бһ„бһ–бҹҒбһүбһӣбҹҒбһүбһҮбһ¶бһҳбһҪбһҷ MCP бһ“бһ·бһ„ AI Toolkit
    - бһҹбһ¶бһ„бһҹбһ„бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҶбҹ’бһӣбһ¶бһҸбһңбҹғбһҠбҹӮбһӣбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҳбҹүбһјбһҠбҹӮбһӣ AI бһҮбһ¶бһҳбһҪбһҷбһ§бһ”бһҖбһҡбһҺбҹҚбһ–бһ·бһ—бһ–бһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„
    - бһҳбҹүбһјбһҢбһ»бһӣбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„бһӮбҹ’бһҡбһ”бҹӢбһҠбҹҶбһҺбһ¶бһҖбҹӢбһҖбһ¶бһӣ бһҡбһҪбһҳбһҳбһ¶бһ“ бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“, бһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“ бһ“бһ·бһ„бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ•бҹ’бһҹбһ–бҹ’бһңбһ•бҹ’бһҹбһ¶бһҷ
    - **бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҳбһ“бҹ’бһ‘бһёбһҡбһҹбһ·бһҖбҹ’бһҹбһ¶**бҹ–
      - бһҳбһ“бҹ’бһ‘бһёбһҡ бҹЎбҹ– бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ MCP
      - бһҳбһ“бҹ’бһ‘бһёбһҡ бҹўбҹ– бһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ MCP бһҒбҹ’бһ–бһҹбҹӢ
      - бһҳбһ“бҹ’бһ‘бһёбһҡ бҹЈбҹ– бһҹбһҳбһ¶бһ бһҡбһҺбһҖбһҳбҹ’бһҳ AI Toolkit
      - бһҳбһ“бҹ’бһ‘бһёбһҡ бҹӨбҹ– бһ•бҹ’бһҹбһ–бҹ’бһңбһ•бҹ’бһҹбһ¶бһҷ бһ“бһ·бһ„бһҖбҹҶбһҺбһҸбҹӢбһҖбһҳбҹ’бһҡбһ·бһҸбһ•бһӣбһ·бһҸбһ•бһӣ
    - бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһҹбһ·бһҖбҹ’бһҹбһ¶бһҳбҹүбҹ„бһ„бһҸбһ¶бһҳбһҳбһ“бҹ’бһ‘бһёбһҡбһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһҮбҹҶбһ бһ¶бһ“бҹ—

12. **бһҳбһ“бҹ’бһ‘бһёбһҡбһҹбһ·бһҖбҹ’бһҹбһ¶бһҹбһҳбһ¶бһ бһҡбһҺбһҖбһҳбҹ’бһҳбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ MCP (11-MCPServerHandsOnLabs/)**
    - **бһ•бҹ’бһӣбһјбһңбһҹбһ·бһҖбҹ’бһҹбһ¶ бҹЎбҹЈ бһҳбһ“бҹ’бһ‘бһёбһҡ** бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһ¶бһ„бһҹбһ„бҹӢбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ MCP бһҡбһҪбһ…бһҡбһ¶бһӣбҹӢбһҮбһ¶бһҳбһҪбһҷбһ…бһјбһӣбһҡбһҪбһҳ PostgreSQL
    - **бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһңбһ·бһ—бһ¶бһӮбһӣбһҖбҹӢбһ…бҹҒбһүбһ–бһ·бһ—бһ–бһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„** бһҮбһ¶бһҳбһҪбһҷбһҖбһҡбһҺбһёбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ Zava Retail
    - **бһӮбҹҶбһҡбһјбһҹбһ бһӮбҹ’бһҡбһ¶бһҹ** бһҡбһҪбһҳбһҳбһ¶бһ“ Row Level Security (RLS), бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ бһҸбҹ’бһҗбһ–бһ·бһ—бһ¶бһҖбҹ’бһҹбһ¶ бһ“бһ·бһ„бһ…бһјбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ…бҹ’бһҡбһҫбһ“бһўбҹ’бһ“бһҖбһҮбһҪбһӣ
    - **бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҳбһ“бҹ’бһ‘бһёбһҡбһ–бҹҒбһүбһӣбҹҒбһү**бҹ–
      - бһҳбһ“бҹ’бһ‘бһёбһҡ бҹ бҹ -бҹ бҹЈбҹ– бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“, бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’, бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–, бһҸбҹ’бһҡбҹҖбһҳбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹ
      - бһҳбһ“бҹ’бһ‘бһёбһҡ бҹ бҹӨ-бҹ бҹҰбҹ– бһҖбһ¶бһҡбһҹбһ¶бһ„бһҹбһ„бҹӢбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ MCP - бһҡбһ…бһ“бһ¶бһ•бҹ’бһ‘бҹғбһҖбҹ’бһ“бһ»бһ„, бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ, бһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ§бһ”бһҖбһҡбһҺбҹҚ
      - бһҳбһ“бҹ’бһ‘бһёбһҡ бҹ бҹ§-бҹ бҹ©бҹ– бһҳбһ»бһҒбһ„бһ¶бһҡбһҒбҹ’бһ–бһҹбҹӢ - бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҡбһ”бҹҖбһ”бһңбҹ„бһ бһ¶бһҡ, бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„ бһ“бһ·бһ„бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҖбҹҶбһ бһ»бһҹ, бһҹбһҳбһ¶бһ бһҡбһҺбһҖбһҳбҹ’бһҳ VS Code
      - бһҳбһ“бҹ’бһ‘бһёбһҡ бҹЎбҹ -бҹЎбҹўбҹ– бһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ бһ“бһ·бһ„бһӣбҹ’бһ”бһ„бһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸ - бһ•бҹ’бһҹбһ–бҹ’бһңбһ•бҹ’бһҹбһ¶бһҷ, бһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ, бһ”бһ„бҹ’бһҖбһҫбһҸбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–
    - **бһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһҠбҹӮбһӣбһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҸбһ”бҹӢ**бҹ– FastMCP framework, PostgreSQL, Azure OpenAI, Azure Container Apps, Application Insights
    - **бһӣбһ‘бҹ’бһ’бһ•бһӣбһҹбһ·бһҖбҹ’бһҹбһ¶**бҹ– бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ MCP бһҡбһҪбһ…бһҡбһ¶бһӣбҹӢ, бһӮбҹҶбһҡбһјбһҹбһҳбһ¶бһ бһҡбһҺбһҖбһҳбҹ’бһҳбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ, бһңбһ·бһ—бһ¶бһӮбһ•бҹ’бһўбҹӮбһҖ AI, бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҹбһ бһӮбҹ’бһҡбһ¶бһҹ

## бһҡбһ”бҹҖбһ”бһ”бҹ’бһҡбһҫд»“еӮЁбһ“бҹҒбҹҮ

1. **бһҹбһ·бһҖбҹ’бһҹбһ¶бһҸбһ¶бһҳбһӣбҹҶбһҠбһ¶бһ”бҹӢ**бҹ– бһҸбһ¶бһҳбһҠбһ¶бһ“бһҮбҹҶбһ–бһјбһҖбһ…бһ¶бһ”бҹӢбһ–бһё бҹ бҹ  бһҠбһӣбҹӢ бҹЎбҹЎ бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһҹбһ·бһҖбҹ’бһҹбһ¶бһҹбһ»бһ‘бҹ’бһ’

3. **бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„**бҹ– бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ–бһёбһ•бҹ’бһ“бҹӮбһҖ "бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ" бһҠбһҫбһҳбҹ’бһ”бһёбһҸбҹ’бһҡбҹҖбһҳбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹ бһ“бһ·бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ MCP бһ“бһ·бһ„бһўбһҸбһ·бһҗбһ·бһҮбһ“бһҠбҹҶбһ”бһјбһ„бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ
