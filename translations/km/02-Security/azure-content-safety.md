# бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– MCP бһҮбһ¶бһҗбҹ’бһҳбһёвҖӢбһҮбһ¶бһҳбһҪбһҷ Azure Content Safety

> **бһӮбҹ’бһҡбҹ„бҹҮбһҗбҹ’бһ“бһ¶бһҖбҹӢ OWASP MCP бһҠбҹӮбһӣбһ”бһ¶бһ“бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ**: [MCP06 - бһҖбһ¶бһҡбһңбһ¶бһҷбһ”бҹ’бһҡбһ бһ¶бһҡбһҮбҹҶбһҡбһ»бһүбһҸбһ¶бһҳбһҡбһҷбҹҲбһ”бһҺбҹ’бһҸбһ¶бһүбһ”бҹ’бһҡбһҖбһ¶бһҹ Contextual Payloads](https://microsoft.github.io/mcp-azure-security-guide/mcp/mcp06-prompt-injection/)

Azure Content Safety бһ•бҹ’бһҠбһӣбҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹҸбһҳбһ¶бһ“бһҘбһ‘бҹ’бһ’бһ·бһ–бһӣбһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бһҠбҹӮбһӣбһўбһ¶бһ…бһ”бһ“бҹ’бһҗбҹӮбһҳбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһ“бҹғбһҖбһ¶бһҡвҖӢбһўбһ“бһ»бһңбһҸбҹ’бһҸ MCP бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҠбҹ„бһҷбһҠбҹғ бһҹбһјбһҳбһҳбһҫбһӣ [бһҹбһ·бһҖбҹ’бһҒбһ¶бһҹбһ¶бһӣбһ¶бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– MCP Summit (Sherpa)](https://azure-samples.github.io/sherpa/) бһҠбҹҶбһ”бһјбһ„ 3: бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– I/O бҹ”

## бһҖбһ“бҹ’бһ‘бһ»бһҷбһҖбһ¶бһҡбһ–бһ¶бһҡ Prompt

бһҖбһ“бҹ’бһ‘бһ»бһҷбһҖбһ¶бһҡбһ–бһ¶бһҡ AI Prompt бһҡбһ”бһҹбҹӢ Microsoft бһ•бҹ’бһҠбһӣбҹӢбһҖбһ¶бһҡбһҖбһ¶бһҡбһ–бһ¶бһҡбһҠбҹҸбһҡбһ№бһ„бһҳбһ¶бҹҶбһ”бҹ’бһҡбһҶбһ¶бҹҶбһ„бһ“бһ№бһ„бһҖбһ¶бһҡбһңбһ¶бһҷбһ”бҹ’бһҡбһ бһ¶бһҡбһҮбҹҶбһҡбһ»бһүбһҸбһ¶бһҳ Prompt бһҠбҹ„бһҷбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ•бҹ’бһӣбһјбһңбһ…бҹҶбһ–бҹ„бҹҮбһҖбһ¶бһҡбһңбһ¶бһҷбһ”бҹ’бһҡбһ бһ¶бһҡбһ•бҹ’бһ‘бһ¶бһӣбҹӢ бһ“бһ·бһ„бһўбһ–бҹ’бһ…бһ¶бһҡбҹ’бһҷбһҸбһ¶бһҳбһҡбһҷбҹҲбҹ–

1. **бһҖбһ¶бһҡбһҡбһҖбһғбһҫбһүбһҮбҹҶбһ“бһ¶бһүбһҒбҹ’бһ–бһҹбҹӢ**бҹ– бһ”бҹ’бһҡбһҫбһҳбҹүбһ¶бһҹбҹҠбһёбһ“вҖӢбһ бҹ’бһңбһ№бһҖбһ бһ¶бһҸбҹӢ бһҠбһҫбһҳбҹ’бһ”бһёбһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһҺбҹӮбһ“бһ¶бҹҶбһҠбҹӮбһӣбһҳбһ¶бһ“бһӮбҹ’бһҡбҹ„бҹҮбһҗбҹ’бһ“бһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбһ¶бһҸбһ·бһҖбһ¶бҹ”
2. **бһҖбһ¶бһҡбһ—бҹ’бһӣбһәбһ…бҹ’бһ”бһ¶бһҹбҹӢ**бҹ– бһ”бһҳбҹ’бһӣбҹӮбһ„бһўбһҸбҹ’бһҗбһ”бһ‘бһ”бһүбҹ’бһ…бһјбһӣбһҠбһҫбһҳбҹ’бһ”бһёбһҮбһҪбһҷбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ AI бһҶбҹҖбһ„бһҒбһ»бһҹбһҸбҹ’бһҡбһјбһңбһ…бҹҶбһ–бҹ„бҹҮбһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһҺбҹӮбһ“бһ¶бҹҶбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң бһ“бһ·бһ„бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ…бҹ”
3. **Delimiters бһ“бһ·бһ„ Datamarking**бҹ– бһҡбҹҖбһ”бһ…бҹҶбһҹбһүбҹ’бһүбһ¶бһ–бҹ’бһҡбҹҶбһҠбҹӮбһ“бһҡбһңбһ¶бһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹӮбһӣбһ‘бһ»бһҖбһ…бһ·бһҸбҹ’бһҸ бһ“бһ·бһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҳбһ·бһ“бһ‘бһ»бһҖбһ…бһ·бһҸбҹ’бһҸбҹ”
4. **бһҖбһ¶бһҡбһҸбҹ’бһҡбһјбһңбһҳбһ¶бһ“ Azure Content Safety**бҹ– бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷ Azure AI Content Safety бһҠбһҫбһҳбҹ’бһ”бһёбһҡбһҖбһғбһҫбһүбһҖбһ¶бһҡбһ–бҹ’бһҷбһ¶бһҷбһ¶бһҳбһ”бҹҶбһ”бҹ’бһӣбҹӮбһ„бһ…бҹҒбһү бһ“бһ·бһ„бһҳбһ¶бһҸбһ·бһҖбһ¶бһҠбҹӮбһӣбһ’бҹ’бһңбһҫбһ бһ¶бһ“бһ·бһ—бҹҗбһҷбҹ”
5. **бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ—бһ¶бһ–бһҮбһ¶бһ”бһ“бҹ’бһҸ**бҹ– Microsoft бһ’бҹ’бһңбһҫбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ—бһ¶бһ–бһҷбҹүбһ¶бһ„бһ‘бҹҖбһ„бһ‘бһ¶бһҸбҹӢбһ“бһјбһңбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһ¶бһҡбһҖбһ¶бһҡбһ–бһ¶бһҡбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбһҷбһ»бһ‘бҹ’бһ’бһ”бҹ’бһҡбһҶбһ¶бҹҶбһ„бһӮбҹ’бһҡбҹ„бҹҮбһҗбҹ’бһ“бһ¶бһҖбҹӢбһҗбҹ’бһҳбһёбҹ—бҹ”

## бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸ Azure Content Safety бһҮбһ¶бһҳбһҪбһҷ MCP

бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ“бҹҒбҹҮбһ•бҹ’бһҠбһӣбҹӢбһҖбһ¶бһҡбһҖбһ¶бһҡбһ–бһ¶бһҡбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҹбҹ’бһҡбһ‘бһ¶бһ”бҹӢбҹ–
- бһҹбҹ’бһҖбҹҒбһ“бһ”бһүбҹ’бһ…бһјбһӣбһҳбһ»бһ“бһ–бҹҒбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ
- бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһӣбһ‘бҹ’бһ’бһ•бһӣбһҳбһ»бһ“бһҹбһ„
- бһ”бҹ’бһҡбһҫбһ”бһүбҹ’бһҮбһёбһҡбһ¶бһҡбһ¶бҹҶбһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҡбһ¶бһ„бһҒбһјбһ…бһҠбҹӮбһӣбһ”бһ¶бһ“бһҹбҹ’бһӮбһ¶бһӣбҹӢ
- бһ”бҹ’бһҡбһҫбһҳбҹүбһјбһҠбҹӮбһӣбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҳбһ¶бһҸбһ·бһҖбһ¶ Azure бһҠбҹӮбһӣбһ’бҹ’бһңбһҫбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ—бһ¶бһ–бһҮбһ¶бһ”бһ“бҹ’бһҸ

## бһ”бҹ’бһҡбһ—бһ–бһҡбһ”бһҹбҹӢ Azure Content Safety

бһҠбһҫбһҳбҹ’бһ”бһёбһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһ”бһ“бҹ’бһҗбҹӮбһҳбһ–бһёбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸ Azure Content Safety бһҮбһ¶бһҳбһҪбһҷбһҳбҹүбһ¶бһҹбҹҠбһёбһ“ MCP бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһҹбһјбһҳбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ”бҹ’бһҡбһ—бһ–бһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбҹ–

1. [бһҜбһҖбһҹбһ¶бһҡбһҮбһ¶бһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡбһўбҹҶбһ–бһё Azure AI Content Safety](https://learn.microsoft.com/azure/ai-services/content-safety/) - бһҜбһҖбһҹбһ¶бһҡбһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Azure Content Safetyбҹ”
2. [бһҜбһҖбһҹбһ¶бһҡбһҖбһ“бҹ’бһ‘бһ»бһҷбһҖбһ¶бһҡбһ–бһ¶бһҡ Prompt](https://learn.microsoft.com/azure/ai-services/content-safety/concepts/prompt-shield) - бһҡбҹҖбһ“бһўбҹҶбһ–бһёбһҡбһ”бҹҖбһ”бһ–бһ¶бһҡбһҖбһ¶бһҡбһңбһ¶бһҷбһ”бҹ’бһҡбһ бһ¶бһҡбһҮбҹҶбһҡбһ»бһүбһ‘бҹҶбһҡбһ„бҹӢ Promptбҹ”
3. [бһҷбҹ„бһ„ API Content Safety](https://learn.microsoft.com/rest/api/contentsafety/) - бһҷбҹ„бһ„ API бһӣбһҳбҹ’бһўбһ·бһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбһ“бһ»бһңбһҸбҹ’бһҸ Content Safetyбҹ”
4. [Quickstart: Azure Content Safety бһҮбһ¶бһҳбһҪбһҷ C#](https://learn.microsoft.com/azure/ai-services/content-safety/quickstart-csharp) - бһҳбһӮбҹ’бһӮбһ»бһ‘бҹҒбһҹбһҖбҹҚбһ”бһ“бҹ’бһ‘бһ¶бһ“бҹӢбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ C#бҹ”
5. [бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷ Client Content Safety](https://learn.microsoft.com/azure/ai-services/content-safety/quickstart-client-libraries-rest-api) - бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷ client бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бһ¶бһҹбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ”
6. [бһҡбһҖбһғбһҫбһүбһҖбһ¶бһҡбһ–бҹ’бһҷбһ¶бһҷбһ¶бһҳбһ”бҹҶбһ”бҹ’бһӣбҹӮбһ„бһ…бҹҒбһү](https://learn.microsoft.com/azure/ai-services/content-safety/concepts/jailbreak-detection) - бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбһўбҹҶбһ–бһёбһҖбһ¶бһҡбһҡбһҖбһғбһҫбһү бһ“бһ·бһ„бһҖбһ¶бһҡбһ–бһ¶бһҡ jailbreakбҹ”
7. [бһ–бһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Content Safety](https://learn.microsoft.com/azure/ai-services/content-safety/concepts/best-practices) - бһ–бһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбһ“бһ»бһңбһҸбҹ’бһҸ Content Safety бһҷбҹүбһ¶бһ„бһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бҹ”

бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҮбһ¶бһҳбһ»бһ“ бһўбһ¶бһ…бһҳбһҫбһӣ [бһҳбһӮбҹ’бһӮбһ»бһ‘бҹҒбһҹбһҖбҹҚбһўбһ“бһ»бһңбһҸбҹ’бһҸ Azure Content Safety](./azure-content-safety-implementation.md) бһҡбһ”бһҹбҹӢбһҷбһҫбһ„бҹ”
