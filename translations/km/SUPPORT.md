# គាំទ្រ

## វិធីសាស្រ្តដាក់បញ្ហា និងទទួលបានជំនួយ  

គម្រោងនេះប្រើ GitHub Issues ដើម្បីតាមដានកាយវិការនិងការស្នើសុំមុខងារ។ សូមស្វែងរកបញ្ហាដែលមានរួចមុនពេលដាក់បញ្ហាថ្មីដើម្បីជៀសវាងការចម្លង។ សម្រាប់បញ្ហាថ្មីៗ សូមដាក់កើតកំហុស ឬ ស្នើសុំមុខងាររបស់អ្នកជាបញ្ហា Issue ថ្មីមួយ។

សម្រាប់ជំនួយ និងសំណួរចំពោះការប្រើប្រាស់គម្រោងនេះ សូម៖
- ពិនិត្យមើល [Azure AI Foundry Discord](https://discord.com/invite/ByRwuEEgH4) ដែលជាកន្លែងដែលអ្នកជំនាញ និងអ្នកអភិវឌ្ឍន៍ផ្សេងទៀតប្រមុខគ្នា
- ប្រើប្រាស់ធនធាន [Model Context Protocol Community](https://modelcontextprotocol.io/community/) សម្រាប់សំណួរទាក់ទងនឹងពិធីការ Protocol

## នយោបាយគាំទ្រ Microsoft  

ការគាំទ្រសម្រាប់គម្រោង Model Context Protocol (MCP) សម្រាប់អ្នកចាប់ផ្តើម មានការកំណត់ត្រឹមធនធានដែលបានរាយឈ្មោះខាងលើ។
