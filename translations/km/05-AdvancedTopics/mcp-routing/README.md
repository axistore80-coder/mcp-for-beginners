# бһҖбһ¶бһҡбһ•бҹ’бһӣбһјбһңбһ…бһҡбһ¶бһ…бһҡбһҺбҹҚбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһӮбҹҶбһҡбһј

бһҖбһ¶бһҡбһ•бҹ’бһӣбһјбһңбһ…бһҡбһ¶бһ…бһҡбһҺбҹҚбһӮбһәбһҹбҹҶбһҒбһ¶бһ“бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһүбҹ’бһҮбһјбһ“бһҹбҹҶбһҺбһҫбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһӮбҹҶбһҡбһј бһҸбһ»бһ”бһ—бһ¶бһӮ бһ¬бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһҹбһҳбһҡбһҳбҹ’бһҷбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ“бҹғ MCPбҹ”

## бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶ

бһҖбһ¶бһҡбһ•бҹ’бһӣбһјбһңбһ…бһҡбһ¶бһ…бһҡбһҺбҹҚбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһӮбҹҶбһҡбһј (MCP) бһҳбһ¶бһ“бһ“бҹҗбһҷбһҗбһ¶бһ”бһүбҹ’бһҮбһјбһ“бһҹбҹҶбһҺбһҫбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһӮбҹҶбһҡбһј бһ¬бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһҠбҹӮбһӣбһҹбһҳбһҡбһҳбҹ’бһҷбһ”бҹҶбһ•бһ»бһҸбһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһӣбһҖбҹ’бһҒбһҺбҹҲбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҠбһјбһ…бһҮбһ¶бһ”бҹ’бһҡбһ—бҹҒбһ‘бһҒбҹ’бһӣбһ№бһҳбһҹбһ¶бһҡ бһ”бһҡбһ·бһ”бһ‘бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ бһ“бһ·бһ„бһ•бҹ’бһ‘бһ»бһҖбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бҹ” бһңбһ¶бһ’бһ¶бһ“бһ¶бһҗбһ¶бһҖбһ¶бһҡбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҮбҹ„бһӮбһҮбҹҗбһҷ бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ’бһ“бһ’бһ¶бһ“бһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бҹ”

## бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶

бһ“бҹ…бһ…бһ»бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһўбһ¶бһ…:

- бһҷбһӣбҹӢбһҠбһ№бһ„бһ–бһёбһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚбһ“бҹғбһҖбһ¶бһҡбһ•бҹ’бһӣбһјбһңбһ…бһҡбһ¶бһ…бһҡбһҺбҹҚбһ“бҹ… MCPбҹ”
- бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһ¶бһҡбһ•бҹ’бһӣбһјбһңбһ…бһҡбһ¶бһ…бһҡбһҺбҹҚбһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһҒбҹ’бһӣбһ№бһҳбһҹбһ¶бһҡбһҠбһҫбһҳбҹ’бһ”бһёбһ”бһүбҹ’бһҮбһјбһ“бһҹбҹҶбһҺбһҫбһ‘бҹ…бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһҮбһ¶бһҹбҹ’бһ‘бһҫбһҡбһҸбҹӮбһҡбҹ”
- бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһҸбһ»бһӣбҹ’бһҷбһ—бһ¶бһ–бһ•бҹ’бһ‘бһ»бһҖбһҷбҹүбһ¶бһ„бһҶбҹ’бһӣбһ¶бһҸбһңбҹғ бһҠбһҫбһҳбҹ’бһ”бһёбһ’бҹ’бһңбһҫбһІбҹ’бһҷбһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ·бһ—бһ¶бһ–бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ’бһ“бһ’бһ¶бһ“бҹ”
- бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһ¶бһҡбһ•бҹ’бһӣбһјбһңбһ…бһҡбһ¶бһ…бһҡбһҺбҹҚбһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣбһҳбһ¶бһ“бһ бҹҒбһҸбһ»бһ•бһӣбһ”бҹӮбһ”бһӣбһҖбҹ’бһҒбһҺбҹҲбһўбһ¶бһҹбҹ’бһҡбҹҗбһҷбһӣбһҫбһ”бһҡбһ·бһ”бһ‘бһҹбҹҶбһҺбһҫбҹ”

## бһҖбһ¶бһҡбһ•бҹ’бһӣбһјбһңбһ…бһҡбһ¶бһ…бһҡбһҺбҹҚбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһҒбҹ’бһӣбһ№бһҳбһҹбһ¶бһҡ

бһҖбһ¶бһҡбһ•бҹ’бһӣбһјбһңбһ…бһҡбһ¶бһ…бһҡбһҺбҹҚбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһҒбҹ’бһӣбһ№бһҳбһҹбһ¶бһҡбһ”бһүбҹ’бһҮбһјбһ“бһҹбҹҶбһҺбһҫбһ‘бҹ…бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһҮбһ¶бһҹбҹ’бһ‘бһҫбһҡбһҸбҹӮбһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһҒбҹ’бһӣбһ№бһҳбһҹбһ¶бһҡбһ“бҹғбһҹбҹҶбһҺбһҫбҹ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ бһҹбҹҶбһҺбһҫбһҠбҹӮбһӣбһ‘бһ¶бһҖбҹӢбһ‘бһ„бһ“бһ№бһ„бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһјбһҠбһўбһ¶бһ…бһ”бһүбҹ’бһҮбһјбһ“бһ‘бҹ…бһӮбҹҶбһҡбһјбһҖбһјбһҠбһҮбһ¶бһҹбҹ’бһ‘бһҫбһҡбһҸбҹӮ бһҒбһҺбҹҲбһҠбҹӮбһӣбһҹбҹҶбһҺбһҫбһҖбһ¶бһҡбһҹбһҡбһҹбҹҒбһҡбһ…бҹ’бһ“бҹғбһ”бҹ’бһҡбһҢбһ·бһҸбһўбһ¶бһ…бһ”бһүбҹ’бһҮбһјбһ“бһ‘бҹ…бһӮбҹҶбһҡбһјбһҖбһ¶бһҡбһҹбһҡбһҹбҹҒбһҡбһ…бҹ’бһ“бҹғбһ”бҹ’бһҡбһҢбһ·бһҸбҹ”

бһ…бһ„бҹӢбһҠбһ№бһ„бһўбҹҶбһ–бһёбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ“бҹҚбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ—бһ¶бһҹбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ”

```csharp
// .NET Example: Content-based routing in MCP
public class ContentBasedRouter
{
    private readonly Dictionary<string, McpClient> _specializedClients;
    private readonly RoutingClassifier _classifier;
    
    public ContentBasedRouter()
    {
        // Initialize specialized clients for different domains
        _specializedClients = new Dictionary<string, McpClient>
        {
            ["code"] = new McpClient("https://code-specialized-mcp.com"),
            ["creative"] = new McpClient("https://creative-specialized-mcp.com"),
            ["scientific"] = new McpClient("https://scientific-specialized-mcp.com"),
            ["general"] = new McpClient("https://general-mcp.com")
        };
        
        // Initialize content classifier
        _classifier = new RoutingClassifier();
    }
    
    public async Task<McpResponse> RouteAndProcessAsync(string prompt, IDictionary<string, object> parameters = null)
    {
        // Classify the prompt to determine the best specialized service
        string category = await _classifier.ClassifyPromptAsync(prompt);
        
        // Get the appropriate client or fall back to general
        var client = _specializedClients.ContainsKey(category) 
            ? _specializedClients[category] 
            : _specializedClients["general"];
            
        Console.WriteLine($"Routing request to {category} specialized service");
        
        // Send request to the selected service
        return await client.SendPromptAsync(prompt, parameters);
    }
    
    // Simple classifier for routing decisions
    private class RoutingClassifier
    {
        public Task<string> ClassifyPromptAsync(string prompt)
        {
            prompt = prompt.ToLowerInvariant();
            
            if (prompt.Contains("code") || prompt.Contains("function") || 
                prompt.Contains("program") || prompt.Contains("algorithm"))
            {
                return Task.FromResult("code");
            }
            
            if (prompt.Contains("story") || prompt.Contains("creative") || 
                prompt.Contains("imagine") || prompt.Contains("design"))
            {
                return Task.FromResult("creative");
            }
            
            if (prompt.Contains("science") || prompt.Contains("research") || 
                prompt.Contains("analyze") || prompt.Contains("study"))
            {
                return Task.FromResult("scientific");
            }
            
            return Task.FromResult("general");
        }
    }
}
```

- бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһ¶бһҡбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһўбһ¶бһҹбҹҠбһёбһ“бһҖбҹ’бһҡбҹ„бһҳбһ”бҹӮбһ” asynchronous бһҠбһҫбһҳбҹ’бһ”бһёбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбҹҶбһҺбһҫбһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бҹ”

- бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ…бҹҶбһҺбһ¶бһҸбҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢбһҹбһ¶бһҳбһүбҹ’бһүбһҳбһҪбһҷбһҠбҹӮбһӣбһңбһ·бһ—бһ¶бһӮбһҹбҹҶбһҺбһҫ бһ бһҫбһҷбһҸбҹ’бһҡбһӣбһ”бҹӢбһ”бҹ’бһҡбһ—бҹҒбһ‘бһҹбһҳбһҡбһҳбҹ’бһҷбҹ”
- бһ”бҹ’бһҡбһҫбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҮбһ¶бһҹбҹ’бһ‘бһҫбһҡбһҸбҹӮбһҠбһҫбһҳбҹ’бһ”бһёбһ•бҹ’бһүбһҫбһҹбҹҶбһҺбһҫ бһ“бһ·бһ„бһ‘бһ‘бһҪбһӣбһҖбһ¶бһҡбһҶбҹ’бһӣбһҫбһҷбһҸбһ”бҹ”
- бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҖбһҡбһҺбһёбһҠбҹӮбһӣбһҹбҹҶбһҺбһҫбһҳбһ·бһ“бһҸбҹ’бһҡбһјбһңбһ“бһ№бһ„бһ”бҹ’бһҡбһ—бҹҒбһ‘бһҮбһ¶бһҹбҹ’бһ‘бһҫбһҡбһҸбҹӮбһҺбһ¶бһҳбһҪбһҷбһҠбҹ„бһҷбһ”бһүбҹ’бһҮбһјбһ“бһ‘бҹ…бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһ‘бһјбһ‘бҹ…бҹ”

</details>

## бһҸбһ»бһӣбҹ’бһҷбһ—бһ¶бһ–бһ•бҹ’бһ‘бһ»бһҖбһҶбҹ’бһӣбһ¶бһҸбһңбҹғ

бһҸбһ»бһӣбҹ’бһҷбһ—бһ¶бһ–бһ•бҹ’бһ‘бһ»бһҖбһ’бҹ’бһңбһҫбһІбҹ’бһҷбһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ’бһ“бһ’бһ¶бһ“ бһ“бһ·бһ„бһ’бһ¶бһ“бһ¶бһӣбһ‘бҹ’бһ’бһ•бһӣбһҒбҹ’бһ–бһҹбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳ MCPбҹ” бһҳбһ¶бһ“бһңбһ·бһ’бһёбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҸбһ»бһӣбҹ’бһҷбһ—бһ¶бһ–бһ•бҹ’бһ‘бһ»бһҖ бһҠбһјбһ…бһҮбһ¶ round-robin, weighted response time, бһ¬бһҷбһ»бһ‘бҹ’бһ’бһ“бһ¶бһҖбһ¶бһҡбһҠбҹӮбһӣбһҮбҹ’бһҡбһ¶бһ”бһҒбҹ’бһӣбһ№бһҳбһҹбһ¶бһҡбҹ”

бһҳбһҖбһҳбһҫбһӣбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҠбҹӮбһӣбһ”бҹ’бһҡбһҫбһҷбһ»бһ‘бҹ’бһ’бһ“бһ¶бһҖбһ¶бһҡбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳ:

- **Round Robin**: бһ…бҹӮбһҖбһ…бһ¶бһҷбһҹбҹҶбһҺбһҫбһ…бҹҒбһүбһҸбһ¶бһҳбһ’бһҳбҹ’бһҳбһҸбһ¶бһ‘бҹ…бһӣбһҫбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒбһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбҹ’бһҡбһ¶бһ”бҹӢбҹ”
- **Weighted Response Time**: бһ”бһүбҹ’бһҮбһјбһ“бһҹбҹҶбһҺбһҫбһ‘бҹ…бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒбһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһ…бҹҶбһӣбһҫбһҷбһ—бһ¶бһӮбһҡбһҷбҹ”
- **Content-Aware**: бһ”бһүбҹ’бһҮбһјбһ“бһҹбҹҶбһҺбһҫбһ‘бҹ…бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒбһҮбһ¶бһҹбҹ’бһ‘бһҫбһҡбһҸбҹӮбһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһҒбҹ’бһӣбһ№бһҳбһҹбһ¶бһҡбһ“бҹғбһҹбҹҶбһҺбһҫбҹ”

<details>
<summary>Java</summary>

```java
// бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ Java: бһҖбһ¶бһҡбһ…бҹӮбһҖбһ…бһ¶бһҷбһ”бһ“бҹ’бһ‘бһ»бһҖбһҶбҹ’бһӣбһ¶бһҸбһңбҹғбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ MCP
public class McpLoadBalancer {
    private final List<McpServerNode> serverNodes;
    private final LoadBalancingStrategy strategy;
    
    public McpLoadBalancer(List<McpServerNode> nodes, LoadBalancingStrategy strategy) {
        this.serverNodes = new ArrayList<>(nodes);
        this.strategy = strategy;
    }
    
    public McpResponse processRequest(McpRequest request) {
        // бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫбһҠбҹҸбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡ
        McpServerNode selectedNode = strategy.selectNode(serverNodes, request);
        
        try {
            // бһ”бһүбҹ’бһҮбһјбһ“бһҹбҹҶбһҺбһҫбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһҖбҹҶбһҺбһҸбҹӢбһҸбҹҶбһҺбһҡбһҠбҹӮбһӣбһ”бһ¶бһ“бһҮбҹ’бһҡбһҫбһҹ
            return selectedNode.processRequest(request);
        } catch (Exception e) {
            // бһҡбҹҖбһ”бһ…бҹҶбһ–бҹҒбһӣбһҳбһ¶бһ“бһ”бһҡбһ¶бһҮбҹҗбһҷ - бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҮбһҳбҹ’бһҡбһ»бһүбһҳбҹ’бһҸбһ„бһ‘бҹҖбһҸ бһ¬бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ”бҹ’бһҸбһјбһҡбһҮбһҳбҹ’бһҡбһҫбһҹ
            System.err.println("Error processing request on node " + selectedNode.getId() + ": " + e.getMessage());
            
            // бһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһҖбҹҶбһҺбһҸбҹӢбһҸбҹҶбһҺбҹ’бһҡбһҗбһ¶бһ–бһ·бһҸбһҮбһ¶бһҳбһ¶бһ“бһ”бһүбҹ’бһ бһ¶бһҹбһ»бһҒбһ—бһ¶бһ–
            selectedNode.recordFailure();
            
            // бһ–бҹ’бһҷбһ¶бһҷбһ¶бһҳбһҖбҹҶбһҺбһҸбҹӢбһҸбҹҶбһҺбһҠбҹӮбһӣбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҮбһ¶бһҮбһҳбҹ’бһҡбһҫбһҹбһ”бҹ’бһҸбһјбһҡ
            List<McpServerNode> remainingNodes = new ArrayList<>(serverNodes);
            remainingNodes.remove(selectedNode);
            
            if (!remainingNodes.isEmpty()) {
                McpServerNode fallbackNode = strategy.selectNode(remainingNodes, request);
                return fallbackNode.processRequest(request);
            } else {
                throw new RuntimeException("All MCP server nodes failed to process the request");
            }
        }
    }
    
    // бһ”бһҳбҹ’бһҡбҹӮбһ”бһҳбҹ’бһҡбһҪбһӣбһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҹбһ»бһҒбһ—бһ¶бһ–бһҖбҹҶбһҺбһҸбҹӢбһҸбҹҶбһҺ
    public void startHealthChecks(Duration interval) {
        ScheduledExecutorService scheduler = Executors.newScheduledThreadPool(1);
        scheduler.scheduleAtFixedRate(() -> {
            for (McpServerNode node : serverNodes) {
                try {
                    boolean isHealthy = node.checkHealth();
                    System.out.println("Node " + node.getId() + " health status: " + 
                                      (isHealthy ? "HEALTHY" : "UNHEALTHY"));
                } catch (Exception e) {
                    System.err.println("Health check failed for node " + node.getId());
                    node.setHealthy(false);
                }
            }
        }, 0, interval.toMillis(), TimeUnit.MILLISECONDS);
    }
    
    // бһ…бҹҶбһҺбһ»бһ…бһҮбһҪбһ”бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ…бҹӮбһҖбһ…бһ¶бһҷбһ”бһ“бҹ’бһ‘бһ»бһҖ
    public interface LoadBalancingStrategy {
        McpServerNode selectNode(List<McpServerNode> nodes, McpRequest request);
    }
    
    // бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҮбһ¶бһҡбһ„бҹ’бһңбһ·бһӣбһҮбһ¶бһ”бҹӢбһҡбһ„бҹ’бһңбһ„бҹӢ
    public static class RoundRobinStrategy implements LoadBalancingStrategy {
        private AtomicInteger counter = new AtomicInteger(0);
        
        @Override
        public McpServerNode selectNode(List<McpServerNode> nodes, McpRequest request) {
            List<McpServerNode> healthyNodes = nodes.stream()
                .filter(McpServerNode::isHealthy)
                .collect(Collectors.toList());
            
            if (healthyNodes.isEmpty()) {
                throw new RuntimeException("No healthy nodes available");
            }
            
            int index = counter.getAndIncrement() % healthyNodes.size();
            return healthyNodes.get(index);
        }
    }
    
    // бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ–бҹҒбһӣбһҶбҹ’бһӣбһҫбһҷбһҸбһ”бһҠбҹӮбһӣбһ”бһ¶бһ“бһ•бҹ’бһҠбһӣбҹӢбһ‘бҹҶбһ„бһ“бҹӢ
    public static class ResponseTimeStrategy implements LoadBalancingStrategy {
        @Override
        public McpServerNode selectNode(List<McpServerNode> nodes, McpRequest request) {
            return nodes.stream()
                .filter(McpServerNode::isHealthy)
                .min(Comparator.comparing(McpServerNode::getAverageResponseTime))
                .orElseThrow(() -> new RuntimeException("No healthy nodes available"));
        }
    }
    
    // бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҠбҹӮбһӣбһҳбһ¶бһ“бһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһ–бһёбһҳбһ¶бһҸбһ·бһҖбһ¶бһҡбһҹбһ¶бҹ’бһҸ
    public static class ContentAwareStrategy implements LoadBalancingStrategy {
        @Override
        public McpServerNode selectNode(List<McpServerNode> nodes, McpRequest request) {
            // бһҖбҹҶбһҺбһҸбҹӢбһӣбһҖбҹ’бһҒбһҺбҹҲбһҹбҹҶбһҺбһҫ
            boolean isCodeRequest = request.getPrompt().contains("code") || 
                                   request.getAllowedTools().contains("codeInterpreter");
            
            boolean isCreativeRequest = request.getPrompt().contains("creative") || 
                                       request.getPrompt().contains("story");
            
            // бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҖбҹҶбһҺбһҸбҹӢбһҸбҹҶбһҺбһҠбҹӮбһӣбһҳбһ¶бһ“бһҮбҹҶбһ“бһ¶бһүбһ–бһ·бһҹбҹҒбһҹ
            Optional<McpServerNode> specializedNode = nodes.stream()
                .filter(McpServerNode::isHealthy)
                .filter(node -> {
                    if (isCodeRequest && node.getSpecialization().equals("code")) {
                        return true;
                    }
                    if (isCreativeRequest && node.getSpecialization().equals("creative")) {
                        return true;
                    }
                    return false;
                })
                .findFirst();
            
            // бһҸбҹ’бһҡбһЎбһ”бҹӢбһҖбҹҶбһҺбһҸбҹӢбһҸбҹҶбһҺбһ–бһ·бһҹбҹҒбһҹ бһ¬бһҖбҹҶбһҺбһҸбҹӢбһҸбҹҶбһҺбһҠбҹӮбһӣбһҳбһ¶бһ“бһ”бһ“бҹ’бһ‘бһ»бһҖбһҸбһ·бһ…бһ”бҹҶбһ•бһ»бһҸ
            return specializedNode.orElse(
                nodes.stream()
                    .filter(McpServerNode::isHealthy)
                    .min(Comparator.comparing(McpServerNode::getCurrentLoad))
                    .orElseThrow(() -> new RuntimeException("No healthy nodes available"))
            );
        }
    }
}
```

бһҖбҹ’бһ“бһ»бһ„бһҖбһјбһҠбһҒбһ¶бһ„бһӣбһҫ бһҷбһҫбһ„бһ”бһ¶бһ“бҹ–

- бһ”бһ„бҹ’бһҖбһҫбһҸбһҗбҹ’бһ“бһ¶бһҖбҹӢ `McpLoadBalancer` бһҠбҹӮбһӣбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ”бһүбҹ’бһҮбһёбһңбҹүбҹӮбһ”бһ•бҹ’бһ‘бһ»бһҖбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP бһ бһҫбһҷбһ”бһүбҹ’бһҮбһјбһ“бһҹбҹҶбһҺбһҫбһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһҷбһ»бһ‘бҹ’бһ’бһ“бһ¶бһҖбһ¶бһҡбһҸбһ»бһӣбҹ’бһҷбһ—бһ¶бһ–бһ•бҹ’бһ‘бһ»бһҖбһҠбҹӮбһӣбһ”бһ¶бһ“бһҮбҹ’бһҡбһҫбһҹбҹ”
- бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҷбһ»бһ‘бҹ’бһ’бһ“бһ¶бһҖбһ¶бһҡбһҸбһ»бһӣбҹ’бһҷбһ—бһ¶бһ–бһ•бҹ’бһ‘бһ»бһҖбһ•бҹ’бһҹбҹҒбһ„бҹ—: `RoundRobinStrategy`, `ResponseTimeStrategy`, бһ“бһ·бһ„ `ContentAwareStrategy`бҹ”
- бһ”бҹ’бһҡбһҫ `ScheduledExecutorService` бһҠбһҫбһҳбҹ’бһ”бһёбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҹбһ»бһҒбһ—бһ¶бһ–бһ“бҹғбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒбһҮбһ¶бһҗбҹ’бһҳбһёбҹ—бһҡбҹҖбһ„бһҡбһ¶бһӣбҹӢбһҡбһҷбҹҲбһ–бҹҒбһӣбҹ”
- бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҷбһ“бҹ’бһҸбһҖбһ¶бһҡбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҹбһ»бһҒбһ—бһ¶бһ– бһҠбҹӮбһӣбһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒбһҮбһ¶бһҹбһ»бһҒбһ—бһ¶бһ– бһ¬бһҳбһ·бһ“бһҹбһ»бһҒбһ—бһ¶бһ–бһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһҖбһ¶бһҡбһҶбҹ’бһӣбһҫбһҷбһҸбһ”бҹ”
- бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҹбҹҶбһҺбһҫбһҠбҹ„бһҷбһҳбһ¶бһ“бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбҹҶбһ бһ»бһҹ бһ“бһ·бһ„бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһ”бҹ’бһҡбһ¶бһҖбҹӢбһҸбҹ’бһҡбһЎбһ”бҹӢбһҠбһҫбһҳбҹ’бһ”бһёбһ’бһ¶бһ“бһ¶бһӣбһ‘бҹ’бһ’бһ•бһӣбһҒбҹ’бһ–бһҹбҹӢбҹ”
- бһ”бҹ’бһҡбһҫбһҗбҹ’бһ“бһ¶бһҖбҹӢ `McpServerNode` бһҠбһҫбһҳбҹ’бһ”бһёбһҸбҹҶбһҺбһ¶бһ„бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP бһҜбһҖбһҮбһ“ бһҡбһҪбһҳбһҳбһ¶бһ“бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҹбһ»бһҒбһ—бһ¶бһ– бһ–бҹҒбһӣбһ…бҹҶбһӣбһҫбһҷбһҳбһ’бҹ’бһҷбһҳ бһ“бһ·бһ„бһ•бҹ’бһ‘бһ»бһҖбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бҹ”
- бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҗбҹ’бһ“бһ¶бһҖбҹӢ `McpRequest` бһҠбһҫбһҳбҹ’бһ”бһёбһ•бҹ’бһ‘бһ»бһҖбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһӣбһҳбҹ’бһўбһ·бһҸбһҹбҹҶбһҺбһҫбһҠбһјбһ…бһҮбһ¶ бһҹбҹҶбһҺбһҫ бһ“бһ·бһ„бһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣбһүбһ¶бһ”бҹӢбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбҹ”
- бһ”бҹ’бһҡбһҫ Java Streams бһҠбһҫбһҳбҹ’бһ”бһёбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҸбһҳбҹ’бһҡбҹҖбһ” бһ“бһ·бһ„бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒбһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһҹбһ»бһҒбһ—бһ¶бһ– бһ“бһ·бһ„бһҖбһ¶бһҡбһҮбһ¶бһҹбҹ’бһ‘бһҫбһҡбһҸбҹӮбҹ”

## бһҖбһ¶бһҡбһ•бҹ’бһӣбһјбһңбһ…бһҡбһ¶бһ…бһҡбһҺбҹҚбһ§бһ”бһҖбһҡбһҺбҹҚбһ‘бһёбһ•бҹ’бһҹбһ¶бһҡбһҮбһҝбһ“бһӣбһҝбһ“

бһҖбһ¶бһҡбһ•бҹ’бһӣбһјбһңбһ…бһҡбһ¶бһ…бһҡбһҺбҹҚбһ§бһ”бһҖбһҡбһҺбҹҚбһ’бһ¶бһ“бһ¶бһҗбһ¶бһҖбһ¶бһҡбһ бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһүбҹ’бһҮбһјбһ“бһ‘бҹ…бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһҹбһҳбһҡбһҳбҹ’бһҷбһ”бҹҶбһ•бһ»бһҸбһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһ”бһҡбһ·бһ”бһ‘бҹ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ бһҖбһ¶бһҡбһ бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚбһўбһ¶бһҖбһ¶бһҹбһ’бһ¶бһҸбһ»бһўбһ¶бһ…бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһүбҹ’бһҮбһјбһ“бһ‘бҹ…бһ…бҹҶбһҺбһ»бһ…бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһҸбҹҶбһ”бһ“бҹӢбһҸбһ¶бһҳбһ‘бһёбһҸбһ¶бҹҶбһ„бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ бһ¬бһ§бһ”бһҖбһҡбһҺбҹҚбһӮбһҺбһ“бһ¶бһўбһ¶бһ…бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһҮбһ¶бһҳбһҪбһҷбһҖбҹҶбһҺбҹӮбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбһ“бҹғ APIбҹ”

бһ…бһ„бҹӢбһҹбҹ’бһӮбһ¶бһӣбҹӢбһ–бһёбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһҖбһ¶бһҡбһ•бҹ’бһӣбһјбһңбһ…бһҡбһ¶бһ…бһҡбһҺбҹҚбһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣбһўбһ¶бһ…бһ”бһҸбҹӢбһ”бҹӮбһ“бһ”бһ¶бһ“бһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһңбһ·бһ—бһ¶бһӮбһҹбҹҶбһҺбһҫ бһ…бҹҶбһҺбһ»бһ…бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһҸбҹҶбһ”бһ“бҹӢ бһ“бһ·бһ„бһҖбһ¶бһҡбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҖбҹҶбһҺбҹӮбҹ”

```python
# бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ Python: бһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһјбһ“бһ§бһ”бһҖбһҡбһҺбҹҚбһҢбһёбһҺбһ¶бһҳбһ·бһ…бһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһңбһ·бһ—бһ¶бһӮбһҹбҹҶбһҺбһҫ
class McpToolRouter:
    def __init__(self):
        # бһ…бһ»бҹҮбһ”бһүбҹ’бһҮбһёбһ…бҹҶбһ“бһ»бһ…бһ”бһүбҹ’бһ…бһ”бҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣбһҳбһ¶бһ“
        self.tool_endpoints = {
            "weatherTool": "https://weather-service.example.com/api",
            "calculatorTool": "https://calculator-service.example.com/compute",
            "databaseTool": "https://database-service.example.com/query",
            "searchTool": "https://search-service.example.com/search"
        }
        
        # бһ…бҹҶбһ“бһ»бһ…бһ”бһүбҹ’бһ…бһ”бҹӢбһҸбҹҶбһ”бһ“бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ…бҹӮбһҖбһ…бһ¶бһҷбһҮбһ¶бһҖбһ·бһ…бҹ’бһ…бһҠбҹӮбһ“бһҹбһҖбһӣ
        self.regional_endpoints = {
            "us": {
                "weatherTool": "https://us-west.weather-service.example.com/api",
                "searchTool": "https://us.search-service.example.com/search"
            },
            "europe": {
                "weatherTool": "https://eu.weather-service.example.com/api",
                "searchTool": "https://eu.search-service.example.com/search"
            },
            "asia": {
                "weatherTool": "https://asia.weather-service.example.com/api",
                "searchTool": "https://asia.search-service.example.com/search"
            }
        }
        
        # бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҖбҹҶбһҺбҹӮбһ§бһ”бһҖбһҡбһҺбҹҚ
        self.tool_versions = {
            "weatherTool": {
                "default": "v2",
                "v1": "https://weather-service.example.com/api/v1",
                "v2": "https://weather-service.example.com/api/v2",
                "beta": "https://weather-service.example.com/api/beta"
            }
        }
    
    async def route_tool_request(self, tool_name, parameters, user_context=None):
        """Route a tool request to the appropriate endpoint based on context"""
        endpoint = self._select_endpoint(tool_name, parameters, user_context)
        
        if not endpoint:
            raise ValueError(f"No endpoint available for tool: {tool_name}")
        
        # бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҹбҹҶбһҺбһҫбһ–бһ·бһҸбһ‘бҹ…бһ…бҹҶбһ–бҹ„бҹҮбһ…бҹҶбһ“бһ»бһ…бһ”бһүбҹ’бһ…бһ”бҹӢбһҠбҹӮбһӣбһ”бһ¶бһ“бһҮбҹ’бһҡбһҫбһҹ
        return await self._execute_tool_request(endpoint, tool_name, parameters)
    
    def _select_endpoint(self, tool_name, parameters, user_context=None):
        """Select the most appropriate endpoint based on context"""
        # бһ…бҹҶбһ“бһ»бһ…бһ”бһүбҹ’бһ…бһ”бҹӢбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ–бһёбһғбҹ’бһӣбһ¶бҹҶбһ„
        if tool_name not in self.tool_endpoints:
            return None
            
        base_endpoint = self.tool_endpoints[tool_name]
        
        # бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣбһҗбһ¶бһҸбһҫбһҷбһҫбһ„бһҸбҹ’бһҡбһјбһңбһ”бҹ’бһҡбһҫбһҖбҹҶбһҺбҹӮбһ§бһ”бһҖбһҡбһҺбҹҚбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбһ¬бһўбһҸбҹӢ
        if tool_name in self.tool_versions:
            version_info = self.tool_versions[tool_name]
            
            # бһ”бҹ’бһҡбһҫбһҖбҹҶбһҺбҹӮбһ“бһ·бһҷбһҳбһ¬бһҖбҹҶбһҺбҹӮбһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһүбҹ’бһҮбһ¶бһҖбҹӢ
            requested_version = parameters.get("_version", version_info["default"])
            if requested_version in version_info:
                base_endpoint = version_info[requested_version]
        
        # бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһјбһ“бһҸбһ¶бһҳбһҸбҹҶбһ”бһ“бҹӢбһ”бһҫбһҸбҹҶбһ”бһ“бҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бһ¶бһ“бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ¶бһ“бҹӢ
        if user_context and "region" in user_context:
            user_region = user_context["region"]
            
            if user_region in self.regional_endpoints:
                regional_tools = self.regional_endpoints[user_region]
                
                if tool_name in regional_tools:
                    # бһ”бҹ’бһҡбһҫбһ…бҹҶбһ“бһ»бһ…бһ”бһүбҹ’бһ…бһ”бҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбһҮбһ¶бһҸбҹҶбһ”бһ“бҹӢ
                    return regional_tools[tool_name]
        
        # бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбһ¶бһҡбһ‘бһ¶бһҳбһ‘бһ¶бһҡбһҹбҹ’бһ“бһ¶бһҖбҹӢбһ“бҹ…бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ
        if user_context and "data_residency" in user_context:
            # бһ“бҹҒбҹҮбһ“бһ№бһ„бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҸбһ»бһӣбҹ’бһҷбҹҗбһҷбһҠбһҫбһҳбҹ’бһ”бһёбһ’бҹ’бһңбһҫбһІбҹ’бһҷбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһўбһ¶бһҺбһ¶бһ…бһҖбҹ’бһҡбһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһүбҹ’бһҮбһ¶бһҖбҹӢ
            pass
        
        # бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣбһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһјбһ“бһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһ–бҹҒбһӣбһҷбһәбһҸбһҷбҹүбһ¶бһң
        if user_context and "latency_sensitive" in user_context and user_context["latency_sensitive"]:
            # бһ“бҹҒбҹҮбһ“бһ№бһ„бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҸбһ»бһӣбҹ’бһҷбҹҗбһҷбһҠбһҫбһҳбҹ’бһ”бһёбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһ…бҹҶбһ“бһ»бһ…бһ”бһүбҹ’бһ…бһ”бҹӢбһҠбҹӮбһӣбһҳбһ¶бһ“бһ–бҹҒбһӣбһҷбһәбһҸбһҷбҹүбһ¶бһңбһ‘бһ¶бһ”бһ”бҹҶбһ•бһ»бһҸ
            pass
            
        return base_endpoint
        
    async def _execute_tool_request(self, endpoint, tool_name, parameters):
        """Execute the actual tool request to the selected endpoint"""
        try:
            async with aiohttp.ClientSession() as session:
                async with session.post(
                    endpoint,
                    json={"toolName": tool_name, "parameters": parameters},
                    headers={"Content-Type": "application/json"}
                ) as response:
                    if response.status == 200:
                        result = await response.json()
                        return result
                    else:
                        error_text = await response.text()
                        raise Exception(f"Tool execution failed: {error_text}")
        except Exception as e:
            # бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҸбһ»бһӣбҹ’бһҷбҹҗбһҷбһ–бҹ’бһҷбһ¶бһҷбһ¶бһҳбһҳбҹ’бһҸбһ„бһ‘бҹҖбһҸбһ¬бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ”бҹ’бһҠбһјбһҡбһҮбһҳбҹ’бһҡбһҫбһҹ
            print(f"Error executing tool {tool_name} at {endpoint}: {str(e)}")
            raise
```

## бһҹбҹҶбһҺбһ¶бҹҶ бһ“бһ·бһ„бһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳбһҖбһ¶бһҡбһ•бҹ’бһӣбһјбһңбһ…бһҡбһ¶бһ…бһҡбһҺбҹҚбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ MCP

бһҹбҹҶбһҺбһ¶бҹҶбһӮбһәбһҮбһ¶бһ’бһ¶бһҸбһ»бһҹбҹҶбһҒбһ¶бһ“бҹӢбһҳбһҪбһҷбһ“бҹғбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһӮбҹҶбһҡбһј (MCP) бһҠбҹӮбһӣбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһҳбһ¶бһ“бһҖбһ¶бһҡбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҹбҹҶбһҺбһҫ бһ“бһ·бһ„бһҖбһ¶бһҡбһ•бҹ’бһӣбһјбһңбһ…бһҡбһ¶бһ…бһҡбһҺбҹҚбһҷбҹүбһ¶бһ„бһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бҹ” бһңбһ¶ бһҡбһҪбһҳбһҳбһ¶бһ“бһҖбһ¶бһҡбһңбһ·бһ—бһ¶бһӮбһҹбҹҶбһҺбһҫбһҠбҹӮбһӣбһҳбһҖбһҠбһӣбҹӢ бһҠбһҫбһҳбҹ’бһ”бһёбһҖбҹҶбһҺбһҸбҹӢбһӮбҹҶбһҡбһј бһ¬бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһҹбһҳбһҡбһҳбҹ’бһҷбһ”бҹҶбһ•бһ»бһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ–бһҪбһҖбһңбһ¶ бһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһӣбһҖбҹ’бһҒбһҺбҹҲбһ•бҹ’бһҹбҹҒбһ„бҹ—бһҠбһјбһ…бһҮбһ¶бһ”бҹ’бһҡбһ—бҹҒбһ‘бһҒбҹ’бһӣбһ№бһҳбһҹбһ¶бһҡ бһ”бһҡбһ·бһ”бһ‘бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ бһ“бһ·бһ„бһ•бҹ’бһ‘бһ»бһҖбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бҹ”

бһҖбһ¶бһҡбһҹбҹҶбһҺбһ¶бҹҶ бһ“бһ·бһ„бһҖбһ¶бһҡбһ•бҹ’бһӣбһјбһңбһ…бһҡбһ¶бһ…бһҡбһҺбҹҚбһўбһ¶бһ…бһҸбҹ’бһҡбһјбһңбһҸбҹӮбһҳбһҖбһҡбһҪбһҳбһӮбҹ’бһ“бһ¶ бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳбһҡбһ№бһ„бһҳбһ¶бҹҶ бһҠбҹӮбһӣбһ’бҹ’бһңбһҫбһІбҹ’бһҷбһ”бҹ’бһҡбһҹбһҫбһҡбһ“бҹғбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ’бһ“бһ’бһ¶бһ“ бһ“бһ·бһ„бһ’бһ¶бһ“бһ¶бһӣбһ‘бҹ’бһ’бһ•бһӣбһҒбҹ’бһ–бһҹбҹӢбҹ” бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҹбҹҶбһҺбһ¶бҹҶбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ…бҹҶбһҺбһ¶бһҸбҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢбһҹбҹҶбһҺбһҫ бһҒбһҺбҹҲбһҠбҹӮбһӣбһҖбһ¶бһҡбһ•бҹ’бһӣбһјбһңбһ…бһҡбһ¶бһ…бһҡбһҺбҹҚбһ”бһүбҹ’бһҮбһјбһ“бһ–бһҪбһҖбһңбһ¶бһ‘бҹ…бһӮбҹҶбһҡбһј бһ¬бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһҠбҹӮбһӣбһҹбһҳбһҡбһҳбҹ’бһҷбҹ”

бһҡбһјбһ”бһ—бһ¶бһ–бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһҡбһ”бҹҖбһ”бһҹбҹҶбһҺбһ¶бҹҶ бһ“бһ·бһ„бһҖбһ¶бһҡбһ•бҹ’бһӣбһјбһңбһ…бһҡбһ¶бһ…бһҡбһҺбҹҚбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһӮбҹ’бһ“бһ¶ бһҖбҹ’бһ“бһ»бһ„бһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳ MCP бһҠбҹҸбһ‘бһјбһӣбҹҶбһ‘бһјбһӣбһ¶бһҷбҹ–
