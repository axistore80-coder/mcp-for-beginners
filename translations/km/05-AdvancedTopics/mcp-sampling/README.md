# Sampling in Model Context Protocol

Sampling бһӮбһәбһҮаёІаёҹбһёбһҲбһә MCP бһҠбҹӮбһӣбһҳбһ¶бһ“бһўбҹҶбһҺбһ¶бһ… бһҠбҹӮбһӣбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫбһҹбҹ’бһ“бһҫбһҹбһ»бҹҶбһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһ”бҹӢ LLM бһҸбһ¶бһҳбһҡбһҷбҹҲбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҠбҹ„бһҷбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһҳбһ¶бһ“бһўбһ¶бһҖбһ”бҹ’бһ”бһҖбһ·бһҡбһ·бһҷбһ¶ agentic бһҠбҹӮбһӣбһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһү бһҒбһҺбҹҲбһҠбҹӮбһӣбһҖбһ¶бһ“бҹӢбһҖбһ¶бһ”бҹӢбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– бһ“бһ·бһ„бһ—бһ¶бһ–бһҜбһҖбһҮбһ“бҹ” бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҖбһ¶бһҡбһҹбҹ’бһ‘бһ„бҹӢбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһўбһ¶бһ…бһ”бһ„бҹ’бһҖбһҫбһ“бһӮбһ»бһҺбһ—бһ¶бһ–бһ“бһ·бһ„бһӣбҹ’бһ”бһҝбһ“бһ“бҹғбһҖбһ¶бһҡбһҶбҹ’бһӣбһҫбһҷбһҸбһ”бһҷбҹүбһ¶бһ„бһҒбҹ’бһӣбһ¶бҹҶбһ„бҹ” MCP бһ•бҹ’бһҸбһӣбҹӢбһ“бһјбһңбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҸбһҳбҹ’бһҡбһјбһңбһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡ бһҠбһҫбһҳбҹ’бһ”бһёбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһңбһ·бһ’бһёбһҠбҹӮбһӣбһӮбҹҶбһҡбһјбһ”бһ„бҹ’бһҖбһҫбһҸбһўбһҸбҹ’бһҗбһ”бһ‘бһҮбһ¶бһҳбһҪбһҷбһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһ–бһ·бһҹбҹҒбһҹбһҠбҹӮбһӣбһҳбһ¶бһ“бһҘбһ‘бҹ’бһ’бһ·бһ–бһӣбһӣбһҫбһҖбһ¶бһҡбһ…бҹғбһҠбһ“бҹ’бһҷ бһҖбһ¶бһҡбһ…бҹ’бһ“бҹғбһ”бҹ’бһҡбһҢбһ·бһҸ бһ“бһ·бһ„бһ—бһ¶бһ–бһҹбҹ’бһҡбһ”бһҸбһ¶бһҳбһӮбҹ’бһ“бһ¶бҹ”

## бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶ

бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһҷбһҫбһ„бһ“бһ№бһ„бһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһңбһ–бһёбһңбһ·бһ’бһёбһҖбҹҶбһҺбһҸбҹӢбһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһҹбҹ’бһ‘бһ„бҹӢбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҹбҹ’бһ“бһҫбһҡбһҹбһ»бҹҶ MCP бһ“бһ·бһ„бһҠбһ№бһ„бһ–бһёбһҷбһ¶бһ“бһ·бһҖбһҹбҹҶбһҺбһ»бҹҶбһ”бҹӮбһ”бһ”бһ‘бһҹбҹ’бһ‘бһ„бҹӢбһҖбҹ’бһ“бһ»бһ„бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ”

## бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶

бһ“бҹ…бһ…бһ»бһ„бһ”бһүбҹ’бһ…бһ”бҹӢбһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһўбһ¶бһ…:

- бһҷбһӣбҹӢбһҠбһ№бһ„бһўбҹҶбһ–бһёбһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһҹбҹ’бһ‘бһ„бҹӢбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—бһҠбҹӮбһӣбһҳбһ¶бһ“бһҖбҹ’бһ“бһ»бһ„ MCPбҹ”
- бһҖбҹҶбһҺбһҸбҹӢбһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһҹбҹ’бһ‘бһ„бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҡбһҺбһёбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ”
- бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһ¶бһҡбһҹбҹ’бһ‘бһ„бҹӢбһІбҹ’бһҷбһҖбһ¶бһ“бҹӢбһҸбҹӮбһҳбһ¶бһ“бһ—бһ¶бһ–бһҖбҹҶбһҺбһҸбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӣбһ‘бҹ’бһ’бһ•бһӣбһўбһ¶бһ…бһ’бҹ’бһңбһҫбһҳбҹ’бһҸбһ„бһ‘бҹҖбһҸбһ”бһ¶бһ“бҹ”
- бһ”бһҸбҹӢбһ”бҹӮбһ“бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһҹбҹ’бһ‘бһ„бҹӢбһұбҹ’бһҷбһҹбһҳбһҹбҹ’бһҡбһ”бһҸбһ¶бһҳбһ”бһҡбһ·бһ”бһ‘ бһ“бһ·бһ„бһ…бҹҶбһҺбһ„бҹӢбһ…бҹҶбһҺбһјбһӣбһ…бһ·бһҸбҹ’бһҸбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбҹ”
- бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҹбҹ’бһ‘бһ„бҹӢбһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһ“бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ“бҹҚбһҡбһ”бһҹбҹӢбһӮбҹҶбһҡбһјбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҹаё–аёІаёҷаёҒаёІаёЈаё“а№Ңбһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ”
- бһҷбһӣбҹӢбһ–бһёбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһҖбһ¶бһҡбһҹбҹ’бһ‘бһ„бҹӢбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҮбҹҶбһ“бһҪбһүбһҡбһңбһ¶бһ„бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ бһ“бһ·бһ„бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҡбһ”бһҹбҹӢ MCPбҹ”

## бһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһҖбһ¶бһҡбһҹбҹ’бһ‘бһ„бҹӢбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡвҖӢбһҖбҹ’бһ“бһ»бһ„ MCP

бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҹбҹ’бһ‘бһ„бҹӢбһҖбҹ’бһ“бһ»бһ„ MCP бһ”бһ“бҹ’бһҸбһ–бһёбһҮбҹҶбһ бһ¶бһ“бһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбҹ–

1. бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫбһ•бҹ’бһүбһҫбһҹбҹҶбһҺбһҫ `sampling/createMessage` бһ‘бҹ…бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ
2. бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҹбҹҶбһҺбһҫ бһ“бһ·бһ„бһўбһ¶бһ…бһҖбҹӮбһ”бҹ’бһҡбҹӮбһңбһ¶
3. бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһӣбһ‘бҹ’бһ’бһ•бһӣбһ–бһё LLM
4. бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһ”бҹӢ
5. бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҸбҹ’бһҡбһЎбһ”бҹӢбһӣбһ‘бҹ’бһ’бһ•бһӣбһ‘бҹ…бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ

бһҡбһ…бһ“бһ¶бһ”бҹҗбһ‘бҹ’бһҳбһҠбҹӮбһӣбһҳбһ¶бһ“бһҳбһ“бһ»бһҹбҹ’бһҹбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ…бҹҶбһҺбһ»бһ…бһ“бҹҒбҹҮбһ’бһ¶бһ“бһ¶бһҗбһ¶ бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ“бҹ…бһ”бҹ’бһҡбһҫбһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһӣбһҫбһўбҹ’бһңбһёбһҠбҹӮбһӣ LLM бһҳбһҫбһӣбһғбһҫбһү бһ“бһ·бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһЎбһҫбһ„бҹ”

## бһ‘бһ·бһҠбҹ’бһӢбһ—бһ¶бһ–бһ‘бһјбһ‘бҹ…бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһҹбҹ’бһ‘бһ„бҹӢ

MCP бһҖбҹҶбһҺбһҸбҹӢбһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһҹбҹ’бһ‘бһ„бҹӢбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҠбҹӮбһӣбһўбһ¶бһ…бһҖбҹҶбһҺбһҸбҹӢбһ”бһ¶бһ“бһҖбҹ’бһ“бһ»бһ„бһҹбҹҶбһҺбһҫбһҡбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбҹ–

```csharp
// .NET Example: Configuring sampling parameters in MCP
public class SamplingExample
{
    public async Task RunWithSamplingAsync()
    {
        // Create MCP client with sampling configuration
        var client = new McpClient("https://mcp-server-url.com");
        
        // Create request with specific sampling parameters
        var request = new McpRequest
        {
            Prompt = "Generate creative ideas for a mobile app",
            SamplingParameters = new SamplingParameters
            {
                Temperature = 0.8f,     // Higher temperature for more creative outputs
                TopP = 0.95f,           // Nucleus sampling parameter
                TopK = 40,              // Limit token selection to top K options
                FrequencyPenalty = 0.5f, // Reduce repetition
                PresencePenalty = 0.2f   // Encourage diversity
            },
            AllowedTools = new[] { "ideaGenerator", "marketAnalyzer" }
        };
        
        // Send request using specific sampling configuration
        var response = await client.SendRequestAsync(request);
        
        // Output results
        Console.WriteLine($"Generated with Temperature={request.SamplingParameters.Temperature}:");
        Console.WriteLine(response.GeneratedText);
    }
}
```

    - `frequencyPenalty` бһ“бһ·бһ„ `presencePenalty` бһҠбһҫбһҳбҹ’бһ”бһёбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһ¶бһҡбһҖбһҫбһҸбһЎбһҫбһ„бһҳбҹ’бһҸбһ„бһ‘бҹҖбһҸ бһ“бһ·бһ„бһ—бһ¶бһ–бһ…бһҳбҹ’бһҡбһ»бҹҮбһҖбҹ’бһ“бһ»бһ„бһӣбһ‘бҹ’бһ’бһ•бһӣбһ…бҹҒбһүбҹ”
    - `temperature` бһҠбһҫбһҳбҹ’бһ”бһёбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ—бһ¶бһ–бһ…бҹғбһҠбһ“бҹ’бһҷбһҖбҹ’бһ“бһ»бһ„бһӣбһ‘бҹ’бһ’бһ•бһӣ бһҠбҹӮбһӣбһҸбһҳбҹ’бһӣбҹғбһҒбҹ’бһ–бһҹбҹӢбһ“бһ¶бҹҶбһІбҹ’бһҷбһҳбһ¶бһ“бһ…бһҳбҹ’бһӣбһҫбһҷбһ…бҹ’бһ“бҹғбһ”бҹ’бһҡбһҢбһ·бһҸбһ…бҹ’бһҡбһҫбһ“бҹ”
    - `top_p` бһҠбһҫбһҳбҹ’бһ”бһёбһҖбҹҶбһҺбһҸбҹӢбһҖбһ¶бһҡбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҸбһјбһҖбһ·бһ“бһ‘бҹ…бһҸбҹӮбһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҹбһҡбһ»бһ”бһҒбҹ’бһ–бһҹбҹӢбһ”бҹҶбһ•бһ»бһҸ, бһҖбҹӮбһӣбһҳбҹ’бһўбһӮбһ»бһҺбһ—бһ¶бһ–бһўбһҸбҹ’бһҗбһ”бһ‘бһ”бһ„бҹ’бһҖбһҫбһҸбҹ”
    - `top_k` бһҠбһҫбһҳбҹ’бһ”бһёбһҖбҹҶбһҺбһҸбҹӢбһҖбҹҶбһҺбһҸбҹӢбһӮбҹҶбһҡбһјбһ‘бҹ…бһҸбҹӮ K бһҸбһјбһҖбһ·бһ“бһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҒбҹ’бһ–бһҹбҹӢбһ”бҹҶбһ•бһ»бһҸ, бһҮбһҪбһҷбһ•бһӣбһ·бһҸбһӣбһ‘бҹ’бһ’бһ•бһӣбһҳбһ¶бһ“бһ—бһ¶бһ–бһҮбһ¶бһ”бҹӢбһ‘бһ¶бһҖбҹӢбһ‘бһ„бһӣбҹ’бһўбҹ”
    - `frequencyPenalty` бһ“бһ·бһ„ `presencePenalty` бһҠбһҫбһҳбҹ’бһ”бһёбһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһҖбһ¶бһҡбһ”бһүбҹ’бһ…бҹҒбһүбһҳбҹ’бһҸбһ„бһ‘бҹҖбһҸ бһ бһҫбһҷбһӣбһҫбһҖбһ‘бһ№бһҖбһ…бһ·бһҸбҹ’бһҸбһ—бһ¶бһ–бһ…бһҳбҹ’бһҡбһ»бҹҮбһҖбҹ’бһ“бһ»бһ„бһўбһҸбҹ’бһҗбһ”бһ‘бһ”бһ„бҹ’бһҖбһҫбһҸбҹ”

- бһ”бҹ„бҹҮбһ–бһ»бһҳбҹ’бһ–бһӣбһ‘бҹ’бһ’бһ•бһӣбһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸ бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбҹЎбҹ бһӣбһ‘бҹ’бһ’бһ•бһӣбһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһҹбҹ’бһ‘бһ„бҹӢбһҒбһ»бһҹбҹ—бһӮбҹ’бһ“бһ¶бҹ”
- бһ”бҹ’бһҡбһҫ `allowedTools` бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣбһӮбҹҶбһҡбһјбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ–бҹҒбһӣбһ”бһ„бҹ’бһҖбһҫбһҸбҹ” бһ“бҹ…бһ‘бһёбһ“бҹҒбҹҮ бһҷбһҫбһ„бһ”бһ¶бһ“бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһ·бһ§бһ”бһҖбһҡбһҺбҹҚ `ideaGenerator` бһ“бһ·бһ„ `environmentalImpactTool` бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһ…бҹ’бһ“бҹғбһ”бҹ’бһҡбһҢбһ·бһҸ бһ бһҫбһҷ `factChecker` бһ“бһ·бһ„ `dataAnalysisTool` бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠбҹ”
- бһ”бҹ’бһҡбһҫ `temperature` бһҠбһҫбһҳбҹ’бһ”бһёбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ—бһ¶бһ–бһ…бҹғбһҠбһ“бҹ’бһҷбһ“бҹғбһӣбһ‘бҹ’бһ’бһ•бһӣ бһҠбҹӮбһӣбһҸбһҳбҹ’бһӣбҹғбһҒбҹ’бһ–бһҹбҹӢбһ“бһ¶бҹҶбһІбҹ’бһҷбһҳбһ¶бһ“бһ…бһҳбҹ’бһӣбһҫбһҷбһ…бҹ’бһ“бҹғбһ”бҹ’бһҡбһҢбһ·бһҸбһ…бҹ’бһҡбһҫбһ“бҹ”
- бһ”бҹ’бһҡбһҫ `top_p` бһҠбһҫбһҳбҹ’бһ”бһёбһҖбҹҶбһҺбһҸбҹӢбһҖбһ¶бһҡбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҸбһјбһҖбһ·бһ“бһ‘бҹ…бһ‘бҹ…бһҖбҹҶбһ–бһјбһӣбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҹбһҡбһ»бһ”, бһҖбҹӮбһӣбһҳбҹ’бһўбһӮбһ»бһҺбһ—бһ¶бһ–бһўбһҸбҹ’бһҗбһ”бһ‘бһ”бһ„бҹ’бһҖбһҫбһҸбҹ”
- бһ”бҹ’бһҡбһҫ `frequencyPenalty` бһ“бһ·бһ„ `presencePenalty` бһҠбһҫбһҳбҹ’бһ”бһёбһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһҖбһ¶бһҡбһ”бһүбҹ’бһ…бҹҒбһүбһҳбҹ’бһҸбһ„бһ‘бҹҖбһҸ бһ бһҫбһҷбһӣбһҫбһҖбһ‘бһ№бһҖбһ…бһ·бһҸбҹ’бһҸбһ—бһ¶бһ–бһ…бһҳбҹ’бһҡбһ»бҹҮбһҖбҹ’бһ“бһ»бһ„бһӣбһ‘бҹ’бһ’бһ•бһӣбһ…бҹҒбһүбҹ”
- бһ”бҹ’бһҡбһҫ `top_k` бһҠбһҫбһҳбҹ’бһ”бһёбһҖбҹҶбһҺбһҸбҹӢбһҖбҹҶбһҺбһҸбҹӢбһӮбҹҶбһҡбһјбһ‘бҹ…бһҸбҹӮ K бһҸбһјбһҖбһ·бһ“бһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҒбҹ’бһ–бһҹбҹӢбһ”бҹҶбһ•бһ»бһҸ, бһҮбһҪбһҷбһ•бһӣбһ·бһҸбһ…бһҳбҹ’бһӣбһҫбһҷбһҳбһ¶бһ“бһ—бһ¶бһ–бһҮбһ¶бһ”бҹӢбһ‘бһ¶бһҖбҹӢбһ‘бһ„бһӣбҹ’бһўбҹ”

## Sampling бһҖбҹҶбһҺбһҸбҹӢвҖӢбһ”бһ¶бһ“вҖӢбһ”бҹ’бһҡбһҖбһ”бһҠбҹ„бһҷбһ—бһ¶бһ–бһҖбҹҶбһҺбһҸбҹӢ

бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбҹӮбһӣбһҸбһҳбҹ’бһҡбһјбһңбһІбҹ’бһҷбһҳбһ¶бһ“бһӣбһ‘бҹ’бһ’бһ•бһӣбһҹбҹҠбһ№бһҳбһҮбһ¶бһ”бҹӢ бһҷбһ¶бһ“бһ·бһҖбһҹбҹ’бһ‘бһ„бҹӢбһҳбһ¶бһ“бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһ’бҹ’бһңбһҫбһІбҹ’бһҷбһҳбһ¶бһ“бһӣбһ‘бҹ’бһ’бһ•бһӣбһўбһ¶бһ…бһ’бҹ’бһңбһҫбһҳбҹ’бһҸбһ„бһ‘бҹҖбһҸбһ”бһ¶бһ“бҹ” бһңбһ¶бһ’бҹ’бһңбһҫбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһ…бҹҶбһҺбһ¶бҹҶбһ…бҹғбһҠбһ“бҹ’бһҷбһҗбҹҒбһҡ бһ“бһ·бһ„бһҖбҹҶбһҺбһҸбҹӢбһ‘бҹҶбһ–бҹҗбһҡбһҒбҹ’бһ–бһҹбҹӢбһ‘бҹ…бһҹбһјбһ“бҹ’бһҷбҹ”

бһҳбһҖбһҳбһҫбһӣбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ“бҹҚбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ„бҹ’бһ бһ¶бһүбһҹбҹ’бһ‘бһ„бҹӢбһҖбҹҶбһҺбһҸбҹӢбһ”бһ¶бһ“бһҖбҹ’бһ“бһ»бһ„бһ—бһ¶бһҹбһ¶бһҖбһјбһҠбһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ”

```javascript
    const response1 = await client.sendPrompt(prompt, {
```

## бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҹбҹ’бһ‘бһ„бҹӢбһҶбҹ’бһӣбһ¶бһҸбһңбҹғ

бһҖбһ¶бһҡбһҹбҹ’бһ‘бһ„бҹӢбһҶбҹ’бһӣбһ¶бһҸбһңбҹғбһ”бһҸбҹӢбһ”бҹӮбһ“бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһҸбһ¶бһҳбһ”бһҡбһ·бһ”бһ‘ бһ“бһ·бһ„бһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҡбһ”бһҹбҹӢбһҹбҹҶбһҺбһҫбһ“бһёбһҳбһҪбһҷбҹ—бҹ” бһ“бҹҒбҹҮбһҳбһ¶бһ“бһ“бҹҗбһҷбһҗбһ¶бһҖбһ¶бһҡбһ”бһҸбҹӢбһ”бҹӮбһ“бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһҠбһјбһ…бһҮбһ¶ temperature, top_p, бһ“бһ·бһ„бһ–бһ·бһ“бҹҗбһҷ бһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбһҸбһ¶бһҳбһ”бҹ’бһҡбһ—бҹҒбһ‘бһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ… бһ…бҹҶбһҺбһ„бҹӢбһ…бҹҶбһҺбһјбһӣбһ…бһ·бһҸбҹ’бһҸбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ бһ¬бһӣбһ‘бҹ’бһ’бһ—бһ¶бһ–бһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбҹ”

бһҳбһҖбһҳбһҫбһӣбһҡбһ”бҹҖбһ”бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһ¶бһҡбһҹбҹ’бһ‘бһ„бҹӢбһҶбҹ’бһӣбһ¶бһҸбһңбҹғбһҖбҹ’бһ“бһ»бһ„бһ—бһ¶бһҹбһ¶бһҖбһјбһҠбһҒбһ»бһҹбҹ—бһӮбҹ’бһ“бһ¶бҹ”

```python
# бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ Pythonбҹ– бһҖбһ¶бһҡбһ‘бһ¶бһүбһҷбһҖбһӮбҹҶбһҡбһјбһҜбһҖбһҡбһ¶бһҮбҹ’бһҷбһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһ”бһҡбһ·бһ”бһ‘бһҹбҹҶбһҺбһҫ
class DynamicSamplingService:
    def __init__(self, mcp_client):
        self.client = mcp_client
        
    async def generate_with_adaptive_sampling(self, prompt, task_type, user_preferences=None):
        """Uses different sampling strategies based on task type and user preferences"""
        
        # бһҖбҹҶбһҺбһҸбҹӢбһҖбһ¶бһҡбһ”бҹ’бһҡбһҳбһјбһӣбһӮбҹҶбһҡбһјбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһ—бҹҒбһ‘бһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһ•бҹ’бһҹбҹҒбһ„бҹ—
        sampling_presets = {
            "creative": {"temperature": 0.9, "top_p": 0.95, "frequency_penalty": 0.7},
            "factual": {"temperature": 0.2, "top_p": 0.85, "frequency_penalty": 0.2},
            "code": {"temperature": 0.3, "top_p": 0.9, "frequency_penalty": 0.5},
            "analytical": {"temperature": 0.4, "top_p": 0.92, "frequency_penalty": 0.3}
        }
        
        # бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“
        sampling_params = sampling_presets.get(task_type, sampling_presets["factual"])
        
        # бһҖбҹӮбһҸбһҳбҹ’бһҡбһјбһңбһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһ…бҹҶбһҺбһјбһӣбһ…бһ·бһҸбҹ’бһҸбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ бһ”бһҫбһҳбһ¶бһ“
        if user_preferences:
            if "creativity_level" in user_preferences:
                # бһ бһҸбҹ’бһҗбһ–бһӣбһ—бһ¶бһ–бһҹбһёбһҸбһ»бһҺбҹ’бһ бһ—бһ¶бһ–бһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһ…бҹҶбһҺбһјбһӣбһ…бһ·бһҸбҹ’бһҸбһ“бҹғбһҖбһ¶бһҡбһ…бҹ’бһ“бҹғбһ”бҹ’бһҡбһҢбһ·бһҸ (1-10)
                creativity = min(max(user_preferences["creativity_level"], 1), 10) / 10
                sampling_params["temperature"] = 0.1 + (0.9 * creativity)
            
            if "diversity" in user_preferences:
                # бһҖбҹӮбһҸбһҳбҹ’бһҡбһјбһң top_p бһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһ—бһ¶бһ–бһ…бһҳбҹ’бһҡбһ»бҹҮбһ“бҹғбһ…бһҳбҹ’бһӣбһҫбһҷбһҠбҹӮбһӣбһ…бһ„бҹӢбһ”бһ¶бһ“
                diversity = min(max(user_preferences["diversity"], 1), 10) / 10
                sampling_params["top_p"] = 0.6 + (0.39 * diversity)
        
        # бһ”бһ„бҹ’бһҖбһҫбһҸбһ“бһ·бһ„бһ•бҹ’бһүбһҫбһҹбҹҶбһҺбһҫбһҮбһ¶бһҳбһҪбһҷбһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһҖбһ¶бһҡбһ”бҹ’бһҡбһҳбһјбһӣбһӮбҹҶбһҡбһјбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“
        response = await self.client.send_request(
            prompt=prompt,
            temperature=sampling_params["temperature"],
            top_p=sampling_params["top_p"],
            frequency_penalty=sampling_params["frequency_penalty"]
        )
        
        # бһҸбҹ’бһҡбһЎбһ”бҹӢбһ…бһҳбҹ’бһӣбһҫбһҷбһҮбһ¶бһҳбһҪбһҷбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҳбҹҒбһҸбһ¶бһўбҹҶбһ–бһёбһҖбһ¶бһҡбһ”бҹ’бһҡбһҳбһјбһӣбһӮбҹҶбһҡбһјбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бһ¶бһ–бһ”бһҡбһ·бһҷбһ¶бһҷбһ—бһ¶бһ–бһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢ
        return {
            "text": response.generated_text,
            "applied_sampling": sampling_params,
            "task_type": task_type
        }
```

- бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҹбҹ’бһ‘бһ„бҹӢбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһ”бҹ’бһҡбһ—бҹҒбһ‘бһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бҹ”
- бһ”бһҸбҹӢбһ”бҹӮбһ“бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһҹбҹ’бһ‘бһ„бҹӢбһҸбһ¶бһҳбһ…бҹҶбһҺбһ„бҹӢбһ…бҹҶбһҺбһјбһӣбһ…бһ·бһҸбҹ’бһҸбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ бһҠбһјбһ…бһҮбһ¶бһҖбһҳбҹ’бһҡбһ·бһҸбһ…бҹ’бһ“бҹғбһ”бҹ’бһҡбһҢбһ·бһҸ бһ“бһ·бһ„бһ—бһ¶бһ–бһ…бһҳбҹ’бһҡбһ»бҹҮбҹ”
- бһ•бҹ’бһүбһҫбһҹбҹҶбһҺбһҫбһҮбһ¶бһҳбһҪбһҷбһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһҹбҹ’бһ‘бһ„бҹӢбһҠбҹӮбһӣбһҖбҹҶбһҺбһҸбҹӢбһҶбҹ’бһӣбһ¶бһҸбһңбҹғбҹ”
- бһҸбҹ’бһҡбһЎбһ”бҹӢбһўбһҸбҹ’бһҗбһ”бһ‘бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһҪбһҳбһҮбһ¶бһҳбһҪбһҷбһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһҹбҹ’бһ‘бһ„бҹӢ бһўбҹҮбһўбһ¶бһ„бһҖбһ¶бһҡбһ”бҹҶбһ–бҹҒбһү бһ“бһ·бһ„бһ”бҹ’бһҡбһ—бҹҒбһ‘бһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бһ¶бһ–бһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢбҹ”
- бһ”бҹ’бһҡбһҫ `temperature` бһҠбһҫбһҳбҹ’бһ”бһёбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ—бһ¶бһ–бһ…бҹғбһҠбһ“бҹ’бһҷбһҖбҹ’бһ“бһ»бһ„бһӣбһ‘бҹ’бһ’бһ•бһӣ бһҠбҹӮбһӣбһҸбһҳбҹ’бһӣбҹғбһҒбҹ’бһ–бһҹбҹӢбһ“бһ¶бҹҶбһІбҹ’бһҷбһҳбһ¶бһ“бһ…бһҳбҹ’бһӣбһҫбһҷбһ…бҹ’бһ“бҹғбһ”бҹ’бһҡбһҢбһ·бһҸбһ…бҹ’бһҡбһҫбһ“бҹ”
- бһ”бҹ’бһҡбһҫ `top_p` бһҠбһҫбһҳбҹ’бһ”бһёбһҖбҹҶбһҺбһҸбҹӢбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҸбһјбһҖбһ·бһ“бһ‘бҹ…бһҖбҹҶбһ–бһјбһӣбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҹбһҡбһ»бһ” бһ”бһ„бҹ’бһҖбһҫбһ“бһӮбһ»бһҺбһ—бһ¶бһ–бһўбһҸбҹ’бһҗбһ”бһ‘бһ”бһ„бҹ’бһҖбһҫбһҸбҹ”
- бһ”бҹ’бһҡбһҫ `frequency_penalty` бһҠбһҫбһҳбҹ’бһ”бһёбһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһҖбһ¶бһҡбһ”бһүбҹ’бһ…бҹҒбһүбһҳбҹ’бһҸбһ„бһ‘бҹҖбһҸ бһ“бһ·бһ„бһӣбһҫбһҖбһ‘бһ№бһҖбһ…бһ·бһҸбҹ’бһҸбһ—бһ¶бһ–бһ…бһҳбҹ’бһҡбһ»бҹҮбҹ”
- бһ”бҹ’бһҡбһҫ `user_preferences` бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһҖбҹҶбһҺбһҸбҹӢбһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһҹбҹ’бһ‘бһ„бҹӢбһҸбһ¶бһҳбһҖбһҳбҹ’бһҡбһ·бһҸбһ…бҹ’бһ“бҹғбһ”бҹ’бһҡбһҢбһ·бһҸ бһ“бһ·бһ„бһ—бһ¶бһ–бһ…бһҳбҹ’бһҡбһ»бҹҮбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһҖбҹҶбһҺбһҸбҹӢбҹ”
- бһ”бҹ’бһҡбһҫ `task_type` бһҖбҹҶбһҺбһҸбҹӢбһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҹбҹ’бһ‘бһ„бҹӢбһҹбһҳбһҡбһҳбҹ’бһҷ бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹҶбһҺбһҫбһ“бһёбһҳбһҪбһҷбҹ—бҹ”
- бһ”бҹ’бһҡбһҫбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸ `send_request` бһҠбһҫбһҳбҹ’бһ”бһёбһ•бҹ’бһүбһҫбһҹбҹҶбһҺбһҪбһҡбһҠбҹ„бһҷбһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһҹбҹ’бһ‘бһ„бҹӢбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢ бһ бһҫбһҷбһ’бһ¶бһ“бһ¶бһҗбһ¶бһӮбҹҶбһҡбһјбһ”бһ„бҹ’бһҖбһҫбһҸбһўбһҸбҹ’бһҗбһ”бһ‘бһҸбһ¶бһҳбһӣбһҖбҹ’бһҒбһҺбҹҲбһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбҹ”
- бһ”бҹ’бһҡбһҫ `generated_text` бһҠбһҫбһҳбҹ’бһ”бһёбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһ…бһҳбҹ’бһӣбһҫбһҷбһҡбһ”бһҹбҹӢбһӮбҹҶбһҡбһј бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҸбҹ’бһҡбһЎбһ”бҹӢбһҮбһ¶бһҳбһҪбһҷбһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһҹбҹ’бһ‘бһ„бҹӢ бһ“бһ·бһ„бһ”бҹ’бһҡбһ—бҹҒбһ‘бһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһңбһ·бһ—бһ¶бһӮбһ¬бһ”бһ„бҹ’бһ бһ¶бһүбһ”бһ“бҹ’бһҗбҹӮбһҳбҹ”
- бһ”бҹ’бһҡбһҫбһҳбһ»бһҒбһ„бһ¶бһҡ `min` бһ“бһ·бһ„ `max` бһҠбһҫбһҳбҹ’бһ”бһёбһ’бһ¶бһ“бһ¶бһҗбһ¶бһ…бҹҶбһҺбһ„бҹӢбһ…бҹҶбһҺбһјбһӣбһ…бһ·бһҸбҹ’бһҸбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҳбһ¶бһ“бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҮбһҪбһҡбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң бһҠбһҫбһҳбҹ’бһ”бһёбһ…бҹҖбһҹбһңбһ¶бһ„бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҹбҹ’бһ‘бһ„бҹӢбһҳбһ·бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбҹ”

# [JavaScript Dynamic](#tab/javascript-dynamic)

```javascript
// бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ JavaScriptбҹ– бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҖбһ¶бһҡбһҹбҹӮбһ„бһҸбҹҶбһ”бһ“бҹӢбһҢбһёбһҺбһ¶бһҳбһ·бһҖ dб»ұa trГӘnбһ”бһҡбһ·бһ”бһ‘бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ
class AdaptiveSamplingManager {
  constructor(mcpClient) {
    this.client = mcpClient;
    
    // бһҖбҹҶбһҺбһҸбҹӢбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҹбҹӮбһ„бһҠбһҫбһҳ
    this.samplingProfiles = {
      creative: { temperature: 0.85, topP: 0.94, frequencyPenalty: 0.7, presencePenalty: 0.5 },
      factual: { temperature: 0.2, topP: 0.85, frequencyPenalty: 0.3, presencePenalty: 0.1 },
      code: { temperature: 0.25, topP: 0.9, frequencyPenalty: 0.4, presencePenalty: 0.3 },
      conversational: { temperature: 0.7, topP: 0.9, frequencyPenalty: 0.6, presencePenalty: 0.4 }
    };
    
    // бһҸбһ¶бһҳбһҠбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҖбҹ’бһ“бһ»бһ„бһ”бһ»бһ–бҹ’бһңбһ бҹҒбһҸбһ»
    this.performanceHistory = [];
  }
  
  // бһҡбһҖбһғбһҫбһүбһ”бҹ’бһҡбһ—бҹҒбһ‘бһҖбһ¶бһҡбһ„бһ¶бһҡбһ–бһёбһҖбһ¶бһҡбһҮбҹҶбһҡбһ»бһү
  detectTaskType(prompt, context = {}) {
    const promptLower = prompt.toLowerCase();
    
    // бһҖбһ¶бһҡбһҡбһҖбһғбһҫбһүбһҠбҹ„бһҷ heuristic бһ„бһ¶бһҷбҹ— - бһўбһ¶бһ…бһ’бҹ’бһңбһҫбһІбҹ’бһҷбһҖбһ¶бһ“бҹӢбһҸбҹӮбһӣбҹ’бһўбһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһ”бҹӮбһ„бһ…бҹӮбһҖ ML
    if (context.taskType) return context.taskType;
    
    if (promptLower.includes('code') || 
        promptLower.includes('function') || 
        promptLower.includes('program')) {
      return 'code';
    }
    
    if (promptLower.includes('explain') || 
        promptLower.includes('what is') || 
        promptLower.includes('how does')) {
      return 'factual';
    }
    
    if (promptLower.includes('creative') || 
        promptLower.includes('imagine') || 
        promptLower.includes('story')) {
      return 'creative';
    }
    
    // бһ”бһ“бҹ’бһӣбһ¶бһҹбҹӢбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһҖбһ¶бһҡбһҮбһҮбҹӮбһҖбһ”бһҫбһӮбҹ’бһҳбһ¶бһ“бһ”бҹ’бһҡбһ—бҹҒбһ‘бһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбһҖбһғбһҫбһү
    return 'conversational';
  }
  
  // бһӮбһҺбһ“бһ¶бһ–бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһҹбҹҮбһҠбҹ„бһҸ dб»ұa trГӘnбһ”бһҡбһ·бһ”бһ‘ бһ“бһ·бһ„бһ…бҹҶбһҺбһјбһӣбһ…бһ·бһҸбҹ’бһҸбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ
  getSamplingParameters(prompt, context = {}) {
    // бһҡбһҖбһғбһҫбһүбһ”бҹ’бһҡбһ—бҹҒбһ‘бһҖбһ¶бһҡбһ„бһ¶бһҡ
    const taskType = this.detectTaskType(prompt, context);
    
    // бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҠбһҫбһҳ
    let params = {...this.samplingProfiles[taskType]};
    
    // бһ”бһ„бҹ’бһңбҹӮбһҖбһҸбһ¶бһҳбһ…бҹҶбһҺбһјбһӣбһ…бһ·бһҸбҹ’бһҸбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ
    if (context.userPreferences) {
      const { creativity, precision, consistency } = context.userPreferences;
      
      if (creativity !== undefined) {
        // бһ”бһҳбҹ’бһӣбҹӮбһ„бһ–бһё бҹЎ-бҹЎбҹ  бһ‘бҹ…бһҮбһ¶бһҮбһҪбһҡбһ– temperatura бһІбҹ’бһҷбһҹбһҳбһҡбһҳбҹ’бһҷ
        params.temperature = 0.1 + (creativity * 0.09); // бҹ .бҹЎ-бҹЎ.бҹ 
      }
      
      if (precision !== undefined) {
        // бһ—бһ¶бһ–бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһҒбҹ’бһ–бһҹбҹӢбһҳбһ¶бһ“бһ“бҹҗбһҷбһҗбһ¶ topP бһ‘бһ¶бһ”бһҮбһ¶бһ„ (бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҳбһ¶бһ“бһ•бҹ’бһҸбҹ„бһҸбһҒбҹ’бһӣбһ¶бҹҶбһ„)
        params.topP = 1.0 - (precision * 0.05); // бҹ .бҹҘ-бҹЎ.бҹ 
      }
      
      if (consistency !== undefined) {
        // бһ—бһ¶бһ–бһ…бҹҶбһҡбһјбһ„бһҒбҹ’бһ–бһҹбҹӢбһҳбһ¶бһ“бһ“бҹҗбһҷбһҗбһ¶вҖӢбһҖбһ¶бһҡ бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ‘бҹ„бһҹбһ‘бһ¶бһ”бһҮбһ¶бһ„
        params.frequencyPenalty = 0.1 + ((10 - consistency) * 0.08); // бҹ .бҹЎ-бҹ .бҹ©
      }
    }
    
    // бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһ¶бһҡбһҖбҹӮбһ”бҹ’бһҡбҹӮбһҠбҹӮбһӣбһҡбҹҖбһ“бһ–бһёбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–
    this.applyLearnedAdjustments(params, taskType);
    
    return params;
  }
  
  applyLearnedAdjustments(params, taskType) {
    // бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҹбһ¶бһҳбһүбҹ’бһүбһ§бһ”бһҸбҹ’бһҗбһҳбҹ’бһ— - бһўбһ¶бһ…бһ”бһ“бҹ’бһҗбҹӮбһҳз®—жі•бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбһҮбһ¶бһ„бһ“бҹҒбҹҮ
    const relevantHistory = this.performanceHistory
      .filter(entry => entry.taskType === taskType)
      .slice(-5); // бһӮбһ·бһҸбһҸбҹӮбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҗбҹ’бһҳбһёбҹ—бһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮ
    
    if (relevantHistory.length > 0) {
      // бһӮбһҺбһ“бһ¶бһ–бһ·бһ“бҹ’бһ‘бһ»бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҳбһ’бҹ’бһҷбһҳ
      const avgScore = relevantHistory.reduce((sum, entry) => sum + entry.score, 0) / relevantHistory.length;
      
      // бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһ‘бһ¶бһ”бһҮбһ¶бһ„бһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡ бһҖбҹӮбһ”бҹ’бһҡбҹӮбһ–бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡ
      if (avgScore < 0.7) {
        // бһҖбһ¶бһҡбһҖбҹӮбһ”бҹ’бһҡбҹӮбһҸбһ·бһ…бһҸбһҪбһ…бһ‘бҹ…бһҖбһ¶бһ“бҹӢбһҸбһҳбҹ’бһӣбҹғбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–
        params.temperature = Math.max(params.temperature * 0.9, 0.1);
        params.topP = Math.max(params.topP * 0.95, 0.5);
      }
    }
  }
  
  recordPerformance(prompt, samplingParams, response, score) {
    // бһ”бһүбҹ’бһҮбһјбһ“бһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҖбҹӮбһ”бҹ’бһҡбҹӮбһҖбҹ’бһ“бһ»бһ„бһ–бҹҒбһӣбһўбһ“бһ¶бһӮбһҸ
    this.performanceHistory.push({
      timestamp: Date.now(),
      taskType: this.detectTaskType(prompt),
      samplingParams,
      responseLength: response.generatedText.length,
      score // бһ—бһ¶бһӮбһҡбһҷ бҹ -бҹЎ бһ“бҹғбһӮбһ»бһҺбһ—бһ¶бһ–бһҖбһ¶бһҡбһҶбҹ’бһӣбһҫбһҷбһҸбһ”
    });
    
    // бһҖбҹҶбһҺбһҸбҹӢбһ‘бҹҶбһ бҹҶбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·
    if (this.performanceHistory.length > 100) {
      this.performanceHistory.shift();
    }
  }
  
  async generateResponse(prompt, context = {}) {
    // бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһ–бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһҹбҹ’бһ‘бҹҮбһҢбһёбһҺбһ¶бһҳбһ·бһҖбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸ
    const samplingParams = this.getSamplingParameters(prompt, context);
    
    // бһ•бҹ’бһүбһҫбһҹбҹҶбһҺбһҫбһҮбһ¶бһҳбһҪбһҷбһ–бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһҸбҹӢбһ”бҹӮбһ“
    const response = await this.client.sendPrompt(prompt, {
      ...samplingParams,
      allowedTools: context.allowedTools || []
    });
    
    // бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ•бҹ’бһҸбһӣбҹӢбһҳбһҸбһ·бһҷбҹ„бһ”бһӣбҹӢ бһҖбһҸбҹӢбһҸбҹ’бһҡбһ¶бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбҹӮбһӣбһҳбҹ’бһўбһ”бһ“бҹ’бһҸ
    if (context.recordPerformance) {
      this.recordPerformance(prompt, samplingParams, response, context.feedbackScore || 0.5);
    }
    
    return {
      response,
      appliedSamplingParams: samplingParams,
      detectedTaskType: this.detectTaskType(prompt, context)
    };
  }
}
```

    - `recordPerformance` бһҠбһҫбһҳбҹ’бһ”бһёбһ…бһ»бҹҮбһ”бһүбҹ’бһҮбһёбһӣбһ‘бҹ’бһ’бһ•бһӣбһ”бһүбҹ’бһ…бҹҒбһүбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһ“бһ¶бһ–бҹҒбһӣбһҖбҹ’бһҡбҹ„бһҷбҹ”
    - `applyLearnedAdjustments` бһҠбһҫбһҳбҹ’бһ”бһёбһҖбҹӮбһ”бҹ’бһҡбҹӮбһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһҹбҹ’бһ‘бһ„бҹӢбһҮбһ¶бһҳбһҪбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһўбһ“бһ»бһңбһҸбҹ’бһҸ бһ“бһ¶бҹҶбһұбҹ’бһҷбһҳбһ¶бһ“бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһ…бһҳбҹ’бһӣбһҫбһҷбһҳбһ¶бһ“бһӮбһ»бһҺбһ—бһ¶бһ–бһҒбҹ’бһ–бһҹбҹӢбҹ”
    - `generateResponse` бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһүбҹ’бһ…бһјбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҹбһҡбһҹбҹҒбһҡбһ…бһҳбҹ’бһӣбһҫбһҷбһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһҹбҹ’бһ‘бһ„бҹӢбһҶбҹ’бһӣбһ¶бһҸбһңбҹғбһҮбһ¶бһҹбһҡбһ»бһ” бһ’бҹ’бһңбһҫбһұбҹ’бһҷбһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҮбһ¶бһҳбһҪбһҷбһҹбҹҶбһҺбһҪбһҡ бһ“бһ·бһ„бһ”бһҡбһ·бһ”бһ‘бһҒбһ»бһҹбҹ—бһӮбҹ’бһ“бһ¶бҹ”
    - `allowedTools` бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣбһӮбҹҶбһҡбһјбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ–бҹҒбһӣбһ”бһ„бҹ’бһҖбһҫбһҸ бһІбҹ’бһҷбһҖбһ¶бһ“бҹӢбһҸбҹӮбһҹбһ¶бһҖбһҹбһҳбһҮбһ¶бһ”бһҡбһ·бһ”бһ‘бҹ”
    - `feedbackScore` бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ•бҹ’бһҸбһӣбҹӢбһҳбһҸбһ·бһңбһ·бһ…бһ¶бһҡбһҺбһ¶ бһӣбһҫбһӮбһ»бһҺбһ—бһ¶бһ–бһ…бһҳбҹ’бһӣбһҫбһҷ бһҠбһҫбһҳбҹ’бһ”бһёбһ’бҹ’бһңбһҫбһұбҹ’бһҷбһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбһҪбһӣбһӮбҹҶбһҡбһјбһҖбһ¶бһ“бҹӢбһҸбҹӮбһ”бҹ’бһҡбһҹбһҫбһҡбһЎбһҫбһ„бҹ”
    - `performanceHistory` бһҡбһҖбҹ’бһҹбһ¶бһ”бһүбҹ’бһҮбһёбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһ”бҹ’бһҡбһҸбһ·бһҖбһҳбҹ’бһҳбһҳбһ»бһ“, бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҡбҹҖбһ“бһ–бһёбһҮбҹ„бһӮбһҮбҹҗбһҷ бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бһҡбһ¶бһҮбҹҗбһҷбҹ”
    - `getSamplingParameters` бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһҸбҹӢбһ”бҹӮбһ“бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһҹбҹ’бһ‘бһ„бҹӢбһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһҶбһ¶бһ”бҹӢбһҡбһ бҹҗбһҹбһҸбһ¶бһҳбһ”бһҡбһ·бһ”бһ‘бһҹбҹҶбһҺбһҫ, бһ”бһ„бҹ’бһҖбһҫбһҸбһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһӮбҹҶбһҡбһјбһ”бһҸбҹ’бһҠбһ·бһ—бһ¶бһ–бһ”бһ¶бһ“бһӣбҹ’бһўбҹ”
    - `detectTaskType` бһҠбһҫбһҳбҹ’бһ”бһёбһ…бһ¶бһҸбҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһ–бһёбһҹбҹҶбһҺбһҪбһҡ, бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһ”бҹ’бһҡбһҫбһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҹбҹ’бһ‘бһ„бҹӢбһҠбҹӮбһӣбһҹбһҳбһҡбһҳбҹ’бһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһ—бҹҒбһ‘бһҹбҹҶбһҺбһҫбһҒбһ»бһҹбҹ—бһӮбҹ’бһ“бһ¶бҹ”
    - `samplingProfiles` бһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҹбҹ’бһ‘бһ„бҹӢбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһ—бҹҒбһ‘бһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһ•бҹ’бһҹбҹҒбһ„бҹ—, бһ’бҹ’бһңбһҫбһІбҹ’бһҷбһ”бһ“бҹ’бһӣбҹӮбһ„бһҖбҹӮбһ”бҹ’бһҡбҹӮбһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҸбһ¶бһҳбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҹбҹҶбһҺбһҫбҹ”
