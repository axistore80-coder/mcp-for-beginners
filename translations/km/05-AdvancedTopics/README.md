# бһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ»бҹҶвҖӢбһҖбһҳбҹ’бһ–бһҹбҹӢвҖӢбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ MCP

[![Advanced MCP: Secure, Scalable, and Multi-modal AI Agents](../../../translated_images/km/06.42259eaf91fccfc6.webp)](https://youtu.be/4yjmGvJzYdY)

_(бһ…бһ»бһ…бһҡбһјбһ”бһ—бһ¶бһ–бһҒбһ¶бһ„бһӣбһҫбһҠбһҫбһҳбҹ’бһ”бһёбһҳбһҫбһӣбһңбһёбһҠбҹҒбһўбһјбһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ)_

бһҮбҹҶбһ–бһјбһҖбһ“бҹҒбҹҮбһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҸбһ”бҹӢбһӣбһҫбһҳбҹҒбһҡбҹҖбһ“бһҖбһҳбҹ’бһҡбһ·бһҸбһҒбҹ’бһ–бһҹбҹӢбһҒбҹ’бһӣбҹҮбҹ—бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ“бҹҚ Model Context Protocol (MCP) бһҡбһҪбһҳбһҳбһ¶бһ“бһҖбһ¶бһҡбһ…бһ„бһҖбҹ’бһҡбһ„бһ–бһ бһ»бһҳбһ»бһҒбһ„бһ¶бһҡ бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһұбҹ’бһҷбһўбһ¶бһ…бһ–бһ„бҹ’бһҡбһёбһҖбһ”бһ¶бһ“ бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸ бһ“бһ·бһ„бһҖбһ¶бһҡбһ…бһ„бһҖбҹ’бһҡбһ„бһҮбһ¶бһҳбһҪбһҷбһҹбһ бһӮбҹ’бһҡбһ¶бһҹбҹ” бһҳбҹҒбһҡбҹҖбһ“бһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһҹбҹҶбһҒбһ¶бһ“бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҹбһ¶бһ„бһҹбһ„бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё MCP бһҠбҹӮбһӣбһҳбһ¶бһ“бһ—бһ¶бһ–бһҡбһ№бһ„бһҳбһ¶бҹҶ бһ“бһ·бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһӣбҹ’бһўбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ AI бһҹбһҳбҹҗбһҷбһҗбҹ’бһҳбһёбҹ”

## бһ‘бһ·бһҠбҹ’бһӢбһ—бһ¶бһ–бһ‘бһјбһ‘бҹ…

бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһ–бһё бһӮбҹҶбһ“бһ·бһҸбһҖбһҳбҹ’бһҡбһ·бһҸбһҒбҹ’бһ–бһҹбҹӢбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸ Model Context Protocol бһҠбҹ„бһҷбһ•бҹ’бһҠбҹ„бһҸбһӣбһҫбһҖбһ¶бһҡбһ…бһ„бһҖбҹ’бһҡбһ„бһ–бһ бһ»бһҳбһ»бһҒбһ„бһ¶бһҡ бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһұбҹ’бһҷбһўбһ¶бһ…бһ–бһ„бҹ’бһҡбһёбһҖбһ”бһ¶бһ“ бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸ бһ“бһ·бһ„бһҖбһ¶бһҡбһ…бһ„бһҖбҹ’бһҡбһ„бһҮбһ¶бһҳбһҪбһҷбһҹбһ бһӮбҹ’бһҡбһ¶бһҹбҹ” бһҳбҹҒбһҡбҹҖбһ“бһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһӮбһәбһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҹбһ¶бһ„бһҹбһ„бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё MCP бһҠбҹӮбһӣбһҳбһ¶бһ“бһӮбһ»бһҺбһ—бһ¶бһ–бһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳбһўбһ¶бһ…бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһӣбҹҶбһ”бһ¶бһҖбҹ—бһҖбҹ’бһ“бһ»бһ„бһ”бһҡбһ·бһ”бһ‘бһҹбһ бһӮбҹ’бһҡбһ¶бһҹбҹ”

## бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҡбҹҖбһ“

бһ”бһүбҹ’бһ…бһ”бҹӢбһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһўбһ¶бһ…бҹ–

- бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҳбһ»бһҒбһ„бһ¶бһҡбһ–бһ бһ»бһҳбһ»бһҒбһ„бһ¶бһҡбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҹбҹ’бһ“бһјбһӣ MCP
- бһҡбһ…бһ“бһ¶бһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳ MCP бһҠбҹӮбһӣбһўбһ¶бһ…бһ–бһ„бҹ’бһҡбһёбһҖбһ”бһ¶бһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹ’бһҗбһ¶бһ“бһҖбһ¶бһҡбһҺбҹҚбһҠбҹӮбһӣбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҒбҹ’бһ–бһҹбҹӢ
- бһўбһ“бһ»бһңбһҸбҹ’бһҸбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸ бһҠбҹӮбһӣбһҹбһҳбһ“бһ№бһ„бһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҡбһ”бһҹбҹӢ MCP
- бһ…бһ„бһҖбҹ’бһҡбһ„ MCP бһҮбһ¶бһҳбһҪбһҷбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ бһ“бһ·бһ„бһҹбҹ’бһ“бһјбһӣ AI бһҹбһ бһӮбҹ’бһҡбһ¶бһҹ
- бһ”бҹ’бһҸбһјбһҡбһҸбһҳбҹ’бһҡбһ„бһ“бһ·бһ„бһ—бһ¶бһ–бһ‘бҹҶбһ“бһҫбһ”бһҮбһ¶бһ“бһ·бһ…бҹ’бһ…бһ“бҹ…бһ”бһҡбһ·бһ”бһ‘бһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ

> **бһҗбҹ’бһҳбһёбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бһүбҹ’бһҮбһ¶бһҖбҹӢ MCP бҹўбҹ бҹўбҹҘ-бҹЎбҹЎ-бҹўбҹҘ**: бһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһҘбһЎбһјбһңбһ“бһ¶бҹҶбһҳбһҖбһ“бһјбһңбһҖбһ¶бһҡбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ **бһҖбһ¶бһҡбһ„бһ¶бһҡ** (бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҡбһҷбҹҲбһ–бҹҒбһӣбһңбҹӮбһ„бһҠбҹӮбһӣбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһ”бһ‘бһ”бһүбҹ’бһҮбһ¶бһҖбһ¶бһҡбһҠбҹҶбһҺбһҫбһҡ), **бһҖбһ¶бһҡбһңбһ¶бһҷбһ”бһүбҹ’бһ…бһјбһӣбһ§бһ”бһҖбһҡбһҺбҹҚ** (metadata бһўбҹҶбһ–бһёбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһ§бһ”бһҖбһҡбһҺбҹҚбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–), **бһҖбһ¶бһҡбһҹбҹ’бһ“бһҫбһҡбһҹбһ»бҹҶбһҡбһ”бҹҖбһ” URL** (бһҹбҹ’бһ“бһҫбһҹбһ»бҹҶбһҳбһ¶бһҸбһ·бһҖбһ¶ URL бһ–бһ·бһҹбҹҒбһҹбһ–бһёбһўбҹ’бһ“бһҖбһӮбһ¶бҹҶбһ‘бҹ’бһҡ), бһ“бһ·бһ„ **Roots** бһ”бһ¶бһ“бһҖбҹӮбһӣбһҳбҹ’бһў (бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ context бһҖбһ“бҹ’бһӣбҹӮбһ„бһҖбһ¶бһҡбһ„бһ¶бһҡ)бҹ” бһ…бһјбһӣбһ‘бҹ…бһҳбһҫбһӣ [MCP Specification changelog](https://spec.modelcontextprotocol.io/) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһӣбһҳбҹ’бһўбһ·бһҸбҹ”

## бһ…бҹҶбһҺбһ»бһ…бһҹбҹҶбһҒбһ¶бһ“бҹӢ

- бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸ MCP бһ–бһ бһ»бһҳбһ»бһҒбһ„бһ¶бһҡбһ–бһ„бҹ’бһҡбһёбһҖбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ– AI бһ“бҹ…бһ–бҹ’бһҡбһ»бһҷбһӣбһҫбһҖбһ¶бһҡбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһўбһҖбҹ’бһҹбһҡ
- бһҖбһ¶бһҡбһ”бһ„бҹ’бһҡбһҪбһҳбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҠбҹ„бһҷбһҹбһ¶бһҡ MCP бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ…бһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣбһҹбһ бһӮбҹ’бһҡбһ¶бһҹбһўбһ¶бһ…бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҸбһ¶бһҳбһҡбһҷбҹҲбһҖбһ¶бһҡбһ–бһ„бҹ’бһҡбһёбһҖбһҠбҹҒбһҖ бһ“бһ·бһ„бһЎбһҫбһ„
- бһңбһ·бһ’бһ¶бһ“бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһ“бһ·бһ„бһ’бһ¶бһ“бһ¶бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ…бһјбһӣбһ”бҹ’бһҡбһҫбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң
- бһҖбһ¶бһҡбһ…бһ„бһҖбҹ’бһҡбһ„бһ…бҹ’бһҡбһҫбһ“бһҮбһ¶бһҳбһҪбһҷбһңбҹҒбһ‘бһ·бһҖбһ¶бһҠбһјбһ…бһҮбһ¶ Azure OpenAI бһ“бһ·бһ„ Microsoft AI Foundry бһ”бһ„бҹ’бһҖбһҫбһ“бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ– MCP
- бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸ MCP бһӣбҹҶбһўбһ·бһҸбһ•бҹ’бһҸбҹ„бһҸбһӣбһҫбһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳбһҠбҹӮбһӣбһ”бҹ’бһҡбһҹбһҫбһҡбһ“бһ·бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ’бһ“бһ’бһ¶бһ“бһҷбҹүбһ¶бһ„бһ”бҹ’бһҡбһ»бһ„бһ”бҹ’бһҡбһҷбҹҗбһҸбҹ’бһ“

## бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸ

бһҡбһ…бһ“бһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё MCP бһҮбһ¶бһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбһҹбһ бһӮбҹ’бһҡбһ¶бһҹбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҡбһҺбһёбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбһҳбһҪбһҷбҹ–

1. бһҖбҹҶбһҺбһҸбҹӢбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ–бһ бһ»бһҳбһ»бһҒбһ„бһ¶бһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҡбһҺбһёбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ
2. бһҹбһ“бҹ’бһ“бһ·бһҠбҹ’бһӢбһ¶бһ“бһңбһ·бһ’бһ¶бһ“бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҖбһ¶бһҡбһ–бһ¶бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһңбһ·бһҮбҹ’бһҮбһҳбһ¶бһ“
3. бһҡбһ…бһ“бһ¶бһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳбһўбһ¶бһ…бһ–бһ„бҹ’бһҡбһёбһҖбһ”бһ¶бһ“бһҠбҹӮбһӣбһўбһ¶бһ…бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһ‘бҹҶбһ“бһҪбһӣбһҒбһ»бһҹбһҸбҹ’бһҡбһјбһңбһ•бҹ’бһҹбҹҒбһ„бҹ—
4. бһӮбҹ’бһҡбҹ„бһ„бһҖбһ“бҹ’бһӣбҹӮбһ„бһ…бһ„бһҖбҹ’бһҡбһ„бһҮбһ¶бһҳбһҪбһҷбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ AI бһҹбһ бһӮбҹ’бһҡбһ¶бһҹ
5. бһ…бһ»бҹҮбһ”бһүбҹ’бһҮбһёбһ…бҹҶбһҺбһ»бһ…бһ”бһүбҹ’бһ бһ¶бһҳбһ»бһҒбһ„бһ¶бһҡбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҗбҹӮбһҳбһ‘бҹҖбһҸ бһ“бһ·бһ„бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҖбһ¶бһҡбһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷ
