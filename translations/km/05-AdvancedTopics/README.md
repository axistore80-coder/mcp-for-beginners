# бһ”бҹ’бһҡбһ’бһ¶бһ“бһ”бһ‘бһҖбһҳбҹ’бһҡбһ·бһҸбһҒбҹ’бһ–бһҹбҹӢбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ MCP

[![Advanced MCP: Secure, Scalable, and Multi-modal AI Agents](../../../translated_images/km/06.42259eaf91fccfc6.webp)](https://youtu.be/4yjmGvJzYdY)

_(бһ…бһ»бһ…бһӣбһҫбһҡбһјбһ”бһ—бһ¶бһ–бһҒбһ¶бһ„бһӣбһҫбһҠбһҫбһҳбҹ’бһ”бһёбһҳбһҫбһӣбһңбһёбһҠбҹҒбһўбһјбһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ)_

бһҮбҹҶбһ–бһјбһҖбһ“бҹҒбҹҮбһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҠбһ”бҹӢбһ”бҹ’бһҡбһ’бһ¶бһ“бһ”бһ‘бһҖбһҳбҹ’бһҡбһ·бһҸбһҒбҹ’бһ–бһҹбҹӢбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ“бҹҚ Model Context Protocol (MCP) бһҡбһҪбһҳбһҳбһ¶бһ“бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһ”бҹ’бһҡбһ—бҹҒбһ‘бһ–бһ бһ»бһҳбҹүбһјбһҠ, бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһ–бһ бһ»бһҹбҹ’бһ‘бһҪбһ“, бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҹбһ“бҹ’бһҸбһ·бһҹбһ»бһҒбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸ бһ“бһ·бһ„бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҹбһ бһӮбҹ’бһҡбһ¶бһҹбҹ” бһ”бҹ’бһҡбһ’бһ¶бһ“бһ”бһ‘бһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһҳбһ¶бһ“бһҹбһ¶бһҡбҹҲбһҹбҹҶбһҒбһ¶бһ“бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҹбһ¶бһ„бһҹбһ„бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё MCP бһҠбҹӮбһӣбһҡбһ№бһ„бһҳбһ¶бҹҶ бһ“бһ·бһ„бһҸбҹ’бһҡбҹҖбһҳбһ”бһ„бҹ’бһ бҹ„бҹҮбһҠбҹӮбһӣбһўбһ¶бһ…бһ”бҹҶбһ–бҹҒбһүбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҡбһ”бһҹбҹӢбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ AI бһҹбһҳбҹҗбһҷбһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷбҹ”

## бһ‘бһ·бһҠбҹ’бһӢбһ—бһ¶бһ–бһ‘бһјбһ‘бҹ…

бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһӮбҹҶбһ“бһ·бһҸбһҖбһҳбҹ’бһҡбһ·бһҸбһҒбҹ’бһ–бһҹбҹӢбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ“бҹҚ Model Context Protocol бһҠбҹӮбһӣбһ•бҹ’бһҸбҹ„бһҸбһӣбһҫбһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһ–бһ бһ»бһҳбҹүбһјбһҠ бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһ–бһ бһ»бһҹбҹ’бһ‘бһҪбһ“ бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҹбһ“бҹ’бһҸбһ·бһҹбһ»бһҒбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸ бһ“бһ·бһ„бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҹбһ бһӮбҹ’бһҡбһ¶бһҹбҹ” бһ”бҹ’бһҡбһ’бһ¶бһ“бһ”бһ‘бһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһҮбһ¶бһҡбһҝбһ„бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҹбһ¶бһ„бһҹбһ„бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё MCP бһҳбһ¶бһ“бһӮбһ»бһҺбһ—бһ¶бһ–бһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳбһҠбҹӮбһӣбһўбһ¶бһ…бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһүбһ№бһҖбһүбһ¶бһ”бҹӢбһҖбҹ’бһ“бһ»бһ„бһ”бһҡбһ·бһҠбҹ’бһӢбһ¶бһ“бһҹбһ бһӮбҹ’бһҡбһ¶бһҹбҹ”

## бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҹбһ·бһҖбҹ’бһҹбһ¶

бһ“бҹ…бһ…бһ»бһ„бһ”бһүбҹ’бһ…бһ”бҹӢбһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһўбһ¶бһ…бҹ–

- бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһ–бһ бһ»бһҳбҹүбһјбһҠбһҖбҹ’бһ“бһ»бһ„бһҹбҹҠбһ»бһҳ MCP
- бһҡбһ…бһ“бһ¶бһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳ MCP бһҠбҹӮбһӣбһўбһ¶бһ…бһҖбҹҶбһҺбһҸбҹӢбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҒбҹ’бһ–бһҹбҹӢбһҖбҹ’бһ“бһ»бһ„бһҹбҹ’бһҗбһ¶бһ“бһҖбһ¶бһҡбһҺбҹҚбһҳбһ¶бһ“бһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҒбҹ’бһ–бһҹбҹӢ
- бһўбһ“бһ»бһңбһҸбҹ’бһҸбһўбһ“бһ»бһҹбҹ’бһҹбһҡбһҺбҹҲбһҹбһ“бҹ’бһҸбһ·бһҹбһ»бһҒбһІбҹ’бһҷбһҹбһҳбһҡбһҳбҹ’бһҷбһҸбһ¶бһҳбһ‘бҹ’бһҡбһ№бһҹбҹ’бһҸбһёбһҹбһ“бҹ’бһҸбһ·бһҹбһ»бһҒбһҡбһ”бһҹбҹӢ MCP
- бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ MCP бһҮбһ¶бһҳбһҪбһҷбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ бһ“бһ·бһ„бһҹбҹҠбһ»бһҳ AI бһҹбһ бһӮбҹ’бһҡбһ¶бһҹ
- бһ”бһ„бҹ’бһҖбһҫбһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ– бһ“бһ·бһ„бһ—бһ¶бһ–бһ‘бһ»бһҖбһ…бһ·бһҸбҹ’бһҸбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бһҡбһ·бһҠбҹ’бһӢбһ¶бһ“бһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ

> **бһҗбҹ’бһҳбһёбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбҹҶбһҺбҹӮ MCP Specification 2025-11-25**: бһҖбҹҶбһҺбҹӮбһ“бҹҒбҹҮбһҳбһ¶бһ“бһҖбһ¶бһҡбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһ”бҹ’бһҡбһ—бҹҒбһ‘бһҸбҹҒбһҹбҹ’бһҸ **Tasks** (бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҡбһҷбҹҲбһ–бҹҒбһӣбһңбҹӮбһ„бһҮбһ¶бһҳбһҪбһҷбһҸбһ¶бһҳбһҠбһ¶бһ“бһңбһҢбҹ’бһҚбһ“бһ—бһ¶бһ–), **Tool Annotations** (бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһӣбһҳбҹ’бһўбһ·бһҸбһўбҹҶбһ–бһёбһҡбһ”бҹҖбһ”бһ”бҹ’бһҡбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–), **URL Mode Elicitation** (бһҹбҹ’бһ“бһҫбһҹбһ»бҹҶбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“ URL бһ…бҹ’бһ”бһ¶бһҹбҹӢбһ–бһёбһўбһҸбһ·бһҗбһ·бһҮбһ“), бһ“бһ·бһ„ **Roots** бһ”бһ¶бһ“бһҖбһ¶бһ“бҹӢбһҸбҹӮбһҳбһ¶бһ“бһҖбһҳбҹ’бһӣбһ¶бҹҶбһ„ (бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ”бһҡбһ·бһ”бһ‘бһҖбһ“бҹ’бһӣбҹӮбһ„бһ”бһҳбҹ’бһҡбһҫбһҖбһ¶бһҡбһ„бһ¶бһҡ)бҹ” бһҹбһјбһҳбһҳбһҫбһӣ [MCP Specification changelog](https://spec.modelcontextprotocol.io/) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһӣбһҳбҹ’бһўбһ·бһҸбһ‘бһ¶бҹҶбһ„бһҳбһјбһӣбҹ”

## бһ…бҹҶбһҺбһ»бһ…бһҹбҹҶбһҒбһ¶бһ“бҹӢбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҷбһҖбһ…бһ·бһҸбҹ’бһҸбһ‘бһ»бһҖбһҠбһ¶бһҖбҹӢ

- бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ–бһ бһ»бһҳбҹүбһјбһҠ MCP бһ•бҹ’бһҸбһӣбҹӢбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ– AI бһ‘бҹ…бһӣбһҫбһҳбһ·бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹӮбһҖбһ¶бһҡбһ”бҹ’бһҡбҹӮбһ”бҹ’бһҡбһҪбһӣбһўбһҸбҹ’бһҗбһ”бһ‘бһҸбҹӮбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮ
- бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһӮбһәбһҳбһ¶бһ“бһҹбһ¶бһҡбҹҲбһҹбҹҶбһҒбһ¶бһ“бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ”бһ„бҹ’бһ бҹ„бҹҮбһ“бҹ…бһҹбһ бһӮбҹ’бһҡбһ¶бһҹ бһ бһҫбһҷбһўбһ¶бһ…бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҸбһ¶бһҳбһҡбһ”бҹҖбһ”бһ”бһүбҹ’бһҮбҹҶбһ–бҹ’бһҷбһҪбһҡбһ“бһ·бһ„бһ”бһүбҹ’бһҮбҹҶбһҮбһ¶бһ“бҹӢ
- бһңбһ·бһ’бһ¶бһ“бһҹбһ“бҹ’бһҸбһ·бһҹбһ»бһҒбһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҸбһ”бҹӢбһ‘бһ¶бҹҶбһ„бһҳбһјбһӣбһҮбһҪбһҷбһҖбһ¶бһҡбһ–бһ¶бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһ“бһ·бһ„бһ’бһ¶бһ“бһ¶бһҡбһ”бһҹбҹӢбһҖбҹҶбһҺбһҸбҹӢбһҖбһ¶бһҡбһ…бһјбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ
- бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҹбһ бһӮбҹ’бһҡбһ¶бһҹбһҮбһ¶бһҳбһҪбһҷбһңбҹҒбһ‘бһ·бһҖбһ¶бһҠбһјбһ…бһҮбһ¶ Azure OpenAI бһ“бһ·бһ„ Microsoft AI Foundry бһҖбҹӮбһӣбһҳбҹ’бһўбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ– MCP
- бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸ MCP бһӮбһәбһ‘бһ‘бһҪбһӣбһ•бһӣбһ–бһёбһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳбһ”бһ¶бһ“бһ”бҹ’бһҡбһҹбһҫбһҡбһЎбһҫбһ„ бһ“бһ·бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ’бһ“бһ’бһ¶бһ“бһҷбҹүбһ¶бһ„бһ”бҹ’бһҡбһ»бһ„бһ”бҹ’бһҡбһҷбҹҗбһҸбҹ’бһ“

## бһӣбҹҶбһ бһ¶бһҸбҹӢ

бһҡбһ…бһ“бһ¶бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸ MCP бһҳбһ¶бһ“бһӮбһ»бһҺбһ—бһ¶бһ–бһҹбһ бһӮбҹ’бһҡбһ¶бһҹбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҡбһҺбһёбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбҹ–

1. бһҖбҹҶбһҺбһҸбҹӢбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ”бҹ’бһҡбһ—бҹҒбһ‘бһ–бһ бһ»бһҳбҹүбһјбһҠбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҡбһҺбһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ
2. бһҡбҹҖбһ”бһ…бҹҶбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбһ“бҹ’бһҸбһ·бһҹбһ»бһҒбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһӣбҹ’бһўбһҠбһҫбһҳбҹ’бһ”бһёбһҖбһ¶бһҡбһ–бһ¶бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢ
3. бһҡбһ…бһ“бһ¶бһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳбһўбһ¶бһ…бһ”бһ„бҹ’бһҖбһҫбһ“бһ‘бҹҶбһ бҹҶбһҠбҹӮбһӣбһўбһ¶бһ…бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ‘бһҳбҹ’бһ„бһ“бҹӢбһҖбһ¶бһҡбһ•бҹ’бһ‘бһ»бһҖбһ”бҹҶбһ”бҹ’бһҡбҹӮбһҖ
4. бһӮбҹ’бһҡбҹ„бһ„бһ…бҹҶбһҺбһ»бһ…бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҮбһ¶бһҳбһҪбһҷбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ AI бһҹбһ бһӮбҹ’бһҡбһ¶бһҹ
5. бһҜбһҖбһҹбһ¶бһҡбһўбҹҶбһ–бһёбһҖбһ“бҹ’бһӣбҹӮбһ„бһ”бһүбҹ’бһ…бҹҒбһүбһ•бһӣбһ”бҹүбҹҮбһ–бһ¶бһӣбҹӢ бһ“бһ·бһ„бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһҖбһ¶бһҡбһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷ
