# бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһӮбҹ’бһ“бһ¶ бһ“бһ·бһ„бһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһӣбҹ’бһ”бһҝбһ“бһҒбҹ’бһ–бһҹбҹӢ MCP

бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ…бҹҒбһүбһҖбҹ’бһ“бһ»бһ„бһҖбҹ’бһҡбһ»бһҳбһ бҹҠбһ»бһ“ бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸ MCP бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҖбһ¶бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶бһ…бҹҶбһ“бһҪбһ“бһ…бҹ’бһҡбһҫбһ“бһ“бҹғбһҖбһ¶бһҡбһҹбҹ’бһ“бһҫбһҹбһ»бҹҶбһҠбҹ„бһҷбһҳбһ¶бһ“бһ–бҹҒбһӣбһҖбҹ’бһҡбҹ„бһҷбһ–бҹҒбһҖбһҸбһ·бһ…бһ”бҹҶбһ•бһ»бһҸбҹ”

## бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶ

бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһ–бһҪбһҖбһҷбһҫбһ„бһ“бһ№бһ„бһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһ–бһёбһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ–бһ„бҹ’бһҡбһёбһҖбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP бһҠбһҫбһҳбҹ’бһ”бһёбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһ’бҹҶбҹ—бһҷбҹүбһ¶бһ„бһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бҹ” бһ–бһҪбһҖбһҷбһҫбһ„бһ“бһ№бһ„бһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҸбһ”бҹӢбһҖбһ¶бһҡбһ–бһ„бҹ’бһҡбһёбһҖбһ•бҹ’бһҠбҹҒбһҖ бһ“бһ·бһ„бһ”бһүбҹ’бһҲбһҡ бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһ’бһ“бһ’бһ¶бһ“ бһ“бһ·бһ„бһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳбһ…бҹӮбһҖбһ…бһ¶бһҷбҹ”

## бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҹбһ·бһҖбҹ’бһҹбһ¶

бһ“бҹ…бһ…бһ»бһ„бһ”бһүбҹ’бһ…бһ”бҹӢбһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһўбһ¶бһ…:

- бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһ¶бһҡбһ–бһ„бҹ’бһҡбһёбһҖбһ•бҹ’бһҠбҹҒбһҖбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһҖбһ¶бһҡбһ”бһ„бҹ’бһ бһҪбһ…бһ•бҹ’бһ‘бһ»бһҖбһ“бһ·бһ„бһҖбһ¶бһҡбһ…бһ„бһ…бһ¶бҹҶбһ…бҹӮбһҖбһ…бһ¶бһҷбҹ”
- бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ–бһ„бҹ’бһҡбһёбһҖбһ”бһүбҹ’бһҲбһҡбһ“бһ·бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ’бһ“бһ’бһ¶бһ“бҹ”
- бһҡбһ…бһ“бһ¶бһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳ MCP бһ…бҹӮбһҖбһ…бһ¶бһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бһ¶бһ–бһ…бҹҒбһүбһ”бҹ’бһҡбһҫбһҒбҹ’бһ–бһҹбҹӢ бһ“бһ·бһ„бһҖбһ¶бһҡбһ–бһ¶бһҡбһҖбҹҶбһ бһ»бһҹбҹ”
- бһ”бҹ’бһҡбһҫбһ§бһ”бһҖбһҡбһҺбҹҚ бһ“бһ·бһ„бһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹбһҖбһҳбҹ’бһҡбһ·бһҸбһҒбҹ’бһ–бһҹбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ“бһ·бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бҹ”
- бһ”бҹ’бһҡбһҫбһҮбҹҶбһ“бһ¶бһүбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ–бһ„бҹ’бһҡбһёбһҖбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳбҹ”

## бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ–бһ„бҹ’бһҡбһёбһҖ

бһҳбһ¶бһ“бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ…бҹ’бһҡбһҫбһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ–бһ„бҹ’бһҡбһёбһҖбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP бһҷбҹүбһ¶бһ„бһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бҹ–

- **бһҖбһ¶бһҡбһ–бһ„бҹ’бһҡбһёбһҖбһ•бҹ’бһҠбҹҒбһҖ**бҹ– бһҠбһ¶бһҖбҹӢбһ…бҹҒбһүбһңбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP бһ–бһ бһ»бһ…бҹҶбһ“бһҪбһ“бһ“бҹ…бһ–бһёбһҖбҹ’бһҡбҹ„бһҷбһ”бһ„бҹ’бһ бһҪбһ…бһ•бҹ’бһ‘бһ»бһҖ бһҠбһҫбһҳбҹ’бһ”бһёбһ…бҹӮбһҖбһ…бһ¶бһҷбһҖбһ¶бһҡбһҹбҹ’бһ“бһҫбһҹбһ»бҹҶбһ“бҹ…бһҳбһ»бһҒбһҠбһјбһ…бһҹбҹ’бһҳбһҫбҹ”
- **бһҖбһ¶бһҡбһ–бһ„бҹ’бһҡбһёбһҖбһ”бһүбҹ’бһҲбһҡ**бҹ– бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP бһҳбһҪбһҷбһ…бһјбһӣбһҹбҹҶбһҡбһ¶бһ”бҹӢбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҖбһ¶бһҡбһҹбҹ’бһ“бһҫбһҹбһ»бҹҶбһ…бҹ’бһҡбһҫбһ“бһҮбһ¶бһ„бһҸбһ¶бһҳбһҡбһҷбҹҲбһҖбһ¶бһҡбһҖбһҫбһ“бһ’бһ“бһ’бһ¶бһ“ (CPU, бһўбһ„бҹ’бһӮбһ…бһ„бһ…бһ¶бҹҶ) бһ“бһ·бһ„бһҖбҹҶбһҺбһҸбҹӢбһҖбҹҶбһҺбҹӮбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“бҹ”
- **бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһ’бһ“бһ’бһ¶бһ“бһІбҹ’бһҷбһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ·бһ—бһ¶бһ–**бҹ– бһ”бҹ’бһҡбһҫбһӮбһҺбһ“бһ¶бһ”бҹ’бһҡбһ¶бһҖбҹӢбһ…бҹҶбһҺбҹҒбһү бһ”бҹ’бһҡбһҳбһјбһӣбһ…бҹҶбһҺбһ¶бҹҶ бһ“бһ·бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҠбҹ„бһҷбһҒбҹ’бһҷбһӣбҹӢбһҠбһҫбһҳбҹ’бһ”бһёбһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ’бһ“бһ’бһ¶бһ“ бһ“бһ·бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҶбҹ’бһӣбһҫбһҷбһҸбһ”бҹ”
- **бһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳбһ…бҹӮбһҖбһ…бһ¶бһҷ**бҹ– бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ…бҹӮбһҖбһ…бһ¶бһҷбһҠбҹӮбһӣбһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бһ“бҹғбһҖбҹ’бһҡбһ»бһҳбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҹбһ бһҖбһ¶бһҡбһӮбҹ’бһ“бһ¶ бһҠбһҫбһҳбҹ’бһ”бһёбһ…бҹӮбһҖбһ…бһ¶бһҷбһҖбһ¶бһҡ бһ“бһ·бһ„бһ•бҹ’бһҸбһӣбҹӢбһҖбһ¶бһҡбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҳбһ·бһ“бһҒбһ¶бһ“бҹ”

## бһҖбһ¶бһҡбһ–бһ„бҹ’бһҡбһёбһҖбһ•бҹ’бһҠбҹҒбһҖ

бһҖбһ¶бһҡбһ–бһ„бҹ’бһҡбһёбһҖбһ•бҹ’бһҠбҹҒбһҖбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҠбҹ„бһҷбһҠбһ¶бһҖбҹӢбһ…бҹҒбһүбһңбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP бһ…бҹ’бһҡбһҫбһ“ бһ“бһ·бһ„бһ”бҹ’бһҡбһҫбһ”бһ„бҹ’бһ бһҪбһ…бһ•бҹ’бһ‘бһ»бһҖбһҠбһҫбһҳбҹ’бһ”бһёбһ…бҹӮбһҖбһ…бһ¶бһҷбһҖбһ¶бһҡбһҹбҹ’бһ“бһҫбһҹбһ»бҹҶбһҠбҹӮбһӣбһҳбһҖбһҠбһӣбҹӢбҹ” бһңбһ·бһҹбҹҗбһҷбһ“бҹҒбҹҮбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҖбһ¶бһҡбһҹбҹ’бһ“бһҫбһҹбһ»бҹҶбһ”бһ¶бһ“бһ…бҹ’бһҡбһҫбһ“бһҮбһ¶бһҳбһҪбһҷбһӮбҹ’бһ“бһ¶ бһ“бһ·бһ„бһ•бҹ’бһҸбһӣбҹӢбһҖбһ¶бһҡбһ–бһ¶бһҡбһҖбҹҶбһ бһ»бһҹбҹ”

бһҳбһҖбһҳбһҫбһӣбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ–бһёбһңбһ·бһ’бһёбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҖбһ¶бһҡбһ–бһ„бҹ’бһҡбһёбһҖбһ•бҹ’бһҠбҹҒбһҖ бһ“бһ·бһ„ MCPбҹ”

### [.NET](#tab/dotnet)

```csharp
// ASP.NET Core MCP load balancing configuration
public class McpLoadBalancedStartup
{
    public void ConfigureServices(IServiceCollection services)
    {
        // Configure distributed cache for session state
        services.AddStackExchangeRedisCache(options =>
        {
            options.Configuration = Configuration.GetConnectionString("RedisConnection");
            options.InstanceName = "MCP_";
        });
        
        // Configure MCP with distributed caching
        services.AddMcpServer(options =>
        {
            options.ServerName = "Scalable MCP Server";
            options.ServerVersion = "1.0.0";
            options.EnableDistributedCaching = true;
            options.CacheExpirationMinutes = 60;
        });
        
        // Register tools
        services.AddMcpTool<HighPerformanceTool>();
    }
}
```

бһҖбҹ’бһ“бһ»бһ„бһҖбһјбһҠбһҒбһ¶бһ„бһӣбһҫ бһҷбһҫбһ„бһ”бһ¶бһ“бҹ–

- бһҖбҹҶбһҺбһҸбҹӢбһ”бҹ’бһҡбһҳбһјбһӣбһ…бҹҶбһҺбһ¶бҹҶбһ…бҹӮбһҖбһ…бһ¶бһҷбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ Redis бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ•бҹ’бһ‘бһ»бһҖбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҹбһҳбҹҗбһҷ бһ“бһ·бһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ§бһ”бһҖбһҡбһҺбҹҚбҹ”
- бһҸбһҳбҹ’бһҖбһӣбҹӢбһҖбһ¶бһҡбһ…бһ„бһ…бһ¶бҹҶбһ…бҹӮбһҖбһ…бһ¶бһҷбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCPбҹ”

public class OptimizedMcpServer {
    public static McpServer createOptimizedServer() {
        // бһҖбҹҶбһҺбһҸбҹӢбһҖбһ¶бһҡбһ”бҹ’бһҡбһҳбһјбһӣбһҒбҹ’бһҹбҹӮбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸ
        int processors = Runtime.getRuntime().availableProcessors();
        int optimalThreads = processors * 2; // бһңбһ·бһ’бһ¶бһ“бһ‘бһјбһ‘бҹ…бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҠбҹӮбһӣбһ–бһ№бһ„бһ•бҹ’бһўбҹӮбһҖбһӣбһҫ I/O
        
        ExecutorService executorService = new ThreadPoolExecutor(
            processors,       // бһ‘бҹҶбһ бҹҶбһ”бҹ’бһҡбһҳбһјбһӣбһҹбҹ’бһ“бһјбһӣ
            optimalThreads,   // бһ‘бҹҶбһ бҹҶбһ”бҹ’бһҡбһҳбһјбһӣбһўбһҸбһ·бһ”бһҡбһҳбһ¶
            60L,              // бһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһҡбһҹбҹӢ
            TimeUnit.SECONDS,
            new ArrayBlockingQueue<>(1000), // бһ‘бҹҶбһ бҹҶбһҮбһҪбһҡбһҲбһҡбһ–бһ¶бһҖбҹ’бһҷбһҹбҹ’бһ“бһҫбһҹбһ»бҹҶ
            new ThreadPoolExecutor.CallerRunsPolicy() // бһңбһ·бһ’бһ¶бһ“бһҖбһ¶бһҡбһҸбһ”бһҹбҹ’бһ‘бҹӮбһӣбһҖбҹ’бһҡбҹ„бһҷ
        );
        
        // бһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ бһ“бһ·бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ”бһҳбҹ’бһҡбһҫ MCP бһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһҠбҹҶбһҺбһ¶бһ”бҹӢбһ’бһ“бһ’бһ¶бһ“
        return new McpServer.Builder()
            .setName("High-Performance MCP Server")
            .setVersion("1.0.0")
            .setPort(5000)
            .setExecutor(executorService)
            .setMaxRequestSize(1024 * 1024) // 1MB
            .setMaxConcurrentRequests(100)
            .setRequestTimeoutMs(5000) // 5 бһңбһ·бһ“бһ¶бһ‘бһё
            .build();
    }
}

```python
# бһҳбҹүбһ¶бһҹбҹҠбҹӮбһң MCP Python бһҖбҹ’бһ“бһ»бһ„бһҹбҹ’бһҗаёІаёӣбһҸбҹ’бһҷбһҖбһҳбҹ’бһҳбһ…бҹӮбһҖбһ…бһ¶бһҷ
from mcp_server import AsyncMcpServer
import asyncio
import aioredis
import uuid

class DistributedMcpServer:
    def __init__(self, node_id=None):
        self.node_id = node_id or str(uuid.uuid4())
        self.redis = None
        self.server = None
    
    async def initialize(self):
        # бҖҒбҖ»бҖӯбҖҗбҖәбҖҶбҖҖбҖәбһ‘бҹ… Redis бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣ
        self.redis = await aioredis.create_redis_pool("redis://redis-master:6379")
        
        # бһ…бһ»бҹҮбһ”бһүбҹ’бһҮбһёбһ“бһјбһңбһ…бҹҶбһҺбһ»бһ…бһ“бҹҒбҹҮбһҮбһ¶бһҳбһҪбһҷбһҖбҹ’бһҡбһ»бһҳ
        await self.redis.sadd("mcp:nodes", self.node_id)
        await self.redis.hset(f"mcp:node:{self.node_id}", "status", "starting")
        
        # бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбҹүбһ¶бһҹбҹҠбҹӮбһң MCP
        self.server = AsyncMcpServer(
            name=f"MCP Node {self.node_id[:8]}",
            version="1.0.0",
            port=5000,
            max_concurrent_requests=50
        )
        
        # бһ…бһ»бҹҮбһ”бһүбҹ’бһҮбһёбһ§бһ”бһҖбһҡбһҺбҹҚ - бһ…бҹҶбһҺбһ»бһ…бһ“бһёбһҳбһҪбһҷбҹ—бһўбһ¶бһ…бһҳбһ¶бһ“бһҮбҹҶбһ“бһ¶бһүбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбһ“бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚбһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“
        self.register_tools()
        
        # бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ heartbeat
        asyncio.create_task(self._heartbeat())
        
        # бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҳбҹүбһ¶бһҹбҹҠбҹӮбһң
        await self.server.start()
        
        # бһҖбҹӮбһ”бҹ’бһҡбҹӮбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ“бҹғбһ…бҹҶбһҺбһ»бһ…
        await self.redis.hset(f"mcp:node:{self.node_id}", "status", "running")
        print(f"MCP Node {self.node_id[:8]} running on port 5000")
    
    def register_tools(self):
        # бһ…бһ»бҹҮбһ”бһүбҹ’бһҮбһёбһ§бһ”бһҖбһҡбһҺбҹҚбһ‘бһјбһ‘бҹ…бһҶбҹ’бһӣбһ„бһҖбһ¶бһҸбҹӢбһ…бҹҶбһҺбһ»бһ…бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢ
        self.server.register_tool(CommonTool1())
        self.server.register_tool(CommonTool2())
        
        # бһ…бһ»бҹҮбһ”бһүбҹ’бһҮбһёбһ§бһ”бһҖбһҡбһҺбҹҚбһҮбҹҶбһ“бһ¶бһүбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ…бҹҶбһҺбһ»бһ…бһ“бҹҒбҹҮ (бһўбһ¶бһ…бһ•бҹ’бһўбҹӮбһҖбһӣбһҫ node_id бһ¬бһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’)
        if int(self.node_id[-1], 16) % 3 == 0:  # бһңбһ·бһ’бһёбһҹбһ¶бһҳбһүбҹ’бһүбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ…бҹӮбһҖбһ…бһ¶бһҷбһ§бһ”бһҖбһҡбһҺбҹҚбһҮбҹҶбһ“бһ¶бһү
            self.server.register_tool(SpecializedTool1())
        elif int(self.node_id[-1], 16) % 3 == 1:
            self.server.register_tool(SpecializedTool2())
        else:
            self.server.register_tool(SpecializedTool3())
    
    async def _heartbeat(self):
        """Periodic heartbeat to indicate node health"""
        while True:
            try:
                await self.redis.hset(
                    f"mcp:node:{self.node_id}", 
                    mapping={
                        "lastHeartbeat": int(time.time()),
                        "load": len(self.server.active_requests),
                        "maxLoad": self.server.max_concurrent_requests
                    }
                )
                await asyncio.sleep(5)  # Heartbeat бһҡбҹҖбһ„бһҡбһ¶бһӣбҹӢ бҹҘ бһңбһ·бһ“бһ¶бһ‘бһё
            except Exception as e:
                print(f"Heartbeat error: {e}")
                await asyncio.sleep(1)
    
    async def shutdown(self):
        await self.redis.hset(f"mcp:node:{self.node_id}", "status", "stopping")
        await self.server.stop()
        await self.redis.srem("mcp:nodes", self.node_id)
        await self.redis.delete(f"mcp:node:{self.node_id}")
        self.redis.close()
        await self.redis.wait_closed()
```

- бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP бһ…бҹӮбһҖбһ…бһ¶бһҷбһҠбҹӮбһӣбһ…бһ»бҹҮбһ”бһүбҹ’бһҮбһёбһҒбҹ’бһӣбһҪбһ“бһҮбһ¶бһҳбһҪбһҷ Redis бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбҹ”
- бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ”бҹҒбҹҮбһҠбһјбһ„бһҠбһҫбһҳбҹ’бһ”бһёбһ’бҹ’бһңбһҫбһІбҹ’бһҷбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ“бһ·бһ„бһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҡбһ”бһҹбҹӢбһҖбҹ’бһҡбһ»бһҳбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһ”бһ“бҹ’бһ‘бһ¶бһ“бҹӢбһ“бҹ… Redisбҹ”
- бһ…бһ»бҹҮбһ”бһүбҹ’бһҮбһёбһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣбһўбһ¶бһ…бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹҶбһ”бҹӮбһҖбһҸбһ¶бһҳбһўбһҸбҹ’бһҸбһҹбһүбҹ’бһүбһ¶бһҺбһ“бҹғбһҖбҹ’бһҡбһ»бһҳбһҳбҹүбһ¶бһҹбҹҠбһёбһ“ бһҠбһҫбһҳбҹ’бһ”бһёбһ…бҹӮбһҖбһ…бһ¶бһҷбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһӣбһҫбһҖбҹ’бһҡбһ»бһҳбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бҹ”
- бһ•бҹ’бһҸбһӣбҹӢбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ”бһ·бһ‘бһҖбҹ’бһҡбһЎбһ¶бһҠбһҫбһҳбҹ’бһ”бһёбһҹбһҳбҹ’бһўбһ¶бһҸбһ’бһ“бһ’бһ¶бһ“ бһ“бһ·бһ„бһҠбһҖбһҖбҹ’бһҡбһ»бһҳбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ…бҹҒбһүбһ–бһёбһҖбҹ’бһҡбһ»бһҳбҹ”
- бһ”бҹ’бһҡбһҫбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһўбһҹбһүбҹ’бһүбһ¶бһ’бһҳбҹҢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҹбҹҶбһҺбһҫбһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ– бһ“бһ·бһ„бһҡбһҖбҹ’бһҹбһ¶бһҖбһ¶бһҡбһҶбҹ’бһӣбһҫбһҷбһҸбһ”бҹ”
- бһ”бҹ’бһҡбһҫ Redis бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣ бһ“бһ·бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ–бһ¶бһҹбһ–бҹҒбһүбһҖбҹ’бһҡбһ»бһҳбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бҹ”
