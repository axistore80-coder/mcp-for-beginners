# бһҖбһҡбһҺбһёбһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһң: бһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ—бһ¶бһ–бһ’бһ¶бһҸбһ» Azure DevOps бһ–бһёбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ YouTube бһҮбһ¶бһҳбһҪбһҷ MCP

> **бһҖбһ¶бһҡбһ”бһҠбһ·бһҹбҹҒбһ’:** бһҳбһ¶бһ“бһ§бһ”бһҖбһҡбһҺбҹҚ бһ“бһ·бһ„бһҡбһ”бһ¶бһҷбһҖбһ¶бһҡбһҺбҹҚбһҸбһ¶бһҳбһўбҹҠбһёбһ“бһ’бһәбһҺбһ·бһҸбһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбҹ’бһҡбһ¶бһ”бҹӢ бһҠбҹӮбһӣбһўбһ¶бһ…бһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҖбһҳбҹ’бһҳбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ—бһ¶бһ–бһ’бһ¶бһҸбһ» Azure DevOps бһҮбһ¶бһҳбһҪбһҷбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ–бһёбһңбҹҒбһ‘бһ·бһҖбһ¶бһҠбһјбһ…бһҮбһ¶ YouTubeбҹ” бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ•бҹ’бһҠбһӣбҹӢбһҮбһ¶бһӮбҹҶбһҡбһјбһӮбҹ’бһҡбһ¶бһ“бҹӢбһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһ бһ¶бһүбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһ§бһ”бһҖбһҡбһҺбҹҚ MCP бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҖбһҳбҹ’бһҳ бһ“бһ·бһ„бһҖбһ¶бһҡбһӣбһ¶бһҷбһ”бһүбҹ’бһ…бһјбһӣбҹ”

## бһҹбһ„бҹ’бһҒбҹҒбһ”

бһҖбһҡбһҺбһёбһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһңбһ“бҹҒбҹҮбһ”бһ„бҹ’бһ бһ¶бһүбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҳбһҪбһҷбһ“бҹғбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣ Model Context Protocol (MCP) бһ“бһ·бһ„бһ§бһ”бһҖбһҡбһҺбҹҚбһҡбһ”бһҹбҹӢбһңбһ¶бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҖбһҳбҹ’бһҳбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ—бһ¶бһ–бһ’бһ¶бһҸбһ»бһҖбһ¶бһҡбһ„бһ¶бһҡ Azure DevOps (ADO) бһҮбһ¶бһҳбһҪбһҷбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҠбҹӮбһӣбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһ–бһёбһңбҹҒбһ‘бһ·бһҖбһ¶ бһўбҹҠбһёбһ“бһ’бһәбһҺбһ·бһҸ бһҠбһјбһ…бһҮбһ¶ YouTubeбҹ” бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҠбҹӮбһӣбһ”бһ¶бһ“бһ–бһ·бһ–бһҺбҹҢбһ“бһ¶бһӮбһәбһҮбһ¶бһӮбҹҶбһҡбһјбһҸбҹӮбһҳбһҪбһҷбһ“бҹғбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһ‘бһјбһӣбҹҶбһ‘бһјбһӣбһ¶бһҷбһҡбһ”бһҹбҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮ бһҠбҹӮбһӣбһўбһ¶бһ…бһ”бҹ’бһҠбһјбһҡбһҸбһ¶бһҳбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҠбһјбһ…бһӮбҹ’бһ“бһ¶бҹ”

бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҳбҹ’бһ“бһ¶бһҖбҹӢбһҸбһ¶бһҳбһҠбһ¶бһ“бһҹбһҳбҹҗбһҷбһ•бҹ’бһҹбһ¶бһҷбһ•бҹ’бһҹбҹҒбһ„бҹ—бһҸбһ¶бһҳбһўбҹҠбһёбһ“бһ’бһәбһҺбһ·бһҸбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһ’бһ¶бһҸбһ» ADO бһҠбҹӮбһӣбһҳбһ¶бһ“ URL бһңбһёбһҠбҹҒбһўбһј YouTube бһҮбһ¶бһҡбһ”бһҹбҹӢбһҒбҹ’бһӣбһҪбһ“бҹ” бһҠбҹ„бһҷбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ§бһ”бһҖбһҡбһҺбҹҚ MCP бһўбҹ’бһ“бһҖбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһўбһ¶бһ…бһҡбһҖбҹ’бһҹбһ¶бһҗбһ¶бһҸбһҫбһ’бһ¶бһҸбһ» ADO бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ—бһ¶бһ–бһҮбһ¶бһҳбһҪбһҷбһҳбһ¶бһҸбҹ’бһҡбһҠбҹ’бһӢбһ¶бһ“бһңбһёбһҠбҹҒбһўбһјбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“ бһҠбһјбһ…бһҮбһ¶ бһ…бҹҶбһ“бһҪбһ“бһҳбһҫбһӣ бһ“бҹ…бһҸбһ¶бһҳбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһўбһ¶бһ…бһ’бҹ’бһңбһҫбһЎбһҫбһ„бһҮбһ¶бһ”бһ“бҹ’бһҸбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ“бһ·бһ„бһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бҹ” бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ“бҹҒбҹҮбһўбһ¶бһ…бһ‘бһјбһ‘бҹ…бһ‘бҹ…бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ•бҹ’бһҹбҹҒбһ„бҹ— бһҠбҹӮбһӣбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ–бһёбһ бҹҒбһҸбһ»бһ•бһӣбһўбҹҠбһёбһ“бһ’бһәбһҺбһ·бһҸбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһӣбһ¶бһҷбһ”бһүбҹ’бһ…бһјбһӣбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„ ADO бһ¬бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбҹ”

## бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–

бһўбҹ’бһ“бһҖбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҳбҹ’бһ“бһ¶бһҖбҹӢбһ‘бһ‘бһҪбһӣбһ”бһ“бҹ’бһ‘бһ»бһҖбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҸбһ¶бһҳбһҠбһ¶бһ“бһ•бһӣбһ”бҹүбҹҮбһ–бһ¶бһӣбҹӢбһ“бҹғбһҹбһҳбҹҗбһҷбһ•бҹ’бһҹбһ¶бһҷбһҸбһ¶бһҳбһўбҹҠбһёбһ“бһ’бһәбһҺбһ·бһҸбһ“бһ·бһ„бһҖбһ¶бһҡбһ…бһјбһӣбһҡбһҪбһҳбһҹбһ бһӮбһҳбһ“бҹҚбҹ” бһҹбһҳбҹҗбһҷбһ“бһёбһҳбһҪбһҷбҹ—бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбһҸбҹӢбһҸбҹ’бһҡбһ¶бһҡбһҪбһҳбһҮбһ¶бһ’бһ¶бһҸбһ»бһҖбһ¶бһҡбһ„бһ¶бһҡ ADO бһҖбҹ’бһ“бһ»бһ„бһӮбһҳбҹ’бһҡбҹ„бһ„ 'DevRel' бһ бһҫбһҷбһ’бһ¶бһҸбһ»бһҖбһ¶бһҡбһ„бһ¶бһҡбһ“бҹ„бҹҮбһҳбһ¶бһ“бһңбһ¶бһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ URL бһңбһёбһҠбҹҒбһўбһј YouTubeбҹ” бһҠбһҫбһҳбҹ’бһ”бһёбһҡбһ¶бһҷбһҖбһ¶бһҡбһҺбҹҚбһҷбҹүбһ¶бһ„бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһ–бһёбһҖбһ¶бһҡбһ•бһӣбһ·бһҸбһҹбһҳбҹҗбһҷбһ“бҹ„бҹҮ бһўбҹ’бһ“бһҖбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ—бһ¶бһ–бһ’бһ¶бһҸбһ» ADO бһҮбһ¶бһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бһҳбһҫбһӣбһңбһёбһҠбҹҒбһўбһјбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“ бһ“бһ·бһ„бһҖбһ¶бһӣбһ”бһҡбһ·бһ…бҹ’бһҶбҹҒбһ‘бһҠбҹӮбһӣбһ”бһ¶бһ“бһҷбһҖбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ“бҹҒбҹҮбҹ”

## бһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣбһ”бҹ’бһҡбһҫ

- [Azure DevOps MCP](https://github.com/microsoft/azure-devops-mcp): бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһ…бһјбһӣбһ”бҹ’бһҡбһҫбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ“бһ·бһ„бһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ—бһ¶бһ–бһ’бһ¶бһҸбһ»бһҖбһ¶бһҡбһ„бһ¶бһҡ ADO бһҸбһ¶бһҳбһҡбһҷбҹҲ MCPбҹ”
- [Playwright MCP](https://github.com/microsoft/playwright-mcp): бһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҖбһҳбҹ’бһҳбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһҖбһҳбһҫбһӣбһҠбһҫбһҳбҹ’бһ”бһёбһ‘бһ¶бһүбһҷбһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһ–бһёбһ‘бҹҶбһ–бҹҗбһҡбһ”бһҺбҹ’бһҠбһ¶бһү бһҠбһјбһ…бһҮбһ¶ бһҹбҹ’бһҗбһ·бһҸбһ·бһңбһёбһҠбҹҒбһўбһј YouTubeбҹ”

## бһҠбҹҶбһҺбһҫбһҡбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҮбһ¶бһ„бһҮбҹҶбһ бһ¶бһ“

1. **бһҖбҹҶбһҺбһҸбҹӢбһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһ’бһ¶бһҸбһ» ADO**бҹ– бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҮбһ¶бһҳбһҪбһҷбһӣбҹҒбһҒбһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһ’бһ¶бһҸбһ»бһҖбһ¶бһҡбһ„бһ¶бһҡ ADO (бһ§. 1234) бһҖбҹ’бһ“бһ»бһ„бһӮбһҳбҹ’бһҡбҹ„бһ„ 'DevRel'бҹ”
2. **бһҷбһҖ URL YouTube**бҹ– бһ”бҹ’бһҡбһҫбһ§бһ”бһҖбһҡбһҺбҹҚ Azure DevOps MCP бһҠбһҫбһҳбҹ’бһ”бһёбһҷбһҖ URL YouTube бһ–бһёбһ’бһ¶бһҸбһ»бһҖбһ¶бһҡбһ„бһ¶бһҡбҹ”
3. **бһ‘бһ¶бһүбһ…бҹҶбһ“бһҪбһ“бһҳбһҫбһӣбһңбһёбһҠбҹҒбһўбһј**бҹ– бһ”бҹ’бһҡбһҫбһ§бһ”бһҖбһҡбһҺбҹҚ Playwright MCP бһҠбһҫбһҳбҹ’бһ”бһёбһ…бһјбһӣбһ‘бҹ…бһҖбһ¶бһ“бҹӢ URL YouTube бһ бһҫбһҷбһ‘бһ¶бһүбһ…бҹҶбһ“бһҪбһ“бһҳбһҫбһӣбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бҹ”
4. **бһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ—бһ¶бһ–бһ’бһ¶бһҸбһ» ADO**бҹ– бһҹбһҡбһҹбҹҒбһҡбһ…бҹҶбһ“бһҪбһ“бһҳбһҫбһӣбһҗбҹ’бһҳбһёбһ“бһ·бһ„бһҖбһ¶бһӣбһ”бһҡбһ·бһ…бҹ’бһҶбҹҒбһ‘бһҠбҹӮбһӣбһ”бһ¶бһ“бһҷбһҖ бһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһ•бҹ’бһ“бҹӮбһҖ 'бһўбһҸбҹ’бһҗбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚ бһ“бһ·бһ„бһҖбһ¶бһҡбһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢ' бһ“бҹғбһ’бһ¶бһҸбһ»бһҖбһ¶бһҡбһ„бһ¶бһҡ ADO бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһ§бһ”бһҖбһҡбһҺбҹҚ Azure DevOps MCPбҹ”
